# Sandbox Component Showcase: Pure-CSS Interactive Star Rating

## Overview
A zero-dependency, pure-CSS interactive star feedback ranking component isolated completely within a sandbox pathway. It manages state highlights natively using strategic combinator cascades.

## 📁 Sandbox Configuration Files
* `demo.html` — Self-contained feedback showcase card loading crisp inline SVG symbols to test user hover tracking paths.
* `style.css` — Scoped style controller applying direction modifications and child selectors linked back to root tokens.

## 🚀 Key Layout Mechanics
1. **Reverse DOM Cascade Mapping:** By arranging input elements from 5 down to 1 inside the HTML text structure but visually sorting them from 1 to 5 using a `flex-direction: row-reverse` flex layout rule, the CSS general sibling combinator (`~`) is able to seamlessly trace and highlight preceding elements to its left.
2. **Zero-Overhead Vector Paints:** The stars render using sharp vector coordinates scaling color parameters cleanly through `currentColor` mapping links. This configuration bypasses heavy image sprite swaps or dynamic script re-renders.
3. **Decoupled Sandbox Isolation:** Maintains full compliance with the core framework freeze rules by keeping the entire feature logic enclosed within `submissions/examples/star-rating/`.
