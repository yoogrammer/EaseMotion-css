Excessive -v1 Suffixes #4578

123
folders with -v1/-v2/-v3
148
multi-version base names

Examples of Redundant Naming

core_changes-v1-v1-v1 → core_changes-issue-XXXX
ease-btn-v1 → ease-btn
ease-tooltip-component-v1 → ease-tooltip-component
ease-tooltip-component-v2 → ease-tooltip-component-advanced
shake-effect-v1-v1 → shake-effect
flip-v1 → flip
printing-utilities-v1 → printing-utilities

Proposed Convention

Single submission → no suffix
Variants → descriptive suffix (e.g., -filled, -outlined)
Core changes → core_changes-issue-{number}
