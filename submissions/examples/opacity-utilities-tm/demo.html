Usage

Apply the .opacity-utilities class to any element. Uses --ease-* design tokens from the EaseMotion CSS framework.

Responsive Behavior

Resize the browser window to see how these utilities respond to different viewport sizes. All utilities respect the prefers-color-scheme and prefers-reduced-motion media queries.
