# Scroll To Top Button

## Description

This example demonstrates a floating scroll-to-top button for long documentation pages.

## Features

- Floating action button
- Appears after scrolling down
- Smooth scrolling to the top
- Responsive layout

## How It Works

The button remains hidden initially.

When the user scrolls beyond a predefined threshold, the button becomes visible.

Clicking the button smoothly scrolls the page back to the top.

## Files

- demo.html
- style.css
- README.md

## Use Case

Useful for documentation websites, blogs, and long-form content pages where users need quick navigation back to the top.
