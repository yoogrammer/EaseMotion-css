# Vertical Align Utilities

CSS utility classes for vertical alignment of inline and table-cell elements.

## Classes

- `.align-baseline` — `vertical-align: baseline`
- `.align-top` — `vertical-align: top`
- `.align-middle` — `vertical-align: middle`
- `.align-bottom` — `vertical-align: bottom`

## Why

Controlling vertical alignment is essential for fine-tuning the positioning of inline elements like images, text, and icons alongside each other. These utilities provide quick, consistent access to the most common vertical-align values without writing custom CSS each time.
