Hover Animations Missing from docs

ease-hover-shimmer
Hover me Shimmer sweeps across on hover
<div class="ease-hover-shimmer">...</div>

A light shimmer sweep effect that slides across the element on hover. Great for cards and buttons.

ease-hover-underline
Hover this link Underline grows from left on hover
<a class="ease-hover-underline">Link</a>

An animated underline that expands from left to right on hover using a CSS pseudo-element.

ease-card-lift
Hover me Lifts up 8px with shadow
<div class="ease-card-lift">...</div>

Lifts the element up by 8px and adds a deep shadow on hover. Stronger than ease-hover-lift.

Skeleton / Loading Entire section missing

ease-skeleton-shimmer
<div class="ease-skeleton-shimmer ease-skeleton-block"></div>

Animated shimmer gradient that slides across — the standard skeleton loading pattern for content placeholders.

ease-skeleton-pulse
<div class="ease-skeleton-pulse ease-skeleton-block"></div>

Fades opacity in and out infinitely — an alternative skeleton loading style that pulses instead of shimmers.

ease-skeleton-block
<div class="ease-skeleton-block"></div>

Base block shape helper — adds border-radius and display: block. Combine with shimmer or pulse.
