# Shimmer Loading Skeleton

A placeholder loading skeleton that mimics a card layout during content loading. Uses a moving gradient animation (shimmer) across rectangular blocks representing an image and text lines. The animation loops continuously until real content replaces it.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser to see the shimmer skeleton.

## Accessibility notes

This is a purely visual placeholder. It should be replaced with real content or hidden from screen readers via `aria-hidden` in production. Reduced motion disables the animation.
