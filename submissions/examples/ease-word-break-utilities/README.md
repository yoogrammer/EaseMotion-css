# Ease Word Break Utilities

A collection of lightweight CSS utilities for controlling word wrapping and text breaking behavior.

## Features

- Pure CSS utilities
- Lightweight and reusable
- Cross-browser compatible
- Beginner-friendly implementation

## Included Utilities

- `ease-break-word`
- `ease-break-all`
- `ease-break-keep`

## Usage

```html
<p class="ease-break-word">
  Long text here
</p>

<p class="ease-break-all">
  Long text here
</p>

<p class="ease-break-keep">
  Long text here
</p>
```

## Utility Details

- `ease-break-word` prevents overflow using overflow-wrap
- `ease-break-all` breaks text at any character
- `ease-break-keep` keeps words together whenever possible
