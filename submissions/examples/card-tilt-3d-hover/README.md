# Card Tilt 3D Hover

**What does this do?**

Creates an interactive 3D card tilt effect that responds to cursor movement. The card rotates in real time based on the cursor position and includes a subtle shine overlay.

**How is it used?**

```html
<div class="tilt-container">
  <div class="tilt-card">
    Content
  </div>
</div>
```

**Why is it useful?**

The 3D tilt effect adds depth and interactivity to cards, making interfaces feel more dynamic and engaging. The effect remains lightweight by using CSS transforms with minimal JavaScript.
