# Section Header

Reusable section heading with title, subtitle, optional badge, action link with arrow icon, and divider. Multiple alignment and size variants.

## Usage

```html
<div class="ease-section-header-az">
  <div class="ease-section-header-top-az">
    <span class="ease-section-badge-az">&#x2728; New</span>
  </div>
  <h2 class="ease-section-title-az">Section title here</h2>
  <p class="ease-section-subtitle-az">Optional description text.</p>
  <div class="ease-section-actions-az">
    <a class="ease-section-link-az" href="#">
      View all <span class="ease-section-link-icon-az">&#x2192;</span>
    </a>
  </div>
</div>
```

## Variants

- **`--center`**: Centered text and actions
- **`--lg`**: Larger title (2.25rem)
- **`--sm`**: Smaller title (1.25rem)
- **`--inline`**: Title + action on the same row
- **`--with-divider`**: Bottom border divider

## Elements

| Class | Purpose |
|-------|---------|
| `.ease-section-badge-az` | Pill badge label |
| `.ease-section-title-az` | Main heading |
| `.ease-section-subtitle-az` | Description text |
| `.ease-section-link-az` | Action link |
| `.ease-section-divider-az` | Horizontal rule |
