# Content Disclosure Grid

## What does this do?
An expandable card grid where clicking a card smoothly reveals hidden content. Each card toggles open/close with a height transition and fade-in effect.

## How is it used?
Add the class to any HTML element:

    <div class="disclosure-card">
      <div class="disclosure-header">Title</div>
      <div class="disclosure-body"><p>Hidden content here.</p></div>
    </div>

## Why is it useful?
Great for FAQs, project showcases, and feature lists where you want to show previews and let users expand for details — keeps the UI clean while being interactive.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
