# Text Transform Utilities

Adds readable text transformation utility classes.

## Classes

| Class            | CSS Property               |
| ---------------- | -------------------------- |
| ease-uppercase   | text-transform: uppercase  |
| ease-lowercase   | text-transform: lowercase  |
| ease-capitalize  | text-transform: capitalize |
| ease-normal-case | text-transform: none       |

## Usage

```html
<p class="ease-uppercase">hello world</p>
<p class="ease-lowercase">HELLO WORLD</p>
<p class="ease-capitalize">hello world</p>
<p class="ease-normal-case">Hello World</p>
```

## Example Output

```text
HELLO WORLD
hello world
Hello World
Hello World
```
