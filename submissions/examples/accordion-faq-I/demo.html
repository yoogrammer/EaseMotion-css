Basic Accordion

What is EaseMotion CSS? ▼

EaseMotion CSS is an animation-first CSS framework with zero dependencies. Just link the CDN and start building beautiful animated interfaces with human-readable class names.

How do I install it? ▼

Simply add the CDN link: <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/easemotion-css/easemotion.min.css" /> or install via npm: npm install easemotion-css

Is it free to use? ▼

Yes! EaseMotion CSS is completely free and open-source under the MIT License. You can use it for personal and commercial projects.

FAQ Section

What makes EaseMotion different from Tailwind? ▼

EaseMotion focuses on human-readable class names like ease-fade-in and ease-center, while Tailwind uses atomic utility classes. EaseMotion is animation-first and works immediately with zero configuration.

Can I use it with React or Vue? ▼

Absolutely! EaseMotion CSS works with any framework that renders standard HTML. Just import the CSS file and use the classes directly in your components.

Does it work without a build step? ▼

Yes! That is one of the core goals of EaseMotion CSS. You can use the CDN link or local CSS files directly without configuring a bundler, compiler, or build pipeline.

How do I customize the framework? ▼

Override any CSS custom property to theme the entire framework. No Sass, no PostCSS, just CSS. You can change colors, motion speeds, shapes, and shadows easily.

Usage

<details class="ease-accordion">
    <summary class="ease-accordion-summary">
        Question Title
        <span class="accordion-icon">▼</span>
    </summary>
    <div class="ease-accordion-content">
        <p>Answer content here</p>
    </div>
</details>

<!-- Color variants -->
<details class="ease-accordion-success">...</details>
<details class="ease-accordion-warning">...</details>

<!-- Rounded variant -->
<details class="ease-accordion-rounded">...</details>
