# ease-place-self utilities

Controls `place-self` (shorthand for `align-self` + `justify-self`) for individual grid/flexbox item alignment.

## Usage

```html
<div class="ease-grid">
  <div class="ease-place-self-center">Centered item</div>
</div>
```

## Classes

| Class                     | CSS Property          |
| ------------------------- | --------------------- |
| `ease-place-self-auto`    | `place-self: auto`    |
| `ease-place-self-start`   | `place-self: start`   |
| `ease-place-self-center`  | `place-self: center`  |
| `ease-place-self-end`     | `place-self: end`     |
| `ease-place-self-stretch` | `place-self: stretch` |
