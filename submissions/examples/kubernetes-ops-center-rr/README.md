# Kubernetes Cluster Operations Center Component

## What does this do?

Provides a centralized dashboard for monitoring Kubernetes cluster health, node utilization, deployments, workloads, and operational alerts.

## How is it used?

```html
<div class="node-card healthy">
    <h3>worker-node-01</h3>
</div>
```

## Why is it useful?

Modern cloud-native platforms rely on Kubernetes for workload orchestration and infrastructure management. This component provides a reusable operations dashboard for monitoring cluster health, node status, deployments, and resource utilization while demonstrating cloud-native platform engineering design patterns.
