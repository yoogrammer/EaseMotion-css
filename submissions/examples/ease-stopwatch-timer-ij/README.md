# Stopwatch Timer

Digital stopwatch with flip-digit animation, lap recording, and millisecond precision.

## Features

- Start/Stop and Reset controls
- Flip animation (rotateX) when digits change
- Lap time recording with slide-in entry animation
- Centisecond (10ms) precision display

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--st-duration` | `0.2s` | Digit flip animation speed |
| `--st-digit-color` | `#f1f5f9` | Digit text color |
| `--st-btn-color` | `#334155` | Default button background |
| `--st-start-color` | `#22c55e` | Start button color |
| `--st-stop-color` | `#ef4444` | Stop/Reset button color |
| `--st-radius` | `12px` | Border radius |
