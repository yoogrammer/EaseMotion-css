# 3D Particle Field

## What it does

Interactive 3D particle system using Three.js. Particles float in a spherical field with mouse interaction, color cycling, and explosion mode.

## How to use it

Include Three.js and the script. The particle field automatically creates a fullscreen interactive background.

## Features

- Mouse drag to rotate view
- Scroll to zoom (camera controls)
- Change Colors button cycles through 5 color palettes
- Explode Mode scatters particles outward
- 4000+ particles with additive blending
- Star background layer

## Variants (CSS classes)

- .ease-particle-field - Container class
- .ease-particle-btn - Button styling

## Why it fits EaseMotion CSS

- Full 3D interaction
- Smooth animations
- Color customization
- Performance optimized
