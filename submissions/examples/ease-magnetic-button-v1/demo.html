Magnetic Button

An interactive, micro-responsive button that attracts and follows the cursor on hover. Combines custom transition speeds for real-time mouse tracking and snaps back with an elastic animation on leave.

Design Variations

Move the cursor near any button to experience the magnetic attraction. Notice how the inner text and icons float slower than the outer container, creating 3D depth.

Floating Action Circles (FAB)

Often used in navigations or dashboard corners, these circular icons follow the cursor, making them highly tactile and engaging.

Integration Code

Understand the wrapper layout, required classes, and coordinate tracking logic. Code is optimized to perform zero DOM queries inside the mouse loop.

HTML Markup 
<!-- Outer target captures mouse events early -->
<div class="ease-magnetic-wrap">
  <!-- Button follows cursor with base strength -->
  <button class="ease-magnetic-btn">
    <!-- Inner text/content floats slower, creating 3D depth -->
    <span class="ease-magnetic-content">
      Get Started
    </span>
  </button>
</div>
JavaScript Track logic 
<script>
  // Perform once during initialisation
  document.querySelectorAll('.ease-magnetic-wrap').forEach(wrap => {
    const btn = wrap.querySelector('.ease-magnetic-btn');
    const content = wrap.querySelector('.ease-magnetic-content');

    wrap.addEventListener('mousemove', function(e) {
      const rect = wrap.getBoundingClientRect();
      const relX = e.clientX - (rect.left + rect.width / 2);
      const relY = e.clientY - (rect.top + rect.height / 2);

      // Max displacement in pixels
      const btnStrength = 18;
      const contentStrength = 8;

      const xRatio = relX / (rect.width / 2);
      const yRatio = relY / (rect.height / 2);

      btn.style.transform = `translate(${xRatio * btnStrength}px, ${yRatio * btnStrength}px)`;
      if (content) {
        content.style.transform = `translate(${xRatio * contentStrength}px, ${yRatio * contentStrength}px)`;
      }
    });

    wrap.addEventListener('mouseleave', function() {
      // snap back automatically using elastic CSS transitions
      btn.style.transform = '';
      if (content) content.style.transform = '';
    });
  });
</script>
