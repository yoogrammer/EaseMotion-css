# ⚡ Ease Magnetic Button Effect

A premium micro-interaction component for EaseMotion CSS that provides multi-layered parallax cursor tracking attraction.

## ✨ What it does
By leveraging a localized event hit wrapper container, elements seamlessly move toward the user's cursor once it enters the trigger bounds. Distinct structural offsets translate the button frame and the nested textual/SVG symbols independently, generating an eye-catching 3D spatial depth feel. On mouse leave, the components execute an elastic rubber-band return to home base.

## 🚀 How to Use
```html
<div class="ease-magnetic-wrap">
  <button class="ease-magnetic-btn">
    <span class="ease-magnetic-content">Get Started</span>
  </button>
</div>
