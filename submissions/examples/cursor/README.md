# ease-cursor

Custom cursor styles and animated hover effects using pure CSS. Zero JavaScript required.

## Usage

```html
<div class="ease-cursor-pointer">Pointer cursor</div>
<div class="ease-cursor-grow">Grows on hover</div>
<div class="ease-cursor-lift">Lifts on hover</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-cursor-pointer` | Pointer cursor |
| `ease-cursor-crosshair` | Crosshair cursor |
| `ease-cursor-zoom-in` | Zoom-in cursor |
| `ease-cursor-zoom-out` | Zoom-out cursor |
| `ease-cursor-grab` | Grab cursor |
| `ease-cursor-grabbing` | Grabbing cursor |
| `ease-cursor-not-allowed` | Not-allowed cursor |
| `ease-cursor-wait` | Wait/loading cursor |
| `ease-cursor-text` | Text cursor |
| `ease-cursor-none` | Hidden cursor |
| `ease-cursor-grow` | Scale up + shadow on hover |
| `ease-cursor-shrink` | Scale down on hover |
| `ease-cursor-grow-rotate` | Scale up + rotate on hover |
| `ease-cursor-lift` | Translate up + shadow on hover |
| `ease-cursor-press` | Scale down on active/click |
| `ease-cursor-drag` | Grab cursor with grabbing on active |
