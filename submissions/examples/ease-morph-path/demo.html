🔄 ease-morph-path

Smoothly animates between two SVG paths

Basic Morph Examples

Triangle → Circle

Star → Circle

Heart → Circle

Speed Variations

Fast (1s)

Normal (2s)

Slow (4s)

Blob Morph (Organic Shape)

Circle → Organic Blob

Circle → Organic Blob 2

Custom Speed (CSS Variable)

Custom 3s speed

💡 Note: CSS `d` property animation works in Chrome 93+, Edge 93+, and modern browsers. For older browsers, the shape will remain static or use SMIL fallback.

Usage

<!-- Basic morph -->
<svg viewBox="0 0 100 100" class="ease-morph-path">
    <path d="M50,20 L80,80 L20,80 Z" />
</svg>

<!-- Custom speed -->
<svg viewBox="0 0 100 100" class="ease-morph-path" style="--ease-morph-speed: 3s;">
    <path d="M50,20 L80,80 L20,80 Z" />
</svg>

<!-- Star morph -->
<svg viewBox="0 0 100 100" class="ease-morph-path-star">
    <path d="M50,15 L61,35 L85,40 L68,58 L72,82 L50,70 L28,82 L32,58 L15,40 L39,35 Z" />
</svg>
