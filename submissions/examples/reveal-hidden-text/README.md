# Reveal Hidden Text Hover

## What does this do?
Reveals hidden subtitle text with a smooth upward slide animation on hover.

## How is it used?
Wrap your text in a parent with class reveal-wrapper and add
ease-hover-reveal-text to the child element:

    <div class="reveal-wrapper">
      <p class="ease-hover-reveal-text">Your hidden text here</p>
    </div>

## Why is it useful?
Keeps cards clean by default and reveals extra content only on interaction,
following EaseMotion CSS's philosophy of smooth, purposeful motion.
