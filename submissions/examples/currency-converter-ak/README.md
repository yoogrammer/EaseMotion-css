# Currency Converter

## What does this do?
A currency converter with 20 major world currencies, real-time conversion, swap functionality, and exchange rate display.

## How is it used?
Open demo.html in a browser. Enter an amount, select source and target currencies. The converted value updates instantly. Click the swap button to reverse the pair.

## Why is it useful?
This provides a complete currency conversion tool for e-commerce, travel planning, or financial dashboards. It demonstrates select-based input handling and bidirectionally safe conversion math.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to use the converter.
