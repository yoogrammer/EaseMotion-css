# Poll Voting Widget

## What
A poll voting widget with selectable options, vote counting, percentage bars with visual results, and a total vote counter.

## How
Uses a JavaScript array of option objects with vote counts. Clicking "Vote" increments the count for that option, calculates percentages, and renders horizontal bar fills via CSS width transitions.

## Why
Demonstrates an interactive polling interface with live visual feedback, suitable for surveys, feedback forms, and community voting.
