# Gradient Border Animation

## What does this do?
A modern animated border effect where a colorful gradient continuously
flows around the edges of a card or button, creating a premium glowing
appearance.

## How is it used?

```html
<div class="gradient-border-card">
  <h2>Gradient Border</h2>
  <p>Animated glowing border effect.</p>
</div>
```

## Why is it useful?
Transforms a static border into a visually engaging animated element.
Suitable for feature cards, pricing sections, CTA buttons, and dashboard
widgets, commonly used in modern SaaS interfaces and landing pages.
Pure CSS using mask-composite for the border ring, no JS required,
with prefers-reduced-motion support.
