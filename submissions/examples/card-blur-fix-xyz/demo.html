✨ Fixed / Hardware Layer Promotion

❌ Unoptimized / Subpixel Blur

This card uses raw CSS scaling without hardware-acceleration overrides. When animated, the rendering engine recalculates text anti-aliasing alignments dynamically mid-flight, making typography appear blurry and unstable until the transition window finishes.
