# Glow Icon Hover

A lightweight CSS utility that creates a glowing hover effect for icons. The animation combines a subtle scale-up effect with a soft glow to make icons feel more interactive and visually engaging.

## Features

* Pure CSS implementation
* Smooth glow animation on hover
* Subtle scale effect for emphasis
* Lightweight and reusable
* Cross-browser compatible
* No JavaScript required

## Usage

```html
<i class="fa-solid fa-star ease-glow-icon-hover"></i>
```

## Behavior

* Soft glow appears on hover
* Icon slightly enlarges for emphasis
* Smooth easing animation
* Returns to its original state when hover ends

## Why is it Useful?

It improves visual feedback for icon-based actions and navigation elements, helping users identify interactive components while keeping the interface clean and modern.
