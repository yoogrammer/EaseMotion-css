# Color Scheme Utilities

CSS utility classes for controlling the color scheme of an element.

## Classes

- `.color-scheme-light` — `color-scheme: light`
- `.color-scheme-dark` — `color-scheme: dark`
- `.color-scheme-normal` — `color-scheme: normal`

## Why

Allows elements to opt into light or dark color schemes independently of the page, useful for themed sections, embedded widgets, and form controls that should match a specific appearance mode.
