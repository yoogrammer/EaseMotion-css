## ease-table

**What does this do?**
A styled data table component with striped rows, hover effects,
and a responsive wrapper.

**How is it used?**
<div class="ease-table-responsive">
  <table class="ease-table ease-table-striped ease-table-hover">
    <thead>
      <tr><th>Name</th><th>Status</th></tr>
    </thead>
    <tbody>
      <tr><td>Item</td><td>Active</td></tr>
    </tbody>
  </table>
</div>

Modifiers: ease-table-striped, ease-table-hover
Wrapper: ease-table-responsive
Color variant: add class `green` to `.ease-table`

**Why is it useful?**
Provides a clean, responsive data table with smooth hover transitions
and status badge support — consistent with EaseMotion CSS's component philosophy.
