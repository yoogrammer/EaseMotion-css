# Scale Background Utility

A smooth scale-style background animation utility for interactive components.

## Features

- Lightweight pure CSS animation
- Smooth background scaling effect
- Cross-browser compatible
- Reusable utility class

## Usage

```html
<div class="ease-scale-background-utility">
  Hover Me
</div>
```

## Animation Details

- Background smoothly scales on interaction
- Uses lightweight CSS keyframes
- Modern and reusable utility effect
