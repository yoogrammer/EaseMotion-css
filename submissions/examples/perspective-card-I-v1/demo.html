Usage

<!-- HTML Structure -->
<div class="ease-perspective-card" id="myCard">
    <div class="card-inner">
        <div class="card-front">
            <!-- Your content here -->
        </div>
    </div>
</div>

<!-- JavaScript (required for cursor tracking) -->
<script>
    const card = document.getElementById('myCard');
    card.addEventListener('mousemove', (e) => {
        const rect = card.getBoundingClientRect();
        const x = (e.clientX - rect.left) / rect.width;
        const y = (e.clientY - rect.top) / rect.height;
        card.style.setProperty('--ease-mx', x);
        card.style.setProperty('--ease-my', y);
    });
    card.addEventListener('mouseleave', () => {
        card.style.setProperty('--ease-mx', 0.5);
        card.style.setProperty('--ease-my', 0.5);
    });
</script>

🎴 3D Perspective Card Features

  • 🎯 Cursor-tracked 3D tilt effect
  • ✨ Smooth reset on mouse leave
  • 💡 Shine layer that follows the cursor
  • 🎨 Configurable via --ease-tilt-max and --ease-tilt-perspective
  • 🖱️ Minimal JS (only reads cursor position)
  • ⚡ All visual work is CSS
