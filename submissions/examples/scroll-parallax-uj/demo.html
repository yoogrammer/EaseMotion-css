Scroll-Driven Parallax

Historically, parallax scrolling required heavy JavaScript event listeners tracking the scroll offset, causing main thread delays and page jitter. Modern browsers support CSS scroll-driven animations, binding keyframes directly to the viewport scroll timeline.
