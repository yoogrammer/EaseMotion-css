# Ease Notification Center

A CSS-first notification center dropdown component for EaseMotion CSS.

## Features

- Bell icon notification trigger
- Animated notification badge
- Dropdown notification panel
- Read / unread notification states
- Mark all as read action
- CSS-driven animations
- Responsive design
- Supports prefers-reduced-motion

## Files

- demo.html
- style.css
- script.js

## Usage

Open demo.html in a browser.

Click the bell icon to open the notification center.

Click "Mark all as read" to clear unread states.
