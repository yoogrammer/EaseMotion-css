# Cloud Infrastructure Relationship Graph Component

## What does this do?

Provides a visual cloud architecture graph showing relationships between infrastructure services such as load balancers, API gateways, application servers, databases, caches, and storage systems.

## How is it used?

```html
<div class="node">
    API Gateway
</div>
```

Infrastructure nodes are connected using visual connectors to represent service dependencies and traffic flow.

## Why is it useful?

Modern cloud applications often require architecture visualizations to communicate system design and service relationships. This component provides a reusable infrastructure diagram while demonstrating responsive layouts, animations, and enterprise-focused design patterns.
