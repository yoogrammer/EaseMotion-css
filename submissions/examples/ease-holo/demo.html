Developer Integration

To implement this holographic foil effect, simply wrap your card inside a container, apply the .ease-holo utility class, and optionally adjust the strength using the custom CSS property.

HTML Structure

<!-- Simply add the ease-holo class to any card or button -->
<div class="my-card ease-holo">
  <h2>Foil Card Title</h2>
  <p>Card content goes here...</p>
</div>

CSS Customization

.my-card {
  background: #12101e;
  border-radius: 20px;
  
  /* Override foil strength (Default is 0.25) */
  --ease-holo-intensity: 0.35;
}

How It Works Under the Hood

The .ease-holo utility relies on a ::before pseudo-element styled with a multi-stop conic-gradient. Using mix-blend-mode: overlay, the colors blend seamlessly with whatever assets or background you put inside your card. On hover, the pseudo-element smoothly fades in and rotates to simulate light moving across the card's surface.
