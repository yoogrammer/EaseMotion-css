# ease-btn-ghost

Ghost/transparent button variant with no background or border by default. On hover, it fills with a subtle background.

## Usage

```html
<button class="ease-btn ease-btn-ghost">Ghost button</button>
```

## Variants

| Class | Description |
|---|---|
| `ease-btn-ghost` | Ghost button with hover fill |

## Notes

- No background or border in resting state
- Hover adds subtle `--ease-color-neutral-100` background
- Standard component in virtually every design system
