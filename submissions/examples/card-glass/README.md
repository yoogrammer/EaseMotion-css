# ease-card-glass

Glassmorphism card component with `backdrop-filter: blur(16px)`, semi-transparent background, and a light border. Includes a `@supports` fallback.

## Usage

```html
<div class="ease-card ease-card-glass">Glass card content</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-card-glass` | Default glass card with blur |
| `ease-card-glass-dark` | Dark variant for light backgrounds |

## Notes

- Uses `backdrop-filter: blur()` for the glass effect
- Includes `@supports` fallback for non-supporting browsers
- Respects dark mode
- Works best over vibrant backgrounds
