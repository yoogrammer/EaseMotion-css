# CSS-only Multi-step Progress / Stepper Component

A clean, responsive, and fully CSS-only multi-step progress indicator for guiding users through sequential workflows.

This component includes:

- A visual progress line connecting steps.
- Clear visual states for `completed` (checkmark), `active` (highlighted/scaled), and upcoming/pending steps.
- Numbered circular indicators.
- Responsive layout that ensures the progress line stays aligned perfectly with the step circles.
- Smooth transition animations when step states change.
- Support for `prefers-color-scheme: dark`.

## Why is this useful for EaseMotion CSS?

Multi-step progress trackers (or steppers) are crucial for onboarding flows, multi-page registration forms, checkout experiences, and setup wizards. Providing a clean, dependency-free CSS-only version gives developers a ready-to-use structural pattern that adheres to EaseMotion CSS's philosophy of animation-first, readable utilities without the bloat of JavaScript logic.
