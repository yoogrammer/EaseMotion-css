# Pulse Border

**EaseMotion Submission** · Contributor: kavin553

---

## 1. What does this do?

Adds a soft, continuously pulsing glow around an element's border using an animated `box-shadow`, without ever changing the element's size or layout.

## 2. How is it used?

Apply the `pulse-border-btn` class to any element:

```html
<button class="pulse-border-btn">
  Get Started
</button>
