# Biometric Synthetic Biology Laboratory Console — Phase #855

A futuristic laboratory dashboard for synthetic biology research. Built entirely with **HTML5**, **CSS3**, and **vanilla JavaScript**. Part of the EaseMotion CSS design system showcase.

---

## Features

- Animated cellular background with floating cells and DNA helix
- Microscope grid overlay
- Live biometric vitals with wave animation
- 4 culture vessel cards with liquid simulation and bubbles
- Gene sequencer with color-coded base pairs
- Interactive microscope with zoom and focus controls
- Animated cell view with nucleus and organelles
- Incubator temperature dial
- Safety status grid
- Toast notification system
- Responsive layout
- Reduced motion support
- Zero external dependencies
