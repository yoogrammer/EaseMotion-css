# Statistics Dashboard Cards

## What does this do?

A responsive statistics dashboard card component for displaying key metrics such as users, revenue, projects, and conversion rates.

## How is it used?

```html
<div class="stat-card">
  <h3>Total Users</h3>
  <div class="value">12,540</div>
  <span class="positive">↑ 12% this month</span>
</div>
```

## Why is it useful?

Statistics cards are commonly used in dashboards, analytics platforms, SaaS products, admin panels, and business applications. They help users quickly understand important metrics and trends.
