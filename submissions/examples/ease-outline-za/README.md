## What does this do?

Provides `ease-outline` utility classes for controlling outline style, width, color, and offset — useful for focus rings and decorative borders that don't affect layout.

## How is it used?

```html
<button
  class="ease-outline-2-za ease-outline-primary-za ease-outline-offset-2-za"
>
  Focus ring
</button>
<div class="ease-outline-dashed-za">Dashed outline</div>
```

## Why is it useful?

EaseMotion core has no outline utilities. Outlines are essential for accessible focus indicators and decorative borders that don't take up layout space like borders do.
