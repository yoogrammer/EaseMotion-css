# Flip Card 3D Hover

A card that flips 180 degrees on hover to reveal content on the back face. Uses CSS 3D transforms with `perspective` and `backface-visibility`. The front shows a gradient with title, and the back shows additional content with an action button.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser. Hover over the card to see it flip.

## Accessibility notes

The card also flips on `focus-within` for keyboard users. Reduced motion disables the flip transition.
