# Pomodoro Timer Showcase

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A premium Pomodoro Timer dashboard showcasing focus sessions, productivity statistics, session history, animated timer rings, and modern dashboard layouts using pure HTML and CSS.

## Features

- Circular timer display
- Animated progress ring
- Focus / Break modes
- Productivity statistics
- Session timeline
- Dashboard layout
- Floating glow effects
- Responsive design
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="timer-ring">
  <div class="timer-inner">
    <h2>25:00</h2>
  </div>
</div>
```

## Main Classes

```css
.timer-ring
.timer-inner
.stat-card
.mode-card
.timeline-card
```
