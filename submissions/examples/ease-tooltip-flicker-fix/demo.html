Tooltip Flicker Fix

Fixed ease-tooltip flickering on rapid mouse movement by adding pointer-events: none and a hover delay transition to all tooltip elements.

📄 The Fix

Added pointer-events: none and transition delay to prevent flicker:

/* ✅ Fix 1: pointer-events none prevents tooltip
   from intercepting mouse events */
.ease-tooltip::before,
.ease-tooltip::after {
  pointer-events: none;
}

/* ✅ Fix 2: Transition delay prevents flicker
   on rapid mouse movement */
.ease-tooltip::before,
.ease-tooltip::after {
  opacity: 0;
  transition: opacity 0.2s ease 0.1s;
}

.ease-tooltip:hover::before,
.ease-tooltip:hover::after {
  opacity: 1;
  transition: opacity 0.2s ease 0.1s;
}
