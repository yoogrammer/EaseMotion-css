# Expense Splitter App Template
A utilitarian, trustworthy, and efficiency-focused landing page template for finance-sharing apps, roommate cost-trackers, and group-expense management tools.

- **Design**: Uses a "Precision-Settlement" palette with split-indigo primary branding, settle-green success accents, and clean, modular layouts designed to minimize financial friction between users.
- **Use Case**: Expense splitting apps, shared-living finance trackers, and group trip cost management tools.
