Spotlight Card

A radial glow follows your cursor inside each card. Move your mouse across the cards below.

Basic Usage

✦

Animation First

Motion is a first-class citizen, not an afterthought. Every interaction feels alive.

◈

Human Readable

Class names read like plain English. No documentation lookup required.

⬡

Zero Dependencies

Link one file. No build steps, no config, no npm. Just write HTML and ship.

With Border Glow Variant

⬟

Success State

Combine spotlight with border glow for emphasis cards.

◉

Info State

The spotlight intensity and color are fully customizable via CSS variables.

◆

Warning State

Works with any content — icons, text, images, buttons.

How to Use

html
<!-- Basic spotlight card -->
<div class="spotlight-card">
  <h3>Card Title</h3>
  <p>Card content here.</p>
</div>

<!-- With border glow variant -->
<div class="spotlight-card spotlight-card--border">
  <h3>Card Title</h3>
  <p>Card content here.</p>
</div>
css vars
/* Customize the spotlight color and intensity */
.spotlight-card {
  --spotlight-color: rgba(108, 99, 255, 0.15); /* default: primary */
  --spotlight-size: 400px;                      /* default: 400px   */
}
