CSS-Only Scroll Progress Bar

A pure CSS scroll progress indicator using scroll-driven animations. Zero JavaScript. Scroll down to see the progress bar fill.

How It Works

This component uses the modern CSS animation-timeline: scroll() property to track page scroll position and animate a progress bar from 0% to 100%. No JavaScript event listeners, no IntersectionObserver — pure CSS.

The progress bar sits fixed at the top of the viewport and scales horizontally as the user scrolls. It supports color variants, glow effects, and respects prefers-reduced-motion.
