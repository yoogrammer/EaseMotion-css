# Animated Responsive Navbar

**What does this do?**
Creates a modern, responsive navigation bar with smooth hover animations, active link indicators, and mobile-friendly behavior. The component provides a clean and intuitive navigation experience while remaining lightweight and easy to customize.

**How is it used?**

```html
<nav class="navbar">
  <a href="#" class="logo"> EaseMotion </a>

  <ul class="nav-links">
    <li><a href="#" class="active">Home</a></li>
    <li><a href="#">Components</a></li>
    <li><a href="#">Examples</a></li>
    <li><a href="#">Docs</a></li>
    <li><a href="#">About</a></li>
  </ul>
</nav>
```

**Why is it useful?**
Navigation bars are one of the most common UI patterns across websites, dashboards, landing pages, documentation sites, and portfolio projects. Developers frequently rebuild similar navigation structures from scratch.

This component provides a reusable solution with subtle motion, clear active-state feedback, and responsive behavior while remaining entirely CSS-based. The hover underline animation and active indicator improve user orientation without introducing unnecessary complexity.

The component aligns with EaseMotion CSS by offering a human-readable, composable, animation-focused building block that can be easily standardized into framework utilities and components.
