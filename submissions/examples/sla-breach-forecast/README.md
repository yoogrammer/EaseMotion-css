# SLA Breach Forecast

SLA Breach Forecast is a responsive reliability dashboard that estimates how close a service is to a breach window. It combines availability, error budget, ticket pressure, and mitigation pace in a compact operational view.

## Features

- Breach timing summary
- Four forecast metric cards
- Progress bars for quick scan value
- Responsive layout for mobile screens

## Files

- `demo.html` contains the SLA forecast markup.
- `style.css` contains the metric cards and responsive styling.
