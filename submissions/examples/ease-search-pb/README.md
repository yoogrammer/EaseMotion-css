# Ease Search Input Component (`ease-search-pr`)

A self-contained, highly interactive search input field element featuring custom alignment variants, expandable width micro-interactions, and a CSS-only automatic entry removal clear configuration button.

## Features
- **Integrated Search Icon:** Anchored cleanly onto the viewport structure using clean padding rules.
- **CSS-Only Clear Button:** Activates conditionally using structural `:placeholder-shown` attributes without relying on script listeners.
- **Dynamic Micro-expansions:** Smooth transitions expand input focus states cleanly.
- **Responsive Variants:** Includes distinct `default` and `compact` modular size setups.
