# Glow Pulse Utility

## What does this do?
A pulsing glow animation that cycles a box-shadow effect using CSS keyframes. Multiple color variants are available for different contexts.

## How is it used?
Add the class to any HTML element:

    <button class="glow-pulse glow-blue">Click me</button>

## Why is it useful?
Great for notification indicators, CTA buttons, live badges, and status dots that need to draw attention without JavaScript. The glow pulse creates a subtle breathing effect.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
