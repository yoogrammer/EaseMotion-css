Usage

<div class="ease-pagination">
    <div class="ease-page-item ease-page-disabled">
        <a href="#" class="ease-page-link">«</a>
    </div>
    <div class="ease-page-item ease-page-active">
        <a href="#" class="ease-page-link">1</a>
    </div>
    <div class="ease-page-item">
        <a href="#" class="ease-page-link">2</a>
    </div>
    <div class="ease-page-item">
        <a href="#" class="ease-page-link">3</a>
    </div>
    <div class="ease-page-item">
        <a href="#" class="ease-page-link">»</a>
    </div>
</div>

<div class="ease-pagination-small">...</div>

<!-- Rounded variant -->
<div class="ease-pagination-rounded">...</div>

📄 Pagination Features

  • 📑 Multiple size variants (small, normal, large)
  • 🎨 Color variants (primary, success, danger, dark)
  • 🟢 Active, disabled, and ellipsis states
  • ✨ Smooth hover animations
  • 🔄 Rounded variant option
  • 🎯 Fully accessible with links
