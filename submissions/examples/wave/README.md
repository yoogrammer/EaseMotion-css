# ease-wave

Waving/hand-gesture animation with translateX and rotate. Slows down on hover for interactive effect.

## Usage

```html
<span class="ease-wave">👋</span>
```

## Variants

| Class | Description |
|---|---|
| `ease-wave` | Default wave animation |
| `ease-wave:hover` | Slows down on hover |

## Notes

- Applies translateX and rotation for natural wave motion
- Hover reduces speed for interactive feel
- Works best on `display: inline-block` elements
- Respects `prefers-reduced-motion`
