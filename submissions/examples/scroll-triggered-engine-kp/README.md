# Scroll-Triggered Animation Engine

1. What does this do? Triggers premium viewport reveal transitions and staggered sequence animations on scroll with automatic prefers-reduced-motion safety.
2. How is it used? Apply `.reveal-on-scroll` along with a preset like `.reveal-fade-up` to an element, or use `.reveal-group` with `.reveal-item` for staggered sequences.
3. Why is it useful? It enables rich, viewport-aware layouts using simple classes and CSS variables, keeping markup clean while separating animation definitions from scroll tracking logic.
