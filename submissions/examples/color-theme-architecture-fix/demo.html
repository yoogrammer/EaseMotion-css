CSS Architecture Fix: Color Variables

This demo showcases how to properly structure color tokens using CSS Custom Properties, fixing the hardcoded color bug that actively blocks dark mode and dynamic theming.

Incorrect Architecture

This card uses hardcoded #FFFFFF and #111827.

When dark mode is toggled, this card remains completely broken and unreadable.

Correct Architecture

This card uses var(--bg-primary) and var(--text-primary).

This card flawlessly transitions to dark mode because it uses centralized variables.
