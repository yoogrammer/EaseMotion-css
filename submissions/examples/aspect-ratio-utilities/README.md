# Aspect Ratio Utilities

## What does this do?

Adds utility classes for common aspect ratios using the native CSS `aspect-ratio` property.

## How is it used?

```html
<div class="ease-aspect-square"></div>
<div class="ease-aspect-video"></div>
<div class="ease-aspect-portrait"></div>
```

## Why is it useful?

These utilities make it easy to create predictable square, video, and portrait layouts without extra wrappers, JavaScript, or custom sizing hacks. This fits EaseMotion CSS's philosophy of simple, reusable utilities that keep interface building lightweight and composable.
