# Animation Code Preview Toggle

## Description

This example demonstrates a toggle that allows users to switch between an animation preview and its implementation code.

## Features

- Preview mode
- Code mode
- View preference persistence using localStorage
- Responsive layout

## How It Works

Users can switch between Preview and Code tabs.

The selected view is stored in localStorage and automatically restored when the page is reopened.

## Files

- demo.html
- style.css

## Use Case

Useful for documentation websites where users may prefer either a visual demonstration or direct access to implementation code.
