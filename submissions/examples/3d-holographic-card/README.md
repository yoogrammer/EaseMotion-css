# Holographic Card

## What it does

A premium card component with a holographic shine effect that moves across the surface like light catching a holographic material.

## How to use it

<div class="ease-holo-card">
    <div class="ease-holo-card-inner">
        <div class="ease-holo-icon">🌟</div>
        <h3>Title</h3>
        <p>Description</p>
    </div>
</div>

## Variants

- ease-holo-fast - Fast shine animation (1.5s)
- ease-holo-slow - Slow shine (6s)
- ease-holo-intense - More vibrant colors
- ease-holo-subtle - Gentle effect
- ease-holo-gold - Gold/amber color scheme

## Why it fits EaseMotion CSS

- Pure CSS gradient animation
- Premium visual effect
- Hover lift interaction
- Multiple intensity variants
