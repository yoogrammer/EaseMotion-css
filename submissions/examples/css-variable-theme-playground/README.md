# CSS Variable Theme Playground

## Description

An interactive playground that demonstrates how EaseMotion CSS variables can be customized to theme the entire framework.

Users can modify core design tokens such as colors and instantly preview the changes on buttons, cards, and text components.

## Features

* Live CSS variable editing
* Instant visual preview
* Before and after comparison
* Generated CSS override code
* Copy-paste ready theme configuration

## Why is it useful?

While the existing documentation explains how CSS variables work, this playground allows users to experiment with them visually and understand their impact across the framework.

## Variables Demonstrated

* --ease-color-primary
* --ease-color-bg
* --ease-color-surface
* --ease-color-text

## Usage

Open `demo.html` in a browser.

Select colors using the controls and observe the live preview update automatically.

The generated CSS block can be copied directly into a project to apply the same theme.
