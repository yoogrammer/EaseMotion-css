# Feature Flag Management Component

## What does this do?

Displays feature flags, rollout status, environments, and enable/disable states in a modern dashboard layout.

## How is it used?

```html
<div class="flag-card enabled">
    <h3>Dark Mode</h3>
    <span class="status on">Enabled</span>
</div>
```

## Why is it useful?

Feature flags are widely used in modern SaaS platforms for gradual rollouts, beta testing, A/B testing, and controlled deployments. This component provides a clean and reusable interface for visualizing feature states using pure HTML and CSS.
