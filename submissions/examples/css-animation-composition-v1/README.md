# CSS animation-composition Property Example

Demo of the CSS `animation-composition` property — control how multiple keyframe animations combine on the same element.

## Features

- Three boxes showing `replace`, `add`, and `accumulate` composition modes
- Two overlapping animations run simultaneously: scale and rotate
- Visual comparison of composition behavior
- Dark theme with `prefers-reduced-motion` support

## Files

- `demo.html` — Three-box comparison layout
- `style.css` — Self-contained CSS (95 lines)

## Preview

Three animated boxes demonstrating how `replace`, `add`, and `accumulate` handle overlapping transform animations.
