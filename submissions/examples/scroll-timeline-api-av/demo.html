Native CSS Intersection Observer

I just faded and slid up purely using the new CSS view-timeline API. I did not violently trigger a heavy JavaScript IntersectionObserver on the browser's main thread!

Flawless 60fps Scrolling

Because this animation is offloaded entirely to the browser's compositor thread via native CSS, scrolling remains perfectly smooth on low-end mobile devices, saving massive amounts of battery life.
