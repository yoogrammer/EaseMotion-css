# Motion Design System Builder

### 1. What does this do?
This is a comprehensive motion design system management dashboard designed for EaseMotion CSS. It allows developers to define, visual-sequence, test, and export reusable animation design tokens (such as duration, easing, delay, scale, and translate values) for scalable and maintainable UI engineering.

### 2. How is it used?
Open `demo.html` directly in any modern web browser to access the interactive builder:
- **Preset Collections**: Select built-in system presets (like "System Clean", "Cinematic Spring", "Ambient Pulse", or "Safe Stable") to instantly populate the token sandbox.
- **Token Sandboxing**: Adjust sliders for base duration, item delay offsets, travel translate offsets, and scaling profiles.
- **Timeline Visualizer**: Observe the horizontal sequencer tracks representing the timing delay offsets for Avatar, Title, and Action button animations.
- **Mock Workspace**: View and trigger the animation sequence live on the mockup preview UI.
- **Export Utility**: Select and copy the generated responsive CSS design tokens and variables directly into your layouts.

### 3. Why is it useful?
It provides a single source of truth for motion systems. Instead of hardcoding random durations or coordinates throughout a project, designers and engineers can visually map staggering delays, check vestibular accessibility metrics, test viewports, and export synchronized token rules using EaseMotion CSS variables.
