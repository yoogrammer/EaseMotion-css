Scroll-Driven Reveal

Fades and slides elements up dynamically as they scroll into view. Runs entirely on native CSS scroll timelines with zero JavaScript listeners, zero layout thrashing, and graceful fallback.

1. Hardware Accelerated

Runs directly on the browser compositor thread. No main-thread scroll event listeners, preventing layout stutter and lag.

2. Graceful Fallback

Elements default to fully visible (`opacity: 1`) outside the supports query, ensuring older browsers or unsupported render engines display layout content immediately.

3. Accessibility Guard

Respects system settings. If prefers-reduced-motion is toggled, all scroll translations freeze to ensure comfortable reading.

Developer Implementation

To use this animation, apply the class .ease-scroll-reveal to any element you wish to reveal during scroll. 

<!-- The card slides up and fades in as the page scrolls down -->
<div class="ease-card ease-scroll-reveal">
  <h3>Scroll Reveal Card</h3>
</div>

How it works (CSS Underneath):

/* Base state is visible (safe fallback) */
.ease-scroll-reveal {
  opacity: 1;
  transform: translateY(0);
}

/* Progressive enhancement binding native timelines */
@supports (animation-timeline: scroll()) {
  .ease-scroll-reveal {
    animation: ease-kf-scroll-reveal linear both;
    animation-timeline: scroll(root block);
    animation-range: entry 0% cover 40%;
  }
}

@keyframes ease-kf-scroll-reveal {
  from {
    opacity: 0;
    transform: translateY(40px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}
