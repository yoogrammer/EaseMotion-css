## ease-blur-shadow-exit

**What does this do?**
Animates an element's exit with a blur and shadow fade effect, simulating depth as the element disappears.

**How is it used?**
<div class="blur-shadow-exit">Content</div>
<div class="blur-shadow-exit-slow">Content</div>

**Why is it useful?**
Adds a smooth, physical exit transition by combining blur, scale, and shadow animations — 
complementing EaseMotion CSS's entrance animation utilities with a matching exit counterpart.
