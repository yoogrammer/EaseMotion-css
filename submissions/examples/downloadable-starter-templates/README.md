# Downloadable Starter Templates

## Description

This example demonstrates a starter template gallery for EaseMotion CSS.

It showcases common project templates that can help developers get started quickly.

## Included Templates

- Landing Page Template
- Portfolio Template
- Dashboard Template

## Features

- Responsive card layout
- Template showcase section
- Download action buttons
- Beginner-friendly onboarding concept

## Files

- demo.html
- style.css
- README.md

## Use Case

Useful for documentation websites that want to provide starter project references for new users.
