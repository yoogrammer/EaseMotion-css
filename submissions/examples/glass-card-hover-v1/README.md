# Glass Card Highlight Effect

A glassmorphism-inspired card hover effect with subtle blur and glow feedback.

## Features

- Pure CSS
- Responsive layout
- No JavaScript
- Glassmorphism styling
- Smooth hover animation

## Usage

```html
<div class="glass-card">
  <h2>Glass Card</h2>
  <p>Your content here</p>
</div>
```

The card uses backdrop blur, transparency, and a glow effect to create a modern glass-like appearance.
