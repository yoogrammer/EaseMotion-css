# CSS Dialog Styling with Animation Example

## Description
A clean, self-contained implementation of the native HTML `<dialog>` element styled with human-readable CSS animations. Demonstrates layout control utilizing the native `::backdrop` pseudo-element layer along with body scroll-locking state controls.

## Features
- **Smooth Modals**: Handles entry animations dynamically via attributes.
- **Backdrop Filters**: Adds a blur backdrop shader using pure modern CSS selectors.
- **Scroll Locking**: Includes a script layer helper to lock primary content viewports seamlessly.
