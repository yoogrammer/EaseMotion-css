# Fraction Calculator

## What does this do?
A fraction calculator supporting addition, subtraction, multiplication, and division with automatic simplification, mixed number conversion, and step-by-step solution display.

## How is it used?
Enter numerators and denominators for two fractions, select an operation (+, −, ×, ÷), and view the result as a simplified fraction, decimal, and mixed number with detailed steps.

## Why is this useful?
This helps students and educators understand fraction arithmetic by showing the complete step-by-step process, including cross-multiplication and simplification.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to use the calculator.
