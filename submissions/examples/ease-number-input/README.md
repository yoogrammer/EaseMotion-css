\# ease-number-input



Styled number input with animated increment/decrement buttons (+/-).



\## Classes



| Class | Description |

|-------|-------------|

| `ease-number-input` | Base number input container |

| `ease-number-decrement` | Decrement button (−) |

| `ease-number-increment` | Increment button (+) |

| `ease-number-primary` | Primary color variant |

| `ease-number-success` | Success color variant |

| `ease-number-danger` | Danger color variant |

| `ease-number-sm` | Small size |

| `ease-number-lg` | Large size |



\## Features



\- 🔢 + and − buttons with hover animation

\- 💫 Bounce animation on value change

\- 🎨 Multiple color variants

\- 📏 Size variants (small, normal, large)

\- 🔒 Min/max clamping

\- 🖱️ Smooth interactions



\## Usage



```html

<div class="ease-number-input">

&#x20;   <button class="ease-number-decrement">−</button>

&#x20;   <input type="number" value="0" min="0" max="100">

&#x20;   <button class="ease-number-increment">+</button>

</div>
