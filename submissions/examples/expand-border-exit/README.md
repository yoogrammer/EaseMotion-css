# ease-expand-border-exit

Exit animation where the element's border expands outward (scale + border-width increase) as it fades out. A creative dismissal effect.

## Usage

```html
<div class="ease-expand-border-exit">Disappears with border expansion</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-expand-border-exit` | Default expand border exit |

## Notes

- Scales element up while border grows
- Useful for dismiss/remove animations
- Respects `prefers-reduced-motion`
