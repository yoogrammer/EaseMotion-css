Text Decoration Skip Ink Utility Classes

Control how underlines render around descenders (g, j, p, q, y) using text-decoration-skip-ink. Best seen on words with descenders, underlined.

💡 Note: Most browsers default to auto, which already skips descenders. none forces the line straight through every glyph — useful for strikethrough-style emphasis or deliberate brutalist underlines.

Common use cases

  • Decorative heading underlines with extra clearance — .ease-decoration-skip-ink-all
