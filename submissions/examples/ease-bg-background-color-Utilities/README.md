# Background Color Utilities

**What does this do?**
Provides semantic utility classes for applying common background colors directly in HTML. These utilities allow developers to quickly style elements using consistent color meanings such as primary, secondary, success, danger, warning, muted, and transparent.

**How is it used?**

```html
<div class="bg-primary">Primary</div>

<div class="bg-secondary">Secondary</div>

<div class="bg-muted">Muted</div>

<div class="bg-success">Success</div>

<div class="bg-danger">Danger</div>

<div class="bg-warning">Warning</div>

<div class="bg-transparent">Transparent</div>
```

**Why is it useful?**
Background colors are among the most frequently used styling utilities in modern web development. Developers often need quick access to semantic color classes for buttons, alerts, badges, cards, status indicators, and layout sections.

Providing these utilities improves consistency across projects while reducing repetitive custom CSS. The class names are intuitive, easy to remember, and align with EaseMotion CSS's philosophy of human-readable utilities that can be composed directly in HTML.

These utilities also create a foundation for future theming systems by associating colors with semantic meaning rather than specific color values.
