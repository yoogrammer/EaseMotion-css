# ease-fade-in utility

A lightweight utility class that fades an element in over **0.5 s** using the `easeFadeIn` keyframes.

## Usage
```html
<link rel="stylesheet" href="../../easemotion.css" />
<link rel="stylesheet" href="style.css" />
<div class="ease-fade-in">Fade‑in content</div>
```

## Demo
Open `demo.html` to see the animation in action.
