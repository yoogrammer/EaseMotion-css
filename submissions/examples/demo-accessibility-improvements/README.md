# Demo Accessibility Improvements

## What does this do?

Provides an accessibility-focused demo page showcasing responsive layouts, keyboard-friendly animation cards, search functionality, live previews, copy feedback, and theme switching.

---

## How is it used?

Open `demo.html` directly in a browser.

Example animation card:

```html
<button
  type="button"
  class="animation-card"
  data-animation="Bounce">
  Bounce
</button>
```

Example grid layout:

```html
<div class="grid-layout">
  <div class="card">Card 1</div>
  <div class="card">Card 2</div>
</div>
```

---

## Why is it useful?

This submission demonstrates modern accessibility and UX best practices that align with EaseMotion CSS goals:

* Keyboard-accessible interactions
* Proper semantic HTML
* Responsive mobile layouts
* Searchable animation catalog
* Live preview experience
* Copy-to-clipboard feedback
* Theme toggle support
* Improved readability and usability
