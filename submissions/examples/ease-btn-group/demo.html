Button Group

A layout component wrapper for combining multiple buttons into a cohesive control group with aligned borders and coordinated boundary radius corners.

Primary Group

Joins primary color-filled buttons. Hover borders elevate dynamically above adjacent buttons.

Outline Group

Overlaps borders by 2px in outline buttons to prevent double-border thickness.

Size Variations

Handles boundary corner radii overrides matching sm, md, or lg button padding sizes.

Action Controls

Integrates mixed button variants (colors, solid, and outline states) in a single row layout.

Developer Guide

Wrap buttons inside the `.ease-btn-group` class and optionally apply scale sizes (`.ease-btn-group-sm`, `.ease-btn-group-lg`): 

<div class="ease-btn-group ease-btn-group-lg">
  <button class="ease-btn ease-btn-outline">Option A</button>
  <button class="ease-btn ease-btn-outline">Option B</button>
</div>
