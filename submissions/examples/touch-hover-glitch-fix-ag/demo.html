Touch Hover Glitch Fix

On touch devices, rapidly tapping elements with :hover animations causes glitching or stuck states. The fix: gate hover with @media (hover: hover) and (pointer: fine) and use :active for touch feedback.

⚡ Fast Actions Hover on desktop, tap on mobile — no glitch.
🎯 Pointer-Aware Uses pointer media queries to detect device type.
