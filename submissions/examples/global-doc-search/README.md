# Global Documentation Search Component

A lightweight, fully interactive Command Palette search modal designed for the EaseMotion CSS documentation platform.

## Features

- **Keyboard Shortcut Trigger:** Use `Ctrl + K` (or `Cmd + K`) to immediately toggle the search window.
- **Dynamic Categorization:** Results are neatly bundled under logical headings (e.g., Animations, Utilities, Docs).
- **Full Keyboard Traversal:** Use Up/Down arrows to scroll focus index, `Enter` to select, and `Esc` to instantly close.
