Technical Specifications

  • Seamless Loop: Achieved by duplicating identical text tracks side-by-side and translating the marquee track exactly from 0% to -50%, snapping back instantly.
  • Houdini Speed Interpolation: Registers the custom variable --marquee-speed using @property. This allows CSS transitions to smoothly interpolate the animation duration from 26s to 6.5s upon hover.
  • 3D Perspective: Applies transform-style: preserve-3d and transitions the container tilt (rotateX() rotateY() rotateZ()) dynamically.
  • Performance Optimization: Utilizes will-change: transform to prompt browser layering, keeping rendering at a solid 60fps during acceleration transitions.
  • Accessibility: Marquee speed freezes to a safe, slow motion-free crawl and tilts are bypassed under @media (prefers-reduced-motion: reduce).
