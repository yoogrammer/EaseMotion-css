# AI Cost Analytics Dashboard Component

## What does this do?

Provides a dashboard interface for visualizing AI usage metrics, token consumption, and operational costs.

## How is it used?

```html
<div class="stat-card">
    <span class="label">Monthly Cost</span>
    <h2>$124.50</h2>
</div>
```

## Why is it useful?

Modern AI applications require visibility into token usage, model costs, and resource consumption. This component provides a reusable analytics dashboard while demonstrating responsive layouts and modern SaaS design patterns.
