# Ease Laser Grid Beacon

## Overview

A futuristic beacon animation featuring expanding laser pulses over a subtle grid with a glowing center, built entirely using HTML and CSS.

## Features

- Expanding beacon pulse
- Grid background effect
- Glowing center node
- Responsive layout
- Lightweight implementation
- Pure HTML & CSS

## Folder Structure

```text
ease-laser-grid-beacon-z1/
├── demo.html
├── style.css
└── README.md
```

## Installation

Open `demo.html` in any modern browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari

## Responsive Design

Optimized for desktop, tablet, and mobile devices.

Introduces a reusable beacon animation suitable for monitoring dashboards, AI interfaces, security systems, mapping tools, and futuristic UI concepts.
