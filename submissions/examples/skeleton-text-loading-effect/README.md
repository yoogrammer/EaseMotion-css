# Skeleton Text Loading Effect

## Description

A modern skeleton loading animation that mimics text content while data is loading.

## Features

- Smooth shimmer animation
- Lightweight CSS-only implementation
- Responsive design
- Easy customization

## Files

- demo.html
- style.css
- script.js

## Usage

Open `demo.html` directly in a browser to view the loading animation.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
