# Glow Text Utility

A smooth glow-style animation utility for text components.

## Features

- Lightweight pure CSS animation
- Smooth glowing text interaction
- Cross-browser compatible
- Reusable utility class

## Usage

```html
<h1 class="ease-glow-text-utility">
  Glow Effect
</h1>
```

## Animation Details

- Text emits a smooth glow effect during interaction
- Uses CSS keyframes and text-shadow
- Lightweight and modern utility animation
