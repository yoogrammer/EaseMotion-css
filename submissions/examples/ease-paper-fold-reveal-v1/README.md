# Ease Paper Fold Reveal

## Overview

Ease Paper Fold Reveal is a 3D interaction utility that mimics a sheet of paper unfolding to reveal hidden content using CSS transforms and perspective.

---

## Features

- 📄 Realistic paper folding animation
- 🎯 CSS 3D transforms
- ✨ Smooth perspective transition
- 🎨 Three card variations
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---

## Demo

Includes:

- Default fold card
- Blue variation
- Purple variation
- Hover reveal interaction

---

## Usage

```html
<div class="paper-card">
...
</div>
```

---

## Customization

You can customize:

- Card size
- Fold speed
- Colors
- Shadow intensity
- Border radius

---

## Browser Support

Works in all modern browsers supporting CSS transforms and perspective.
