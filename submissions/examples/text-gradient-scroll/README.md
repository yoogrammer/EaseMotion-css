# Text Gradient Scroll

## What does this do?
Gradient text animates its background position based on scroll progress.

## How is it used?
Add class `gradient-text` to any heading:

    <h1 class="gradient-text">Your Text</h1>

## Why is it useful?
Adds dynamic scroll-linked typography effects for landing pages and storytelling.

## Tech Stack
- HTML
- CSS
- Vanilla JS

## Preview
Open demo.html directly in your browser to see the effect.
