# Flip Clock Countdown

## What it does
A **retro split-flap countdown timer** with 3D flip animations on each digit change, using CSS `perspective` and `rotateX` keyframes.

## Animations & Techniques
- **3D flip**: Top half rotates away on `-90deg` X-axis.
- **Blinking colon**: `step-end` opacity toggle every second.
- **Monospace digits**: JetBrains Mono for authentic clock feel.
