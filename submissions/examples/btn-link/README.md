# ease-btn-link

Link-style button variant — renders as an underlined text link while maintaining button semantics. Part of `components/buttons.css`.

## Usage

```html
<button class="ease-btn ease-btn-link">Link button</button>
```

## Variants

| Class | Description |
|---|---|
| `ease-btn-link` | Underlined text link style |
| `ease-btn-link-icon` | Adds animated arrow suffix |

## Notes

- Customize color with inline `style="color: ..."` or CSS
- Pairs with `ease-btn-sm`/`ease-btn-lg` for sizing
- Accessible: maintains button role and keyboard interaction
