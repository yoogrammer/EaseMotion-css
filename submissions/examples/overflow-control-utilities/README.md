# Overflow Control Utilities

Utility classes for controlling content overflow behavior.

## Classes
| Class | CSS |
|-------|-----|
| `.ease-overflow-hidden` | `overflow: hidden` |
| `.ease-overflow-auto` | `overflow: auto` |
| `.ease-overflow-scroll` | `overflow: scroll` |
| `.ease-overflow-visible` | `overflow: visible` |
| `.ease-overflow-x-hidden` | `overflow-x: hidden` |
| `.ease-overflow-y-hidden` | `overflow-y: hidden` |
| `.ease-overflow-x-auto` | `overflow-x: auto` |
| `.ease-overflow-y-auto` | `overflow-y: auto` |

## Usage
```html
<div class="ease-overflow-hidden">Hidden overflowing content</div>
<div class="ease-overflow-auto">Scroll when necessary</div>
```

## Why EaseMotion CSS?
Human-readable, composable, lightweight — no dependencies.
