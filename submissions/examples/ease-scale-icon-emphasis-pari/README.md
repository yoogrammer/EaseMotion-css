## ease-scale-icon-emphasis

**What does this do?**
Animates icon components with scale effects during emphasis interactions like hover.

**How is it used?**
<button class="icon-btn scale-up">🚀</button>
<button class="icon-btn scale-down">🔥</button>
<button class="icon-btn scale-pop">⭐</button>
<button class="icon-btn scale-pulse">💡</button>

**Why is it useful?**
Adds expressive scale-based micro-interactions to icon buttons using pure CSS,
making UI feel responsive and alive — consistent with EaseMotion CSS's emphasis animation philosophy.
