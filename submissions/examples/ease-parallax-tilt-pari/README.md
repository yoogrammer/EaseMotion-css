## ease-parallax-tilt

**What does this do?**
Tilts card elements in 3D space on hover using CSS perspective and rotateX/rotateY transforms.

**How is it used?**
<div class="tilt-card tilt-left">Content</div>
<div class="tilt-card tilt-right">Content</div>
<div class="tilt-card tilt-up">Content</div>
<div class="tilt-card tilt-down">Content</div>

**Why is it useful?**
Creates a parallax depth illusion using pure CSS 3D transforms and perspective —
no JavaScript needed, consistent with EaseMotion CSS's lightweight animation philosophy.
