# Hover Lift Product Card

A responsive product card that lifts slightly and enhances its shadow on hover.

## Features

- Pure CSS
- Transform-based hover animation
- Responsive layout
- No JavaScript required

## Usage

```html
<div class="product-card">
  <div class="product-image"></div>

  <h2>Product Name</h2>

  <p>Product description</p>

  <span class="price">$99</span>
</div>
```

The card uses transform and shadow transitions to create a subtle lift effect on hover.
