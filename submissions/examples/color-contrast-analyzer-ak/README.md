# Color Contrast Analyzer

## What does this do?
A color contrast analyzer with color pickers, real-time contrast ratio calculation, WCAG AA/AAA compliance indicators for normal and large text sizes with visual preview.

## How is it used?
Pick foreground and background colors using the color inputs. The contrast ratio is calculated in real time. WCAG compliance is shown for AA and AAA levels for normal (≥18px) and large (≥24px) text.

## Why is this useful?
This helps designers and developers ensure their color combinations meet accessibility standards for readable text.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to use the analyzer.
