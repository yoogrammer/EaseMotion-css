# CSS Aspect Ratio Responsive Gallery

A responsive image gallery using the CSS `aspect-ratio` property to maintain consistent card proportions across viewports.

## Features

- Grid layout with `aspect-ratio: 3/4` cards
- Responsive columns (3 → 2 → 1)
- Hover scale effect with overlay
- Dark theme with `prefers-reduced-motion` support

## Files

- `demo.html` — Gallery grid with placeholder cards
- `style.css` — Self-contained CSS (75 lines)

## Preview

A responsive gallery where every card maintains a 3:4 aspect ratio regardless of screen width.
