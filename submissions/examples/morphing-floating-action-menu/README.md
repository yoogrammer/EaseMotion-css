# Morphing Floating Action Menu

A modern **CSS-only Floating Action Button (FAB)** that expands into an animated action menu without using JavaScript.

## Features

- ✅ Pure HTML & CSS
- ✅ Floating Action Button
- ✅ Morphing "+" into "×"
- ✅ Smooth expand/collapse animation
- ✅ Hover scaling effects
- ✅ Responsive design
- ✅ No JavaScript required

## Folder Structure

```
morphing-floating-action-menu/
├── demo.html
├── style.css
└── README.md
```

## How to Run

1. Open `demo.html` in any modern browser.
2. Click the floating action button in the bottom-right corner.
3. The action menu expands with smooth CSS transitions.

## Customization

You can easily customize:

- Button color
- Menu icons
- Animation duration
- Position of the FAB
- Number of menu items

## Technologies Used

- HTML5
- CSS3 (Transforms, Transitions, Flexbox)

## Preview

A floating circular button expands into four animated action buttons using only CSS, making it suitable for dashboards, mobile-inspired interfaces, and modern web applications.
