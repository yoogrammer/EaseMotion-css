# CSS Content Visibility Lazy Render Example

This example demonstrates CSS `content-visibility: auto` for lazy rendering of off-screen sections using `contain-intrinsic-size`.

## Features

- Lazy rendering for performance optimization
- Smooth scrolling
- Placeholder sizing with `contain-intrinsic-size`

## Files

- `demo.html`
- `style.css`

## Usage

Open `demo.html` in the browser to see the example.
