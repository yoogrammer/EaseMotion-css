# Expandable Data Table Showcase

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A modern expandable data table component that allows users to reveal additional information beneath each row through a smooth slide-down animation.

Inspired by admin dashboards, CRM systems, analytics panels, and enterprise management interfaces.

## Features

- Pure HTML and CSS
- Expandable table rows
- Smooth slide-down animation
- Rotating chevron indicator
- Status badges
- Employee avatars
- Responsive layout
- Hover interactions
- Modern dashboard styling
- No JavaScript required

## Usage

```html
<details class="table-row">
  <summary>
    Row Content
  </summary>

  <div class="expanded-content">
    Additional Details
  </div>
</details>
```

## Main Classes

```css
.table-row
.expanded-content
.chevron
.status
.avatar
```
