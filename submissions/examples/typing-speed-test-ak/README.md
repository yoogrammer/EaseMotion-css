# Typing Speed Test

## What does this do?
A typing speed test that measures WPM and accuracy with real-time character highlighting, configurable duration, and detailed results.

## How is it used?
Select a duration (30s/60s/120s), click Start, and type the displayed words. Correct characters turn green, incorrect turn red. Results show WPM, accuracy, and character counts.

## Why is this useful?
Track and improve your typing speed and accuracy over time with a clean, distraction-free interface.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to take the test.
