# FAQ Accordion Component

## What does this do?

A responsive FAQ accordion component built entirely with HTML and CSS that allows users to expand and collapse questions without JavaScript.

## How is it used?

```html
<details class="faq-item">
  <summary>What is EaseMotion CSS?</summary>
  <p>EaseMotion CSS is a lightweight CSS framework.</p>
</details>
```

Multiple FAQ items can be grouped inside a container:

```html
<div class="faq-container">
  <!-- FAQ Items -->
</div>
```

## Why is it useful?

FAQ sections are commonly used in landing pages, documentation websites, SaaS products, portfolios, and business websites. This component provides a lightweight, dependency-free solution that aligns with EaseMotion CSS's philosophy of simple and reusable UI patterns.
