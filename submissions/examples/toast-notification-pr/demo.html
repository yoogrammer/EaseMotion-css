Pure CSS toast notifications using the :target pseudo-class. No JavaScript required.

Trigger Notifications

✓ Show Success ! Show Error ⚠ Show Warning i Show Info

How It Works

  1. Click a trigger button — it links to a toast's id (e.g., #toast-success)
  2. The :target pseudo-class activates, sliding the toast in from the right
  3. A progress bar animates for 5 seconds as a visual timer
  4. Click the × button — it links to #hide, removing the target
  5. The toast slides out and hides

Inside EaseMotion Card

Toast notifications can be triggered from anywhere, including inside cards.
