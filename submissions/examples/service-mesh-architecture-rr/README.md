# Service Mesh Architecture Visualization Component

## What does this do?

Visualizes a modern service mesh architecture where services communicate through sidecar proxies managed by a centralized control plane.

## How is it used?

```html
<div class="service">
    Service A
</div>

<div class="sidecar">
    Sidecar Proxy
</div>
```

## Why is it useful?

Service meshes are widely used in cloud-native and Kubernetes environments to manage communication, observability, security, and traffic routing between services. This component provides a clean visual representation of those concepts using pure HTML and CSS.
