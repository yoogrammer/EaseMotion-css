## ease-shockwave

**What does this do?**
Propagates a visual wave across sibling elements when one is hovered,
creating a chain-reaction scale and opacity effect.

**How is it used?**
<div class="ease-shockwave-group">
  <button class="ease-shockwave-item">One</button>
  <button class="ease-shockwave-item">Two</button>
  <button class="ease-shockwave-item">Three</button>
</div>

**Why is it useful?**
Creates dynamic group-based interactions using pure CSS :has() selector
and sibling combinators — consistent with EaseMotion CSS's
animation-first philosophy.
