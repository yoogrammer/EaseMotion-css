# API Request Lifecycle Visualization Component

## What does this do?

Visualizes the lifecycle of an API request as it travels through infrastructure, authentication, business logic, database access, and response generation.

## How is it used?

```html
<div class="step gateway">
    <h3>API Gateway</h3>
</div>

<div class="step auth">
    <h3>Authentication</h3>
</div>
```

## Why is it useful?

Understanding API request processing is a fundamental concept in backend development and system design. This component provides a clear visualization of request flow through various system layers using pure HTML and CSS.
