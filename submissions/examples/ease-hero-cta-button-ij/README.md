# Hero CTA Button (#36725)

A call-to-action button with a pulsing attract ring animation, designed for hero sections.

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--btn-bg` | `#8b5cf6` | Background color of the button |
| `--btn-color` | `#ffffff` | Text and arrow color |
| `--pulse-color` | `rgba(139,92,246,0.5)` | Color of the expanding pulse ring |
| `--pulse-duration` | `2s` | Duration of one pulse cycle |

## Interactive Demo

Open `demo.html` to see the button in action. Hover to lift and scale the button. Use the controls to customize colors and pulse timing.
