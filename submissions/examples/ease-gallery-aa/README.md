# Visual Animation Reference Gallery

### ❓ What does this do?
This is an interactive dashboard gallery that provides a live visual reference, real-time filtering, and one-click class name copying for EaseMotion CSS animation rules.

### ⚙️ How is it used?
Users can search for specific animation behaviors or filter by categories (Entrance, Exit, Hover, Attention). Hovering over a card shows the live preview, and clicking the code block instantly copies the string to the clipboard.

### 💡 Why is it useful?
It removes the friction of reading through raw CSS source files, allowing developers to visually sample and pick animation styles instantly.
