# Glassmorphism Pricing Card

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A premium glassmorphism pricing card showcase featuring blurred backgrounds, floating gradient glows, pricing tiers, hover animations, and responsive layouts using pure HTML and CSS.

## Features

- Glassmorphism card design
- Three pricing tiers
- Featured plan highlight
- Hover lift animation
- Gradient CTA buttons
- Floating background glows
- Responsive layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="pricing-card">
  Pricing Content
</div>
```

## Main Classes

```css

.featured
.plan-badge
.popular
.pricing-grid
```
