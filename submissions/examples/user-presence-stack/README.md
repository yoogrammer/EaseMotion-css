# User Presence Stack

## What does this do?

Creates a stack of overlapping user avatars with hover animations.

## How is it used?

```html
<div class="presence-stack">
    <img src="avatar.jpg" alt="User">
</div>
```

## Why is it useful?

Presence stacks are widely used in collaboration tools, dashboards, and team management interfaces to display active members in a compact and visually appealing way.
