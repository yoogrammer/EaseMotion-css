# Rotate Border Utility

A lightweight rotating border animation utility built with pure CSS.

## Features

* Animated rotating gradient border
* Pure CSS implementation
* Lightweight and reusable
* Smooth infinite rotation
* No JavaScript required

## Files

* `demo.html`
* `style.css`

## Usage

```html id="aq3f7v"
<button class="ease-rotate-border">
  Hover Me
</button>
```

## Concepts Used

* CSS pseudo-elements
* `@keyframes`
* `conic-gradient`
* transform rotation
* layered border effects
