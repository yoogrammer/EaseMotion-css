# Accordion Expand Indicator

A CSS-only accordion component with a rotating expand indicator.

## Features

- Pure CSS
- Smooth indicator rotation
- Keyboard accessible
- Responsive layout
- No JavaScript required

## Usage

```html
<details class="accordion">
  <summary>
    Accordion Item
    <span class="indicator">+</span>
  </summary>

  <p>Accordion content</p>
</details>
```

The indicator smoothly rotates when the accordion is expanded, providing visual feedback to users.
