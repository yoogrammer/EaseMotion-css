# Conic Progress Ring

## What does this do?
Circular progress indicators using CSS conic-gradient to show fill percentage. Multiple rings display different progress levels with a hover scale effect.

## How is it used?
Add the class to any HTML element:

    <div class="progress-ring" style="background: conic-gradient(#6366f1 0% 50%, #e0e0e0 50% 100%);">
      <span class="ring-value">50%</span>
    </div>

## Why is it useful?
Lightweight, no-JS progress indicators ideal for dashboards, skill meters, storage usage displays, and loading states. Works with any color and percentage.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
