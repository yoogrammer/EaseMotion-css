# Supply Chain Global Ops Command Center

This example showcases a **Supply Chain & Logistics** operations dashboard built with **EaseMotion CSS**. It demonstrates:

- An industrial-themed dark UI with orange and blue accents suitable for operations centers.
- A 3-column dense grid layout optimized for large monitors.
- Over 15 distinct UI sections including a simulated map, timeline, tables, and AI rerouting suggestions.
- Motion effects including continuous radar ping animations, a CSS-only map scanner line, and hover interactions.

## Directory Structure

```
/supply-chain-ops-bv/
│   demo.html   # Interactive demo page
│   style.css   # Custom CSS variables and animations for this example
│   README.md   # This documentation
```

## How to Run

Open `demo.html` in a modern browser. No external dependencies are required; all styling and layout are handled by the included CSS.

## Key Features

- **Map Visualization**: CSS-driven nodes and radar pings on a grid background.
- **Logistics Timeline**: Custom styled tracking steps.
- **Data Table**: High-density table with status badges.
- **AI Reroute**: Call to action cards emphasizing decision making.

Built with **EaseMotion CSS** – a lightweight, utility-first CSS framework for motion-rich interfaces.
