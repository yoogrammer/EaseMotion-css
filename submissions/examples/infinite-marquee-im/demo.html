How to Use

html
<!-- Wrap content in .marquee > .marquee-track -->
<!-- Duplicate all items once for seamless looping -->
<div class="marquee">
  <div class="marquee-track">
    <span class="marquee-item">Item 1</span>
    <span class="marquee-item">Item 2</span>
    <!-- Duplicate -->
    <span class="marquee-item">Item 1</span>
    <span class="marquee-item">Item 2</span>
  </div>
</div>

<!-- Reverse direction -->
<div class="marquee marquee--reverse">...</div>

<!-- Custom speed -->
<div class="marquee" style="--marquee-speed: 35s">...</div>

<!-- Vertical -->
<div class="marquee marquee--vertical">
  <div class="marquee-track marquee-track--vertical">...</div>
</div>
Important: Duplicate all child items exactly once inside .marquee-track — this is what makes the loop seamless. The animation translates exactly -50% (or -50% on Y for vertical), so the duplicated set lines up perfectly with the original.
