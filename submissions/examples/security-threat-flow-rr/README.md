# Security Threat Flow Visualization Component

## What does this do?

Displays the flow of a request through multiple security layers, from external traffic to protected resources.

## How is it used?

```html
<div class="step firewall">
    <h3>Firewall</h3>
</div>
```

## Why is it useful?

Security architectures often involve multiple layers of protection such as firewalls, authentication systems, authorization checks, and secured resources. This component provides a clean visualization of those relationships using pure HTML and CSS.
