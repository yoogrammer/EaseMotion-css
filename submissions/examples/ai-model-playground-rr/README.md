# AI Model Playground Component

## What does this do?

Provides a responsive interface for comparing responses from multiple AI models side by side.

## How is it used?

```html
<div class="model-card">
    <span class="model-badge">GPT-5</span>
    <p>Model response...</p>
</div>
```

## Why is it useful?

AI applications often require model comparison interfaces for testing prompts and evaluating outputs. This component provides a clean and reusable playground layout while showcasing responsive cards and modern AI-focused design patterns.
