## ease-slide-button-entrance

**What does this do?**
Animates button entrance with a smooth slide-in effect from any direction — left, right, top, or bottom.

**How is it used?**
<button class="slide-btn-left">Slide from Left</button>
<button class="slide-btn-right">Slide from Right</button>
<button class="slide-btn-top">Slide from Top</button>
<button class="slide-btn-bottom">Slide from Bottom</button>

**Why is it useful?**
Gives buttons a dynamic entrance feel using pure CSS keyframes with a subtle overshoot bounce,
consistent with EaseMotion CSS's animation philosophy.
