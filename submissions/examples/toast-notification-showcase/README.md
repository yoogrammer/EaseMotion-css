# Toast Notification Showcase

## Description

This example demonstrates animated toast notifications using EaseMotion-style interactions.

## Features

- Success notification
- Error notification
- Warning notification
- Info notification
- Entrance animation
- Exit animation
- Implementation example

## How It Works

Users can trigger different notification types using the provided buttons.

Each toast appears with an entrance animation and automatically disappears using an exit animation.
