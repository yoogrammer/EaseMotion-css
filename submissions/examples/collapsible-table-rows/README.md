# Collapsible Table Rows

A data table where clicking a parent row toggles the visibility of a detail row beneath it. An arrow indicator changes from ▶ to ▼ when expanded. The detail row contains additional information spanning all columns.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser. Click any task row to expand or collapse its details.

## Accessibility notes

Parent rows are clickable with a cursor pointer. Toggle state is indicated by both the arrow symbol and the `.open` class.
