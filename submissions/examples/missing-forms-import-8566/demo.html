Missing forms.css Import

Issue #8566 — easemotion/all.css (modular bundle) omits the forms.css import that exists in the main easemotion.css entry point.

The Problem

easemotion.css imports forms.css at line 44, but easemotion/all.css has no such import. Any user loading the framework via easemotion/all.css gets buttons, cards, navbar, etc. — but no form styles.

--- easemotion.css (line 44) ✓ @import "./components/forms.css"; --- easemotion/all.css ✗ (missing) @import "../components/forms.css"; <-- NOT PRESENT

Suggested Fix

Add the following line to easemotion/all.css (after modals.css):
