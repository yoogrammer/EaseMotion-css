# Wiggle Animation

This example demonstrates the usage of the `ease-wiggle` animation utility.

## Description

The `ease-wiggle` utility class applies a continuous, playful side-to-side rotational animation. 

This animation is commonly used in mobile and web UIs to indicate that an element is in an "editable" or "draggable" state (similar to iOS home screen icons when rearranging apps), or simply to draw attention to an interactive element.

## Usage

Include the `style.css` containing the animation, then simply apply the class to your HTML element.

```html
<div class="ease-wiggle">
  I'm ready to be moved or edited!
</div>
```

## Preview

Open `demo.html` in your browser to test out the wiggle animation on a sample "App Icon" component.
