# ease-pulse

Infinite pulse opacity animation. A subtle attention-drawing effect for notification dots, live indicators, and status badges.

## Usage

```html
<div class="ease-pulse">Pulsing content</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-pulse` | Default pulse (opacity fade loop) |
| `ease-pulse-fast` | Faster pulse |
| `ease-pulse-slow` | Slower pulse |

## Notes

- Opacity oscillates between 1 and 0.5
- Loops continuously
- Respects `prefers-reduced-motion`
