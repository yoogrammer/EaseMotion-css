# QR Code Generator

## What
A QR code generator that converts text input into a visual QR-style dot matrix on canvas with size selection.

## How
Uses an HTML canvas and a deterministic hashing algorithm to produce a binary matrix from input text. Finder patterns (position detection squares) are drawn at three corners for QR-like appearance. Supports small, medium, and large sizes.

## Why
Demonstrates canvas-based rendering and hash-based pattern generation, creating a visual representation of text data without external APIs.
