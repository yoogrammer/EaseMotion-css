# ease-justify-items utilities

Controls `justify-items` for grid/flexbox alignment along the inline axis.

## Usage

```html
<div class="ease-grid ease-justify-items-center">
  <div>Centered</div>
</div>
```

## Classes

| Class                        | CSS Property             |
| ---------------------------- | ------------------------ |
| `ease-justify-items-start`   | `justify-items: start`   |
| `ease-justify-items-center`  | `justify-items: center`  |
| `ease-justify-items-end`     | `justify-items: end`     |
| `ease-justify-items-stretch` | `justify-items: stretch` |
