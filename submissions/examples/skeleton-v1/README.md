# ease-skeleton

CSS loading placeholder shimmer animation using `background-position` keyframes. Zero JavaScript required.

## Usage

```html
<div class="ease-skeleton ease-skeleton-text"></div>
<div class="ease-skeleton ease-skeleton-circle"></div>
<div class="ease-skeleton ease-skeleton-rect"></div>
```

## Variants

| Class | Description |
|---|---|
| `ease-skeleton-text` | Full width text line placeholder |
| `ease-skeleton-circle` | Circle avatar placeholder |
| `ease-skeleton-rect` | Rectangle image/banner placeholder |
| `ease-skeleton-avatar` | Small circle avatar |
| `ease-skeleton-dark` | Dark background shimmer |
| `ease-skeleton-w-3/4` | 75% width |
| `ease-skeleton-w-1/2` | 50% width |
| `ease-skeleton-w-1/3` | 33% width |

## Custom Properties

| Property | Default | Description |
|---|---|---|
| `--ease-skeleton-base` | `#e0e0e0` | Base shimmer color |
| `--ease-skeleton-shine` | `#f5f5f5` | Highlight shimmer color |
