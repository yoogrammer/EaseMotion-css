# ease-place-content utilities

Controls `place-content` (shorthand for `align-content` + `justify-content`) for grid/flexbox layouts.

## Usage

```html
<div class="ease-grid ease-place-content-center">
  <div>Item</div>
</div>
```

## Classes

| Class                        | CSS Property                   |
| ---------------------------- | ------------------------------ |
| `ease-place-content-start`   | `place-content: start`         |
| `ease-place-content-center`  | `place-content: center`        |
| `ease-place-content-end`     | `place-content: end`           |
| `ease-place-content-between` | `place-content: space-between` |
| `ease-place-content-around`  | `place-content: space-around`  |
| `ease-place-content-evenly`  | `place-content: space-evenly`  |
| `ease-place-content-stretch` | `place-content: stretch`       |
