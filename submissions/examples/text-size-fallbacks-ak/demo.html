Fix summary

Add a hardcoded fallback to every var() reference in text-size utilities so they render correctly even when CSS custom properties are not defined.

.ease-text-xs  { font-size: var(--ease-font-size-xs, 0.75rem); }
.ease-text-sm  { font-size: var(--ease-font-size-sm, 0.875rem); }
.ease-text-base { font-size: var(--ease-font-size-base, 1rem); }
.ease-text-lg  { font-size: var(--ease-font-size-lg, 1.125rem); }
.ease-text-xl  { font-size: var(--ease-font-size-xl, 1.25rem); }
.ease-text-2xl { font-size: var(--ease-font-size-2xl, 1.5rem); }
