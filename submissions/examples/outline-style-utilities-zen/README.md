## What does this do?

Provides outline-style and outline-none utility classes for accessible focus indicators.

## How is it used?

```html
<button class="outline-none">No outline on focus</button>
<div class="outline-dashed">Dashed outline</div>
```

## Why is it useful?

Outline control is critical for accessibility and custom focus styles. These complement EaseMotion's existing border utilities.
