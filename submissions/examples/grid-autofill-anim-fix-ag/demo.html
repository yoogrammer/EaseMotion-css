Grid auto-fit + Animation Fix

Animating grid children directly breaks auto-fit column widths. Fix: keep the grid item as a plain layout container, and animate only the inner wrapper — so layout is never disturbed.

Grid layout never shifts during animation.

Auto-fill Safe
