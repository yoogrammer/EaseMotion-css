# Cloud Cost Optimization Dashboard Component

## What does this do?

Provides a cloud cost monitoring dashboard displaying provider spending, optimization opportunities, budget tracking, and potential savings.

## How is it used?

```html
<div class="card aws">
    <h2>$12.4K</h2>
    <span>AWS Spend</span>
</div>
```

## Why is it useful?

Cloud cost optimization is a critical part of FinOps and cloud operations. This component helps visualize spending, savings opportunities, and budget utilization using pure HTML and CSS.
