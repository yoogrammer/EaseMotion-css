# Timezone Converter

## What does this do?
A timezone converter that displays current time across multiple world cities with DST indicators, and converts a selected time between source and target timezones.

## How is it used?
Select source and target timezones from the dropdowns. The live clocks update every second showing current time in each city with DST status. Enter a specific time to convert between zones.

## Why is this useful?
This is ideal for scheduling meetings across time zones, travel planning, or any application needing multi-timezone awareness with live updates.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to use the converter.
