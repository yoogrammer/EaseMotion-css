# ease-container

Max-width centered container utility. The fundamental layout building block for constraining content width.

## Usage

```html
<div class="ease-container">
  <p>Centered content with max-width</p>
</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-container` | Centered container with max-width |
| `ease-container-sm` | Smaller max-width |
| `ease-container-lg` | Larger max-width |

## Notes

- Uses `--ease-container-max` CSS variable for width
- Includes responsive padding at sm+ breakpoint
- Centers with `margin-inline: auto`
- Essential layout utility for every project
