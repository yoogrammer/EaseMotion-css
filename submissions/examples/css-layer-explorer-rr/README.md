# Interactive CSS Layer Explorer Component

## What does this do?

Provides a visual representation of CSS Cascade Layers (`@layer`) and demonstrates how layer priority affects style overrides.

## How is it used?

```html
<div class="layer components">
    <h2>@layer components</h2>
</div>
```

## Why is it useful?

CSS Cascade Layers are a modern CSS feature used to organize styles and control specificity. This component helps developers understand layer hierarchy through an interactive visual interface built with pure HTML and CSS.
