ease-kf-bounce Keyframe Easing Documentation

The framework @keyframes ease-kf-bounce hardcodes cubic-bezier values inside keyframes instead of relying on the animation-timing-function set at the class level. This example documents the correct approach:

/* Current keyframes — hardcodes easing at 0% and 50% */
@keyframes ease-kf-bounce {
  0%, 100% { transform: translateY(0); animation-timing-function: cubic-bezier(0.8, 0, 1, 1); }
  50% { transform: translateY(-20px); animation-timing-function: cubic-bezier(0, 0, 0.2, 1); }
}

/* Recommended: remove hardcoded keyframe timing functions so
   the class-level animation-timing-function (var(--ease-ease-bounce))
   is respected consistently across the animation. */
.ease-bounce {
  animation: ease-kf-bounce 1s var(--ease-ease-bounce) infinite;
}
