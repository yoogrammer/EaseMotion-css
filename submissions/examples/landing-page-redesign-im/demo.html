Animation-first CSS

EaseMotion CSS

A human-readable CSS library. Write UI with simple, English-like class names — no memorizing utilities, no complex configuration, no build steps.

Get started See animations →
💡
EaseMotion CSS bridges the gap between raw vanilla CSS and utility-heavy frameworks like Tailwind. If you can say it in English, you can write it as a class.
Core principles

Design Philosophy

📖
Human-readable

Class names read like plain English — ease-fade-in, not fx-op-0-d400.

✨
Animation-first

Animations are first-class citizens, not accessories bolted on later.

🧩
Composable

Stack modifier classes freely — no specificity wars, no overrides needed.

⚡
Clean & minimal

No dependencies, no build steps, no JavaScript required for core styling.

Setup

Installation

html 
<!-- Recommended — works everywhere, no build step -->
<link rel="stylesheet"
  href="https://cdn.jsdelivr.net/npm/easemotion-css/easemotion.css" />
bash 
npm install easemotion-css
js 
import 'easemotion-css/easemotion.css';
Tokens

Variables

All values are defined as CSS custom properties in :root. Override any token to theme the entire framework.

css 
:root {
  --ease-color-primary:  #6c63ff;
  --ease-speed-fast:     150ms;
  --ease-speed-medium:   300ms;
  --ease-speed-slow:     600ms;
  --ease-radius-md:      0.5rem;
  --ease-font-sans:      'Inter', system-ui, sans-serif;
}
Theming

Dark Mode

EaseMotion CSS ships with full dark mode support via CSS custom properties. You can also drive it with a data-theme attribute (as this page does).

css 
@media (prefers-color-scheme: dark) {
  :root {
    --ease-color-bg:      #0b1121;
    --ease-color-surface: #141e33;
    --ease-color-text:    #e2e8f0;
    --ease-color-muted:   #94a3b8;
  }
}
Layout

Utilities

Layout helpers that read like what they do.

html 
<!-- Center content -->
<div class="ease-center"> ... </div>

<!-- Flex row with gap -->
<div class="ease-flex ease-gap-4 ease-items-center"> ... </div>

<!-- Responsive grid -->
<div class="ease-grid ease-grid-3 ease-gap-4"> ... </div>
Motion

Animations

Add animation with a single class. Every animation respects prefers-reduced-motion out of the box.

Preview Class Effect
ease-fade-in Fades element from 0 → 1 opacity
ease-slide-up Slides element up while fading in
ease-bounce Infinite vertical bounce
ease-hover-grow Scales up on hover (elastic)

Contributing

EaseMotion CSS is curated. You submit raw CSS ideas inside submissions/examples/ — the maintainer reviews, standardizes naming, and integrates into the framework.

Naming Rules

You don't need to follow the ease- convention — write class names that communicate intent clearly. The maintainer handles all standardization.

You write → .hover-grow, .card-lift, .shimmer-btn — whatever reads clearly to you
Maintainer → Renames to ease-hover-grow, replaces values with CSS variables, adds to framework
