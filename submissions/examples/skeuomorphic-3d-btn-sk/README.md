# Skeuomorphic 3D Push Button

A highly realistic, tactile button that mimics a physical keyboard key or heavy machinery button.

## Files
- `demo.html`: The HTML structure for the skeuomorphic button.
- `style.css`: The complex CSS gradients and box-shadows used to draw the 3D button.
- `README.md`: This file.

## Features
- **Pure CSS 3D:** Uses stacked `box-shadow` inset/outset layers and `linear-gradient` backgrounds to simulate light hitting a physical object. No SVG or external assets are used.
- **Realistic Active State:** Clicking the button physically depresses the cap, compressing the shadow layers dynamically.
- **Text Emboss:** The text itself has subtle `text-shadow` applied to make it look engraved into the plastic button cap.

## Usage
Add the `.skeuomorphic-3d-btn` class to your button. The button's colors can be adjusted by changing the CSS variables.
