# Float Animation

This example demonstrates the usage of the `ease-float` animation utility.

## Description

The `ease-float` utility class applies a gentle, continuous vertical translation to an element. 

This animation is extremely popular in modern web design for adding an organic, subtle sense of depth and movement to hero section illustrations, 3D graphics, floating badges, and primary call-to-action buttons. It operates continuously (infinite) and smoothly reverses its direction.

## Usage

Include the `style.css` containing the animation, then apply the class to your HTML element.

```html
<div class="ease-float">
  ☁️ I'm floating gently...
</div>
```

## Preview

Open `demo.html` in your browser to test out the float animation on a sample glassy badge component.
