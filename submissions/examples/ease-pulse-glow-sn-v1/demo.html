Pulse & Glow Animations

Attention-grabbing animation utilities for live indicators, CTAs, active states, and featured content highlights.

Pulse Animation

⚡
.animate-pulse Gentle scale pulse (1 → 1.05)
⚡
.animate-pulse-fast Fast pulse (1s duration)
⚡
.animate-pulse-slow Slow pulse (4s duration)
⚡
.animate-pulse Pulse on circular elements

Use Cases

🔴 Recording / Live Indicator

A pulsing red dot to show live recording or streaming status — perfect for podcast apps, video tools, and live broadcast UIs.

LIVE RECORDING

🎯 Call-to-Action Buttons

Draw attention to primary CTAs, sign-up buttons, and submit actions with a gentle glow effect.

Highlight the currently focused input field with a pulsing glow, guiding the user's eye through the form.

CSS Custom Properties

Property Default Description
--pulse-intensity 0.05 Controls the pulse scale factor. 0.05 = 5% scale increase.
--glow-color rgba(108, 99, 255, 0.5) Sets the glow color and opacity. Use any valid CSS color value.
