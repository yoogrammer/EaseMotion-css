Usage

<!-- Basic table -->
<div class="ease-table-responsive">
    <table class="ease-table">
        <thead>
            <tr><th>Header</th></tr>
        </thead>
        <tbody>
            <tr><td>Data</td></tr>
        </tbody>
    </table>
</div>

<!-- Striped rows -->
<table class="ease-table ease-table-striped">...</table>

<!-- Hover effect -->
<table class="ease-table ease-table-hover">...</table>

<!-- Compact table -->
<table class="ease-table ease-table-compact">...</table>

<!-- Bordered table -->
<table class="ease-table ease-table-bordered">...</table>

📊 Table Features

  • 📋 ease-table - Base table styling
  • 📏 ease-table-striped - Alternating row colors
  • ✨ ease-table-hover - Row hover effect
  • 📐 ease-table-compact - Denser padding
  • 📦 ease-table-bordered - Full borders
  • 📱 ease-table-responsive - Horizontal scroll on mobile
