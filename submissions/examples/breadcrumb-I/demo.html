🍞 Breadcrumb Features

  • 📂 Multiple separator styles (chevron, double, slash, dot)
  • 🔄 Animated separators (rotate, scale, slide on hover)
  • 🎨 Light and dark theme support
  • 🖼️ Icon support with emojis or icons
  • 📱 Responsive and accessible
  • ✨ Smooth CSS transitions
