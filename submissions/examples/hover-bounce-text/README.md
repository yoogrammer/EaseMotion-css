# ease-hover-bounce-text

Bouncy hover animation for text elements. Adds playful vertical bounce with scale on hover — great for nav links, CTAs, and labels.

## Usage

```html
<a href="#" class="ease-hover-bounce-text">Click me</a>
```

## Variants

| Class | Description |
|---|---|
| `ease-hover-bounce-text` | Default bounce (6px, 0.5s) |
| `ease-bounce-fast` | Fast bounce (0.25s) |
| `ease-bounce-slow` | Slow bounce (0.8s) |
| `ease-bounce-lg` | Big bounce (12px) |

## Notes

- Set `display: inline-block` (already included in the class)
- Pairs well with `ease-btn`, nav links, and inline text
- Respects `prefers-reduced-motion`
