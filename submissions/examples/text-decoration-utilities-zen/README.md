## What does this do?

Provides `text-decoration` utility classes: underline, line-through, overline, and none.

## How is it used?

```html
<span class="text-underline">Underlined text</span>
<span class="text-line-through">Strikethrough</span>
```

## Why is it useful?

Text decoration is a core typographic need. These complement EaseMotion's existing text utilities.
