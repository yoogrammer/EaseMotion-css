# Sandbox Component Showcase: Pure-CSS Floating Diagonal Speed-Dial Shortcut Wheel

## Overview
A zero-dependency, pure-CSS interactive floating action button (FAB) speed-dial utility isolated within an examples subdirectory. It coordinates geometric sub-action disclosures natively via structural input checkbox variables, executing multi-axis coordinate translations smoothly without launching script listeners.

## 📁 Sandbox Configuration Files
* `demo.html` — Interactive presentation user cockpit hosting absolute layout corner anchors to evaluate cascading fanned-out shortcuts.
* `style.css` — Scoped layout sheet establishing isolated custom translation tokens and staggered timeline curves linked back to global tokens.

## 🚀 Key Layout Mechanics
1. **Checkbox State Stacking Focus:** Eliminates the necessity for layout event listeners by deploying an absolute checkbox toggle system hook. Sibling lookups handle macro component positions natively to expand child menu trees: `input:checked ~ .alm-speed-dial-node`.
2. **GPU-Accelerated Staggered Extensions:** Translates sub-action shortcut nodes outward along custom layout path channels (`transform: translate(X, Y) scale()`) running over elastic spring curves (`cubic-bezier(0.34, 1.56, 0.64, 1)`). Assigning ascending transition delays builds a fluid cascading sequence natively, offloading calculations to the compositor plane to protect main text rows from rendering lag.
3. **Sandbox Compliance Safeguard:** Fully satisfies repository security policies by placing every asset line strictly inside `submissions/examples/speed-dial-wheel/`.
