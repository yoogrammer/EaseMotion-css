# Multi-Step Progress Indicator

## What does this do?

A responsive multi-step progress indicator that visually shows user progress through a workflow such as checkout, onboarding, registration, or multi-page forms.

## How is it used?

```html
<div class="step completed">
  <div class="circle">✓</div>
  <span>Cart</span>
</div>

<div class="step active">
  <div class="circle">3</div>
  <span>Payment</span>
</div>
```

## Why is it useful?

Progress indicators improve user experience by showing completed, active, and upcoming steps. They are commonly used in e-commerce checkouts, onboarding flows, application forms, and account setup processes.
