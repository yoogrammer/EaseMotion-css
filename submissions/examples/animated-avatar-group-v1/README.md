# Animated Avatar Group

## What does this do?
An avatar group/stack with overlapping circles that expand on hover to reveal individual names.

## How is it used?
Add the avatar group markup to any page:

    <div class="avatar-group">
      <div class="avatar-item" style="--avatar-bg: #6366f1;">
        <div class="avatar-circle">A</div>
        <span class="avatar-name">Name</span>
      </div>
    </div>

## Why is it useful?
Displays team members or user avatars in a compact, interactive stack perfect for team sections, chat headers, and user lists.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
