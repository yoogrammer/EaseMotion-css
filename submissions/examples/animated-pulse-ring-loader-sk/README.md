# Animated Pulse Ring Loader

**What does this do?**  
Creates a loading indicator with concentric expanding rings and a pulsing center dot using staggered CSS animations.

**How is it used?**  
Add a `.pulse-loader` container with `.pulse-ring` elements and a `.pulse-dot` inside. Add `.delay-1` through `.delay-3` classes to stagger the ring expansions.

**Why is it useful?**  
Provides a visually appealing, zero-dependency loading state for async operations, form submissions, or page transitions.
