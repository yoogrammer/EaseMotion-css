# Animation: Elastic Notification Badge

A notification badge that enters with an elastic stretch effect. The badge briefly squashes and stretches before settling into place, creating a playful and attention-grabbing microinteraction.

## Features

- Pure CSS animation
- Elastic stretch effect
- No JavaScript required
- Responsive
- Reduced motion support
- Suitable for notifications, alerts, and counters

## Usage

```html
<button class="notification-btn">
    Messages
    <span class="badge">9</span>
</button>
```

The badge automatically plays the animation when rendered.
