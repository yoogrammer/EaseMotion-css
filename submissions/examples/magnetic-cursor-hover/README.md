# magnetic-cursor-hover — CSS Magnetic Hover Interaction Demo

## Feature overview

This demo presents a CSS-only magnetic cursor hover interaction utility with:

- magnetic hover cards and buttons
- smooth transform animations
- subtle scale and lift motion
- responsive layout across devices
- reusable utility classes
- modern, lightweight UI styling
- browser-friendly implementation without external dependencies

## Usage instructions

Open `demo.html` directly in your browser. The page imports only the local `style.css` file and does not depend on JavaScript or external frameworks.

The demo includes:

- multiple magnetic hover cards
- magnetic buttons with hover motion
- interaction showcase sections for reusable UI patterns
- responsive stacking for mobile screens

## Customization examples

- Adjust the hover motion by modifying the `.magnetic:hover` transform values.
- Reuse `.magnetic-card` and `.magnetic-btn` classes for additional interactive components.
- Change theme colors using CSS gradients and border colors inside `style.css`.
- Add `transition` customizations to tune motion timing for different interaction speeds.

## Browser compatibility

This example targets modern browsers with support for:

- CSS transforms and transitions
- CSS gradients
- responsive grid layout
- `clamp()` and `minmax()` sizing

It works well in Chrome, Edge, Firefox, Safari, and modern mobile browsers.

## Why this fits EaseMotion CSS

This submission enhances EaseMotion CSS with a motion-rich hover interaction pattern that stays true to a lightweight, CSS-first philosophy. The magnetic hover utilities are reusable and easy to integrate into UI designs without extra dependencies or JavaScript.
