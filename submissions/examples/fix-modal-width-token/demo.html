Modal Width Token Fix

Modal width is now customisable via --ease-modal-max-width and --ease-modal-width.

Default Modal

max-width: 500px (default via --ease-modal-max-width fallback)

Small Modal

--ease-modal-max-width: 400px

Large Modal

--ease-modal-max-width: 700px

Custom Modal

--ease-modal-max-width: 800px (inline override)
