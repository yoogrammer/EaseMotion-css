\# ease-flip-card-entrance



Flip card entrance animation - cards flip into view with 3D effect.



\## Classes



| Class | Description | Speed |

|-------|-------------|-------|

| `ease-flip-card-entrance` | Standard flip entrance | 0.6s |

| `ease-flip-card-entrance-fast` | Fast flip entrance | 0.3s |

| `ease-flip-card-entrance-slow` | Slow flip entrance | 1.2s |

| `ease-flip-card-horizontal` | Horizontal flip | 0.6s |

| `ease-flip-card-vertical` | Vertical flip | 0.6s |

| `ease-flip-card-3d` | 3D perspective flip | 0.8s |

| `ease-flip-card-entrance-delay-1` | Delay 0.2s | 0.6s |

| `ease-flip-card-entrance-delay-2` | Delay 0.4s | 0.6s |

| `ease-flip-card-entrance-delay-3` | Delay 0.6s | 0.6s |



\## Usage



```html

<div class="ease-flip-card-entrance">

&#x20;   Card content flips into view

</div>
