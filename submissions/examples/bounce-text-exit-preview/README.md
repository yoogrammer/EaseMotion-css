# Bounce Text Alignment & Border Exit Animation Preview

### 1. What does this do?

This contribution implements a smooth text vertical bounce effect on hover and a scaling border-fading exit animation that transitions element borders smoothly during dismissal.

### 2. How is it used?

Apply the corresponding class to your HTML element:

```html
<div class="hover-bounce-text">Bounce!</div>

<div class="expand-border-exit"></div>
```

### 3. Why is it useful?

It fits EaseMotion CSS's philosophy by offering lightweight, high-performance, animation-first utilities that replace complex JavaScript-driven animation libraries with standard, human-readable CSS declarations.
