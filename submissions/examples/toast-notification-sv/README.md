# Toast Notification Component

## What does this do?

Creates animated toast notifications for displaying success, error, warning, and informational messages.

## How is it used?

```html
<div class="toast success">
  Profile updated successfully!
</div>
```

Additional variants can be used for warnings, errors, and informational messages.

## Why is it useful?

Toast notifications are one of the most common UI patterns used to provide immediate feedback to users. This component offers a lightweight, reusable, and animation-friendly notification system that aligns with EaseMotion CSS's animation-first philosophy.
