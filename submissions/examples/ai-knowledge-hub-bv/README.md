# AI Knowledge Hub

## What does this do?

The AI Knowledge Hub is a centralized document repository and learning management workspace. It features a semantic vector similarity search bar, documentation category selectors, article detail reader panels with integrated laser scanning sweeps, interactive learning path checklists that dynamically update progress metrics, and collaborative team activity logs.

## How is it used?

Open `demo.html` directly in any modern browser:

- **Filter Categories**: Select specific documentation modules (e.g. Neural Architectures, RAG & Vector DBs, LoRA Tuning) in the dropdown to instantly filter document index lists.
- **Run Semantic Search**: Type keywords like "RAG", "Attention", or "LoRA" in the search box and press Search. Watch the laser scanning bar sweep down the explorer card as the similarity list sorts items by calculated relevance.
- **Read Documentation**: Click items in the document list (e.g. Transformer Attention Mechanics). Read the summary text and view markdown-styled structures in the reader panel.
- **Interactive Checklists**: Check or uncheck learning path checkboxes. Watch the Intermediate learning path progress bar fill, animate, and update percentage metrics in real-time.
- **Share Activity Updates**: Type a learning announcement in the activity console and click Post to instantly append the item to the feed.

## Why is it useful?

Centralizing documentation, validating semantic queries, and tracking learning paths requires responsive, interactive, and beautifully styled platforms.

This hub solves knowledge management workflows by:

1. Displaying semantic search similarity scores side-by-side using high-contrast vector badges.
2. Integrating a responsive progress checklist that recalculates and triggers progress transitions dynamically.
3. Incorporating interactive viewpoint search matching to represent embedding indexing operations.

## Tech Stack

- HTML5
- CSS3 (laser scanning sweeps, progress fill transitions, checklist tabbing, responsive grids)
- JavaScript (similarity score matching, progress bar calculators, category filter listeners, activity message post handlers)
