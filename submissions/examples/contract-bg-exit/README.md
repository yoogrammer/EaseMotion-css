# ease-contract-bg-exit

Exit animation where the element scales down and fades out. Clean dismiss/remove effect for modals, toasts, or cards.

## Usage

```html
<div class="ease-contract-bg-exit">Fades out while shrinking</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-contract-bg-exit` | Default contract exit (scale + fade) |

## Notes

- Scales element to 0.85x while fading out
- Useful for dismiss/close animations on modals and toasts
- Respects `prefers-reduced-motion`
