# Animation Class Reference

## What does it do?
A comprehensive interactive reference page documenting every
EaseMotion CSS animation class with live demos, class names,
CSS variables used, and usage recommendations.

## Contents
- Entrance animations (fade, slide, zoom, flip, blur-to-focus)
- Exit animations (fade-out)
- Looping animations (bounce, pulse, rotate, ping, shake)
- Hover animations (grow, shrink, glow, lift, shimmer)
- Skeleton loading (shimmer, pulse)
- Delay helpers (ease-delay-75 through ease-delay-700)
- Duration overrides (fast, medium, slow)

## How to use
Open `demo.html` directly in browser.
Click any card's "↺ Replay" button to re-trigger the animation.

## Tech Stack
- HTML + CSS + minimal vanilla JS (replay function only)
