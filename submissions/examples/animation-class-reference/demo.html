Entrance Animations

Fade In
.ease-fade-in

Fades element from invisible to visible.

Uses: --ease-speed-medium, --ease-ease

Use for: modals, toasts, overlays

Slide Up
.ease-slide-up

Slides element up from below while fading in.

Uses: --ease-speed-medium, --ease-ease

Use for: cards, list items, page sections

Slide Down
.ease-slide-down

Slides element down from above while fading in.

Uses: --ease-speed-medium, --ease-ease

Use for: dropdowns, notifications

Slide Left
.ease-slide-in-left

Slides in from the left side.

Uses: --ease-speed-medium, --ease-ease

Use for: sidebars, drawers

Slide Right
.ease-slide-in-right

Slides in from the right side.

Uses: --ease-speed-medium, --ease-ease

Use for: notifications, panels

Zoom In
.ease-zoom-in

Scales up from 85% while fading in.

Uses: --ease-speed-medium, --ease-ease-bounce

Use for: modals, image previews

Flip
.ease-flip

3D flip entrance from rotateY(90deg).

Uses: --ease-speed-medium, --ease-ease

Use for: cards, badges

Blur to Focus
.ease-blur-to-focus

Transitions from blurred to sharp focus.

Uses: --ease-speed-medium, --ease-ease

Use for: hero text, spotlight elements

Fade Out
.ease-fade-out

Fades element from visible to invisible.

Uses: --ease-speed-medium, --ease-ease

Use for: dismissing elements

Looping Animations

Bounce
.ease-bounce

Continuously bounces up and down.

Uses: 1s infinite

Use for: scroll indicators, CTAs

Pulse
.ease-pulse

Pulsates opacity between 1 and 0.45.

Uses: 2s infinite, --ease-ease

Use for: loading states, live indicators

Rotate
.ease-rotate

Continuously rotates 360 degrees.

Uses: 1.2s linear infinite

Use for: spinners, loading icons

Ping
.ease-ping

Scales out and fades — like a sonar ping.

Uses: 1s cubic-bezier infinite

Use for: notification dots, live badges

Shake (click)
.ease-shake

Quick horizontal shake.

Uses: 0.5s, --ease-ease

Use for: error states, form validation

Hover Animations

Hover: Grow
.ease-hover-grow

Scales up to 1.06 on hover.

Uses: --ease-speed-medium, --ease-ease-bounce

Use for: buttons, cards, images

Hover: Shrink
.ease-hover-shrink

Scales down to 0.95 on hover.

Uses: --ease-speed-fast, --ease-ease

Use for: pressed button effect

Hover: Glow
.ease-hover-glow

Adds primary color glow on hover.

Uses: --ease-glow-primary

Use for: CTAs, highlighted cards

Hover: Lift
.ease-hover-lift

Lifts element up with shadow on hover.

Uses: --ease-shadow-xl

Use for: cards, list items

Card Lift
.ease-card-lift

Deeper lift — 8px translateY + shadow.

Uses: --ease-shadow-xl, --ease-ease-out

Use for: product cards, feature cards

Hover: Shimmer
.ease-hover-shimmer

Light sweep across element on hover.

Uses: --ease-speed-slow, --ease-ease

Use for: premium cards, buttons

Skeleton Loading

.ease-skeleton-shimmer

Sweeping shimmer for loading placeholders.

Uses: --ease-color-neutral-100/200

Use for: content loading states

.ease-skeleton-pulse

Pulsating opacity for loading placeholders.

Uses: --ease-color-neutral-200

Use for: text/image loading states

Delay & Duration Helpers

.ease-delay-75 .ease-delay-100 .ease-delay-150 .ease-delay-200 .ease-delay-300 .ease-delay-500 .ease-delay-700

Stagger animation delays for sequential effects.

Use for: list items, grid cards

.ease-duration-fast .ease-duration-medium .ease-duration-slow

Override animation speed.

Use for: fine-tuning animation feel
