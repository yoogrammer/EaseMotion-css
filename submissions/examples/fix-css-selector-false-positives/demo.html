🧪 Smoke Test Fix Proposal

Resolves Issue #2511 — CSS Selector False Positives.

Vulnerable (Current)

Raw substring matching matches comments and unrelated values.

// ❌ Passes even if .ease-btn is commented out!
expect(css).toContain('.ease-btn');
Robust (Proposed)

Parses the DOM stylesheet to ensure it is a valid, active rule.

// ✅ Uses CSSOM to verify actual defined selectors
const selectors = getAllSelectors(sheet.cssRules);
expect(hasSelectorWithClass(selectors, '.ease-btn')).toBe(true);
