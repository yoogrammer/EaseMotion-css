# ease-divider — Divider Utilities

Lightweight divider utilities: horizontal rule, text-labeled divider, and vertical inline separator.

## Classes

| Class | Description |
|---|---|
| `ease-divider` | Horizontal rule with styling |
| `ease-divider-text` | Divider with centered label text (e.g. "or") |
| `ease-divider-vertical` | Vertical inline separator |

## Usage

```html
<!-- Horizontal -->
<hr class="ease-divider" />

<!-- With text -->
<div class="ease-divider-text">or</div>

<!-- Vertical -->
<span>Home</span>
<span class="ease-divider-vertical"></span>
<span>About</span>
```

## Why it fits EaseMotion CSS

Zero-JS, pure CSS. Uses design tokens for consistent spacing and color.
