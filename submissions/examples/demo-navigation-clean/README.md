# EaseMotion Clean Demo Navigation

## Feature Overview

This demo presents a clean preview interface for animation examples using a sticky navigation sidebar, category buttons, and animation preview cards. It is built with pure HTML and CSS and is designed to open directly in the browser without any dependencies.

## How to Use

- Open `demo.html` in your browser
- Use the sidebar buttons to jump to each animation section
- Observe the animation preview cards in each category
- Resize the browser to confirm responsive layout behavior

## Included Categories

- Fade Animations
- Hover Effects
- Floating Motions
- Rotate Effects
- Slide Transitions

## Why It Fits EaseMotion CSS

- Lightweight CSS-only implementation matches EaseMotion's minimal approach
- Uses modern visual styling with dark theme and subtle gradients
- Demonstrates real animation preview cards without JavaScript
- Includes sticky navigation and responsive layout for modern UI experiences
- Ideal as a standalone demo page for animation examples

## Browser Compatibility

- Chrome: supported
- Firefox: supported
- Safari: supported
- Edge: supported
- Mobile browsers: responsive layout supported

## Files Included

- `demo.html` — main demo page with navigation and preview sections
- `style.css` — all page styling and animation definitions
- `README.md` — documentation and usage notes

## Notes

- The demo uses CSS `position: sticky` for the sidebar
- Smooth scrolling is enabled with `scroll-behavior: smooth`
- The design is responsive and adjusts to tablet and mobile screens
- Accessibility prefers reduced motion if the user has that preference enabled
