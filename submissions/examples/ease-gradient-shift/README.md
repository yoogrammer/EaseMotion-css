# Gradient Shift Animated Background

## What does this do?

Adds an animated gradient background effect that smoothly shifts colors over time.

## How is it used?

```html
<section class="gradient-shift">
  Animated gradient content
</section>
```

## Why is it useful?

Gradient shift animations help create eye-catching hero sections, banners, cards, and call-to-action areas while keeping the effect reusable and easy to understand.
