# SBOM Explorer Component

## What does this do?

Provides a centralized interface for exploring software components, dependencies, licenses, vulnerabilities, and compliance status.

## How is it used?

```html
<div class="package-card healthy">
    <h3>React</h3>
</div>
```

## Why is it useful?

Software Bill of Materials (SBOM) data helps organizations understand application composition, open-source usage, licensing obligations, and security exposure. This component provides a reusable software inventory interface while demonstrating modern software supply chain transparency practices.
