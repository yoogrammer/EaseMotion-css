# Smart Notification Center Component

## What does this do?

Provides a modern notification center UI with read/unread indicators, notification badges, timestamps, and action controls.

## How is it used?

```html
<div class="notification unread">
    <h3>You were mentioned in a project</h3>
</div>
```

## Why is it useful?

Notification centers are a core part of modern SaaS applications, productivity tools, and collaboration platforms. This component offers a clean and responsive notification interface built entirely with HTML and CSS.
