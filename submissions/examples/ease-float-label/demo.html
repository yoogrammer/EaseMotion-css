🏷️ ease-float-label

Label floats above input on focus or when content is filled

Basic Floating Label

Input Types

Textarea

Color Variants

Registration Form

Create Account

Usage

<div class="float-group">
    <input type="text" id="name" placeholder=" " class="ease-float-input">
    <label for="name" class="ease-float-label">Full Name</label>
</div>

⚡ Features

  • 🏷️ Label floats above input on focus or when filled
  • 🎨 Multiple color variants
  • 📝 Works with input and textarea
  • 💡 Uses :placeholder-shown with placeholder=" "
