\# ease-float-label



Floating label input — label floats above input on focus or when content is filled.



\## Files



| File | Description |

|------|-------------|

| \[demo.html](./demo.html) | Interactive demo |

| \[style.css](./style.css) | Floating label styles |

| \[README.md](./README.md) | Documentation |



\## Classes



| Class | Description |

|-------|-------------|

| `ease-float-input` | Input with floating label support |

| `ease-float-label` | Label that floats above input |

| `ease-float-primary` | Primary color variant |

| `ease-float-success` | Success color variant |

| `ease-float-danger` | Danger color variant |

| `ease-float-warning` | Warning color variant |



\## Usage



```html

<div class="float-group">

&#x20;   <input type="text" id="name" placeholder=" " class="ease-float-input">

&#x20;   <label for="name" class="ease-float-label">Full Name</label>

</div>
