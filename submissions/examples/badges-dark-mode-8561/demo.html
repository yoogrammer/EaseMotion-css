Live Active Alert

Suggested Fix

Add the following blocks to the end of components/badges.css:

/* Dark mode */ @media (prefers-color-scheme: dark) { .ease-badge-pulse::after { box-shadow: 0 0 0 0 rgba(160, 154, 248, 0.6); } } /* Reduced motion */ @media (prefers-reduced-motion: reduce) { .ease-badge-pulse::after { animation: none; display: none; } }
