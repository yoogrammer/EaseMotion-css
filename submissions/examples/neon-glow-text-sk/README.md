# Glowing Neon Text Effect

A synthwave/80s inspired text effect that perfectly mimics a glowing gas neon tube.

## Files
- `demo.html`: The HTML structure for the text.
- `style.css`: The CSS that layers the multiple text-shadows and creates the flicker animation.
- `README.md`: This file.

## Features
- **Perfect Shadow Layering:** Employs up to 6 layers of perfectly balanced `text-shadow` values to create a realistic outer glow and an intense inner core.
- **Flicker Animation:** Uses CSS `@keyframes` to randomly dim and brighten the text shadow on hover, simulating a real gas tube struggling for power.
- **Dark Mode Essential:** Designed specifically for high-contrast dark mode interfaces.

## Usage
Simply apply the `.neon-glow-text` class to your headings or paragraphs.
