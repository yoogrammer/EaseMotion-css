# Progress Bar — Variants

A collection of progress bar variants including animated fill, striped patterns, indeterminate state, and size options.

## Features

- Animated shimmer fill effect
- Striped pattern overlay
- Rounded and flat variants
- Percentage label inside bar
- Color variants: default, success, warning, error
- Indeterminate (animated unknown progress)
- Size variants: xs (4px), sm (8px), md (14px), lg (20px)

## Usage

```html
<!-- Basic -->
<div class="progress">
  <div class="progress-bar" style="width: 65%;">65%</div>
</div>

<!-- Animated + striped -->
<div class="progress progress-animated progress-striped">
  <div class="progress-bar" style="width: 70%;"></div>
</div>

<!-- Indeterminate -->
<div class="progress progress-indeterminate">
  <div class="progress-bar"></div>
</div>
```
