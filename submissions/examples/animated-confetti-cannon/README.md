# Animated Confetti Cannon Component

## Description
A high-performance interactive celebration element driven by an HTML5 Canvas drawing loop layer. Captures mouse positions natively to fire structural vector bursts utilizing integrated atmospheric friction models.

## Features
- **150 Particles Per Burst**: Spawns a dense custom array of independent elements concurrently.
- **3 Geometric Varieties Included**: Populates field paths with circles, squares, and balanced layout triangles.
- **Physics Simulators**: Accommodates air drag multipliers, directional gravity drops, and speed fade properties.
