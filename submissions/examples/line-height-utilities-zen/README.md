## What does this do?

Provides line-height utility classes: leading-none (1), leading-tight (1.25), leading-normal (1.5), leading-loose (2).

## How is it used?

```html
<p class="leading-tight">Compact text</p>
<p class="leading-loose">Spacious text</p>
```

## Why is it useful?

Line-height control is essential for typographic rhythm. These utilities bring the existing CSS variables (--ease-leading-*) into easy-to-use class form.
