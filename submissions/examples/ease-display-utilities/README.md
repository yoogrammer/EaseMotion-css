# Ease Display Utilities

Display utility classes for commonly used CSS display properties.

## Available Utilities

1. `.ease-block` → `display: block;`
2. `.ease-inline` → `display: inline;`
3. `.ease-inline-block` → `display: inline-block;`
4. `.ease-flex` → `display: flex;`
5. `.ease-inline-flex` → `display: inline-flex;`
6. `.ease-grid` → `display: grid;`
7. `.ease-inline-grid` → `display: inline-grid;`

## Usage

### Flex Layout

```html
<div class="ease-flex">
    <div>Item 1</div>
    <div>Item 2</div>
    <div>Item 3</div>
</div>
```

### Grid Layout

```html
<div class="ease-grid">
    <div>A</div>
    <div>B</div>
    <div>C</div>
</div>
```
