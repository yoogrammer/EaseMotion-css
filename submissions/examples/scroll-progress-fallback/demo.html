Scroll Progress Fallback

The .ease-scroll-progress bar uses animation-timeline: scroll(), which is not supported in Firefox, Safari < 15.4, or older Chromium-based browsers. This example demonstrates a JavaScript polyfill fallback.

Browser Support

Chrome 115+ Firefox (no support) Safari < 15.4 Edge 115+

How this works: The CSS wraps the native animation-timeline in an @supports rule. When the browser does not support it, a lightweight scroll event listener updates transform: scaleX() inline. The bar works in every browser.
