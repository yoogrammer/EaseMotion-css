CSS overscroll-behavior: contain

Preventing scroll chaining in nested containers

How it works

overscroll-behavior: contain prevents the browser from passing scroll events to the parent element once a scrollable element reaches its boundary. This stops "scroll chaining" — the annoying effect where scrolling past the end of a nested list scrolls the page behind it.

Toggle the sidebar

Use the checkbox in the sidebar to toggle overscroll-behavior: contain on and off. Add more items to the sidebar to make it scrollable, then try scrolling past the bottom with contain both enabled and disabled.
