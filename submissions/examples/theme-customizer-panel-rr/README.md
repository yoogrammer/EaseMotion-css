# Animated Theme Customizer Panel

## What does this do?

Provides an interactive theme customization panel that allows users to preview appearance settings, accent colors, and UI preferences.

## How is it used?

```html
<div class="colors">
    <span class="color purple"></span>
    <span class="color blue"></span>
</div>
```

## Why is it useful?

Theme customization panels are common in modern SaaS applications, documentation sites, and design systems. This component demonstrates theme controls, color selection patterns, and smooth CSS animations using pure HTML and CSS.
