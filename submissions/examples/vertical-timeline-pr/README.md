# Vertical Timeline Component

## What does this do?

A responsive vertical timeline component that displays chronological events using connected timeline nodes and content cards.

## How is it used?

```html
<div class="timeline">
  <div class="timeline-item">
    <div class="timeline-dot"></div>

    <div class="timeline-content">
      <h3>Planning Phase</h3>
      <p>Initial project research and requirement gathering.</p>
    </div>
  </div>
</div>
```

## Why is it useful?

Timeline layouts are commonly used for resumes, portfolios, company histories, project roadmaps, and product development journeys. This component provides a clean and lightweight solution using only HTML and CSS while remaining easy to customize.
