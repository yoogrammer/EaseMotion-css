# ease-border-none Utility

Removes the border from any element. Useful for resetting bordered elements like buttons, cards, inputs, and table cells.

## Usage

```html
<button class="ease-border-none">Borderless button</button>
<div class="ease-card ease-border-none">Card without border</div>
```

## Classes

| Class              | CSS Property |
| ------------------ | ------------ |
| `ease-border-none` | border: none |
