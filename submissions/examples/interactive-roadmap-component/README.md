# Interactive Roadmap Component with Milestone Cards

A responsive roadmap timeline component built for EaseMotion CSS submissions.

## Features

- Vertical roadmap layout
- Connected timeline structure
- Animated milestone cards
- Status indicators (Completed, Active, Upcoming)
- Hover micro-interactions
- Mobile responsive design
- Accessibility-friendly reduced motion support

## Files

- `demo.html` — Example implementation
- `style.css` — Component styling
- `README.md` — Documentation

## Use Cases

- SaaS product roadmaps
- Startup launch timelines
- Developer portfolios
- Learning journeys
- Project milestones
- Feature release tracking
