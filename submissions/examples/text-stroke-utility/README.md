# ease-text-stroke-utility

Outline/stroked text effect using `-webkit-text-stroke` with transparent fill. Creates the popular hollow text look.

## Usage

```html
<h1 class="ease-text-stroke-2 ease-text-stroke-white">OUTLINE</h1>
```

## Variants

| Class | Description |
|---|---|
| `ease-text-stroke-1` | 1px stroke width |
| `ease-text-stroke-2` | 2px stroke width |
| `ease-text-stroke-3` | 3px stroke width |
| `ease-text-stroke-white` | White stroke color |
| `ease-text-stroke-primary` | Primary brand stroke color |
| `ease-text-fill-none` | Transparent fill (default with stroke) |

## Notes

- Set `-webkit-text-stroke-color` to any color for custom stroke colors
- Omit text-fill classes to keep colored text fill
- Great on gradient or image backgrounds
