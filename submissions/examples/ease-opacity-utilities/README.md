# Opacity Utilities

## What does this do?

Provides reusable opacity utility classes for controlling element transparency.

## How is it used?

```html
<div class="opacity-25">Content</div>
<div class="opacity-50">Content</div>
<div class="opacity-75">Content</div>
<div class="opacity-100">Content</div>
```

## Why is it useful?

Opacity utilities help developers quickly control visibility and emphasis while keeping class names simple and readable.
