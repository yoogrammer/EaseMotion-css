# Animated Onboarding Checklist Component

1. **What does this do?**
   A motion-first onboarding checklist that shows completed, active, and pending states with animated check transitions and smooth hover micro-interactions.

2. **How is it used?**
   Use the component markup inside `demo.html` with the `.onboarding-checklist` container and `.checklist-item` items, applying state classes like `.checklist-item--done`, `.checklist-item--active`, and `.checklist-item--pending`.

3. **Why is it useful?**
   It fits EaseMotion CSS by turning progress tracking into a clear, human-friendly interaction pattern—where state changes (pending → active → completed) are communicated through subtle motion, not just color.
