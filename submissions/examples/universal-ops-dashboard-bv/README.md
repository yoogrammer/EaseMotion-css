# Universal Operations Dashboard

A premium enterprise-grade operations dashboard showcasing pure-CSS animations, transitions, and layout capabilities.

## Features
- Ingestion rate stats
- Active sub-system logs
- Configuration controls
- Animated stat cards and log entries

## Usage
Open `demo.html` directly in a browser — no server needed.

## Why EaseMotion CSS?
Demonstrates premium CSS layout, hover animations, and transitions with zero JS dependencies.
