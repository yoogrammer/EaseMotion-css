# Dropdown Navigation

## What does this do?
A dropdown navigation bar with hover-triggered submenus and multi-level menu items. Clean responsive navigation pattern.

## How is it used?
Open demo.html in a browser. Hover or click on menu items with arrows to reveal dropdown submenus. Multi-level nesting is supported.

## Why is this useful?
Demonstrates a common navigation pattern for websites with hierarchical menu structures, supporting multiple levels of dropdown menus.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
