# ease-skeleton-pulse

Pulsing opacity skeleton loading placeholder. A fundamental pattern for improving perceived performance during data fetching.

## Usage

```html
<div class="ease-skeleton-pulse" style="width:100%;height:20px;"></div>
```

## Variants

| Class | Description |
|---|---|
| `ease-skeleton-pulse` | Default opacity pulse skeleton |
| `ease-skeleton-pulse-fast` | Faster pulse variation |
| `ease-skeleton-pulse-slow` | Slower pulse variation |

## Notes

- Use as a placeholder while content loads
- Apply to any element shape (text lines, images, cards)
- Respects `prefers-reduced-motion`
