# GitHub Contribution Heatmap Grid

## What does this do?

Creates a GitHub-inspired contribution heatmap using a responsive CSS grid and multiple activity intensity levels.

## How is it used?

```html
<div class="contribution-grid">
  <div class="cell level-0"></div>
  <div class="cell level-1"></div>
  <div class="cell level-2"></div>
  <div class="cell level-3"></div>
  <div class="cell level-4"></div>
</div>
```

## Why is it useful?

Contribution heatmaps are widely recognized by developers and provide a visually engaging way to display activity, progress, learning streaks, coding consistency, and project metrics without requiring JavaScript.
