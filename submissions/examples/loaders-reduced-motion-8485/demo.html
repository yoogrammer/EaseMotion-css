Loaders Reduced Motion Fix

Issue #8485 — all four loader animations ignore prefers-reduced-motion: reduce

Two ways to test: 1. Click Simulate Reduce Motion below to toggle the fix on/off in this demo.
2. Chrome DevTools → More Tools → Rendering → set Emulate CSS media feature prefers-reduced-motion to reduce — the fix in style.css will stop all loaders automatically.
Current state: Animations running

All four loader variants

.ease-loader-spin Running
.ease-loader-pulse Running
.ease-loader-ping Running
.ease-loader-dots Running

The fix — add to components/loaders.css

@media (prefers-reduced-motion: reduce) {
  .ease-loader-spin,
  .ease-loader-pulse,
  .ease-loader-ping,
  .ease-loader-dot {
    animation: none !important;
  }
}
