Selector Pollution Refactor

Notice how the <h1> and <p> tags *inside* this specific container are styled perfectly, but the tags located above remain completely unaffected.

By forcing all contributors to wrap their demo styles in a namespace class (like .ease-demo-namespace) rather than blindly targeting global HTML tags like body or p, we completely prevent submissions from destroying each other's layouts when loaded into the same application environment.
