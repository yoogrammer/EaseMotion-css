# Ease File Upload Dropzone

## What does this do?

Provides a modern drag-and-drop file upload component with animated idle, uploading, and success states.

---

## How is it used?

```html
<div class="upload-dropzone">
  <div class="upload-icon">📁</div>
  <h3>Drop files here</h3>
  <p>or click to browse</p>
</div>
```

---

## Features

- Drag-and-drop style upload area
- Animated hover effects
- Upload progress indicator
- Success state feedback
- Small, medium, and large variants
- Accessibility-friendly focus styles
- Reduced motion support
- Pure HTML and CSS implementation

---

## Why is it useful?

File upload interfaces are common in dashboards, onboarding flows, profile management systems, and document upload applications. This component provides a reusable, animation-first solution that fits EaseMotion CSS's philosophy while reducing repetitive implementation effort.
