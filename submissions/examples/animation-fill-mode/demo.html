animation-fill-mode

Controls what values the element holds before and after an animation runs.

none (default)

Element snaps back to original style when animation ends.

→

forwards

Element keeps the LAST keyframe style after animation ends.

→

backwards

Element takes the FIRST keyframe style during the delay, before animation starts.

→

both

Applies both forwards and backwards — holds first keyframe during delay, keeps last keyframe after end.

→

Each box has a 2s animation with a 1s delay and 3s pause at the end.
Watch what happens during the delay and after the animation finishes — the fill mode controls this behavior.
