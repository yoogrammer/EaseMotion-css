# Knowledge Graph Visualization Component

## What does this do?

Displays relationships between concepts, technologies, or entities using a graph-based layout with connected nodes.

## How is it used?

```html
<div class="node center">JavaScript</div>
<div class="node react">React</div>
```

## Why is it useful?

Knowledge graphs help visualize complex relationships and interconnected systems. They are commonly used in software architecture, learning roadmaps, dependency mapping, technology ecosystems, and educational platforms. This component provides a reusable graph visualization using pure HTML and CSS.
