Layout

Basic layout helpers and grid utilities.

.ease-block

Sets element to display: block.

Block

.ease-flex

Enables flex container.

Item 2

.ease-gap-4

Applies gap spacing using token `--ease-space-4`.

Animation

Prebuilt animation classes and timing utilities.

.ease-fade-in

Fade-in entrance animation.

<div class="ease-fade-in">...</div>
Fade In

.ease-hover-lift

Hover lift + subtle shadow.

<button class="ease-hover-lift">Hover me</button>

.ease-delay-150

Delay animations by 150ms.

<div class="ease-fade-in ease-delay-150">...</div>
Delayed
