# Animation: Terminal Command Showcase

A terminal-inspired component that simulates command execution with a typing effect, blinking cursor, and animated output messages.

## Features

- Pure CSS animation
- Typing command effect
- Blinking cursor
- Animated output reveal
- Developer-focused aesthetic
- Responsive layout
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="line command">
    <span class="prompt">$</span>
    npm install easemotion-css
</div>
```

Commands appear to type in real time while output messages fade into view.

## Why is it useful?

Terminal showcases are popular on documentation pages, developer tools, open-source projects, and product landing pages. They provide an engaging way to demonstrate installation commands, setup instructions, and workflow examples.
