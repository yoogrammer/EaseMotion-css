# Period Tracker

## What does this do?
A period cycle tracker with a calendar grid view for marking period days, automatic cycle length and period duration tracking, next period prediction, and symptom note-taking.

## How is it used?
Click on calendar dates to mark or unmark period days. Cycle length and period duration are calculated automatically. The next predicted period date is shown. Add symptom notes for each cycle.

## Why is this useful?
This provides a simple period tracking tool for health and wellness applications, helping users understand their cycle patterns and predict future periods.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to use the tracker.
