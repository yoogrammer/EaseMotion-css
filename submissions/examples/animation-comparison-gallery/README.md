# Animation Comparison Gallery

## Description

This example demonstrates a side-by-side animation comparison gallery.

Users can compare multiple animation effects, replay animations, and view the corresponding code snippets.

## Features

- Side-by-side animation comparison
- Replay button for each animation
- Code snippet display
- Responsive gallery layout

## Included Animations

- Fade In
- Slide Up
- Bounce

## How It Works

Each animation card contains:

- Animation preview
- Replay control
- Code snippet

Users can easily compare animation behavior before implementation.

- demo.html

## Use Case

Useful for documentation websites where developers need to compare multiple animation effects simultaneously.
