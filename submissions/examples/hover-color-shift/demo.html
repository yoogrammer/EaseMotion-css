Hover Color Shift

A pure-CSS gradient transition system for buttons, cards, links, and banners. Smooth, flicker-free, zero JavaScript — built on a clever opacity cross-fade technique.

Component · 01

Primary Buttons

The .color-shift-btn class turns a button into a gradient-shifting element. Hover to watch the gradient quietly rotate from navy through ocean blue to teal.

Component · 02

Secondary Buttons

Add .color-shift-btn--secondary alongside .color-shift-btn for a charcoal-to-purple palette shift — ideal as a contrast pairing on light backgrounds.

Component · 03

Gradient-Shift Cards

.gradient-shift cards feature a top accent stripe that cross-fades from cool blue to warm amber on hover, paired with a smooth lift shadow — no JavaScript required.

Reference · 06

How to Use It

Add the stylesheet, then drop in the class names. Customise colours with CSS variables — no source edits needed. 

<!-- 1. Link the stylesheet -->
<link rel="stylesheet" href="hover-color-shift.css" />

<!-- 2. Button -->
<button class="color-shift-btn">
  <span>Get Started</span>
</button>

<!-- 3. Card -->
<article class="gradient-shift">
  <h3>Card Title</h3>
  <p>Card body text goes here.</p>
</article>

<!-- 4. Banner -->
<div class="hover-color-shift--banner">
  <div class="banner-content">…</div>
</div>

<!-- 5. Override colours with CSS variables -->
<style>
  :root {
    --hcs-from: #7c3aed;   /* violet */
    --hcs-mid:  #db2777;   /* pink   */
    --hcs-to:   #ea580c;   /* orange */
  }
</style>
