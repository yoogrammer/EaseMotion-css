# ease-italic / ease-not-italic

Controls the `font-style` property.

## Usage

```html
<p class="ease-italic">Italic text</p>
<p class="ease-not-italic">Normal text (override italic)</p>
```

## Classes

| Class             | CSS Property       |
| ----------------- | ------------------ |
| `ease-italic`     | font-style: italic |
| `ease-not-italic` | font-style: normal |
