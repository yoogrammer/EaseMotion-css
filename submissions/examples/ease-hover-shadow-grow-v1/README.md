# Hover Shadow Grow

## What does this do?

Adds a hover interaction where an element slightly grows and receives a stronger shadow.

## How is it used?

```html
<div class="hover-shadow-grow">
  Hover over me
</div>
```

## Why is it useful?

This effect helps clickable elements feel more interactive by adding depth and motion. It fits EaseMotion CSS because the class name clearly describes the visual behavior in simple, readable words.
