♿ ease-sr-only

Visually hidden but accessible to screen readers — essential for accessible icon buttons and skip links

What is Screen Reader Only?

The text with ease-sr-only is invisible on screen but will be announced by screen readers.

👇 Try using a screen reader or inspect the element below

Basic Example

Visible text: This is visible on screen

This text is only for screen readers (visually hidden)

The second line is hidden visually but screen readers will announce it

Icon Button (Accessible)

Screen readers will announce "Like this post" instead of just "heart emoji"

Skip to Content Link

Skip to main content

Press Tab key - the "Skip to main content" link will appear when focused

Form Labels (Accessible Icons)

Screen readers will announce "Search" for the button

Loading State Announcement

Loading content, please wait...

Screen readers announce loading status

Accordion Buttons (Accessible)

Screen readers announce the button purpose

Close Button (Modal/Dialog)

Screen readers announce "Close dialog" instead of just "X"

Navigation Menu (Mobile)

Mobile menu button with screen reader text

Live Region Announcement

Status updated to active

Screen readers announce status changes automatically

Usage

<!-- Basic screen reader only text -->
<span class="ease-sr-only">This text is only for screen readers</span>

<!-- Accessible icon button -->
<button>
    <span>❤️</span>
    <span class="ease-sr-only">Like this post</span>
</button>

<!-- Skip to content link -->
<a href="#main" class="ease-sr-only">Skip to main content</a>

<!-- Close button -->
<button>
    <span>✕</span>
    <span class="ease-sr-only">Close dialog</span>
</button>
