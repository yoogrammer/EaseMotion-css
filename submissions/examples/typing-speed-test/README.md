# Typing Speed Test

A typing speed test that measures WPM and accuracy against a given prompt. Click Start to begin, then type the displayed text. Stats update every 200ms. When the full prompt is typed, the test stops automatically.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser. Click "Start Test" and begin typing the prompt in the textarea.

## Accessibility notes

The textarea is initially disabled and enabled when the test starts. All stats are displayed as visible text. Buttons are `<button>` elements.
