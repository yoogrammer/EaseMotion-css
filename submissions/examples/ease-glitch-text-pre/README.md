## ease-glitch-text

**What does this do?**
A cyberpunk-inspired text hover animation that creates an RGB
split and glitch distortion effect using CSS pseudo-elements
and clip-path animations.

**How is it used?**
<h1 class="glitch-text" data-text="GLITCH">GLITCH</h1>
<h1 class="glitch-heading" data-text="GLITCH">GLITCH</h1>
<button class="glitch-btn" data-text="LAUNCH">LAUNCH</button>
<a class="glitch-link" data-text="Portfolio">Portfolio</a>

**Why is it useful?**
Adds a futuristic glitch effect to headings, buttons and links
using pure CSS — perfect for portfolios, gaming sites and
creative landing pages. Consistent with EaseMotion CSS's
animation-first philosophy.
