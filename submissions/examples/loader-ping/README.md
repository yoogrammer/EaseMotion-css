# ease-loader-ping

Ping loader variant: a radar-like pulse animation that repeatedly scales up to 2x and fades out. A distinctive alternative to spin and pulse loaders.

## Usage

```html
<div class="ease-loader-ping"></div>
```

## Variants

| Class | Description |
|---|---|
| `ease-loader-ping` | Default ping loader |
| `ease-loader-ping-sm` | Small variant |
| `ease-loader-ping-lg` | Large variant |

## Notes

- Creates a radar/ping effect
- Great for notification badges, live indicators, and loading states
- Respects `prefers-reduced-motion`
