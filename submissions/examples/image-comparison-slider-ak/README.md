## What does this do?

This component provides a before/after image comparison slider with a draggable handle. Two CSS gradients simulate the before (warm tones) and after (cool tones) images, and the user can drag the handle to reveal more of each side.

## How is it used?

Include the `style.css` and the HTML structure. The JavaScript handles mouse and touch drag events to adjust the `clip-path` of the before layer and the handle position.

## Why is this useful?

Image comparison sliders are commonly used in photography, design portfolios, and e-commerce to highlight differences between two versions of an image — such as edited vs. original or old vs. new.

## Tech Stack

- HTML
- CSS
- JavaScript (vanilla)
