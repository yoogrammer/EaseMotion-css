📈 Scale Icon Features

  • ✨ Smooth scale animation on hover
  • 🎯 Multiple intensity levels (1.05x to 1.3x)
  • ⚡ Speed variations (fast, normal, slow)
  • 🎭 Easing variations (bounce, spring, elastic)
  • 💓 Pulse effects (heartbeat, pop, wobble)
  • 🎨 Works on icons, buttons, and images
  • 🖱️ Focus state support for accessibility
