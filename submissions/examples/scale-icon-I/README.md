# Scale Icon Emphasis Animation

Smooth scale animation for icons during emphasis interactions — perfect for attention-grabbing effects.

## Files

- demo.html - Interactive demo
- style.css - Scale animation styles
- README.md - Documentation

## Classes

- ease-scale-icon - Normal scale (1.1x)
- ease-scale-icon-subtle - Subtle (1.05x)
- ease-scale-icon-strong - Strong (1.2x)
- ease-scale-icon-extreme - Extreme (1.3x)
- ease-scale-icon-fast/slow - Speed variants
- ease-scale-icon-bounce/spring/elastic - Easing variants
- ease-scale-icon-pulse - Continuous heartbeat pulse
- ease-scale-icon-pop - Pop on hover
- ease-scale-icon-wobble - Wobble on hover

## Usage

```html
<div class="ease-scale-icon">🎨</div>

<!-- Different intensities -->
<div class="ease-scale-icon-subtle">🌊</div>
<div class="ease-scale-icon-strong">🔥</div>

<!-- Pulse effect -->
<div class="ease-scale-icon-pulse">💓</div>
