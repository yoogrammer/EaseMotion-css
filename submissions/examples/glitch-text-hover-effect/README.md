# Glitch Text Hover Effect

## Description

A futuristic text hover animation featuring RGB split and glitch distortion effects.

## Features

- Cyberpunk-inspired effect
- Pure CSS implementation
- Lightweight and reusable
- Modern UI interaction

## Files

- demo.html
- style.css
- script.js

## Usage

Open demo.html and hover over the text.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
