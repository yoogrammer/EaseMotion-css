# Custom Animated Category Dropdown Component

### 1. What does this do?
Provides a highly optimized, fully styled native select menu container wrapped with custom SVG chevrons that morph smoothly on element activation states.

### 2. How is it used?
```html
<div class="ease-dropdown-select-container-sb">
  <select class="ease-dropdown-select-sb">
    <option value="all">All Categories</option>
  </select>
  <span class="dropdown-arrow-sb">...</span>
</div>
