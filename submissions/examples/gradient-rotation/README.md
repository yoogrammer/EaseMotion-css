# ease-gradient-rotation

Animated gradient background rotation. Applies a moving background-position gradient in a 3s infinite loop for dynamic backgrounds.

## Usage

```html
<div class="ease-gradient-rotation">Animated gradient background</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-gradient-rotation` | Default gradient rotation (3s loop) |

## Notes

- Animates `background-position` for flowing gradient effect
- 3s infinite loop
- Great for hero sections, card backgrounds, and banners
- Respects `prefers-reduced-motion`
