# FAQ Accordion with Smooth Open-Close

A list of FAQ items where clicking a question expands its answer below with a smooth max-height transition. Each question has a chevron that rotates on open. Multiple items can be open simultaneously.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser. Click any question to expand its answer. Click again to collapse.

## Accessibility notes

Each question is a `<button>` element for keyboard access. Reduced motion disables the transition.
