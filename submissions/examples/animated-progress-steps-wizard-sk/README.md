# Animated Progress Steps Wizard

**What does this do?**  
Creates a multi-step progress wizard with animated step indicators, connecting bars, and navigation buttons.

**How is it used?**  
Use `.wizard` with `.steps` containing `.step` elements and `.step-bar` connectors. Active steps get the `.active` class which triggers pop-in and fill-bar animations.

**Why is it useful?**  
Provides a polished, zero-dependency wizard/progress stepper for forms, onboarding, or checkout flows with smooth CSS animations.
