## ease-top-front-button

**What does this do?**
A 3D "top front" button with entrance animation, hover lift effect,
and press-down active state simulating physical depth.

**How is it used?**
<button class="ease-top-front-btn purple">Get Started</button>
<button class="ease-top-front-btn green">Confirm</button>
<button class="ease-top-front-btn orange">Explore</button>
<button class="ease-top-front-btn pink">Subscribe</button>

**Why is it useful?**
Provides a satisfying 3D button interaction with entrance animation
using pure CSS box-shadow and transform — consistent with
EaseMotion CSS's animation-first philosophy.
