# Animation Category Filters

## Description

This example demonstrates category-based filtering for animation galleries.

Users can quickly browse animations by selecting a category filter.

## Features

- Category filter buttons
- Instant client-side filtering
- Responsive grid layout
- Multiple animation categories

## Categories

- Entrance
- Exit
- Attention

## How It Works

Selecting a filter displays only animations belonging to that category.

The filtering happens instantly on the client side without reloading the page.

## Use Case

Useful for documentation websites that contain a large collection of animations and need improved discoverability.
