# Cinematic Parallax Movie Poster

## What it does
A **10-layer parallax movie poster** that responds to mouse movement, creating an illusion of real depth. Foreground elements (the wanderer figure, forest) shift more than background elements (moon, sky) as you move your cursor, exactly like a parallax scrolling effect but in 3D perspective space. The poster also physically tilts toward the cursor.

## Animations & Techniques
- **Multi-Layer Parallax**: 10 `div` layers each have a `data-depth` attribute (0.02–0.65). On mousemove, JavaScript reads `data-depth` and applies a `translate(x, y)` proportional to that depth — deeper layers move barely, closer layers move dramatically.
- **3D Card Tilt**: The poster container applies `rotateX` + `rotateY` based on cursor position relative to center (±10°), sitting inside a `perspective: 900px` scene.
- **Dynamic Shine**: A `radial-gradient` spotlight follows the cursor inside the poster, tracking the exact mouse position for a specular highlight effect.
- **Procedural Stars**: 120 star `div`s are generated by JavaScript with unique size, position, and CSS custom property values (`--dur`, `--del`, `--lo`, `--hi`) for randomised `@keyframes twinkle` behaviour entirely driven by CSS.
- **Continuous Animations**: Lantern swing (`lantern-swing`), star twinkle (`twinkle`), fog drift (`fog-drift`) all run independently in CSS.

## Why it fits EaseMotion CSS
- A show-stopping visual effect that demonstrates CSS `transform-style`, `perspective`, and layered composition at their finest.
- The animation system (stars, lantern, fog) is entirely declarative CSS — JavaScript only handles the interactive tilt math.
- Includes `prefers-reduced-motion` support.
