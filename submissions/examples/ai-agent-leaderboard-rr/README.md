# AI Agent Performance Leaderboard Component

## What does this do?

Displays AI agents in a ranked leaderboard format with performance metrics and ratings.

## How is it used?

```html
<div class="row">
    <span>#1</span>
    <span>CodePilot AI</span>
    <span>98%</span>
</div>
```

## Why is it useful?

AI platforms often need a way to compare agent performance using metrics such as success rate, tasks completed, response time, and ratings. This component provides a modern and reusable leaderboard interface while demonstrating responsive layouts and animations.
