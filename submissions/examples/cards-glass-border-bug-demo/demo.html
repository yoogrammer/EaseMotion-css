Bug: hardcoded rgba border

The glass card uses rgba(255, 255, 255, 0.3) directly instead of var(--ease-glass-border).

Glass Card

Border is hardcoded rgba.

Source

.ease-card-glass {
  border: 1px solid rgba(255, 255, 255, 0.3);  /* hardcoded */
}

Fix

.ease-card-glass {
  border: 1px solid var(--ease-glass-border, rgba(255, 255, 255, 0.20));
}
