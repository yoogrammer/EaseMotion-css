# Smooth Strikethrough Hover

## 1. What does this do?

This adds a smooth horizontal strikethrough transition that draws a line across the center of text elements on hover.

## 2. How is it used?

Apply the `.strike-hover-link` class to any text or link element:

```html
<a href="#" class="strike-hover-link">Complete EaseMotion Task</a>
```

## 3. Why is it useful?

It fits EaseMotion CSS's philosophy by using clean, declarative classes to create micro-interactions suitable for task lists, navigation links, or action lists using performant CSS background transitions.
