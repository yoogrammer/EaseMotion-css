Parallax Scrolling

Pure CSS parallax effect using perspective and transform-style: preserve-3d. Smooth, performant, and accessible.

🐌 Slow Layer

Deep Background

.parallax-slow or --parallax-depth: -100 — Elements scroll much slower than the page, creating a sense of deep distance.

Usage Guide

To use the parallax effect, wrap your sections in a container with class .parallax-container, add individual sections with .parallax-section, and layer elements using .parallax-layer combined with depth classes.

.parallax-container

Parent wrapper that establishes perspective. Sets perspective: 200px and transform-style: preserve-3d.

.parallax-section

Creates a full-viewport section with height: 100vh and centered content.

.parallax-layer

Absolutely positioned layer inside a section. Combine with depth classes for parallax effect.

Depth Classes

.parallax-slow, .parallax-medium, .parallax-fast.

Custom --parallax-depth

Set via inline style, e.g. style="--parallax-depth: -150" for custom scroll speed ratios.

Mobile & Reduced Motion

Parallax is disabled on screens <768px and respects prefers-reduced-motion.
