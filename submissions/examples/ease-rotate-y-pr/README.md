# Ease Rotate-Y Utility

## What does this do?
A utility class that applies a CSS `rotateY` transform with a smooth
transition, rotating an element 180° around the Y-axis on hover.

## How is it used?

```html
<div class="ease-rotate-y">Hover to Flip</div>
```

## Why is it useful?
Adds a polished flip-style interaction commonly used for cards, icons,
and toggles. Pure CSS, no JS required, with full prefers-reduced-motion
support.
