# ease-card-shadow

Card modifier that applies a large box-shadow and removes the border. A simple card variant for depth/emphasis.

## Usage

```html
<div class="ease-card ease-card-shadow">Shadow card</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-card-shadow` | Card with shadow, no border |

## Notes

- Removes the default card border
- Applies `box-shadow-lg` for depth
- Great for elevated cards, modals, and popovers
