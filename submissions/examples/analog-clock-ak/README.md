# Analog Clock

## What does this do?
A real-time analog clock with hour, minute, and second hands, numbered hour markers, and a matching digital time display.

## How is it used?
Open demo.html in a browser. The clock updates every second showing the current time with rotating hands and digital readout.

## Why is this useful?
Demonstrates a CSS-based analog clock with CSS transforms for hand rotation and real-time JavaScript animation.

## Tech Stack
- HTML
- CSS
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
