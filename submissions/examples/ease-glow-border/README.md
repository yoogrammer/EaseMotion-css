# ⚡ Ease Glowing Card Border Animation

Animated rotating glow borders for EaseMotion CSS.

## ✨ What it does
Creates a smooth rotating conic-gradient glow border around 
any card or container element. One class adds a premium 
animated border effect instantly.

## 🚀 How to Use
```html
<div class="ease-glow-border ease-glow-border--rainbow">
  <div class="ease-glow-border__inner">
    <!-- Your content here -->
  </div>
</div>
```

## 🎨 Color Variants
| Class | Color |
|-------|-------|
| `ease-glow-border--rainbow` | Full spectrum rainbow |
| `ease-glow-border--indigo` | Indigo & purple |
| `ease-glow-border--cyan` | Cyan & blue |
| `ease-glow-border--rose` | Rose & pink |
| `ease-glow-border--emerald` | Emerald & green |
| `ease-glow-border--gold` | Gold & amber |

## ⚡ Speed Variants
| Class | Speed |
|-------|-------|
| `ease-glow-border--slow` | 6s rotation |
| *(default)* | 3s rotation |
| `ease-glow-border--fast` | 1.5s rotation |
