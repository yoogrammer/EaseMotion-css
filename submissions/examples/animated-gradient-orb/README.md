# Animated Gradient Orb

## What it does

A stunning 3D sphere with dynamic shader-based gradient colors that shift over time. Features mouse-interactive rotation and speed controls.

## How to use it

The orb automatically animates. Drag mouse to rotate view. Use buttons to control rotation speed.

## Features

- Shader-based dynamic colors
- Mouse drag rotation
- 3 speed presets (Slow/Normal/Fast)
- Starfield background
- 60fps performance

- Three.js WebGL
