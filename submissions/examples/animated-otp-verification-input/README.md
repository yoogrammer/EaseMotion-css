1. What does this do? 
A multi-box OTP verification input UI with modern animated focus micro-interactions.

2. How is it used? 
Add six inputs using `.otp-container` and the `.otp-input` class, e.g. `<div class="otp-container"><input class="otp-input" maxlength="1" /></div>`.

3. Why is it useful? 
It follows EaseMotion CSS’s motion-first UX philosophy by making form focus feel responsive and intuitive with lightweight transitions and reduced-motion support.
