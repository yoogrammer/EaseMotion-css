# Ease Ribbon Zd

## What does this do?
An animated corner ribbon component with swoop entrance animation, 5 color variants, and fold effect — pure HTML and CSS.

## How is it used?
```html
<div class="rb-card">
  <div class="rb-ribbon rb-ribbon-red">Sale</div>
  <div class="rb-title">Card Title</div>
  <div class="rb-desc">Card description text...</div>
</div>
```

Color variants: rb-ribbon-red, rb-ribbon-blue, rb-ribbon-green, rb-ribbon-orange, rb-ribbon-purple
Add `.rb-ribbon-left` for left-side placement.

## Why is it useful?
Ribbons are a classic UI element for marking products or content as "New", "Sale", "Popular", "Free", or "Featured". This component provides a pure CSS implementation with smooth entrance animation and fold effect.
