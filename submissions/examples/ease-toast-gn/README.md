# Ease Toast Notification Component

1. **What does this do?** Adds animated toast/snackbar notifications with success, error, and info variants plus flexible positioning utilities.

2. **How is it used?**
```html
   <div class="ease-toast-container ease-position-top-right">
     <div class="ease-toast ease-toast-success">Action completed!</div>
     <div class="ease-toast ease-toast-error">Something went wrong!</div>
     <div class="ease-toast ease-toast-info">Here is some info.</div>
   </div>
```

3. **Why is it useful?** Provides ready-to-use toast notifications with slide-in and fade-out animations — pure CSS animations with minimal JS for triggering, fitting EaseMotion CSS's animation-first philosophy.
