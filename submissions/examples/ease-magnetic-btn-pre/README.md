## ease-magnetic-btn

**What does this do?**
A button that creates a magnetic attraction effect when hovered —
smoothly shifts toward cursor position while scaling up.

**How is it used?**
<button class="magnetic-btn purple">Hover Me</button>
<button class="magnetic-btn green pill">Subscribe</button>
<button class="magnetic-btn orange lg">Large</button>

Variants: purple, green, orange, outline
Sizes: sm, default, lg
Shapes: pill

**Why is it useful?**
Provides a premium magnetic hover interaction seen on modern
portfolio and SaaS landing pages — consistent with EaseMotion
CSS's animation-first philosophy.
