# ⚡ Animation: Neon Flicker (Irregular Capacitor Shiver)

### What does this do?
This micro-interaction utility simulates a realistic, unstable neon glass sign mechanism. Instead of relying on traditional uniform looping intervals, it uses an erratic, non-uniform keyframe array chain to rapidly cycle between deep, glowing text-shadow envelopes and dead unlit state filament layers.

### How is it used?
Drop the `.ease-neon-flicker` utility class on headings or typographic emphasis anchors and pass custom neon color metrics inline:

```html
<h1 class="ease-neon-flicker" style="--ease-neon-color: #ff0077;">
  LIVE LOUNGE
</h1>
```
