# Ease Notification Toast

Animated toast notification component for EaseMotion CSS.

## Variants

- ease-toast-success
- ease-toast-error
- ease-toast-warning
- ease-toast-info

## Features

- Slide-in animation
- Fade effect
- Hover elevation
- Responsive layout
- Stackable notifications
- Reduced motion support
- Accessible color contrast

## Example

<div class="ease-notification-toast ease-toast-success">
  <span class="toast-icon">✓</span>

  <div class="toast-content">
    <h4>Success</h4>
    <p>Your changes have been saved.</p>
  </div>
</div>
