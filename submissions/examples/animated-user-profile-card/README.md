# Animated User Profile Card

A responsive animated user profile card component built using pure HTML and CSS.

## Features

- Hover elevation effects
- Animated profile image interactions
- Animated social icons
- Responsive design
- Smooth transitions and motion effects
- Pure CSS implementation

## Usage

Open `demo.html` in a browser to view the animated user profile card component.

## Files

- `demo.html` → Profile card structure
- `style.css` → Profile card animations and styles
- `README.md` → Documentation
