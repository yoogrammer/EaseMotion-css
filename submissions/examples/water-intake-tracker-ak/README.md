# Water Intake Tracker

## What does this do?
A water intake tracker with configurable daily goal, interactive glass-by-glass logging with visual fill animation, progress percentage, and a 7-day weekly history chart.

## How is it used?
Set your daily glass goal, then click on individual glasses to log or remove each glass of water. The progress updates in real time showing count, percentage, and remaining glasses.

## Why is this useful?
This provides a simple hydration tracking tool for health and wellness apps, encouraging users to meet their daily water intake goals with visual feedback.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to use the tracker.
