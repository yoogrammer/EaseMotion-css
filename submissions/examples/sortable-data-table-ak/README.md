## What does this do?

This component renders a data table with sortable columns. Clicking any column header sorts the table rows in ascending or descending order, with an arrow indicator showing the active sort direction.

## How is it used?

Include the `style.css` and the provided HTML structure. The table headers use a `data-key` attribute that maps to object keys in the JavaScript data array. Clicking a header toggles the sort direction.

## Why is this useful?

Sortable tables make it easy for users to reorder and find information quickly, improving data exploration and usability in dashboards, reports, and admin panels.

## Tech Stack

- HTML
- CSS
- JavaScript (vanilla)
