EXPLORE THE UNSEEN

A cinematic camera pull-back effect driven entirely by scroll offset. Bypasses all JavaScript execution for pure, GPU-accelerated rendering.

Technical Specifications

  • Scroll-Driven Animation: Powered by the modern CSS properties animation-timeline: scroll(root), linking keyframe progression directly to viewport scroll offset.
  • Camera Pull-Back: Images start clustered at center scaled up (`scale(2.2)`) and translated forward on the Z-axis (`translateZ(380px)`), then zoom backward and disperse as scroll progresses.
  • Unified Keyframe Engine: Positions are mapped using separate CSS custom variables (`--tx`, `--ty`, `--tz`, `--rot`, `--sc`) inside a single, reusable keyframe animation block.
  • Cross-Browser Fallback: Browsers lacking scroll-timeline support automatically ignore timeline rules via @supports and render a clean, standard grid layout.
  • Reduced Motion: Fully disables all sticky behaviors, 3D translations, and timeline animations when prefers-reduced-motion is active.
