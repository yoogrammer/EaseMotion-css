# Animation: Glass Dock Navigation

A dock-style navigation component inspired by desktop operating systems. Icons magnify when hovered, creating a playful and highly interactive navigation experience.

## Features

- Pure CSS interaction
- Dock magnification effect
- Glassmorphism container
- Smooth scaling animation
- Responsive layout
- Reduced motion support
- No JavaScript required

## Usage

```html
<nav class="dock">
    <a href="#" class="dock-item">🏠</a>
    <a href="#" class="dock-item">📁</a>
    <a href="#" class="dock-item">📧</a>
</nav>
```

Hovering an icon enlarges it while nearby icons scale subtly to create a dock-style effect.

## Why is it useful?

Dock navigation provides an engaging way to present frequently used actions while maintaining a compact layout. It works particularly well for dashboards, portfolios, launchpads, and modern application interfaces.
