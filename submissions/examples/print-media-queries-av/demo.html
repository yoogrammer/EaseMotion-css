EaseMotion Sticky Navbar (This will gracefully hide when printing)

Print Media Queries Refactor

Press Ctrl+P or Cmd+P to open your browser's print dialog. Notice how the giant purple sticky navbar is completely stripped from the printed document preview. This natively saves the user's expensive printer ink and guarantees that the actual text content is perfectly readable without UI artifacts blocking or overlapping it.

Currently, the core framework architecture completely lacks these vital @media print rules, causing massive layout breakages when end-users attempt to physically print or save-to-PDF a webpage built with EaseMotion.
