# Pricing Toggle Component

A premium, reusable pricing toggle component designed for SaaS interfaces. This component features a smooth monthly/yearly switcher, animated price transitions, and responsive pricing cards.

## Key Features
- **Monthly/Yearly Switcher**: Seamlessly toggle between billing cycles with a smooth sliding animation.
- **Animated Price Transitions**: Prices fade and slide slightly when switching, providing visual feedback.
- **Savings Badge**: Highlight the benefits of annual billing with a dedicated savings badge.
- **Responsive Pricing Cards**: Fluid layout that looks great on mobile, tablet, and desktop.
- **Accessibility**: Built with semantic HTML and respects `prefers-reduced-motion`.

## How to Use
1. Include `easemotion.css` in your project.
2. Copy the HTML structure from `demo.html`.
3. Apply the custom styles from `style.css`.

## Directory Structure
- `demo.html`: Interactive demo showcasing the pricing toggle.
- `style.css`: Styles for the component and animations.
