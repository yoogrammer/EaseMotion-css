# Knowledge Hub

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS.

## Overview

A responsive Knowledge Hub dashboard for organizing documentation, tutorials, FAQs, and troubleshooting resources. Suitable for documentation portals, internal wikis, developer centers, SaaS help centers, and enterprise knowledge management systems.

## Features

- Hero header
- Search bar
- Category cards
- Knowledge article cards
- Responsive dashboard
- Glassmorphism styling
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="dashboard">

    <section class="categories">
        ...
    </section>

    <section class="articles">
        ...
    </section>

</div>
```
