# Card CTA Spacing Fix

## What this fixes
CTA text (e.g. Explore, Learn More, Get Started) inside .ease-card had no visual separation from body text, making it blend in rather than stand out as an action element.

## Fix
Added display: block and margin-top: 1rem to strong and  elements inside .ease-card.

- style.css — the fix
- demo.html — live demonstration
