# Elastic Notification Bell

## Description

A notification bell animation with an elastic swinging effect on hover.

## Features

- Smooth elastic swing animation
- CSS-only implementation
- Lightweight and reusable
- Ideal for notification systems

## Files

- demo.html
- style.css
- script.js

## Usage

Open `demo.html` in a browser and hover over the bell.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
