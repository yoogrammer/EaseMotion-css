# Modern Animated Ring Loader Spinner

A sleek, modern, pure CSS loading spinner indicator.

## Files
- `demo.html`: The HTML structure for the spinner.
- `style.css`: The CSS that generates the rotating ring.
- `README.md`: This file.

## Features
- **Pure CSS:** No SVGs, GIFs, or canvas elements are required. The entire shape is drawn using CSS `border` properties.
- **Infinite Rotation:** Uses an `@keyframes` animation to spin the ring continuously.
- **Highly Customizable:** Change the `--loader-size`, `--loader-color`, and `--loader-track-color` CSS variables to instantly fit the spinner into any application theme.

## Usage
Add an empty `div` with the class `.animated-ring-loader`. It will automatically spin and indicate loading progress.
