# Pulse Dot Chain Loader

### What does this do?

Creates a polished row of pulsing dots that animate sequentially from left to right. The loader is built with pure HTML and CSS, and is ideal for modern SaaS dashboards and product interfaces.

### How is it used?

Add the markup and include `style.css` in the same folder.

Default Loader:

```html
<link rel="stylesheet" href="style.css">
<div class="loader-row loader-default">
  <span class="dot"></span>
  <span class="dot"></span>
  <span class="dot"></span>
  <span class="dot"></span>
  <span class="dot"></span>
</div>
```

### Why is it useful?

- Provides an immediately visible loading state with clear sequential motion.
- Ideal for SaaS, dashboards, and product experiences where polished UI feedback is important.
- Fully self-contained, responsive, and free of JavaScript or external assets.
