# Microinteraction Showcase

### 1. What does this do?
This is a premium frontend microinteraction catalog dashboard showcasing fine-tuned interactive transitions, loading states, sliding indicators, input effects, and switch states utilizing EaseMotion CSS variables.

### 2. How is it used?
Open `demo.html` directly in any modern web browser to access the showcase:
- **Filter Groups**: Browse widgets categorized into Buttons, Cards, Navigation, Loaders & Progress, and Inputs & Toggles.
- **Interact with Widgets**: Hover, click, and inspect the responsive state changes live on the visualizer cards (e.g. typing in form fields, clicking ripple nodes, toggling active tabs).
- **Inspect Markup**: Select any widget to print its styled implementation template and click "Copy" to grab the code.

### 3. Why is it useful?
It fits EaseMotion CSS's philosophy by demonstrating detailed state interactions. Standard CSS animations look good, but true sensory delight comes from microinteractions (like labels sliding on focus or navbar indicator sliders moving on hover), showcasing how to leverage framework properties for fine-grained interaction states.
