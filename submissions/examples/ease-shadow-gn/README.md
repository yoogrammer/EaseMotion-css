# ease-shadow Box-Shadow Utilities

1. **What does this do?** Adds smooth box-shadow transitions on hover using six utility classes — from no shadow to extra-large and inner shadow.
2. **How is it used?**
```html
<div class="ease-shadow-none">No shadow</div>
<div class="ease-shadow-sm">Small shadow on hover</div>
<div class="ease-shadow">Default shadow on hover</div>
<div class="ease-shadow-lg">Large shadow on hover</div>
<div class="ease-shadow-xl">Extra large shadow on hover</div>
<div class="ease-shadow-inner">Inner shadow on hover</div>
```
3. **Why is it useful?** Provides a complete set of depth utilities with zero JavaScript — pure CSS transitions that follow EaseMotion CSS's naming conventions and philosophy.
