# Fade Text Entrance

A smooth fade-style entrance animation for text components.

## Features

- Lightweight pure CSS animation
- Smooth fade entrance effect
- Cross-browser compatible
- Reusable utility class

## Usage

```html
<h1 class="ease-fade-text-entrance">
  Welcome
</h1>
```

## Animation Details

- Text fades in smoothly
- Slight upward motion during entrance
- Modern and lightweight interaction
