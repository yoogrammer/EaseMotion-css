# Neumorphic Music Player Component

1. What does this do? Creates a highly styled, modern music player widget using pure neumorphic CSS styling methods.
2. How is it used? Apply `.music-player` styles to a wrapper containing nav buttons, album art, sliders, and audio control elements.
3. Why is it useful? It illustrates how to leverage double outset box-shadows alongside double inset box-shadows to craft realistic physical-depth UI widgets.
