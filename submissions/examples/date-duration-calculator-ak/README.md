# Date Duration Calculator

## What does this do?
A precise date duration calculator that computes the difference between two dates across years, months, days, hours, minutes, and seconds with animated counters.

## How is it used?
Pick start and end dates, or use quick presets (This Week, This Month, This Year, Last 30 Days). All duration units update with animated number transitions.

## Why is this useful?
Quickly find the exact duration between any two dates for planning, tracking project timelines, or calculating age and service periods.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to calculate date durations.
