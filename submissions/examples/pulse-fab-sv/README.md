# Pulse Floating Action Button

## What does this do?

Creates a floating action button fixed to the bottom-right corner of the page with a subtle pulse animation to attract attention.

## How is it used?

```html
<button class="pulse-fab">
  +
</button>
```

## Why is it useful?

Floating action buttons are commonly used for primary actions such as creating content, opening forms, starting chats, or triggering workflows. The pulse effect helps users discover the action without being intrusive.
