Entrance Animations

Four ways to enter
the third dimension.

Hover over each card to trigger the animation. Stack with ease-delay-* and ease-duration-* freely.

⬆️
ease-rise-3d

Element rises from behind the screen along the Z axis, combined with a subtle upward Y translate.

translateZ(-120px) translateY(30px)
🔄
ease-flip-3d

Element flips in from a flat horizontal plane — like a card face-down that reveals itself.

rotateX(-80deg) → rotateX(0deg)
↩️
ease-tilt-in

Element swings in from the side on the Y axis, as if hinging open from the left edge.

rotateY(60deg) → rotateY(0deg)
🔭
ease-depth-fade

Element zooms in from deep Z space while fading up — a cinematic depth-of-field entrance.

translateZ(-300px) scale(0.5) → scale(1)

Hover Effect

Cards that tilt
toward your cursor.

ease-hover-3d-lift gives any card a perspective tilt on hover — pure CSS, zero JavaScript.

⚡

Animation-first

Motion is a first-class citizen in EaseMotion CSS — not bolted on after the fact.

🧩

Fully Composable

Stack ease-delay-*, ease-duration-*, and reveal utilities with any 3D class.

📖

Human-readable

Class names read like plain English. If you can say it, you can write it as a class.

Usage

Drop a class.
Get 3D.

No configuration. No build step. Just add the class to any element and it works.

HTML
<!-- Entrance animations --> <h1 class="ease-rise-3d"> Rise from depth </h1> <p class="ease-flip-3d ease-delay-200"> Flip on X axis </p> <div class="ease-tilt-in ease-delay-300"> Swing in from side </div> <section class="ease-depth-fade"> Emerge from Z-space </section> <!-- Hover lift effect --> <div class="ease-card ease-hover-3d-lift"> Tilts toward cursor </div>
CSS (raw — maintainer standardizes)
/* Rise from Z-depth */ .rise-3d { animation: rise3d 0.7s cubic-bezier(0.34,1.56,0.64,1) both; } @keyframes rise3d { from { opacity:0; transform: perspective(800px) translateZ(-120px) translateY(24px); } to { opacity:1; transform: perspective(800px) translateZ(0) translateY(0); } } /* Hover 3D lift */ .hover-3d-lift:hover { transform: perspective(600px) rotateX(8deg) rotateY(-8deg) translateZ(16px); transition: transform 0.3s ease; } /* Reduced motion — always included */ @media (prefers-reduced-motion: reduce) { .rise-3d, .flip-3d, .tilt-in, .depth-fade { animation: none; opacity: 1; } }
