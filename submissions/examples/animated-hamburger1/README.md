# Animated Hamburger Menu

A smooth, accessible, and **100% JavaScript-free** animated hamburger menu toggle. It uses the classic "CSS Checkbox Hack" combined with modern CSS transforms to deliver a buttery-smooth 60fps animation that transitions from a 3-line hamburger into an "X" close icon.

## ✨ Features

- ⚡ **Zero JavaScript**: Pure CSS logic using the `:checked` pseudo-class.
- 🎬 **Smooth 60fps Animations**: GPU-accelerated `transform` and `opacity` transitions.
- ♿ **Accessible**: Keyboard navigable (`Tab` + `Space`/`Enter`), includes `aria-label`, and respects `prefers-reduced-motion`.
- 📱 **Responsive**: Includes a sleek slide-in mobile navigation drawer.
- 🎨 **Easy to Customize**: CSS variables for colors, sizing, and animation speed.
