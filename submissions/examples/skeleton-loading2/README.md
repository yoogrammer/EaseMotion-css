# Shimmer Skeleton Loading

Animated gradient shimmer placeholders for content loading states. Pure CSS — no JavaScript required.

## Demo

Open `demo.html` in any modern browser.

## Features

- ✅ **Pure CSS** — zero JavaScript
- ✅ **Shimmer animation** — sweeping gradient overlay
- ✅ **Multiple shapes** — text lines, avatar, image, circle, table cells
- ✅ **Layout helpers** — card, article, table, row, column
- ✅ **Speed variants** — fast, normal, slow
- ✅ **Dark mode** — automatic via `prefers-color-scheme`
- ✅ **Accessible** — respects `prefers-reduced-motion`

## Usage
