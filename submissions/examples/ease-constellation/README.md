# Constellation Connect

A futuristic hover animation inspired by star constellations and neural networks.

When the user hovers over the card, glowing stars become connected through animated lines, creating a smooth constellation effect that resembles an AI network or space map.


## ✨ Features

* Animated constellation-style connections
* Smooth line reveal on hover
* Glowing star nodes
* Futuristic cyber aesthetic
* Pure HTML + CSS (no JavaScript)
* Lightweight and easy to customize


## Preview

Hover over the card to activate the network and reveal the animated constellation pattern.



## HTML Structure

```html
<div class="constellation-card">

  <!-- lines -->
  <span class="line l1"></span>
  <span class="line l2"></span>
  <span class="line l3"></span>
  <span class="line l4"></span>
  <span class="line l5"></span>

  <!-- stars -->
  <span class="star s1"></span>
  <span class="star s2"></span>
  <span class="star s3"></span>
  <span class="star s4"></span>
  <span class="star s5"></span>
  <span class="star s6"></span>

  <div class="content">
    <h2>AI Network</h2>
    <p>
      Hover to reveal a constellation-inspired neural connection system.
    </p>
  </div>

</div>
```
