# Ease Mesh Background

A beautiful animated mesh gradient background with multiple floating gradient orbs.

## Features

- Animated gradient mesh effect
- Floating blurred color orbs
- Pure CSS implementation
- Responsive design
- Supports reduced motion preference

## Usage

```html
<div class="ease-mesh-bg">
  <div class="ease-mesh-bg__orb ease-mesh-bg__orb--1"></div>
  <div class="ease-mesh-bg__orb ease-mesh-bg__orb--2"></div>
  <div class="ease-mesh-bg__orb ease-mesh-bg__orb--3"></div>

  <div class="ease-mesh-bg__content">
    <h1>Your Content Here</h1>
  </div>
</div>
