# Animated Hero Section - EaseMotion CSS

A stunning, modern, and fully responsive hero section built with pure CSS. Designed for SaaS landing pages, modern web apps, and portfolio sites, featuring smooth staggered entrance animations and a sleek dark-mode aesthetic.

## ✨ Features

- 🎬 **Staggered Animations**: Text and buttons fade in sequentially for a premium feel.
- 🎨 **Modern Dark Theme**: Deep background with animated, floating gradient orbs.
- 📱 **Fully Responsive**: Adapts beautifully from mobile to ultra-wide desktops.
- ⚡ **Zero JavaScript**: 100% pure CSS animations and transitions.
- ♿ **Accessible**: Respects `prefers-reduced-motion` and includes visible focus states.
- 🚀 **Performance Optimized**: Uses GPU-accelerated transforms (`translateY`) for smooth 60fps animations.
