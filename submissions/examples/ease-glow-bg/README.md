\# ease-glow-bg



Glow background entrance animation for cards and containers.



\## Classes



| Class | Description | Speed |

|-------|-------------|-------|

| `ease-glow-bg` | Normal glow entrance | 1s |

| `ease-glow-bg-fast` | Fast glow entrance | 0.5s |

| `ease-glow-bg-slow` | Slow glow entrance | 2s |

| `ease-glow-bg-strong` | Strong intense glow | 1s |

| `ease-glow-bg-subtle` | Subtle gentle glow | 1s |

| `ease-glow-bg-colored` | Rainbow colored glow | 1.5s |



\## Usage



```html

<div class="ease-glow-bg">

&#x20;   Content glows on entrance

</div>
