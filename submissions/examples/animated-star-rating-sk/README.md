# Animated Star Rating Example

1. What does this do? This example provides a pure CSS, interactive star rating component with smooth hover previews and selection animations.
2. How is it used? Use a set of radio inputs with the `.rating-input` class and corresponding labels with the `.rating-label` class inside a `.rating-group` container.
3. Why is it useful? It allows developers to add polished feedback interactions to their forms without any JavaScript, ensuring high performance and accessibility while maintaining a modern aesthetic.
