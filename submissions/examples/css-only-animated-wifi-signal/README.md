# CSS-Only Animated Wi-Fi Signal

A clean and smooth animated Wi-Fi signal indicator built purely with CSS.

## Features
- Pure CSS implementation
- Smooth pulsing and outward signaling animation
- Responsive design
- Clean and modern UI

## Usage
Simply include the `style.css` and use the HTML structure from `demo.html`. No JavaScript is required.

## Preview
Open `demo.html` in your browser to view the component.
