# Backdrop Blur Gallery

## What does this do?
Displays a gallery of cards with varying backdrop-blur levels over a colorful gradient.

## How is it used?
Add a card with a `blur-content` element and set `--blur` custom property:

    <div class="blur-card" style="--blur: 6px">
      <div class="blur-content">Text</div>
    </div>

## Why is it useful?
Demonstrates frosted-glass effects at different intensities for UI design.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
