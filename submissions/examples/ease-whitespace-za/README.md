## What does this do?

Provides `whitespace` utility classes for controlling whitespace and line break handling. Covers all six `white-space` CSS values: `normal`, `nowrap`, `pre`, `pre-wrap`, `pre-line`, and `break-spaces`.

## How is it used?

```html
<p class="ease-whitespace-nowrap-za">No line breaks here</p>
<pre class="ease-whitespace-pre-wrap-za">Wrapping preserved text</pre>
```

## Why is it useful?

EaseMotion has no whitespace utilities, yet they're essential for code blocks, addresses, labels, and text layout control. Every major CSS framework includes these as standard utilities.
