# CSS Loader Gallery

A comprehensive gallery of pure CSS loaders and spinners.

## Features
- 8 loader variants: border spinner, dots, ring, ellipsis, hourglass, pulse, striped bar, indeterminate bar
- Pure CSS keyframe animations
- Reusable component patterns

## Usage
```html
<div class="spinner-border"></div>
<div class="spinner-dots"><span></span><span></span><span></span></div>
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
