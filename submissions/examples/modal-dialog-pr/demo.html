How It Works

  1. Click a trigger link — it points to the modal's id (e.g., #modal-confirm)
  2. The :target pseudo-class activates, showing the modal with backdrop
  3. Modal content scales in with cubic-bezier bounce
  4. Click the backdrop, × button, or any close link — all point to #modal-hide-pr
  5. Target removed, modal fades out and hides

♿ Accessibility: Respects prefers-reduced-motion. In production, add role="dialog", aria-modal="true", and trap focus with JavaScript. Pure CSS version is best for simple use cases.
