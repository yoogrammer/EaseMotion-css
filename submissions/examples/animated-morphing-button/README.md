# Animated Morphing Button

## What it does

A button that morphs shape on hover - changing border-radius, size, and color with elastic easing.

## How to use it

<button class="ease-morph-btn">Hover Me</button>

## Variants

- ease-morph-fast - 0.2s transition
- ease-morph-slow - 0.8s transition
- ease-morph-pill - Pill to rounded rectangle
- ease-morph-square - Rounded to sharp square
- ease-morph-glow - Neon glow effect
- ease-morph-expand - Expands width on hover
- ease-morph-shrink - Shrinks on hover
- ease-morph-rotate - Rotates slightly

## Why it fits EaseMotion CSS

- Pure CSS transitions
- Elastic cubic-bezier easing
- Ripple effect on click
- Multiple morphing styles
