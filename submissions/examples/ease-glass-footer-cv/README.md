# Glass Footer (cv variant)

## What does this do?
A premium glassmorphism-style footer with animated gradient borders, staggered fade-in animations, glowing social icons, and smooth hover interactions — all built with pure CSS.

```html

    <div class="ease-glass-footer-cv-glow"></div>

            <h3 class="ease-glass-footer-cv-logo">

            <p class="ease-glass-footer-cv-tagline">Your tagline here</p>

                <a href="#" class="ease-glass-footer-cv-social-link">...</a>
            </div>
