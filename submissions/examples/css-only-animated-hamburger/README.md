# CSS-only Animated Hamburger Menu

### 1. What does this do?
A sleek, purely CSS-driven animated hamburger menu icon. It uses a hidden checkbox hack to toggle states, transforming three horizontal bars into a smooth, satisfying "X" close icon upon interaction with a bouncy cubic-bezier transition.

### 2. How is it used?
```html
<label class="hamburger-menu">
  <input type="checkbox" />
  <div class="bars">
    <span class="bar bar1"></span>
    <span class="bar bar2"></span>
    <span class="bar bar3"></span>
  </div>
</label>
```

### 3. Why is this useful?
Hamburger menus are ubiquitous in modern responsive web design. Providing a highly polished, interactive toggle animation using strictly CSS aligns perfectly with EaseMotion CSS's goal of offering lightweight, plug-and-play UI utilities.
