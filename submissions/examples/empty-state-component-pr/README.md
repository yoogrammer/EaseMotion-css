# Empty State Component

## What does this do?

A responsive empty state component for displaying a friendly placeholder when no content, messages, notifications, or search results are available.

## How is it used?

```html
<div class="empty-state">
  <div class="empty-icon">📭</div>
  <h2>No Messages Yet</h2>
  <p>Your inbox is currently empty.</p>
</div>
```

## Why is it useful?

Empty states improve user experience by providing helpful context and guidance when there is no data to display. They are commonly used in dashboards, inboxes, search interfaces, and SaaS applications.
