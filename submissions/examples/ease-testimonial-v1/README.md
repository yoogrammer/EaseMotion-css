# ease-testimonial

Beautiful testimonial cards with avatar, quote text, author name slots, and decorative quote marks.

## Files

| File | Description |
|------|-------------|
| 📄 [demo.html](./demo.html) | Interactive demo |
| 🎨 [style.css](./style.css) | Testimonial styles |
| 📖 [README.md](./README.md) | Documentation |

## Classes

| Class | Description |
|-------|-------------|
| `ease-testimonial` | Base testimonial card |
| `ease-testimonial-primary` | Purple gradient variant |
| `ease-testimonial-dark` | Dark theme variant |
| `ease-testimonial-glass` | Glassmorphism variant |
| `ease-testimonial-centered` | Centered layout |

## Structure

```html
<div class="ease-testimonial">
    <div class="testimonial-avatar">
        <img src="avatar.jpg" alt="Avatar">
    </div>
    <div class="testimonial-content">
        <p class="testimonial-quote">Quote text here</p>
        <h4 class="testimonial-author">Author Name</h4>
        <p class="testimonial-title">Author Title</p>
    </div>
</div>
