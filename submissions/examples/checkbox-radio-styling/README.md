# Checkbox & Radio Styling

Custom checkbox and radio input styling using `appearance: none` with animated pseudo-element indicators.

## Features

- `appearance: none` with pure CSS pseudo-element checkmarks and dots
- Animated checkmark (rotating scale) and radio dot (scale) indicators
- Four color variants: blue, green, purple, red
- Focus-visible ring for keyboard navigation
- Disabled and disabled-checked states with reduced opacity
- `prefers-reduced-motion` support

## Files

- `demo.html` — Interactive demo with all variants and states
- `style.css` — Self-contained CSS (201 lines)

## Preview
