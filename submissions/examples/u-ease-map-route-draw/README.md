# ease-map-route-draw

Animated SVG route drawing effect for maps and navigation interfaces.

## Features

- SVG path drawing animation
- Uses stroke-dashoffset technique
- Configurable animation speed
- Start and end map markers
- Pure HTML and CSS
- Suitable for navigation and route visualizations

## Usage

```html
<path class="route-path" d="..." />
```

## Custom Property

```css
--ease-route-draw-speed: 3s;
```

Adjust the drawing speed by changing the CSS variable.

## Animation Details

- Route starts hidden.
- SVG stroke progressively reveals itself.
- Mimics real-world navigation route drawing.
