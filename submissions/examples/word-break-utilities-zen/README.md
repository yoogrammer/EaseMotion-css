## What does this do?

Provides word-break and overflow-wrap utility classes for controlling text wrapping behavior.

## How is it used?

```html
<p class="break-all">LongWordsWillBreakAtAnyCharacter</p>
<p class="break-word">Long words will wrap at word boundaries</p>
```

## Why is it useful?

Essential for handling long URLs, code snippets, and preventing horizontal overflow in narrow containers.
