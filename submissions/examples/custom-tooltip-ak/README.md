## What does this do?

This component provides custom CSS tooltips that appear on hover with configurable positions (top, bottom, left, right) and a smooth fade-in animation.

## How is it used?

Add `data-tip="Your tooltip text"` and `data-pos="top|bottom|left|right"` to any element. The JavaScript automatically injects tooltip spans and the CSS handles positioning and hover visibility.

## Why is this useful?

Custom tooltips enhance user experience by providing contextual information without cluttering the interface. Position control ensures the tooltip stays visible regardless of where the target element is on the page.

## Tech Stack

- HTML
- CSS
- JavaScript (vanilla)
