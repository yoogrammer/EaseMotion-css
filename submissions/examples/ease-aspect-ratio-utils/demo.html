ease-aspect-ratio utilities

Four composable aspect-ratio helpers for images, cards, and embedded media. Each card below uses one utility class to lock its proportions inside a responsive grid.

.ease-aspect-square Ratio: 1 / 1
.ease-aspect-video Ratio: 16 / 9
.ease-aspect-portrait Ratio: 3 / 4
.ease-aspect-4-3 Ratio: 4 / 3

.ease-aspect-video (div) Embedded video placeholder — 16 / 9

Browser note: The CSS aspect-ratio property requires Safari 15+, Chrome 88+, and Firefox 89+. It is not supported in Safari 9.1 despite what the main project browser compatibility table currently states.
