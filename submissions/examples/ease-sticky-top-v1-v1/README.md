 
# ease-sticky-top

Sticks an element to the top of the viewport on scroll.

## Usage
```html
<nav class="ease-sticky-top">Sticky Header</nav>
```

## CSS Output
```css
.ease-sticky-top {
  position: sticky;
  top: 0;
  z-index: 100;
}
```

## Use Case
Sticky headers and navigation bars that stay visible while scrolling.
