# Animation: Interactive Product Roadmap

A roadmap component that displays project milestones along a connected timeline. Hovering a milestone highlights the node, elevates the card, and reveals additional information.

## Features

- Pure CSS interaction
- Animated roadmap timeline
- Interactive milestone cards
- Detail reveal animation
- Responsive layout
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="milestone">

    <div class="node"></div>

    <div class="card">
        <h3>Core Features</h3>

        <div class="details">
            Additional roadmap information.
        </div>
    </div>

</div>
```

Hovering a milestone expands the roadmap card and reveals more information.

## Why is it useful?

Roadmaps are commonly used for product planning, changelogs, project tracking, and documentation. This component presents progress in a visually engaging and easy-to-follow format.
