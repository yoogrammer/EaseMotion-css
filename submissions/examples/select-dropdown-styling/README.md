# Select Dropdown Styling

Custom select dropdown styling with SVG arrow indicators, focus rings, and multiple visual variants.

## Features

- `appearance: none` with custom SVG background arrow indicator
- Six style variants: default, bordered, pill, ghost, dark, glass
- Focus ring with matching accent color per variant
- Styled `optgroup` headers with indented options
- Hover and focus state transitions
- Disabled select state
- `prefers-reduced-motion` support

## Files

- `demo.html` — Interactive demo with all variants and optgroup example
- `style.css` — Self-contained CSS (163 lines)
