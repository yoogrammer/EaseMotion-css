# Sequential Letter Wave Animation

## What does this do?

Creates a ripple-like wave animation where each letter moves vertically in sequence.

## How is it used?

```html
<p class="wave-text">
  <span>H</span>
  <span>e</span>
  <span>l</span>
  <span>l</span>
  <span>o</span>
</p>
```

Optional variants:

```html
<p class="wave-text wave-text-fast">...</p>
<p class="wave-text wave-text-slow">...</p>
```

## Why is it useful?

This effect is useful for hero headings, loading indicators, interactive text, and playful UI elements. It aligns with EaseMotion CSS by providing a lightweight, animation-first effect with no JavaScript dependencies.
