# Copy Link Button

## Description

A lightweight copy-link button that can be placed beside documentation headings.

When clicked, it copies the direct URL of the corresponding section to the clipboard, making it easier to share and navigate documentation content.

---

## Features

- Copies section URLs to the clipboard
- Simple and lightweight implementation
- Visual feedback after successful copy
- Easy integration into documentation pages

---

## Usage

Place the button beside a heading and assign it a section ID.

Example:

```html
<h2 id="philosophy">
  Design Philosophy
  <button class="copy-link-btn">🔗</button>
</h2>
