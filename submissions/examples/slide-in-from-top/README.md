# ease-slide-in-from-top

Entrance animation that slides an element in from the top (-100% Y offset) with fade-in. Great for header reveals and modal entrances.

## Usage

```html
<div class="ease-slide-in-from-top">Slides in from top</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-slide-in-from-top` | Slide from top |
| `ease-slide-in-from-bottom` | Slide from bottom |

## Notes

- Full offset (-100% Y) entrance
- Combines with fade-in
- Respects `prefers-reduced-motion`
