# Animated Gradient Border Cards

A stunning, modern card component featuring animated rotating gradient borders. Built with pure CSS using the cutting-edge `@property` rule and `conic-gradient`, these cards create a mesmerizing effect similar to those seen on Linear, Vercel, and Stripe websites.

## ✨ Features

- 🌈 **Animated Gradient Borders**: Smoothly rotating conic-gradient borders using CSS `@property`
- ⚡ **Zero JavaScript**: 100% pure CSS implementation
- 🎨 **Multiple Color Variants**: Four different gradient themes (blue, purple, green, orange)
- 🖱️ **Hover Effects**: Cards lift up and gradient border appears on hover
- ♿ **Accessible**: Respects `prefers-reduced-motion` and includes focus states
- 📱 **Responsive**: Adapts from single column on mobile to multi-column grid on desktop
- 🚀 **Performance**: GPU-accelerated animations for smooth 60fps rendering
