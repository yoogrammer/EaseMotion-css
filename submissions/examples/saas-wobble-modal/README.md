# SaaS Showcase Wobble-Focus Modal (Pure CSS)

A clean, modern modal framework optimized for professional B2B/SaaS product landing layouts. It relies exclusively on native core CSS utilities for operation.

## Key Features
- **Zero JavaScript Dependencies**: Operates natively via standard declarative `:target` viewport focus states.
- **Wobble-Focus Mechanics**: Implements a snappy, decaying horizontal oscillation sequence that helps retain direct visual focus on system dialog cards without harsh flashes.
- **SaaS Component Visuals**: Uses neutral dashboard tones, clean container elevations, and border structures.

## Modification
Update values in the `:root` pseudo-selector block inside your `style.css` file:
- `--wobble-duration`: Adjusts the decay speed of the entry horizontal oscillation effect.
- `--brand-primary`: Direct custom branding accent color variables.
