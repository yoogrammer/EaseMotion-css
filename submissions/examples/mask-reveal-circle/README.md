# ease-mask-reveal-circle

Circular clip-path reveal animation using `clip-path: circle()` for an expanding circle reveal effect.

## Usage

```html
<div class="ease-mask-reveal-circle">Content revealed</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-mask-reveal-circle` | Circle reveal from center to 75% |
| `ease-mask-reveal-circle-full` | Circle reveal from center to 100% |

## Notes

- Uses `clip-path` for the mask effect
- Great for hero images, profile photos, and splash content
- Respects `prefers-reduced-motion`
