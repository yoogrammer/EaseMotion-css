# Animated Pricing Card Component

A responsive animated pricing card component built using pure HTML and CSS.

## Features

- Multiple pricing tiers
- Highlighted featured plan
- Hover animations
- Responsive card layout
- Smooth transitions and motion effects
- Pure CSS implementation

## Usage

Open `demo.html` in a browser to view the animated pricing card component.

## Files

- `demo.html` → Pricing card structure
- `style.css` → Pricing card animations and styles
- `README.md` → Documentation
