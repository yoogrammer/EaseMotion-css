# EaseMotion CSS Animation Documentation

A comprehensive guide covering all available **EaseMotion CSS** utility classes. Designed to eliminate guesswork and streamline front-end UI onboarding.

## Key Objectives

- 🚀 **High Discoverability:** Centralized catalog of predefined UI motions.
- 🛠️ **Zero Config:** Drag-and-drop CSS utility classes straight into HTML templates.
- 📦 **Efficient Onboarding:** Clear reference specs for new engineers and designers.

## Core Core Utility Classes

| Class Name      | Animation Type | Default Duration | Behavior                                           |
| :-------------- | :------------- | :--------------- | :------------------------------------------------- |
| `ease-fade-in`  | Entrance       | `0.5s`           | Smooth opacity opacity transition from `0` to `1`  |
| `ease-slide-up` | Entrance       | `0.6s`           | Vertical slide tracking upward paired with fade-in |
| `ease-bounce`   | Loop/Attention | `0.8s`           | Spring-action physics bounce effect for CTAs       |

## Usage Quickstart

Simply include the `style.css` stylesheet file into your document header, and apply the utility directly to the target node elements:

```html
<div class="ease-slide-up">This content smoothly slides up into view.</div>
```
