## Tab-Size Utility

**What does this do?**  
Provides utility classes to control the rendering width of tab characters (`\t`) inside `<pre>`, `<code>`, and `<textarea>` elements.

**How is it used?**  
Add `tab-size-2`, `tab-size-4`, or `tab-size-8` to any `<pre>` or `<code>` block:

```html
<pre class="tab-size-2">...</pre>
<pre class="tab-size-4">...</pre>
<pre class="tab-size-8">...</pre>
```

**Why is it useful?**  
Developers displaying code snippets often want consistent tab indentation. These utilities give simple, declarative control over tab rendering width, making code blocks easier to read across different projects and preferences.
