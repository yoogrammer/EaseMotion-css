Tab-Size Utility

Control the rendered width of tab characters (\t) inside <pre> and <code> elements. Each panel below uses a different tab-size value.

tab-size-2 (narrow tabs)

function hello() {
	console.log("Hello");
	if (true) {
		console.log("World");
	}
}

tab-size-4 (default-style tabs)

function hello() {
	console.log("Hello");
	if (true) {
		console.log("World");
	}
}

tab-size-8 (wide tabs)

function hello() {
	console.log("Hello");
	if (true) {
		console.log("World");
	}
}
