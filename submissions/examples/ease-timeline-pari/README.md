## ease-timeline

**What does this do?**
A vertical timeline component with animated slide-left entrance
for each item using staggered nth-child delays.

**How is it used?**
<div class="ease-timeline">
  <div class="ease-timeline-item">
    <p class="item-date">June 2026</p>
    <h3 class="item-title">Title</h3>
    <p class="item-desc">Description</p>
  </div>
</div>

Color variants: add class `green` or `orange` to `.ease-timeline`

**Why is it useful?**
Delivers a clean, animated vertical timeline using pure CSS —
consistent with EaseMotion CSS's component and animation philosophy.
