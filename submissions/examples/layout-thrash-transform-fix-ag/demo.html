Layout Thrash → Transform Fix

Animating margin, padding, or width triggers expensive layout recalculations every frame. This fix replaces all layout-triggering properties with transform: translateY() and transform: scale() — GPU-composited, zero layout cost.

🚀 Lift Effect translateY(-10px) on hover
✦ Grow Effect scale(1.07) on hover
✅ 0 layout recalculations — pure GPU compositing
