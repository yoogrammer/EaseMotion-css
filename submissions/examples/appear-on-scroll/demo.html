Features

Everything you need.
Nothing you don't.

EaseMotion's scroll-reveal utility is designed to be simple, composable, and respectful of user preferences.

⚡

Zero JavaScript

Powered entirely by CSS animation-timeline: view() and animation-range. No observers, no handlers, no scripts.

🎞

Five Variants

Choose from fade-up, fade-down, fade-left, fade-right, or fade-scale with a single modifier class.

🕐

Stagger Delays

Five pre-built delay helpers (.delay-1 → .delay-5) let you choreograph sequential reveals in a grid or list.

♿

Accessible by Default

A prefers-reduced-motion media query disables all animations for users who have requested it in their OS settings.

📦

One File, No Build Step

Drop style.css into any project. No npm install, no config, no PostCSS pipeline required.

🎨

Framework Agnostic

Works with React, Vue, Svelte, static HTML — any environment that renders HTML. The classes are just CSS.

How It Works

Up and running in
four simple steps

No configuration object. No data attributes. Just semantic class names that map directly to CSS keyframes.

1

Include the CSS

Add <link rel="stylesheet" href="easemotion.css"> to your <head>.

2

Add the Class

Apply scroll-reveal to any element you want to animate on scroll.

3

Choose a Variant

Optionally pair it with fade-left, fade-scale, etc. to change the animation direction.

4

Stagger Children

Add delay-1 through delay-5 to orchestrate sequential reveals in a grid.

Example Cards

Works on any element,
any layout

Slap .scroll-reveal on cards, text blocks, images, or any HTML element. Mix variants freely.

🔭
Animation

Fade Up

The default variant. Elements rise from 40 px below their resting position while fading in. Great for vertical content flows.

↙
Animation

Fade Left

Slides in from 48 px to the left. Particularly effective for feature lists and sidebar elements.

↘
Animation

Fade Right

The mirror of Fade Left. Combine both in a two-column layout for a satisfying bilateral reveal effect.

⬆
Animation

Fade Down

Drops in from above. Useful for headers, navigation items, or any element anchored to the top of a section.

🔍
Animation

Fade Scale

Scales up from 88% while fading in. Perfect for cards, modals, and focal elements that should feel like they're emerging.

🎛
Utility

Stagger Delays

Five .delay-N helpers translate to animation-delay values from 80 ms to 540 ms for natural-feeling sequencing.
