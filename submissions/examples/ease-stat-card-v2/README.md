# Ease Stat Card

## What does this do?

Creates a reusable statistics card component with support for icons, metric values, labels, and animated count-up effects.

---

## How is it used?

```html
<div class="ease-stat-card">
    <div class="icon">📈</div>
    <h2>12,500</h2>
    <p>Total Users</p>
</div>
```

The component can be combined into dashboard layouts to display analytics, KPIs, and business metrics.

---

## Why is it useful?

Stat cards are one of the most common UI patterns in:

- Admin dashboards
- Analytics platforms
- SaaS products
- CRM systems
- Reporting interfaces

Including a reusable stat card component helps developers quickly build professional dashboard experiences.

---

## Features

- Responsive card layout
- Icon support
- Metric value display
- Label support
- Count-up animation
- Hover interaction
- Dashboard examples
- Reduced-motion support
