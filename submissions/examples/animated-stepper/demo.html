Quick Start Usage

Implement the stepper component using simple HTML structure:

<div class="ease-stepper"> <div class="ease-step ease-step-completed"> <div class="step-marker">✓</div> <div class="step-label">Step 1</div> </div> <div class="ease-step ease-step-active"> <div class="step-marker">2</div> <div class="step-label">Step 2</div> </div> <div class="ease-step"> <div class="step-marker">3</div> <div class="step-label">Step 3</div> </div> </div>

CSS Classes Reference:

.ease-stepper

Main container for horizontal stepper

.ease-step

Individual step element

.ease-step-active

Marks the current active step

.ease-step-completed

Marks completed steps

.ease-stepper-vertical

Modifier for vertical layout

CSS Variables

Customize colors and animation speed

Customization with CSS Variables

Easily customize the stepper appearance by overriding CSS variables:

:root { --stepper-primary-color: #667eea; --stepper-completed-color: #48bb78; --stepper-bg-color: #f0f4ff; --stepper-animation-speed: 0.3s; }

Available Variables:

--stepper-primary-color

Color for active step (default: #667eea)

--stepper-completed-color

Color for completed steps (default: #48bb78)

--stepper-bg-color

Background color for steps (default: #f0f4ff)

--stepper-animation-speed

Animation duration (default: 0.3s)
