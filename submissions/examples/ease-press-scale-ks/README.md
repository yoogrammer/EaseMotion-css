# Scale-Press Click Utility

## 1. What does this do?

This adds an active state micro-interaction that scales down buttons slightly on click to mimic a physical button press.

## 2. How is it used?

Add the class to any interactive button or clickable element:

```html
<button class="press-scale-btn">Click Me</button>
<button class="press-scale-btn success">Success</button>
<button class="press-scale-btn danger">Danger</button>
```

## 3. Why is it useful?

It fits EaseMotion CSS's philosophy of offering lightweight, intuitive interactive states. It improves tactile user feedback on mobile touch and desktop click events using CSS transitions on `:active` with no Javascript weight.
