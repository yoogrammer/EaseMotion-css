CSS Variable Namespace Scoping

By wrapping the framework's core CSS variables inside a dedicated .ease-theme class and strictly prefixing them with --ease-, the EaseMotion framework can be safely embedded into legacy enterprise applications without fatally colliding with existing Bootstrap, Tailwind, or Material UI CSS variables.
