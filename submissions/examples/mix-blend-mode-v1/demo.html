Mix Blend Mode

Blending elements with their background using mix-blend-mode — no JavaScript required.

Multiply

mix-blend-mode: multiply — darkens where elements overlap

Screen

mix-blend-mode: screen — lightens where elements overlap

Overlay

mix-blend-mode: overlay — combines multiply and screen

Difference

mix-blend-mode: difference — inverts where colors differ

Exclusion

mix-blend-mode: exclusion — similar to difference but lower contrast
