# Floating Breadcrumb Trail

An animated breadcrumb navigation component built with pure HTML and CSS.

---

## Features

- Floating breadcrumb animation
- Glassmorphism design
- Gradient active state
- Hover effects
- Responsive layout
- Pure HTML & CSS

---
