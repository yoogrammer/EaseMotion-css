# Spotlight Glow Card Showcase

Issue: [#10252](https://github.com/SAPTARSHI-coder/EaseMotion-css/issues/10252)

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A premium card hover effect that creates a glowing spotlight border around cards using pure CSS. The card subtly lifts on hover while an animated gradient glow appears around the edges.

## Features

- Animated spotlight glow border
- Card lift hover animation
- Premium glassmorphism styling
- Rotating gradient glow
- Responsive layout
- Multiple showcase cards
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<article class="ease-spotlight-card">
  Content
</article>
```

## Main Classes

```css
.ease-spotlight-card
.card-icon
.card-grid
```

## Why It Fits EaseMotion CSS

- Animation-first interaction
- Human-readable class names
- Reusable card pattern
- Modern premium design
- Lightweight implementation
- Responsive and accessible
