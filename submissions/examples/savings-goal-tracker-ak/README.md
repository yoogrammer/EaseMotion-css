# Savings Goal Tracker

## What does this do?
A savings goal tracker that lets users set financial targets with circular progress visualization, milestone markers, and projected completion date based on monthly contributions.

## How is it used?
Enter a goal name, target amount, current savings, monthly contribution, and start date. The circular progress bar updates in real time with milestone indicators at 25%, 50%, 75%, and 100%.

## Why is this useful?
This helps users visualize their savings journey, track progress toward financial goals, and understand when they'll reach their target based on consistent monthly contributions.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to use the tracker.
