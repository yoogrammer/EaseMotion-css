# Neon Glow Button

A stylish, interactive button that creates a neon cyan glowing effect and scales up slightly when the user hovers over it. 

## Features
- Smooth transitions using standard CSS.
- Multi-layered `box-shadow` to create a realistic neon bloom.
- Utilizes the framework's core `.ease-btn` for baseline padding and layout.

Submitted for GSSoC '26!
