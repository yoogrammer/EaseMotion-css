# Animated Mesh Gradient Background

**What does this do?**  
Creates a full-screen animated mesh gradient background with morphing blobs that smoothly change shape and position using CSS keyframes.

**How is it used?**  
Add a `.mesh-bg` container with up to 5 blob elements (`.mesh-blob-1` through `.mesh-blob-3`) inside. The `::before` and `::after` pseudo-elements create additional blobs. Each blob has a unique animation delay and duration for organic movement.

**Why is it useful?**  
Provides a zero-dependency, visually stunning background effect for hero sections, landing pages, or login screens. Fully responsive and respects `prefers-reduced-motion`.
