# Identity & Access Management Flow Component

## What does this do?

Visualizes the authentication and authorization process from user login through identity verification and application access.

## How is it used?

```html
<div class="stage sso">
    Single Sign-On (SSO)
</div>

<div class="stage token">
    Access Token (JWT)
</div>
```

## Why is it useful?

Identity and Access Management systems are critical for securing modern applications. This component helps explain authentication workflows including SSO, MFA, token generation, and protected resource access using pure HTML and CSS.
