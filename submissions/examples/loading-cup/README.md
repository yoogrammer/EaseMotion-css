# ☕ CSS Coffee Loading Animation

A creative pure CSS loading animation featuring a stretching coffee cup with animated steam effects. The loader uses CSS keyframes and SVG paths to create a smooth, engaging coffee-themed loading experience.

## Features

- ☕ Animated coffee cup
- 🌫️ Realistic steam animation
- 🎨 Pure HTML, CSS, and SVG
- 🌙 Supports light and dark themes
- 📱 Responsive design

## Technologies Used

- HTML5
- CSS3
- SVG Animations

## Preview

A coffee cup stretches from side to side while animated steam rises, creating a unique loading screen effect.
