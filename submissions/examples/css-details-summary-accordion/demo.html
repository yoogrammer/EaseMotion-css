Zero-JS accordion with smooth CSS animations.

What is EaseMotion?

EaseMotion is a pure CSS component library providing ready-to-use animations, transitions, and layout utilities.

How do I install it?

Include easemotion.css via CDN or npm. See the docs for detailed setup instructions.

Is it accessible?

Yes. All components follow WCAG guidelines and support keyboard navigation and screen readers.

Can I customize it?

Absolutely. Every component uses CSS custom properties for easy theming and overrides.
