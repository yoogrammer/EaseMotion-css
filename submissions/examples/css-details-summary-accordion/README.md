# CSS Details/Summary Accordion Example

A pure HTML/CSS accordion using native `<details>` / `<summary>` elements with smooth open/close animations.

## Features

- Uses `<details>` / `<summary>` for zero-JS accordion behavior
- Smooth `scaleY` and opacity animation via `@keyframes`
- Multiple accordion items with independent toggle
- Custom marker styling replacing the default triangle
- Dark theme with `prefers-reduced-motion` support

## Files

- `demo.html` — Accordion FAQ layout
- `style.css` — Self-contained CSS (75 lines)
