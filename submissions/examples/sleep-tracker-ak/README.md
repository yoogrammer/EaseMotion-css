# Sleep Tracker

## What does this do?
A sleep tracker that logs bedtime and wake times, calculates sleep duration, captures quality ratings (1-5 stars), and displays weekly averages with a sleep schedule visualization.

## How is it used?
Enter your bedtime and wake time, rate the sleep quality using stars, and click "Log Tonight's Sleep". The dashboard updates with last duration, weekly averages, and a 7-day bar chart.

## Why is this useful?
This provides a simple sleep logging tool for health and wellness tracking, helping users monitor their sleep patterns and quality over time.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to use the tracker.
