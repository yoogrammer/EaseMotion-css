# Flexbox Visual Playground Component

## What does this do?

Provides a visual playground for exploring common Flexbox properties such as direction, alignment, spacing, wrapping, and justification.

## How is it used?

```html
<div class="demo center">
    <span>A</span>
    <span>B</span>
    <span>C</span>
</div>
```

## Why is it useful?

Flexbox is one of the most important CSS layout systems. This component helps developers understand how different Flexbox properties affect layout behavior through interactive visual examples built entirely with HTML and CSS.
