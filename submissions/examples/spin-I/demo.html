Usage

<!-- Basic continuous spin -->
<div class="ease-spin">🔄</div>

<!-- Different speeds -->
<div class="ease-spin-fast">Fast spin (0.5s)</div>
<div class="ease-spin-slow">Slow spin (2s)</div>

<!-- Reverse spin (counter-clockwise) -->
<div class="ease-spin-reverse">🔄</div>

<!-- Custom speed using CSS variable -->
<div class="ease-spin" style="--spin-speed: 3s;">Custom speed</div>

<!-- Hover to spin -->
<div class="ease-spin-hover">Hover to spin</div>

🔄 Continuous Rotation Features

  • 🔄 Continuous 360° rotation
  • ⚡ Linear timing for smooth spin
  • 🎮 Speed variations (fast: 0.5s, normal: 1s, slow: 2s)
  • 🔄 Reverse spin variant (counter-clockwise)
  • 🎨 Customizable speed via --spin-speed variable
  • 🖱️ Hover trigger option
  • 💫 Perfect for loaders and loading states
