# Orbiting Social Icons

## Description

Animated social icons orbiting around a central avatar or logo.

## Features

- Continuous orbital animation
- Pure CSS implementation
- Lightweight and reusable
- Perfect for profile sections

## Files

- demo.html
- style.css
- script.js

## Usage

Open `demo.html` in a browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
