# Chat Messaging Interface

## What does this do?
A chat messaging UI with sent/received message bubbles, auto-reply simulation, timestamps, and a clean messaging layout.

## How is it used?
Open demo.html in a browser. Type a message and press Enter or click send to see it appear with an automatic simulated reply.

## Why is this useful?
Demonstrates a chat interface pattern with message alignment, scroll-to-bottom behavior, and time-stamped bubbles.

## Tech Stack
- HTML
- CSS
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
