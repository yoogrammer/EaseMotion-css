# Animation: Folding FAQ Cards

A FAQ component that reveals answers using a smooth unfolding animation. The card expands vertically while the content fades into view, creating a clean and engaging reading experience.

## Features

- Pure CSS interaction
- Folding reveal animation
- FAQ-style layout
- Responsive design
- No JavaScript required
- Reduced motion friendly

## Usage

```html
<div class="faq-card">

    <div class="faq-title">
        Question
    </div>

    <div class="faq-content">
        Answer
    </div>

</div>
```

Hovering a card unfolds the answer section and reveals additional content.

## Why is it useful?

FAQ sections are common across documentation websites, product pages, and help centers. The folding animation improves readability while keeping content organized and compact.
