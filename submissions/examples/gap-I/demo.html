Usage

<!-- Gap values 1-12 -->
<div class="ease-gap-1">...</div>
<div class="ease-gap-2">...</div>
...
<div class="ease-gap-12">...</div>

<!-- Horizontal gap only -->
<div class="ease-gap-x-2">...</div>
<div class="ease-gap-x-4">...</div>

<!-- Vertical gap only -->
<div class="ease-gap-y-2">...</div>
<div class="ease-gap-y-4">...</div>

📏 Gap Spacing Values

  • 📐 ease-gap-1 to ease-gap-12 - Full gap spacing (0.25rem increments)
  • ➡️ ease-gap-x-* - Horizontal gap only
  • ⬇️ ease-gap-y-* - Vertical gap only
  • 🎯 Perfect for flex and grid layouts
  • 📱 Responsive and easy to use
