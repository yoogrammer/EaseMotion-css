# ease-glitch

## What does this do?
A pure CSS glitch animation that creates a retro digital 
distortion effect on text using keyframes and clip-path.

## How is it used?

```html
<!-- Basic glitch -->
<h1 class="glitch" data-text="Hello World">Hello World</h1>

<!-- Slow variant -->
<h1 class="glitch glitch-slow" data-text="Hello World">Hello World</h1>

<!-- Fast variant -->
<h1 class="glitch glitch-fast" data-text="Hello World">Hello World</h1>
