# ease-ping

Ping/radar scale-out loop animation. Uses cubic-bezier easing for an elastic feel. Distinct from the loader-specific `ease-loader-ping`.

## Usage

```html
<div class="ease-ping"></div>
```

## Variants

| Class | Description |
|---|---|
| `ease-ping` | Default ping/radar animation |

## Notes

- Scales up to 2x while fading out in a loop
- Great for notification badges and "find me" indicators
- Uses `cubic-bezier(0, 0, 0.2, 1)` for elastic feel
- Respects `prefers-reduced-motion`
