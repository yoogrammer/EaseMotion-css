# FAQ Accordion

## What does this do?
An FAQ accordion component with expand/collapse toggling and single-item-open behavior. Click a question to reveal its answer and hide others.

## How is it used?
Open demo.html in a browser. Click any question to expand its answer. Only one item can be open at a time.

## Why is this useful?
Demonstrates an accordion UI pattern commonly used for FAQ sections, saving vertical space while allowing users to access content on demand.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
