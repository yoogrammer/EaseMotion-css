# Animated Wave Divider

## Overview

Animated Wave Divider is a CSS-only visual effect that creates a smooth and modern transition between sections of a webpage.

It uses layered animated wave shapes moving at different speeds to produce a subtle flowing motion without requiring JavaScript.

---

## Features

- Pure CSS implementation
- No JavaScript required
- Lightweight and reusable
- Smooth infinite animation
- Easy to customize colors, speed, and height
- Ideal for landing pages, portfolios, and hero sections

---

## Folder Structure
```

animated-wave-divider/
├── demo.html
├── style.css
└── README.md

````

---

## Usage

1. Copy the HTML structure from `demo.html`.
2. Add the styles from `style.css`.
3. Customize colors, animation speed, and wave height as needed.

---

## Customization

### Change Animation Speed

```css
animation: waveMove 5s linear infinite;
````

### Change Wave Color

```css
background: rgba(79, 70, 229, 0.4);
```

### Change Divider Height

```css
.wave-divider {
  height: 180px;
}
```
