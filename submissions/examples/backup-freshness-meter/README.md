# Backup Freshness Meter

Backup Freshness Meter is a responsive recovery-readiness panel that shows how current each backup source is. It uses freshness bars to separate current, watch, and stale restore points.

## Features

- Latest restore point summary
- Backup freshness bars
- Current, delayed, and stale backup states
- Responsive layout for smaller screens

## Files

- `demo.html` contains the backup freshness markup.
- `style.css` contains the meter layout and responsive styling.
