# Ease Fade-In-Out Utility

## What does this do?
A utility class that applies a combined fade-in and fade-out animation
using keyframes, allowing developers to easily add subtle, looping
opacity transitions to elements.

## How is it used?

```html
<div class="ease-fade-in-out">Fading In & Out</div>
```

## Why is it useful?
Provides a simple, reusable looping opacity transition for highlighting
elements, notifications, or attention-grabbing UI components. Pure CSS,
no JS required, with full prefers-reduced-motion support.
