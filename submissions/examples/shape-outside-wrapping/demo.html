Shape Outside Wrapping

Text wrapping around CSS shapes with shape-outside — no JavaScript required.

Circle

Text wraps around a circular shape created with shape-outside: circle(). The shape-margin adds spacing between the text and the circle edge. This technique is useful for magazine-style layouts where text should flow around circular images or decorative elements without requiring JavaScript or complex HTML structures.

Ellipse

An elliptical shape creates a wider wrapping path. The ellipse is defined by its horizontal and vertical radii using shape-outside: ellipse(80px 120px at 50% 50%). This works well for portrait-oriented images or decorative oval elements where text should follow a curved path.

Polygon

A custom polygon shape allows wrapping text around any arbitrary shape. This example uses a diagonal cut created with shape-outside: polygon(0 0, 100% 0, 100% 100%, 0 60%). The clipped background element matches the shape using clip-path for a seamless visual effect.

Inset (Rounded Rectangle)

An inset shape creates a rounded rectangle wrapping path using shape-outside: inset(0 0 0 0 round 50%). Combined with a matching clip-path, this creates a smooth pill-shaped element that text flows around naturally.
