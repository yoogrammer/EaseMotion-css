# Font Weight Bold on Hover

Text becomes bold on hover — pure CSS.

## Features
- Smooth font-weight transition from 400 to 700
- Pure CSS, no JavaScript

## Usage
```css
.bold-text { font-weight: 400; transition: font-weight 0.2s; }
.bold-text:hover { font-weight: 700; }
```

## Browser Support
- Chrome 1+, Firefox 3.5+, Safari 3.1+

## Tech Stack
- HTML + CSS only, no JavaScript
