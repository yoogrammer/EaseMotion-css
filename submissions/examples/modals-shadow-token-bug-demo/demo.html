Undefined --ease-shadow-2xl Token

The modal container references var(--ease-shadow-2xl, ...) but --ease-shadow-2xl is never defined in core/variables.css. Only --ease-shadow-sm through --ease-shadow-xl exist.

The shadow on this modal uses --ease-shadow-2xl which is undefined.

The Bug

components/modals.css line 45 uses box-shadow: var(--ease-shadow-2xl, 0 25px 50px -12px rgba(0,0,0,0.25)) but no --ease-shadow-2xl token exists. Only --ease-shadow-sm, --ease-shadow-md, --ease-shadow-lg, and --ease-shadow-xl are defined.

The Fix (in this demo's style.css)

.ease-modal {
  box-shadow: var(--ease-shadow-xl, 0 25px 50px -12px rgba(0,0,0,0.25));
}
