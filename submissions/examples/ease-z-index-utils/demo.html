ease-z-index utilities

Four semantic z-index utility classes map to the framework's --ease-z-* design tokens. The stacked boxes below all share the same containing block; only the utility class on each box controls which layer appears on top.

.ease-z-base z-index: var(--ease-z-base) — token value: 1
.ease-z-overlay z-index: var(--ease-z-overlay) — token value: 100
.ease-z-modal z-index: var(--ease-z-modal) — token value: 200
.ease-z-toast z-index: var(--ease-z-toast) — token value: 300
  • Top — .ease-z-toast (300)
  • .ease-z-modal (200)
  • .ease-z-overlay (100)
  • Bottom — .ease-z-base (1)
