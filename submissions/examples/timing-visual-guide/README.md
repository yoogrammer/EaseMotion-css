# Timing & Easing Visual Reference Guide

## Overview
Interactive visual documentation for `ease-duration-*` and `ease-delay-*` classes in EaseMotion CSS.

## Classes Covered

### Duration Classes
| Class | Value | Use Case |
|-------|-------|----------|
| `ease-duration-fast` | 150ms | Hover effects, micro-interactions |
| `ease-duration-medium` | 300ms | Modals, card transitions |
| `ease-duration-slow` | 600ms | Page entrances, hero sections |

### Delay Classes
`ease-delay-75` · `ease-delay-100` · `ease-delay-150` · `ease-delay-200` · `ease-delay-300` · `ease-delay-500` · `ease-delay-700`

## Demo Features
- Compare duration speeds with playable animations
- Visual comparison of all delay values
- Staggered sequence example
- Best practices reference table

## Usage
Open `demo.html` in any modern browser to explore the interactive guide.

## Files
- `demo.html` - Interactive demonstration
- `style.css` - Demo styling
- `README.md` - Documentation
