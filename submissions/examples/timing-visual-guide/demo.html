💡 When to Use Each

Class Best For Example Use Case
ease-duration-fast Micro-interactions, hover states Button hover effects, tooltip fade-ins
ease-duration-medium Standard UI transitions Modal open/close, card expansions
ease-duration-slow Emphasis, page transitions Hero section entrances, loading animations
ease-delay-75 / 100 Subtle staggered effects Tooltip delays, sequential fades
ease-delay-150 / 200 / 300 Noticeable staggered sequences List item entrances, card grids
ease-delay-500 / 700 Long pauses, dramatic effects After-notification dismissals, timed reveals
🎨 Customization Tip: Override these values globally by setting CSS variables: :root {
  --ease-duration-fast: 200ms;
  --ease-duration-medium: 400ms;
  --ease-duration-slow: 800ms;
}
