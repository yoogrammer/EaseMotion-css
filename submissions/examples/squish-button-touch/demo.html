ease-squish-button Touch Device Workaround

The framework .ease-squish-button only squishes on :hover, which doesn't fire on touch devices. This example adds a @media (hover: none) workaround:

📱 Touch Device Note: On mobile/tablet, tap and hold the button below to see the squish feedback. 
@media (hover: none) {
  .ease-squish-button:active {
    animation: ease-kf-squish-button var(--ease-speed-medium) var(--ease-ease) both;
  }
}

✅ Touch Workaround Active

On touch devices: tap and hold to see squish.
On desktop: hover to see squish.
