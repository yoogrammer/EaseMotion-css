# ease-text-nowrap Utility

Prevents text from wrapping to the next line. Useful for table cells, buttons, badges, and inline elements where wrapping would break the layout.

## Usage

```html
<span class="ease-text-nowrap">This text stays on one line</span>
<td class="ease-text-nowrap">Table cell without wrap</td>
```

## Classes

| Class              | CSS Property        |
| ------------------ | ------------------- |
| `ease-text-nowrap` | white-space: nowrap |
