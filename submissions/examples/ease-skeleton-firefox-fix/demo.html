Skeleton Firefox Fix

Fixed ease-skeleton loading animation on Firefox by adding explicit background-size property to the shimmer animation.

📄 The Fix

Added explicit background-size: 200% 100% required by Firefox:

/* ❌ Before — Broken on Firefox */
.ease-skeleton {
  background: linear-gradient(
    90deg,
    #1e1e2e 25%,
    #2a2a3e 50%,
    #1e1e2e 75%
  );
  /* Missing background-size! */
  animation: skeleton-shimmer 1.5s infinite;
}

/* ✅ After — Works on all browsers */
.ease-skeleton {
  background: linear-gradient(
    90deg,
    #1e1e2e 25%,
    #2a2a3e 50%,
    #1e1e2e 75%
  );
  background-size: 200% 100%; /* ✅ Added for Firefox */
  animation: skeleton-shimmer 1.5s infinite;
}

@keyframes skeleton-shimmer {
  0%   { background-position: 200% 0; }
  100% { background-position: -200% 0; }
}
