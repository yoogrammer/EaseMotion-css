# Animated Number Counter

## What it does

Numbers count up from 0 to target when scrolled into view.

## How to use it

<div class="ease-counter" data-target="10000">0</div>

## Variants

- ease-counter-fast - 1s duration
- ease-counter-slow - 3s duration
- ease-counter-glow - Neon glow effect
- ease-counter-large - 4rem font size
- ease-counter-bold - Extra bold
- ease-counter-purple/pink - Color variants

## JavaScript Required

Add Intersection Observer to trigger counting when in view.

## Why it fits EaseMotion CSS

- Smooth counting animation
- Scroll-triggered
- Multiple style variants
