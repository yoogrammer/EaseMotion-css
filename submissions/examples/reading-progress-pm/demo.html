Reading Progress Bar

A thin bar at the top of the viewport that fills as you scroll — perfect for blog posts, documentation, and long-form content.

What is a Reading Progress Bar?

A reading progress bar is a thin visual indicator fixed at the top of the page that fills from left to right as the user scrolls down. It gives readers a quick sense of how much content remains and how far they've progressed.

This pattern is popular on blogs, documentation sites, and news articles. EaseMotion CSS now provides .ease-reading-progress as a ready-made component.

How It Works

The bar uses a ::after pseudo-element whose width is driven by the CSS custom property --ease-scroll. A lightweight scroll listener updates this value based on the ratio of window.scrollY to the total scrollable document height.

The approach uses requestAnimationFrame for smooth, performant updates and a linear CSS transition for a polished feel.

Customization

The progress bar is fully customizable through CSS custom properties:

--ease-progress-height: 3px; /* Bar thickness */ --ease-progress-color: #6c63ff; /* Bar fill color */ --ease-z-progress: 9999; /* Stack order */ --ease-scroll: 0; /* Driven by JS (0-100) */

Usage

<div class="ease-reading-progress" id="progress"></div> <script> const bar = document.querySelector('.ease-reading-progress'); window.addEventListener('scroll', () => { const scrollTop = window.scrollY; const docHeight = document.documentElement.scrollHeight - window.innerHeight; const percent = docHeight > 0 ? (scrollTop / docHeight) * 100 : 0; bar.style.setProperty('--ease-scroll', percent); }, { passive: true }); </script>
