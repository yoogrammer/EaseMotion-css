# Travel Itinerary App Template
An inspiring, organized, and adventurous landing page template for travel planning apps, itinerary builders, and digital travel diaries.

- **Design**: Uses a "Wander-Horizon" palette with wander-teal adventure headers, horizon-gold highlights, and clean card layouts designed to spark wanderlust while offering clear organizational value.
- **Use Case**: Travel planning platforms, vacation management tools, and digital guidebook apps.
