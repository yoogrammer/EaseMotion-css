# CSS Quiz / Question Card

## What does this do?
CSS quiz/question card with progress bar, multiple choice options with correct/wrong highlighting, feedback messages, and navigation buttons.

## How is it used?
```html
<div class="qz-card"><div class="qz-progress"><div class="qz-progress-bar" style="width:60%"></div></div><div class="qz-question"><h3>Question?</h3></div><label class="qz-option"><input type="radio" name="q" class="qz-radio"><span class="qz-letter">A</span>Answer</label></div>
```

## Why is it useful?
Provides quiz and assessment UI patterns without JavaScript. Multiple choice options with correct/wrong visual states via CSS classes. Progress bar tracking. Question type indicators. Previous/Next buttons. Ideal for e-learning platforms, surveys, and trivia apps.
