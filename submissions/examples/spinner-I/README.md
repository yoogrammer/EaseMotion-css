 
# Spinner Loading Component

Spinner/loader component with circular spinner, size variants, and bouncing dots variant.

## Files

- demo.html - Interactive demo
- style.css - Spinner styles
- README.md - Documentation

## Classes

- ease-spinner - Circular spinner (40px)
- ease-spinner-sm - Small spinner (20px)
- ease-spinner-lg - Large spinner (60px)
- ease-spinner-fast/slow - Speed variants
- ease-spinner-dots - Bouncing dots
- ease-spinner-dots-sm/lg - Dots size variants

## CSS Variables

- --spinner-color - Circular spinner color
- --dots-color - Bouncing dots color

## Usage

```html
<!-- Circular spinner -->
<div class="ease-spinner"></div>

<!-- Size variants -->
<div class="ease-spinner-sm"></div>
<div class="ease-spinner-lg"></div>

<!-- Color variants -->
<div class="ease-spinner" style="--spinner-color: #10b981;"></div>

<!-- Bouncing dots -->
<div class="ease-spinner-dots"></div>
