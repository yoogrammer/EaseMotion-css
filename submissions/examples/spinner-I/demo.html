Usage

<!-- Circular spinner -->
<div class="ease-spinner"></div>

<!-- Size variants -->
<div class="ease-spinner-sm"></div>
<div class="ease-spinner-lg"></div>

<!-- Color variants -->
<div class="ease-spinner" style="--spinner-color: #10b981;"></div>

<!-- Bouncing dots -->
<div class="ease-spinner-dots"></div>

<!-- Speed variants -->
<div class="ease-spinner-fast"></div>
<div class="ease-spinner-slow"></div>

⏳ Spinner Features

  • 🔄 Circular CSS spinner animation
  • 📏 Size variants (small, medium, large)
  • 🎨 Color variants via --spinner-color and --dots-color
  • ⚫ Bouncing dots variant
  • ⚡ Speed variations (fast, normal, slow)
  • 🔘 Button loading state integration
  • 💫 Pure CSS, no JavaScript
