# CSS color-mix() Dynamic Colors

Interactive demo of the CSS `color-mix()` function — blend two colors in various color spaces and ratios.

## Features

- Shows base colors and mixed results in `srgb`, `oklch`, and `hsl` color spaces
- Mix ratios of 25%, 50%, and 75%
- Responsive card grid with hover effects
- Dark theme with `prefers-reduced-motion` support

## Files

- `demo.html` — Color mixing comparison layout
- `style.css` — Self-contained CSS (90 lines)
