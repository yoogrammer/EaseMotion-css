replace
Hover animation replaces the float — the card jumps to a fixed position and stops floating.
add
Hover transform is added to the underlying animation's base value (the to keyframe at 0%), not the current live value.
accumulate
Hover transform is added to the float's current computed value — the float continues, the lift stacks on top. This is the correct fix for "hover breaks my idle animation."
