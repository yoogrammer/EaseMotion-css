# Canvas Waveform Visualizer

## What it does

Real-time audio waveform visualization using Canvas API. Generates synthetic audio and displays 6 different waveform types.

## How to use it

Canvas automatically animates. Click different wave modes to see sine, square, sawtooth, triangle, noise, and bass waveforms.

## Waveform Types

- Sine Wave - Smooth oscillating wave
- Square Wave - Digital on/off signal
- Sawtooth - Ramp up, drop down
- Triangle - Linear up and down
- White Noise - Random signal
- Bass Drop - Decaying low frequency
