Basic Accordion

EaseMotion CSS is an animation-first CSS framework with zero dependencies. Just link the CDN and start building beautiful animated interfaces with human-readable class names.

Simply add the CDN link: <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/easemotion-css/easemotion.min.css" /> or install via npm: npm install easemotion-css

20+ animation classes including fade-in, slide-up, zoom-in, bounce, pulse, rotate, and many more. Also includes hover effects like grow, glow, lift, and shimmer.

Usage

<!-- Basic accordion structure -->
<div class="ease-accordion">
    <div class="ease-accordion-header">
        <button class="ease-accordion-trigger">
            <span>Question title</span>
            <span class="accordion-icon">▼</span>
        </button>
    </div>
    <div class="ease-accordion-panel">
        <div class="ease-accordion-content">
            <p>Answer content here</p>
        </div>
    </div>
</div>

<!-- With color variants -->
<div class="ease-accordion ease-accordion-primary">...</div>
<div class="ease-accordion ease-accordion-success">...</div>

⚡ Accordion Features

  • 📦 Smooth CSS Grid transition (grid-template-rows: 0fr → 1fr)
  • 🎯 No JavaScript required — pure CSS toggle
  • 📏 Dynamic height support (works with any content size)
  • ✨ Fade-in animation for nested content
  • 🎨 Multiple color variants (primary, success, warning)
  • 🔄 Nested accordion support
