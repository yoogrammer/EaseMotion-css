# Skeleton Loader

1. **What does this do?**
   A shimmer-animated placeholder component that mimics content layout while data is loading.

2. **How is it used?**
   ```html
   <div class="skeleton-card">
     <div class="skeleton-circle"></div>
     <div class="skeleton-text"></div>
     <div class="skeleton-text skeleton-text-short"></div>
     <div class="skeleton-block"></div>
   </div>
   ```

3. **Why is it useful?**
   Skeleton loaders are animation-first by nature — the shimmer sweep IS the component. Fits EaseMotion CSS's identity perfectly: composable, readable, zero JavaScript.
