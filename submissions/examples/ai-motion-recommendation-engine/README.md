# AI Motion Recommendation Engine

### 1. What does this do?
This is an intelligent frontend motion recommendation dashboard designed for EaseMotion CSS. It helps developers immediately identify the best animation combinations (entrances, loops, hovers) based on their specific UI goals (e.g. Hero banners, data tables, modals, onboarding). It features an interactive combination builder sandbox, a real-time accessibility and performance score estimator, live previews of animations side-by-side or combined, prefers-reduced-motion simulation, and instant code template generators.

### 2. How is it used?
Open `demo.html` directly in any modern web browser to access the interactive dashboard:
- **UI Goal Recommendation**: Select standard goals (like "Cinematic Hero Page" or "Fast Row Hover") to let the engine recommend the most suitable EaseMotion CSS animation combo.
- **Combination Sandbox**: Manually mix and match Entrance, Looping, and Hover interaction classes.
- **Live Scoring Indicator**: Observe changes in the visual gauges representing Smoothness, Accessibility, Cinematic, and Performance metrics as you adjust timings and combinations.
- **Preview Mockup**: Click "Trigger Preview" to watch the selected animations apply dynamically to a mockup web card and header title.
- **Reduced Motion Simulation**: Enable the "Reduced Motion Preference Simulator" to test the accessibility-safe animation fallbacks.
- **HTML/CSS Snippet Panel**: Inspect code templates and click "Copy" to copy them directly.

### 3. Why is it useful?
It bridges the gap between raw CSS framework classes and practical engineering decisions. Instead of guessing which duration or timing-function fits a design, developers can query their UI goals, instantly preview animations, analyze performance/accessibility impacts, and copy high-quality dogfooded implementation snippets.
