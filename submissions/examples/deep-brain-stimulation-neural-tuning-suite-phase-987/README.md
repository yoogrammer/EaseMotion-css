# Deep‑Brain Stimulation Neural Tuning Suite Showcase

A responsive, interactive medical dashboard UI for a deep‑brain stimulation device. Built with **EaseMotion CSS** utility classes and minimal custom CSS. Zero JavaScript — all interactions are native HTML (range sliders, hover effects).

## Sections
- **Header** — navigation with hover effect
- **Patient Metrics** — ID, stimulus frequency, pulse width, battery life
- **Neural Activity Waveform** — placeholder visualization with live status badge
- **Control Panel** — amplitude, frequency, pulse width sliders + apply button
- **Event Log** — recent stimulation and system events

## EaseMotion Classes Showcased
- **Layout:** `ease-container`, `ease-flex`, `ease-grid`, `ease-gap-*`, `ease-justify-between`, `ease-items-center`, `ease-min-h-screen`, `ease-py-*`, `ease-px-*`, `ease-mb-*`, `ease-mt-*`, `ease-h-64`, `ease-lg-col-span-2`
- **Background:** `ease-bg-gray-900`, `ease-bg-gray-800`, `ease-bg-teal-900`
- **Typography:** `ease-text-*`, `ease-font-bold`, `ease-font-semibold`, `ease-text-gray-*`, `ease-text-teal-*`, `ease-text-green-*`, `ease-text-yellow-*`, `ease-text-white`
- **Components:** `ease-card`, `ease-btn`, `ease-btn-primary`, `ease-badge`
- **Hover Effects:** `ease-hover-scale-105`, `ease-hover-text-teal-300`
- **Animations:** `ease-fade-in`, `ease-delay-*`, `ease-transition`
- **Utilities:** `ease-rounded-lg`, `ease-w-full`, `ease-block`, `ease-uppercase`, `ease-space-y-4`

## How to Use
1. Open `demo.html` in any modern browser.
2. Adjust the sliders and observe the dashboard.
3. Copy the HTML/CSS into your project for a medical device monitoring template.
