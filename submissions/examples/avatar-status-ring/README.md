# Avatar Status Ring

A CSS-only profile avatar component with a rotating status ring to indicate activity or online presence.

## Features

- Pure CSS
- Infinite rotation animation
- Reduced-motion support
- Responsive layout
- No JavaScript required

## Usage

```html
<div class="avatar-wrapper">
  <div class="status-ring"></div>

  <img
    src="avatar.jpg"
    alt="Profile Avatar"
    class="avatar"
  >
</div>
```

The rotating ring provides subtle motion around the avatar while maintaining a clean profile card appearance.
