# Animation: Expandable File Explorer

A file explorer component inspired by modern code editors such as VS Code. Folders expand to reveal files with smooth staggered animations.

## Features

- Pure HTML & CSS
- Native `<details>` support
- Expandable folders
- Staggered file reveal
- Developer-tool aesthetic
- Responsive layout
- Reduced motion support
- No JavaScript required

## Usage

```html
<details>
    <summary>📁 Components</summary>

    <div class="file">📄 buttons.css</div>
    <div class="file">📄 cards.css</div>
</details>
```

Opening a folder reveals files with a smooth animated entrance.

## Why is it useful?

File explorers are common in documentation platforms, dashboards, IDE-inspired interfaces, and developer tools. This component provides a lightweight and visually engaging navigation pattern while remaining fully accessible.
