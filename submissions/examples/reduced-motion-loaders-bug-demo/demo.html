Loaders Missing prefers-reduced-motion Guard

All loader animations (spin, pulse, ping, dots) are infinite loops with no @media (prefers-reduced-motion: reduce) block — a WCAG 2.1 SC 2.3.3 violation.

Bug Demo

Spin
Pulse
Ping
Dots

Under prefers-reduced-motion: reduce, all animations should stop but they continue because no guard exists in loaders.css.

The Bug

components/loaders.css uses animation on spin, pulse, ping, and dot elements but has no @media (prefers-reduced-motion: reduce) block.

The Fix (in this demo's style.css)

@media (prefers-reduced-motion: reduce) {
  .ease-loader-spin,
  .ease-loader-pulse,
  .ease-loader-ping,
  .ease-loader-dots span {
    animation: none !important;
  }
}
