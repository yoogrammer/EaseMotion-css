# User Select Utilities

CSS utility classes for controlling text selection behavior.

## Classes

- `.select-none` — `user-select: none`
- `.select-text` — `user-select: text`
- `.select-all` — `user-select: all`

## Why

Preventing text selection is useful for UI elements like buttons, icons, and draggable items to improve UX. Conversely, `.select-all` makes it easy to select entire content blocks with a single click. These utilities give developers quick control over selection behavior without custom CSS.
