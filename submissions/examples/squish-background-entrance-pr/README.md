# Squish Background Entrance Animation

## What does this do?
Animates a background component into view with a squish effect —
scaling wide and flat before springing into its natural shape.

## How is it used?

**On page load:**
```html
<div class="squish-bg-entrance">Background</div>
```

**On hover:**
```html
<div class="squish-bg-entrance-hover">Hover Me</div>
```

## Why is it useful?
Squish entrance animations add personality and energy to background
components, making UI feel alive and playful. Pure CSS, no JS,
with full prefers-reduced-motion support.
