\# ease-flip-bg



Flip background animation utility for cards and containers.



\## Classes



| Class | Description | Speed |

|-------|-------------|-------|

| `ease-flip-bg` | Normal flip background | 0.6s |

| `ease-flip-bg-fast` | Fast flip background | 0.3s |

| `ease-flip-bg-slow` | Slow flip background | 1.2s |

| `ease-flip-bg-x` | Horizontal flip | 0.6s |

| `ease-flip-bg-y` | Vertical flip | 0.6s |

| `ease-flip-bg-3d` | 3D perspective flip | 0.8s |



\## Usage



```html

<div class="ease-flip-bg">

&#x20;   Hover to flip background

</div>
