RTL Native Layout Refactor

By actively migrating all hardcoded physical CSS properties (like margin-left) to modern logical properties (like margin-inline-start), the EaseMotion framework will natively support international Right-to-Left (RTL) languages like Arabic and Hebrew out of the box, without requiring developers to maintain a second bloated RTL CSS file!
