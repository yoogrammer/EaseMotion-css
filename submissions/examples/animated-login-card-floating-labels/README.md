# Animated Login Card with Floating Labels

## What it does
A self-contained **animated login card** with floating labels for:
- **Username**
- **Password**

Behavior:
- Labels **float upward** on **focus**.
- Labels stay floated when the input has **content**.
- The card has a **hover elevation** effect.
- The **Login** button shows **press feedback** on `:active`.

## Why it fits EaseMotion CSS
- Smooth, modern motion tuned with EaseMotion-like easing curves.
- Pure **HTML + CSS** (no external libraries/frameworks).
- Includes `prefers-reduced-motion` support.
