# Flip Border Hover

A smooth flip-style hover animation for border components.

## Features

- Lightweight pure CSS animation
- Smooth flip hover effect
- Cross-browser compatible
- Reusable utility class

## Usage

```html
<div class="ease-flip-border-hover">
  Hover Me
</div>
```

## Animation Details

- Border component performs a subtle flip effect on hover
- Uses CSS keyframes with smooth easing
- Lightweight and modern interaction utility
