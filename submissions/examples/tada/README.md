# ease-tada

Celebratory tada scale+rotate animation for playful attention effects. Zero JavaScript required.

## Usage

```html
<button class="ease-tada">🎉 Submit</button>
<span class="ease-tada-hover">🏆 Hover me</span>
<div class="ease-tada-loop">⭐ New!</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-tada` | Plays once on load |
| `ease-tada-hover` | Triggers on hover |
| `ease-tada-loop` | Loops continuously |
| `ease-tada-fast` | 500ms duration |
| `ease-tada-slow` | 2s duration |
| `ease-delay-100` to `ease-delay-500` | Staggered start delays |
