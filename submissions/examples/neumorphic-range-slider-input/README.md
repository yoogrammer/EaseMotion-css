# Neumorphic Range Slider Input

Issue: [#11412](https://github.com/SAPTARSHI-coder/EaseMotion-css/issues/11412)

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A neumorphic range slider showcase featuring soft shadows, custom slider thumbs, smooth hover interactions, and multiple themed control examples using pure HTML and CSS.

## Features

- Neumorphic design
- Custom range slider styling
- Soft shadow effects
- Hover animations
- Multiple color variants
- Volume control example
- Brightness control example
- Performance control example
- Responsive layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<input
  type="range"
  class="neo-slider blue"
>
```

## Main Classes

```css
.neo-slider
.blue
.yellow
.green
.slider-card
.control-panel
```

## Why It Fits EaseMotion CSS

- Animation-first interactions
- Human-readable class names
- Reusable UI control pattern
- Modern neumorphic styling
- Lightweight implementation
