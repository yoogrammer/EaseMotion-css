# Pulse Animation Effect

Rhythmic scaling animation for notification badges, live indicators, and call-to-action buttons. Pure CSS.

## Demo

Open `demo.html` in any modern browser.

## Features

- ✅ **Pure CSS** — zero JavaScript
- ✅ **Ring effect** — expanding circles fade out
- ✅ **Double pulse** — two overlapping animations
- ✅ **Ripple effect** — outward expanding waves
- ✅ **Speed variants** — slow, normal, fast
- ✅ **Color variants** — success, warning, error, info
- ✅ **Multiple uses** — badges, avatars, buttons, cards
- ✅ **Accessible** — respects `prefers-reduced-motion`
