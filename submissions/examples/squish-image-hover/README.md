# Squish Image Hover

A smooth squish-style hover animation for image components.

## Features

- Lightweight pure CSS animation
- Smooth squish hover effect
- Cross-browser compatible
- Reusable utility class

## Usage

```html
<img
  src="image.jpg"
  class="ease-squish-image-hover"
  alt="Demo"
/>
```

## Animation Details

- Image performs a subtle squish/stretch effect on hover
- Uses CSS keyframes with smooth easing
- Lightweight and modern interaction utility
