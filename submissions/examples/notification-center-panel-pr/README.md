# Notification Center Panel

## What does this do?

A responsive notification center panel for displaying user notifications, alerts, messages, and updates.

## How is it used?

```html
<div class="notification unread">
  <div class="indicator"></div>

  <div class="content">
    <h4>New Message</h4>
    <p>You received a message from Alice.</p>
    <span>2 min ago</span>
  </div>
</div>
```

## Why is it useful?

Notification centers are commonly used in dashboards, SaaS products, admin panels, and productivity applications. This component provides a lightweight and responsive solution using only HTML and CSS.
