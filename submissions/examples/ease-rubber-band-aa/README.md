# Rubber Band Stretch Utility Component

An optimization utility built on hardware-isolated properties that stretches and compresses target bounding blocks upon pointer triggers.

## Matrix Features Included
- **Squash & Stretch Computations:** Standard physical vector simulations targeting hardware layer dimensions natively.
- **Velocity Modifiers:** Independent classes regulating duration tracking (Standard 600ms, Snap 400ms, Relaxed 1s).
- **Persistent Iteration States:** Loop toggles utilizing persistent keyframe recalculations out-of-the-box.
- **Accessibility Layers:** Standard query intercepts mapping to device `prefers-reduced-motion` safety configurations.
- **Zero Framework Intercepts:** Relies 100% on native CSS animations without layout thread locking.
