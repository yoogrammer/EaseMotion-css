Undefined --ease-color-text-muted Token

The modal body references var(--ease-color-text-muted, #4b5563) but --ease-color-text-muted is never defined in core/variables.css. The fallback #4b5563 doesn't match any palette color.

The Bug

components/modals.css line 127 uses color: var(--ease-color-text-muted, #4b5563) but --ease-color-text-muted is not defined. The existing tokens are --ease-color-text and --ease-color-muted.

The Fix (in this demo's style.css)

.ease-modal-body {
  color: var(--ease-color-muted, #64748b);
}
