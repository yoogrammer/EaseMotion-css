Bug: hardcoded 0.7s reveal duration

The reveal transition uses 0.7s directly instead of a CSS variable. Scroll down.

Revealed with 0.7s hardcoded

Source

.ease-reveal {
  transition:
    opacity 0.7s ...,
    transform 0.7s ...;     /* hardcoded 0.7s */
}

Fix

.ease-reveal {
  transition:
    opacity var(--ease-reveal-duration, 0.7s) ...,
    transform var(--ease-reveal-duration, 0.7s) ...;
}
