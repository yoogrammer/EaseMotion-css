# ease-text-fill-none

Makes text fill transparent (companion to `ease-text-stroke`). Essential for creating outlined/hollow text effects.

## Usage

```html
<h1 class="ease-text-stroke-2 ease-text-fill-none">Outlined text</h1>
```

## Variants

| Class | Description |
|---|---|
| `ease-text-fill-none` | Transparent text fill |

## Notes

- Pairs with `ease-text-stroke-*` for outlined text
- Also sets `color: transparent` for broader browser support
- Works on any text element
