FAQ

Is EaseMotion CSS free to use?

Yes, the core library is free and open source under MIT license.

Does it require JavaScript?

No, most components are pure CSS. A few advanced examples use minimal JS.

Can I customize the colors?

Yes, all components use CSS custom properties for easy theming.
