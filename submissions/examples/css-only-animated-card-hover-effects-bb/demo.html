CSS-Only Animated Card Hover Effects

A collection of 10 distinct card hover effects. Pure CSS, no JavaScript.

lift
Lift Effect
Card lifts up with enhanced shadow on hover.
scale
Scale Effect
Card scales up slightly on hover.
glow
Glow Effect
Purple glow ring appears around the card.
border-slide
Border Slide
Gradient border fades in on hover.
gradient-overlay
Gradient Overlay
Purple gradient overlay covers the card on hover.
shimmer
Shimmer Effect
Light shimmer sweeps across the card on hover.
rotate
Rotate Effect
Card rotates slightly and lifts on hover.
shadow-expand
Shadow Expand
Shadow grows larger and deeper on hover.
border-color
Border Color
Border color transitions to purple on hover.
bg-shift
Background Shift
Background color shifts to light purple on hover.
All effects use CSS transition for smooth interpolation. Each card is focusable via tabindex="0" and includes :focus-visible styles for keyboard navigation.
