# ease-mask-reveal

Horizontal mask reveal animation using `clip-path: inset()` for a left-to-right content reveal effect.

## Usage

```html
<div class="ease-mask-reveal">Reveals from left to right</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-mask-reveal` | Horizontal left-to-right reveal |
| `ease-mask-reveal-y` | Vertical top-to-bottom reveal |

## Notes

- Uses `clip-path: inset()` for the reveal
- Different from circle-based `ease-mask-reveal-circle`
- Great for text reveals, image unveils, and section entrances
- Respects `prefers-reduced-motion`
