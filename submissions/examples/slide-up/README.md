# ease-slide-up

Entrance animation that slides an element up while fading in. The most requested direction variant for scroll-reveal and staggered entries.

## Usage

```html
<div class="ease-slide-up">Slides in from below</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-slide-up` | Default slide up (24px) |
| `ease-slide-up-sm` | Subtle slide (12px) |
| `ease-slide-up-lg` | Strong slide (48px) |

## Notes

- Combine with `ease-delay-*` for staggered reveals
- Works with any element type
- Respects `prefers-reduced-motion`
