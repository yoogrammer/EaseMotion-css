Modal Size Modifier Token Fix

.ease-modal-sm and .ease-modal-lg now set --ease-modal-max-width instead of hardcoding max-width. This means .ease-modal drives the actual dimension via its token.

Default Modal

max-width driven by --ease-modal-max-width default (500px)

Small Modal

.ease-modal-sm sets --ease-modal-max-width: 400px

Large Modal

.ease-modal-lg sets --ease-modal-max-width: 700px

Overridden Small Modal

.ease-modal-sm with --ease-modal-max-width: 300px inline override
