# ease-notification-badge

A CSS-only animated notification badge component for EaseMotion CSS.

## Usage

```html
<!-- Count badge -->
<div class="ease-badge-wrapper">
  <span>🔔</span>
  <span class="ease-badge ease-badge--danger">3</span>
</div>

<!-- Pulsing dot badge -->
<div class="ease-badge-wrapper">
  <span>👤</span>
  <span class="ease-badge ease-badge--dot ease-badge--success"></span>
</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-badge--danger` | Red badge |
| `ease-badge--success` | Green badge |
| `ease-badge--warning` | Yellow badge |
| `ease-badge--info` | Blue badge |
| `ease-badge--dot` | Pulsing dot (no text) |
