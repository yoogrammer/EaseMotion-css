# CSS Masonry Grid

## What does this do?
A Pinterest-style masonry grid layout using CSS columns, with various sized items and hover effects.

## How is it used?
Add the masonry markup to any page:

    <div class="masonry">
      <div class="masonry-item" style="height: 200px;">Content</div>
    </div>

## Why is it useful?
Creates visually appealing, responsive masonry layouts without JavaScript, perfect for galleries, portfolios, and pinboards.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
