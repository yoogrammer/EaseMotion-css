# Animated Landing Hero Showcase

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A modern animated landing page hero section featuring staggered typography reveals, glowing floating particles, a custom grid background, interactive call-to-action buttons, and responsive design.

## Features

- Staggered typography animations
- Animated hero reveal effects
- Custom CSS grid background
- Floating glowing particles
- Interactive CTA buttons
- Gradient text effects
- Statistics section
- Responsive layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<section class="hero">
  ...
</section>
```

## Main Classes

```css
.hero
.hero-title
.hero-actions
.hero-stats
.grid-bg
.particle
.btn
```

## Why It Fits EaseMotion CSS

- Animation-first design
- Human-readable class names
- Reusable landing page component
- Modern visual presentation
- Lightweight implementation
- Responsive and accessible
