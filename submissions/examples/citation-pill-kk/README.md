# Citation Pill

1. **What does this do?**  
   It creates a compact inline citation pill or reference tag for article, documentation, and editorial-style interfaces.

2. **How is it used?**  
   Place a citation pill inline with text to represent references or notes:

   ```html
   <p class="article-copy">
     Motion systems should be readable and reusable
     <a href="#" class="citation-pill citation-pill-primary">[01]</a>
   </p>
   ```

3. **Why is it useful?**  
   It adds a small, readable, motion-forward content component that pairs naturally with long-form text, documentation pages, research summaries, and product explainers.
