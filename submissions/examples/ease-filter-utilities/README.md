# Ease Filter Utilities

A collection of lightweight CSS filter utility classes for blur, brightness, grayscale, and sepia effects.

## Features

- Pure CSS filter utilities
- Lightweight and reusable
- Cross-browser compatible
- Beginner-friendly implementation

## Included Utilities

- `ease-blur-sm`
- `ease-blur`
- `ease-blur-lg`
- `ease-brightness-50`
- `ease-brightness-150`
- `ease-grayscale`
- `ease-sepia`

## Usage

```html
<img src="image.jpg" class="ease-blur" />

<img src="image.jpg" class="ease-grayscale" />
```

## Utility Details

- Blur utilities apply different blur strengths
- Brightness utilities control image brightness
- Grayscale and sepia add color filter effects
