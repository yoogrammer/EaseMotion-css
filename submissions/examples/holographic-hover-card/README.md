# Holographic Hover Card

A modern, glassmorphism-inspired card component featuring a smooth, rotating holographic border that activates on hover. Built with a focus on premium easing curves to match the "EaseMotion" philosophy.

## Features

- ✨ **Holographic Border**: A rotating conic-gradient border that appears on hover.
- 🪟 **Glassmorphism Base**: Frosted glass inner background using `backdrop-filter`.
- 🎢 **Premium Easing**: Uses custom `cubic-bezier` curves for a snappy, natural feel.
- 📱 **Responsive Grid**: Adapts seamlessly to any screen size.
- ♿ **Accessible**: Respects `prefers-reduced-motion` for users who prefer less animation.
- ⚡ **Pure CSS**: Zero JavaScript required.

## Usage

Simply copy the HTML structure and the CSS classes into your project.

### HTML

```html
<div class="holo-card">
    <div class="card-content">
        <div class="card-icon">🚀</div>
        <h2>Your Title</h2>
        <p>Your description goes here.</p>
    </div>
</div>
