# ✨ Reveal on Scroll

## What does this do?

Creates a smooth fade-in and upward entrance animation for elements using pure CSS.

---

## How is it used?

```html
<div class="em-reveal-scroll">
  Content
</div>
```

---

## Why is it useful?

Reveal-on-scroll style animations help guide user attention, improve visual hierarchy, and make interfaces feel more engaging while remaining lightweight and JavaScript-free.
