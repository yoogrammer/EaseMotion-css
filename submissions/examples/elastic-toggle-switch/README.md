# Elastic Toggle Switch

**What does this do?**
A pure CSS toggle switch component that uses a custom `cubic-bezier` transition to create a satisfying, bouncy, and elastic sliding effect when toggled on or off.

**How is it used?**
```html
<label class="elastic-toggle">
  <input type="checkbox" class="elastic-toggle-input">
  <div class="elastic-toggle-track">
    <div class="elastic-toggle-knob"></div>
  </div>
</label>
```

**Why is it useful?**
This fulfills the "Form components (inputs, checkboxes, toggles)" item on the project roadmap. It provides a common interactive form element with premium micro-interactions (bouncing, squishing on hover, smooth color cross-fade) without requiring any JavaScript.
