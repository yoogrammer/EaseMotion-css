ease-object-fit & object-position utilities

Control how replaced elements (images, video) resize and align inside fixed containers. All placeholders below use inline SVG data URIs — no external image files.

Object fit

Same 400×300 SVG inside a 160px-tall frame. Compare how each object-fit value behaves.

.ease-object-cover Fills frame, crops overflow
.ease-object-contain Entire image visible, letterboxed
.ease-object-fill Stretches to fill frame
.ease-object-none Original size, clipped
.ease-object-scale-down None or contain, whichever is smaller
