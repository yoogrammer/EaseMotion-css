ease-contain utilities

Isolate element rendering for performance optimization.

ease-contain-layout-za

Isolates layout — internal layout changes don't affect the outside. Use for widgets, cards, and independent sections.

ease-contain-paint-za

Clips paint to the element's bounds — nothing visually overflows. Use for clipped containers and scrollable regions.

ease-contain-strict-za

Applies all containment types (layout, style, paint, size). Maximum isolation — the element has no effect on anything outside its bounds.

ease-contain-content-za

Applies layout, style, and paint (but not size). Like strict but without size containment — good for elements with dynamic content height.

ease-contain-size-za

Isolates size — the element's size can be computed without checking children. Must set explicit width/height for best results.
