 
# Blur Icon Hover Animation

Smooth blur transition for icons — blurs and regains clarity on hover.

## Files

- demo.html - Interactive demo
- style.css - Blur animation styles
- README.md - Documentation

## Classes

- ease-blur-icon - Normal blur (2px)
- ease-blur-icon-subtle/strong - Intensity variants
- ease-blur-icon-fast/slow - Speed variants
- ease-blur-icon-bounce/elastic - Easing variants

## Usage

```html
<div class="ease-blur-icon">🎨</div>

<!-- Different intensities -->
<div class="ease-blur-icon-subtle">🌊</div>
<div class="ease-blur-icon-strong">🔥</div>

<!-- On buttons -->
<button class="ease-blur-icon">❤️ Like</button>
