# Microservices Dependency Visualization Component

## What does this do?

Displays dependency relationships between microservices, infrastructure components, and supporting services in a modern architecture diagram.

## How is it used?

```html
<div class="node auth">
    Auth Service
</div>

<div class="node database">
    Database
</div>
```

## Why is it useful?

Microservices architectures often involve complex dependency chains between services, databases, caches, and supporting systems. This component provides a clean visual representation of those relationships for documentation, architecture reviews, technical portfolios, and educational content using pure HTML and CSS.
