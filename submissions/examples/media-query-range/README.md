# Media Query Range Syntax

Demonstrates modern `@media (400px <= width <= 800px)` range syntax compared to classic `min-width` / `max-width` approach, with live viewport tracking and responsive cards.

## Features

- Live viewport width indicator with range bar visualization
- Side-by-side syntax comparison (classic vs modern)
- Cards that respond to different range conditions
- Reference table of all range operators

## Usage

Open `demo.html` and resize the browser window. Observe which cards activate at different viewport widths. The range marker tracks your current viewport position.
