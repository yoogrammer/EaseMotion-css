# Perspective & Perspective-Origin

Demonstrates how `perspective` and `perspective-origin` affect the vanishing point of 3D transforms. A rotating 3D cube responds to nine different origin positions via interactive buttons.

## Features

- 3D cube with labelled faces (F, B, R, L, T, D) rotating continuously.
- Nine interactive buttons to change `perspective-origin` (top-left through bottom-right).
- Perspective value comparison: 200px, 500px, 1000px, and none.
- Hover tilt effect on perspective tiles.

## Topics Covered

- `perspective` property with different distance values
- `perspective-origin` keyword pairs
- `transform-style: preserve-3d` for nested 3D children
- `backface-visibility` and 3D cube construction
