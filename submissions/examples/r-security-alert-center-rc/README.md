# Security Alert Notification Center Showcase

## What does this do?

Provides a modern security alert center interface for monitoring account-security events and notifications.

## Features

- Security alert cards
- Severity badges
- Security event timeline
- Responsive layout
- Notification-focused UI patterns
- Smooth entrance animations

## How is it used?

Open `demo.html` directly in a browser.

## Why is it useful?

It demonstrates a common security-notification interface used in modern applications while showcasing alert management and timeline animations.
