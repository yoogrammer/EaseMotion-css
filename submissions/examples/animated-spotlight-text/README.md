# Animated Spotlight Text

## Description
A cinematic heading enhancement style rule that overlays text elements with a linear moving gradient mask sweep, simulating a dynamic spotlight tracking beam across individual letter patterns.

## Features
- **5 Configuration Options Included**: Supports base defaults, bright glow shaders, sapphire space arrays, and luxury gold.
- **Stagger Support**: Pre-calculated animation index timings to chain delays across multi-word headers smoothly.
- **Hover Interactivity**: Built-in micro-interaction event to pause keyframe play state on mouse target entry.
