## ease-breadcrumb-animated-separator

**What does this do?**  
Adds a breadcrumb navigation component with animated separator icons that rotate and scale on hover.

**How is it used?**
<ol class="breadcrumb">
  <li><a href="#">Home</a><span class="separator">›</span></li>
  <li><a href="#">Category</a><span class="separator">›</span></li>
  <li><a href="#">Page</a></li>
</ol>

**Why is it useful?**  
Enhances navigation UX with subtle micro-animations on the separator, 
keeping interaction feedback smooth and consistent with EaseMotion CSS's animation philosophy.
