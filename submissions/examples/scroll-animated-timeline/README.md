# Scroll Animated Timeline

## What does this do?
A vertical timeline where each entry fades and slides in as the user scrolls.

## How is it used?
Structure items inside `.timeline`:

    <div class="timeline">
      <div class="timeline-item">
        <div class="timeline-dot"></div>
        <div class="timeline-content">...</div>
      </div>
    </div>

## Why is it useful?
Ideal for company history, project roadmaps, and storytelling sections.

## Tech Stack
- HTML
- CSS
- Vanilla JS (IntersectionObserver)

## Preview
Open demo.html directly in your browser to see the effect.
