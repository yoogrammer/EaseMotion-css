# Animated Status Badge

## What it does

This component presents a set of animated status badges for dashboard states such as Online, Offline, Busy, Away, Live, Recording, Syncing, and Updating. Each badge includes subtle motion effects and a polished SaaS visual treatment, making it easy to display realtime status indicators in an admin interface.

## How to use it

1. Open `demo.html` directly in the browser.
2. Link `style.css` from the same directory.
3. Add the badge HTML structure inside your dashboard layout.

Example:

```html
<link rel="stylesheet" href="style.css">
<span class="status-badge badge-online">Online</span>
<span class="status-badge badge-live">Live</span>
<span class="status-badge badge-syncing">Syncing</span>
```
