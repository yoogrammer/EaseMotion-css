# Animation Combo Builder

## What does this do?
A fully interactive tool where developers pick entrance, hover, looping, delay, and duration classes from dropdowns, preview the combination live on a card/button/badge/box, and copy the final class string or full HTML snippet in one click.

The page `animation-combo.html` already exists in the repo nav but is empty — this fills it with a real working tool.

## Features
- 5 pickers — entrance, hover, looping, delay, duration
- 4 preview element types — card, button, badge, box
- Live preview updates instantly as you pick
- Replay button to re-trigger entrance animations
- One-click copy of class string
- One-click copy of full HTML snippet
- Class breakdown panel — shows each selected class with its description
- Dark/light mode with localStorage persistence
- Fully responsive

## How to use it
Open `demo.html` directly in a browser — no server needed. Select classes from the dropdowns on the left, see the preview update live on the right, copy and use.

## Why it fits EaseMotion CSS
Right now developers have to guess which classes work well together. The combo builder removes that friction entirely — see it live, tweak it, copy it. It's the most useful developer tool the framework could have and the page is already in the nav waiting to be filled. Directly serves the human-readable, frictionless philosophy.
