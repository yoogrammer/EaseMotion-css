# Calendar Scheduler

## What does this do?
A monthly calendar with day selection, event add/delete per day, month navigation, and localStorage persistence.

## How is it used?
Open demo.html in a browser. Navigate months, click a day to select it, add events with the input field, delete events with the &times; button.

## Why is this useful?
Demonstrates a full calendar widget with event management, date navigation, and persistent storage.

## Tech Stack
- HTML
- CSS
- Vanilla JavaScript (localStorage)

## Preview
Open demo.html directly in your browser.
