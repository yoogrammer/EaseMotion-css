Animation Specificity Fix

High-specificity animation classes prevent inline style="" overrides. Fix: expose timing as CSS custom properties so users can override with style="--anim-duration: 1.5s" — no specificity wars.

Default (0.5s)
--anim-duration not set
Slow (1.2s)
style="--anim-duration:1.2s"
Fast (0.2s)
style="--anim-duration:0.2s"
Inline style sets the CSS var — not the property directly
