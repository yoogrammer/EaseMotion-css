# Team Profile Cards

## What does this do?

A responsive team profile card component for displaying team members, their roles, and social profiles.

## How is it used?

```html
<div class="team-card">
  <div class="avatar">A</div>
  <h3>Alice Johnson</h3>
  <p class="role">Frontend Developer</p>
</div>
```

## Why is it useful?

Team sections are common on company websites, agency portfolios, startup landing pages, and personal portfolios. This component provides a clean and responsive solution using only HTML and CSS.
