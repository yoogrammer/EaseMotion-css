Python Code Block

Python 
def calculate_sum(numbers):
    total = 0
    for num in numbers:
        total += num
    return total

result = calculate_sum([1, 2, 3, 4, 5])
print(f"Sum: {result}")

Usage

<!-- Basic code block -->
<div class="ease-code-block">
    <div class="code-header">
        <span class="code-lang">HTML</span>
        <button class="copy-btn">📋 Copy</button>
    </div>
    <pre><code>Your code here</code></pre>
</div>

<!-- With line numbers -->
<div class="ease-code-block ease-code-block-lines">...</div>

<!-- Rounded variant -->
<div class="ease-code-block ease-code-block-rounded">...</div>

<!-- Dark theme -->
<div class="ease-code-block ease-code-block-dark">...</div>

⚡ Code Block Features

  • 💻 Syntax highlighting friendly styling
  • 📋 One-click copy button with visual feedback
  • 🎨 Multiple language support (HTML, CSS, JS, Python, JSON, bash)
  • 📏 Line numbers variant
  • 🎯 Rounded corners variant
  • 🌙 Dark theme variant
  • ✨ Copy success notification
