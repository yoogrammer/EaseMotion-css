CSS Grid Architecture Refactor

Notice how the modern CSS Grid implementation perfectly aligns and stretches the elements across the 2D space. By using display: grid, we entirely eliminate the need for hacky flex: 1 1 200px mathematics that often break when content wraps unevenly or when an odd number of items are present.
