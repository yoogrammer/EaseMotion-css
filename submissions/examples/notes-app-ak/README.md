# Notes App

## What
A notes app with create, edit, and delete functionality, search filtering, and localStorage persistence for all notes.

## How
Uses JavaScript array to manage notes and localStorage for persistence. Search filters notes by title and body in real-time. Each note gets a unique timestamp-based ID for edit/delete operations.

## Why
Demonstrates a complete CRUD application with client-side persistence, useful for quick note-taking and task management.
