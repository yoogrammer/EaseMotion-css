# Infinite Marquee Logo Strip

## Description

A continuously scrolling horizontal marquee for logos, brands, or text.

## Features

- Infinite smooth scrolling
- Pause on hover
- CSS-only implementation
- Lightweight and reusable

## Files

- demo.html
- style.css
- script.js

## Usage

Open `demo.html` directly in a browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
