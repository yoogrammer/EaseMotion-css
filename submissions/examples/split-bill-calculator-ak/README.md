# Split Bill Calculator

## What
A split bill calculator that computes the amount each person owes including tip, with adjustable tip percentage and number of people.

## How
Uses real-time JavaScript calculations: tip amount = total × tip%, total with tip = sum, per person = total / people. Input events trigger recalculation instantly.

## Why
Provides a quick, accurate way to split restaurant bills or group expenses with customizable tip percentages.
