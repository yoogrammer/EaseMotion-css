# Hover Spotlight Card

A CSS-only card hover effect that reveals a soft spotlight highlight using a radial gradient.

## Features

- Pure CSS
- Uses gradients and opacity transitions
- No JavaScript
- Responsive design
- Lightweight and reusable

## Usage

```html
<div class="spotlight-card">
  <h2>Spotlight Card</h2>
  <p>Your content here</p>
</div>
```

The effect uses a radial gradient overlay that fades in on hover, creating a subtle spotlight appearance.
