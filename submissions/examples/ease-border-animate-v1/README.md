# ease-border-animate

A smooth rotating border animation created with `conic-gradient` and pseudo-elements.

## Features

- Rotating animated border
- Pure CSS solution
- No JavaScript required
- Perfect for cards, highlights, and focus states
- Lightweight and reusable

## Usage

Add the utility class:

```html
<div class="ease-border-animate">
    Content Here
</div>
```

## Example

The animation creates a flowing border effect around the element while preserving the content area.
