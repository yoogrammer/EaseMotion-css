# Pseudo Element Effects Showcase

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A collection of creative UI effects built entirely with CSS pseudo-elements (`::before` and `::after`).

This showcase demonstrates decorative borders, ribbons, overlays, shine animations, layered frames, and hover interactions without requiring additional HTML elements.

## Features

- Animated gradient borders
- Corner ribbon badges
- Hover overlay effects
- Shine sweep animation
- Double border decorations
- Decorative image frames
- Responsive layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="border-card">
  Content
</div>
```

## Main Classes

```css
.border-card
.ribbon-card
.overlay-card
.shine-card
.double-border-card
.frame-card
```

## Pseudo Elements Demonstrated

```css
::before
::after
```
