## ease-spinner

**What does this do?**
A CSS spinner/loader component with circular and dots variants,
size options, and color customization via CSS variables.

**How is it used?**
<!-- Circular spinner -->
<div class="ease-spinner"></div>
<div class="ease-spinner ease-spinner-sm"></div>
<div class="ease-spinner ease-spinner-lg"></div>

<!-- Color variants -->
<div class="ease-spinner green"></div>
<div class="ease-spinner orange"></div>

<!-- Dots variant -->
<div class="ease-spinner-dots">
  <span></span><span></span><span></span>
</div>

**Why is it useful?**
Provides a lightweight, customizable loading indicator using pure CSS
animations and variables — consistent with EaseMotion CSS's component philosophy.
