Rotate Animation Utilities

Spin, swing, and hover-rotate animations for loading indicators, interactive icons, and engaging UI elements.

Spin

.animate-spin continuously rotates 360 degrees. Customize with speed modifiers.

🌀
.animate-spin Default (1s)
⚙️
.animate-spin .spin-fast Fast (0.4s)
🤍
.animate-spin .spin-slow Slow (2s)
🔄
.animate-spin-reverse Reverse spin

Swing

.animate-swing oscillates like a pendulum. Hover to see the effect clearly.

⏰
.animate-swing Default (15deg)
📣
.animate-swing .swing-wide Wide (30deg)
🎵
.animate-swing .swing-narrow Narrow (8deg)
👇
.animate-swing .swing-fast Fast swing

Hover Rotate

Hover over the cards to see rotation effects.

📷
Hover to rotate 45°
.hover-rotate
♻️
Full rotation on hover
.hover-rotate-full
📦
Custom 90° rotation
style="--hover-rotate-angle: 90deg"

CSS Custom Properties

--spin-speed

Default: 1s

Controls rotation speed for spin, swing, and hover animations.

--swing-angle

Default: 15deg

Controls the swing arc for pendulum oscillation.

--hover-rotate-angle

Default: 45deg

Controls rotation amount on hover for .hover-rotate.

Use Cases

🔄 Loading Spinners

Replace bulky SVG spinners with a simple CSS class. Perfect for async operations and content loading.

🌀

🎵 Music & Media

Swinging effects for music visualizers, notification bells, and media playback indicators.

🎵

📷 Interactive Icons

Hover rotations for camera icons, refresh buttons, expand/collapse toggles, and settings gear icons.

⚙️

✨ Decorative Animations

Subtle spinning effects for decorative elements, loading states, and ambient motion.

✨

Accessibility

The component respects prefers-reduced-motion — all spin, swing, and hover rotations are disabled for users who prefer reduced motion.
