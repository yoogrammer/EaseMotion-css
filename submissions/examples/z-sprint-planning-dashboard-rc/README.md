# Sprint Planning Dashboard Showcase

## What does this do?

Provides an agile sprint-planning dashboard for tracking goals, velocity, backlog items, and sprint progress.

## Features

- Sprint overview cards
- Backlog management section
- Priority indicators
- Progress tracking bar
- Activity feed
- Responsive dashboard layout

## How is it used?

Open `demo.html` directly in a browser.

## Why is it useful?

It demonstrates agile project-management patterns while showcasing dashboard animations, progress indicators, and productivity-focused UI design.
