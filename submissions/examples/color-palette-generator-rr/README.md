# Interactive Color Palette Generator Component

## What does this do?

Displays interactive color palettes and gradient previews using modern card-based layouts and CSS hover animations.

## How is it used?

```html
<div class="color-card purple">
    <span>#7C3AED</span>
</div>
```

## Why is it useful?

Color palettes are an important part of design systems and UI development. This component helps showcase color combinations, gradients, and interactive design patterns using pure HTML and CSS.
