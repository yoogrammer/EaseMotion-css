# Expandable Team Member Cards

## What does this do?

Provides responsive team profile cards with expandable content, skill tags, and social links.

## How is it used?

```html
<div class="team-card">
    <div class="card-header">
        <h2>Member Name</h2>
    </div>
</div>
```

## Why is it useful?

Team showcase sections are common in portfolios, company websites, startups, communities, and open-source projects. This component provides an animated and reusable way to present team information while keeping layouts clean and responsive.
