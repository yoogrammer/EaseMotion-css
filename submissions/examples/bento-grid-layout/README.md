# Animated Bento Grid Layout Component

A highly interactive, modern bento grid structural component template implementing performance-optimized micro-interactions.

## Features
- Asymmetrical design composition utilizing explicit CSS Grid mapping parameters.
- Fluid hover tracking depth scales utilizing hardware-accelerated translations (`translateY`).
- Dynamic layout context adaptive fallback scaling rules to flatten column structures safely across mobile screens.

## Usage
Simply style individual grids using the layout control utility anchors:
```html
<div class="bento-grid-wrapper">
  <div class="bento-card card-wide">...</div>
  <div class="bento-card card-tall">...</div>
</div>
