# Ease Zoom-In Utility

## What does this do?
A utility class that scales an element from 0.9 to 1.0 with a smooth
transition, useful for hover zoom effects on cards, images, and buttons.

## How is it used?

```html
<div class="ease-zoom-in">Hover to Zoom</div>
```

## Why is it useful?
A simple, reusable hover zoom effect that adds subtle interactivity to
cards and media elements. Pure CSS, no JS required, with full
prefers-reduced-motion support.
