Usage

<div class="ease-breadcrumb">
    <div class="ease-breadcrumb-item">
        <a href="#">Home</a>
    </div>
    <div class="ease-breadcrumb-item">
        <a href="#">Products</a>
    </div>
    <div class="ease-breadcrumb-item">
        <span>Electronics</span>
    </div>
</div>

<!-- Custom separator -->
<div class="ease-breadcrumb" style="--ease-breadcrumb-sep: '›';">
    ...
</div>

<!-- Different separators -->
<div style="--ease-breadcrumb-sep: '/';">...</div>
<div style="--ease-breadcrumb-sep: '»';">...</div>
<div style="--ease-breadcrumb-sep: '•';">...</div>

🍞 Breadcrumb Features

  • 📂 Navigation breadcrumb component
  • 🔧 Customizable separator via --ease-breadcrumb-sep
  • 🎨 Multiple separator options (chevron, slash, bullet, arrow, double chevron)
  • 🖼️ Icon support
  • 🌙 Dark theme variant
  • 📱 Responsive wrapping
