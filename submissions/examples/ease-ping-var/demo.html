ease-ping CSS Token Override

".ease-ping" hardcodes cubic-bezier(0, 0, 0.2, 1). This example shows the correct override using var(--ease-ease):

.ease-ping {
  animation: ease-kf-ping 1s var(--ease-ease) var(--ease-animation-iterations, infinite);
}
