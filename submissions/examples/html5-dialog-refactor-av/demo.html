HTML5 <dialog> Refactor

Notice how we don't need hundreds of lines of complex JavaScript to trap focus or handle the Escape key anymore! The browser handles all of that natively via the modern showModal() API.

Native HTML5 Modal

This modal automatically traps keyboard focus (try pressing Tab!) and natively closes when you press the Escape key. The background blur overlay is handled natively by the CSS ::backdrop pseudo-element, removing the need for a massive wrapper <div>.
