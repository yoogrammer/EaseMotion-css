## What does this do?

Provides `ease-list-style` utility classes for controlling list marker type and position — including disc, decimal, circle, square, none, and inside/outside position.

## How is it used?

```html
<ul class="ease-list-disc-za">
  <li>Bullet points</li>
</ul>
<ol class="ease-list-decimal-za">
  <li>Numbered items</li>
</ol>
<ul class="ease-list-none-za">
  <li>No markers</li>
</ul>
```

## Why is it useful?

EaseMotion core has no list-style utilities. These are commonly needed for resetting or customizing list marker appearance in navigation, content, and component styling.
