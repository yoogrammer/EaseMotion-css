# Perspective Scroll

## What does this do?
Cards tilt and translate at different speeds as the user scrolls, creating a 3D parallax effect.

## How is it used?
Add a `perspective-card` with a `data-speed` attribute:

    <div class="perspective-card" data-speed="0.5">Content</div>

## Why is it useful?
Adds depth and visual interest to scrollable content sections.

## Tech Stack
- HTML
- CSS
- Vanilla JS

## Preview
Open demo.html directly in your browser to see the effect.
