# Animated Checkbox – Checkmark draws itself, box bounces

> A delightful, production-ready animated checkbox component where the checkmark draws itself and the box bounces on toggle. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Self-drawing checkmark** – SVG stroke animation draws the checkmark
- ✅ **Bouncing box** – Elastic bounce animation on toggle
- ✅ **Multiple color variants** – Primary, Success, Danger, Warning, Dark
- ✅ **Keyboard accessible** – Enter/Space to toggle
- ✅ **Focus indicators** – Visible focus for keyboard navigation
- ✅ **Smooth transitions** – All interactions are animated
- ✅ **Responsive** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
- ✅ **Highly customizable** – Colors, sizes, animations
