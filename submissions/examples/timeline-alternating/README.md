# Alternating Timeline

A vertical timeline with items alternating left and right of a center line. Each item shows a date badge, title, and description. The timeline dot uses a double-ring indigo circle. On small screens the layout collapses to a single left-aligned column.

## EaseMotion CSS classes used

None required — built with plain CSS positioning.

## How to run

Open `demo.html` in a browser. Scroll through the timeline entries.

## Accessibility notes

The timeline is a semantic sequence of `<div>` elements with headings and text. The dots and line are purely visual. The layout is responsive with stacked cards on narrow screens.
