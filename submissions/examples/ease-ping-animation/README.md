# Ease Ping Animation

## What does this do?

Creates an expanding and fading circle animation anchored to an element. Commonly used for live status indicators, notification badges, and activity markers.

## How is it used?

```html
<div class="ease-ping-wrapper">
  <span class="ease-ping"></span>
  <div class="ease-dot"></div>
</div>
```

## Why is it useful?

The ping animation draws attention to important UI elements without being intrusive. It is ideal for indicating live activity, unread notifications, online presence, or recording status.
