# Animated Neon Button Component

A futuristic CSS-only neon button collection featuring glowing borders, pulse animations, smooth hover interactions, and multiple vibrant color themes.

---

## Features

- Neon glowing borders
- Continuous pulse animation
- Smooth hover lift effect
- Multi-color themes
- Gradient-like glow effects
- Fully responsive layout
- Lightweight pure HTML & CSS
- Easy to customize

---

## Included Neon Styles

### 🔵 Blue Neon
Ideal for technology websites and software products.

### 🩷 Pink Neon
Great for creative portfolios and modern landing pages.

### 🟢 Green Neon
Suitable for downloads, success actions, and gaming interfaces.

### 🟣 Purple Neon
Perfect for futuristic and cyber-themed designs.

## Files

- `demo.html` — Demonstrates neon button variants.
- `style.css` — Contains all animations, effects, and responsiveness.
