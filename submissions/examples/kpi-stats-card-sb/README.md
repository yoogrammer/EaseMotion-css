# KPI / Stats Cards Dashboard Component

A modern, responsive, and purely CSS-driven KPI / Stats Card component for building data dashboards and business overviews.

## Features

- **Pure HTML/CSS:** No JavaScript required.
- **Trend Indicators:** Pre-styled variations for positive, negative, and neutral trends.
- **Light & Dark Mode:** Adapts automatically based on `prefers-color-scheme`.
- **Responsive Grid Layout:** Beautifully scales across mobile, tablet, and desktop sizes using CSS Grid.
- **Theming:** Includes CSS variables for fine-tuning colors, shadows, and icon themes (danger, warning, success, default).

## Usage

Simply copy the HTML structure and link the `style.css` file. Ensure you wrap your cards within the `kpi-grid` container for responsive behavior.

## File Structure

- `demo.html`: Contains the example markup for the dashboard and the various stat cards.
- `style.css`: The stylesheet providing the layouts, hover effects, and dark mode configuration.
