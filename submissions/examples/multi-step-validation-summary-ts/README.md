# Multi Step Form Validation Summary

## What does this do?

Shows complete, warning, error, and current states for multi-step form sections.

## How is it used?

```html
<a class="summary-step has-error" href="#">
  <span class="step-number">3</span>
  <span>Security</span>
</a>
```

## Why is it useful?

Long forms need a clear summary of sections that require attention. This example makes that feedback scannable and responsive.
