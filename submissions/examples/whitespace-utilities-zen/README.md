# Whitespace Utilities

CSS utility classes for controlling how whitespace inside an element is handled.

## Classes

- `.whitespace-normal` — `white-space: normal`
- `.whitespace-nowrap` — `white-space: nowrap`
- `.whitespace-pre` — `white-space: pre`
- `.whitespace-pre-wrap` — `white-space: pre-wrap`
- `.whitespace-pre-line` — `white-space: pre-line`

## Why

Controlling whitespace wrapping behavior is essential for code blocks, preformatted text, truncated text that shouldn't wrap, and preserving intentional line breaks. These utilities provide consistent whitespace handling without inline styles or custom CSS.
