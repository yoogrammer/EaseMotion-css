Floating Action Button

Fixed-position circular button with speed dial variant, size options, and position modifiers.

Variants Preview (inline)

.ease-fab-sm .ease-fab (default)

Speed Dial Demo

The speed dial at the bottom-left shows action buttons on hover. The simple FAB at the bottom-right is a standalone button.

CSS Classes Reference

.ease-fab Base FAB (fixed, bottom-right, 3.5rem) .ease-fab-sm Small variant (2.5rem) .ease-fab-bottom-left Position: bottom-left .ease-fab-top-right Position: top-right .ease-fab-top-left Position: top-left .ease-fab-speed-dial Container for FAB + action buttons .ease-fab-actions Action buttons wrapper (hidden by default) .ease-fab-action Individual action button (2.5rem)

Usage

<button class="ease-fab" aria-label="Add"> <!-- plus icon --> </button> <div class="ease-fab-speed-dial"> <div class="ease-fab-actions"> <button class="ease-fab-action">Edit</button> <button class="ease-fab-action">Share</button> </div> <button class="ease-fab"> <!-- more icon --> </button> </div>
