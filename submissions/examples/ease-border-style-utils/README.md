# ease-border-dashed / dotted / solid / double

Controls the `border-style` property.

## Usage

```html
<div class="ease-border-dashed ease-border-2">Dashed border</div>
<div class="ease-border-dotted ease-border-2">Dotted border</div>
```

## Classes

| Class                | CSS Property         |
| -------------------- | -------------------- |
| `ease-border-dashed` | border-style: dashed |
| `ease-border-dotted` | border-style: dotted |
| `ease-border-solid`  | border-style: solid  |
| `ease-border-double` | border-style: double |
