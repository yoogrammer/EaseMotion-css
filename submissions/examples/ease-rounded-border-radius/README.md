# Border Radius Utilities

**What does this do?**
Provides utility classes for controlling the roundness of element corners. These utilities allow developers to quickly apply consistent border-radius values ranging from sharp corners to fully rounded pill-shaped elements.

**How is it used?**

```html
<div class="rounded-none">...</div>

<div class="rounded-sm">...</div>

<div class="rounded">...</div>

<div class="rounded-lg">...</div>

<div class="rounded-xl">...</div>

<div class="rounded-full">...</div>
```

**Why is it useful?**
Border radius is one of the most commonly adjusted design properties in modern UI development. Developers frequently need different levels of corner rounding for buttons, cards, badges, avatars, inputs, and containers.

Providing standardized radius utilities promotes visual consistency across projects while reducing repetitive CSS. The class names are intuitive, human-readable, and easy to compose with other utilities, making them a natural fit for EaseMotion CSS.

These utilities also establish a scalable foundation for future design token integration and theme customization.
