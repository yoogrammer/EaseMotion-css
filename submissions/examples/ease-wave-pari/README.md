## ease-wave

**What does this do?**
Creates a cascading wave animation across child elements (text or icons)
using nth-child animation-delay offsets.

**Why is it useful?**
Delivers a smooth cascading wave effect using pure CSS nth-child delays —
no JavaScript needed, perfect for loading states, hero text, and icon rows.
