# Animated Testimonial Carousel UI

A responsive animated testimonial carousel component built using pure HTML and CSS.

## Features

- Sliding testimonial cards
- Avatar support
- Hover interactions
- Responsive layout
- Smooth transitions and animations
- Pure CSS implementation

## Usage

Open `demo.html` in a browser to view the animated testimonial carousel UI.

## Files

- `demo.html` → Testimonial carousel structure
- `style.css` → Carousel animations and styles
- `README.md` → Documentation
