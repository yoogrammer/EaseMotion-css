# Image Gallery Lightbox Overlay

A thumbnail grid where clicking an image opens a full-screen dark overlay (lightbox) showing the image at full size. Includes previous/next navigation, close button, a caption, and click-outside-to-close behavior. Uses gradient placeholders for the demo.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser. Click a thumbnail to open the lightbox. Navigate with the arrows or close with ✕.

## Accessibility notes

The close button and nav arrows are `<button>` elements. The lightbox overlay can be dismissed by clicking outside the image. Reduced motion disables the thumb hover scale.
