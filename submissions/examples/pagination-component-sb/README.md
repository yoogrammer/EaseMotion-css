# CSS-only Pagination Component

A fully CSS-only, lightweight pagination component for navigating between pages.

This feature includes:

- Previous and next buttons
- Numbered page links
- Visually distinct active/current page state using `aria-current="page"`
- Disabled state for unavailable previous/next actions
- Smooth hover and keyboard focus-visible styles
- An ellipsis item for long page ranges
- A compact, mobile-friendly layout
- Dark mode compatibility

## Why is this useful for EaseMotion CSS?

Pagination is a core navigation pattern for any content-heavy interface, including blogs, tables, documentation sites, product listings, and dashboards.

Adding this component ensures EaseMotion CSS can handle complex navigation structures right out of the box, keeping with its philosophy of zero-dependency, animation-first, and highly readable CSS implementations.
