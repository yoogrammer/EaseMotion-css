Usage Tips

Combine Animations

Mix different text animations in the same page for visual variety. Use timing delays to create sequences.

Performance Tip

Text animations are lightweight but avoid animating entire paragraphs. Best results on headings and key phrases.

Accessibility First

All animations respect prefers-reduced-motion. Users with motion sensitivity will see static text.

Mobile Considerations

Test on mobile devices. Reduce animation duration on smaller screens for better perceived performance.
