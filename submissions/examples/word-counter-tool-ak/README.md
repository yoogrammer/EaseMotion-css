## What does this do?

This component provides a word counter tool with real-time counting of words, characters, sentences, and paragraphs as the user types or pastes text into a textarea.

## How is it used?

Include the `style.css` and the HTML structure. The JavaScript attaches an `input` event listener to the textarea and updates the four statistic counters on every keystroke.

## Why is this useful?

A real-time word counter is essential for writers, students, and content creators who need to stay within length limits for articles, essays, or social media posts.

## Tech Stack

- HTML
- CSS
- JavaScript (vanilla)
