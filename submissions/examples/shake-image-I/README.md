 
# Shake Image Utility

Smooth shake animation for images — perfect for error states and interactive feedback.

## Files

- demo.html - Interactive demo
- style.css - Shake animation styles
- README.md - Documentation

## Classes

- ease-shake-image - Normal shake
- ease-shake-image-subtle/strong - Intensity variants
- ease-shake-image-horizontal/vertical/diagonal - Direction variants
- ease-shake-image-fast/slow - Speed variants

## Usage

```html
<img src="image.jpg" class="ease-shake-image" alt="Image">

<!-- Different intensities -->
<img class="ease-shake-image-subtle" src="image.jpg">
<img class="ease-shake-image-strong" src="image.jpg">
