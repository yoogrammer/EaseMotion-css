# Coming Soon Page

Launch countdown / coming soon section with headline, description, countdown timer, email signup form, and social links.

## Usage

```html
<div class="ease-coming-soon-az">
  <div class="ease-coming-soon-icon-az">&#x1f680;</div>

  <span class="ease-coming-soon-badge-az">&#x1f514; Launching soon</span>

  <h1 class="ease-coming-soon-title-az">Something amazing is on the way</h1>

  <p class="ease-coming-soon-desc-az">We're launching soon. Be the first to know.</p>

  <div class="ease-coming-soon-countdown-az">
    <div class="ease-coming-soon-countdown-item-az">
      <span class="ease-coming-soon-countdown-value-az">14</span>
      <span class="ease-coming-soon-countdown-label-az">Days</span>
    </div>
    <span class="ease-coming-soon-countdown-sep-az">:</span>
    <!-- repeat for Hours, Minutes, Seconds -->
  </div>

  <div class="ease-coming-soon-form-az">
    <input class="ease-coming-soon-input-az" type="email" placeholder="Enter your email" />
    <button class="ease-coming-soon-btn-az">Notify Me</button>
  </div>

  <div class="ease-coming-soon-social-az">
    <a class="ease-coming-soon-social-link-az" href="#">...</a>
  </div>
</div>
```

## CSS Variables

| Variable | Default |
|----------|---------|
| `--ease-space-*` | spacing |
| `--ease-radius-md` | 8px |
| `--ease-color-primary` | #6366f1 |
| `--ease-color-primary-dark` | #4f46e5 |
| `--ease-color-success` | #10b981 |
| `--ease-color-neutral-*` | grays |
| `--ease-font-sans` | system-ui |
