# Floating Toast

A lightweight toast notification component with a smooth slide-in animation.

## Features

- Pure CSS
- Responsive layout
- No JavaScript required
- Lightweight and reusable

## Usage

```html
<div class="toast">
  <strong>Success!</strong>
  <p>Your changes have been saved.</p>
</div>
```

The component uses a simple slide-in animation and can be positioned anywhere on the screen.
