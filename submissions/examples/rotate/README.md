# ease-rotate

Infinite 360-degree rotation animation. A fundamental spinner/loader pattern distinct from the styled `ease-loader-spin`.

## Usage

```html
<div class="ease-rotate">Rotating</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-rotate` | Default rotation (1.2s, linear, infinite) |
| `ease-rotate-fast` | Faster rotation |
| `ease-rotate-slow` | Slower rotation |

## Notes

- Pure 360-degree rotation loop
- Use on icons, arrows, or any element needing spin
- Respects `prefers-reduced-motion`
