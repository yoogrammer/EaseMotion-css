# Animated Tooltips Component

## What does this do?

Creates animated tooltips with multiple positions, animation styles, and theme variations using only HTML and CSS.

## How is it used?

```html
<div class="tooltip-wrap">
  <button>Hover Me</button>
  <span class="tooltip top fade dark">
    Hello!
  </span>
</div>
```

## Why is it useful?

Tooltips are one of the most common UI patterns for providing contextual information. This component offers reusable, animation-friendly tooltip variations that align with EaseMotion CSS's animation-first philosophy while remaining lightweight and dependency-free.
