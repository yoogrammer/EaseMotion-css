# Sticky Navigation Showcase

Issue: [#8802](https://github.com/SAPTARSHI-coder/EaseMotion-css/issues/8802)

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A self-contained example demonstrating sticky navigation, smooth scrolling, active navigation states, responsive layouts, and modern section-based page structure using pure HTML and CSS.

## Features

- Sticky navigation bar
- Smooth scrolling behavior
- Active navigation link styling
- Scroll progress indicator
- Hero section
- Feature cards
- Pricing cards
- Contact section
- Glassmorphism navigation
- Responsive design
- Pure HTML and CSS

## Usage

```html
<nav class="sticky-nav">
  ...
</nav>
```

```css
.sticky-nav{
  position: sticky;
  top: 0;
}
```

## Main Classes

```css
.sticky-nav
.nav-links
.section
.hero
.feature-grid
.pricing-grid
```

## Why It Fits EaseMotion CSS

- Demonstrates practical layout techniques
- Uses human-readable class names
- Encourages reusable navigation patterns
- Showcases smooth interactions
- Lightweight and dependency-free
