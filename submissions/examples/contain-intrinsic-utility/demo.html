Contain-Intrinsic Utility

Sets the intrinsic size of elements using content-visibility: auto to prevent layout shift and scrollbar jank.

Fixed intrinsic sizes

contain-intrinsic-size-100
content truncated to 100px height
contain-intrinsic-size-200
content truncated to 200px height
contain-intrinsic-size-300
content truncated to 300px height

With content-visibility: auto (simulated laziness)

Each box below uses content-visibility: auto — the browser reserves the intrinsic size before rendering the full content.

auto intrinsic 200px
.content-visibility-auto.contain-intrinsic-size-200

This element's content is lazy-rendered. The browser reserves 200px of vertical space before the content paints.

auto intrinsic 500px
.content-visibility-auto.contain-intrinsic-size-500

Larger intrinsic size for a bigger block of lazy content. The scrollbar track accounts for 500px from the start.

Auto-size mode

The auto keyword tells the browser to measure the element once rendered, then reuse that size for future off-screen occurrences.
