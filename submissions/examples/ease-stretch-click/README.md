\# ease-stretch-click



Element stretches horizontally on click then snaps back like stretching rubber.



\## Classes



| Class | Description | ScaleX | ScaleY |

|-------|-------------|--------|--------|

| `ease-stretch-click` | Normal stretch | 1.15 | 0.9 |

| `ease-stretch-click-light` | Light stretch | 1.05 | 0.95 |

| `ease-stretch-click-strong` | Strong stretch | 1.25 | 0.85 |

| `ease-stretch-click-fast` | Fast snap (0.15s) | 1.15 | 0.9 |

| `ease-stretch-click-slow` | Slow snap (0.5s) | 1.15 | 0.9 |



\## Features



\- 🔲 Horizontal stretch on click

\- 💫 Spring-back effect with cubic-bezier

\- 📏 Multiple intensity options

\- ⚡ Speed variations



\## Usage



```html

<!-- Basic stretch -->

<button class="ease-stretch-click">Click Me</button>



<!-- Light stretch -->

<div class="ease-stretch-click-light">Light stretch</div>



<!-- Works on any element -->

<div class="ease-stretch-click">Clickable div</div>
