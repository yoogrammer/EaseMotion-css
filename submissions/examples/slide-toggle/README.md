# ease-slide-toggle

CSS-only animated toggle switch component using checkbox hack. Zero JavaScript required.

## Usage

```html
<label class="ease-toggle">
  <input type="checkbox" />
  <div class="ease-toggle-track">
    <div class="ease-toggle-thumb"></div>
  </div>
  <span class="ease-toggle-label">Toggle me</span>
</label>
```

## Variants

| Class | Description |
|---|---|
| `ease-toggle` | Default indigo toggle |
| `ease-toggle-success` | Green toggle |
| `ease-toggle-danger` | Red toggle |
| `ease-toggle-warning` | Orange toggle |
| `ease-toggle-dark` | Dark toggle |
| `ease-toggle-sm` | Small size |
| `ease-toggle-lg` | Large size |
| `ease-toggle-disabled` | Disabled state |
