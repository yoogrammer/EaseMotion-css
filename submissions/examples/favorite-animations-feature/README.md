# Favorite Animations Feature

## Description

This example demonstrates a simple favorites system for animation galleries.

Users can bookmark their preferred animations and access them later from a dedicated favorites section.

## Features

- Favorite / unfavorite animations
- Dedicated favorites section
- localStorage persistence
- Responsive card layout

## How It Works

Click the star icon on any animation card.

The selected animation is stored in localStorage and automatically restored when the page reloads.

Useful for documentation websites where developers want quick access to frequently used animations.
