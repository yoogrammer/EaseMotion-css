# Pulse Button Hover

A smooth pulse-style hover animation for button components.

## Features

- Lightweight pure CSS animation
- Smooth pulse hover effect
- Cross-browser compatible
- Reusable utility class

## Usage

```html
<button class="ease-pulse-button-hover">
  Hover Me
</button>
```

## Animation Details

- Button slightly enlarges on hover
- Smooth easing interaction
- Lightweight and responsive
