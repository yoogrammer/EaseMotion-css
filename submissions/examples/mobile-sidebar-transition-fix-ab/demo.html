Mobile Transition Test (#10177)

The mobile collapsible sidebar previously used display: none / display: block. Because toggling display in the same frame as transform causes the browser to bypass transitions, the sidebar would flash open/closed instantly without sliding on viewports under 768px.

This fix swaps display properties with visibility interpolation.

💡 Testing Tip

  1. Resize your browser window below 768px (or toggle Device Simulation in browser DevTools).
  2. Click the floating menu trigger button at the bottom-right corner.
  3. Observe the mobile sidebar slide in and out smoothly from the left edge instead of instantly flashing!
