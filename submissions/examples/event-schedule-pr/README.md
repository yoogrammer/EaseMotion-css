# Event Schedule Component

## What does this do?

A responsive event schedule component for displaying upcoming events, meetings, workshops, and activities.

## How is it used?

```html
<div class="event-card">
  <div class="event-date">
    <span class="day">12</span>
    <span class="month">AUG</span>
  </div>

  <div class="event-content">
    <h3>Event Title</h3>
    <p>Event description.</p>
    <span>10:00 AM - 12:00 PM</span>
  </div>
</div>
```

## Why is it useful?

Event schedules are widely used for conferences, workshops, schools, communities, and business events. This component provides a clean and responsive layout using only HTML and CSS.
