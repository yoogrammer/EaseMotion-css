# Password Strength Meter – Bars animate with color grading

> A production-ready password strength meter that provides real-time feedback with animated bars and color grading. Built with vanilla HTML, CSS, and JavaScript—no dependencies required.

## ✨ Features

- ✅ **Real-time strength checking** – Updates as you type
- ✅ **Animated bars** – Smooth fill animation with spring effect
- ✅ **Color grading** – Red → Yellow → Blue → Green
- ✅ **4 strength levels** – Weak, Fair, Good, Strong
- ✅ **Detailed requirements** – Shows which criteria are met
- ✅ **Password visibility toggle** – Show/hide password
- ✅ **Score display** – Shows score out of 4
- ✅ **Password generator** – Generate strong passwords
- ✅ **Responsive design** – Adapts to any screen size
- ✅ **Light/Dark mode** – Auto-adjusts to system preference
- ✅ **Zero dependencies** – No libraries or frameworks
- ✅ **Highly customizable** – Colors, thresholds, messages
