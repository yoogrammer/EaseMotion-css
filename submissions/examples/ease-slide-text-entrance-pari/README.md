## ease-slide-text-entrance

**What does this do?**
Animates text components (headings, labels, paragraphs) with a smooth slide-in entrance from any direction.

**How is it used?**
<h1 class="slide-text-left">Heading</h1>
<p class="slide-text-right">Paragraph</p>
<span class="slide-text-bottom">Label</span>
<p class="slide-text-top">Text</p>

**Why is it useful?**
Gives text elements dynamic entrance animations with a subtle overshoot bounce using pure CSS keyframes,
consistent with EaseMotion CSS's animation philosophy.
