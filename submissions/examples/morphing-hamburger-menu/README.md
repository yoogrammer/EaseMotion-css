# Morphing Hamburger Menu Icon

An animated hamburger menu icon that smoothly transforms into a close (X) icon when clicked.

## Usage
```html
<button class="menu-toggle">
  <span></span>
  <span></span>
  <span></span>
</button>
```

## Why EaseMotion CSS?
Animation-first, lightweight, meaningful motion on a common UI element, zero dependencies.
