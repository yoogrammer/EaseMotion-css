## What does this do?

Provides vertical alignment utility classes (`vertical-align`). Essential for aligning inline and inline-block elements (icons, badges, images) with surrounding text.

## How is it used?

```html
<span class="ease-align-middle-za">Vertically centered with text</span> H<span
  class="ease-align-sub-za"
  >2</span
>O
```

## Why is it useful?

EaseMotion has no vertical alignment utilities, yet they're essential whenever inline elements like icons, badges, or avatars need precise alignment with surrounding text. Without these, developers resort to fragile inline styles.
