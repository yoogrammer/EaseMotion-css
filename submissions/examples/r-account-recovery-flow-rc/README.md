# Account Recovery Flow Showcase

## What does this do?

Provides a modern account recovery interface demonstrating password reset and account recovery workflows.

## Features

- Email recovery form
- Recovery action button
- Success confirmation state
- Recovery workflow steps
- Responsive layout
- Smooth entrance animations

## How is it used?

Open `demo.html` directly in a browser.

## Why is it useful?

It demonstrates a common authentication and account-recovery UI pattern while showcasing user-friendly feedback and transition animations.
