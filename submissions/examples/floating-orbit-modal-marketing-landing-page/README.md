# Floating Orbit Modal – Marketing Landing Page

A lightweight **CSS-only Floating Orbit Modal** created for modern marketing landing pages.

## Features

- Pure HTML & CSS
- No JavaScript
- Floating Orbit entrance animation
- Responsive layout
- Keyboard accessible (`tabindex`)
- CSS Custom Properties
- Easy to customize
- Modern landing page appearance

## Folder Structure

```
floating-orbit-modal-marketing-landing-page/
│── demo.html
│── style.css
└── README.md
```

## Usage

Include the stylesheet:

```html
<link rel="stylesheet" href="style.css">
```

Use the modal:

```html
<div class="orbit-modal">
    ...
</div>
```

## CSS Variables

```css
:root{
--duration:.75s;
--orbit-distance:40px;
--modal-width:430px;
--radius:22px;
}
```
