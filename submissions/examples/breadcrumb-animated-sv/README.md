# Animated Breadcrumb Navigation

## What does this do?

Creates a breadcrumb navigation trail with animated chevron separators and hover interactions for navigation links.

## How is it used?

```html
<nav aria-label="Breadcrumb">
  <ol class="breadcrumb">
    <li><a href="#">Home</a></li>
    <li><a href="#">Components</a></li>
    <li class="active">Breadcrumb</li>
  </ol>
</nav>
```

## Why is it useful?

Breadcrumb navigation improves usability by helping users understand their current location within a website hierarchy. The animated separators and hover interactions add subtle visual feedback while maintaining accessibility.
