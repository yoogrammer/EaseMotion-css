# Fade Shadow Utility

A smooth fade-style shadow animation for utility interactions.

## Features

- Lightweight pure CSS animation
- Smooth fading shadow effect
- Cross-browser compatible
- Reusable utility class

## Usage

```html
<div class="ease-fade-shadow-utility">
  Hover Me
</div>
```

## Animation Details

- Shadow smoothly fades in and out during interaction
- Uses CSS keyframes with easing
- Lightweight and modern utility animation
