# Cinematic Reveal Card Component

1. What does this do? Creates a visually premium card containing hidden descriptions and action buttons that reveal on hover.
2. How is it used? Nest `.image-wrapper` and `.card-content` within the `.cinematic-card` wrapper.
3. Why is it useful? It demonstrates how to orchestrate multiple nested hover animations (scaling background, card translation, staggering element opacity) cleanly in CSS.
