# CSS-Only Animated Typing Indicator

A classic chat UI component showing an animated "typing..." indicator, built purely with CSS keyframes.

## Features
- ✨ **100% CSS-only**: Uses CSS `@keyframes` to animate the dots.
- 💬 **Contextual UI**: Includes a responsive chat container with mock sent and received messages to provide context.
- ⏱️ **Staggered Animation**: The dots bounce sequentially using precisely timed `animation-delay` offsets.
- 📱 **Responsive & Clean**: Adapts nicely to small screens with a sleek dark mode aesthetic.

## Files
- `demo.html`: The HTML structure of the chat interface and the typing indicator.
- `style.css`: The styling rules and the `bounce` keyframe animation that controls the scale and color of the dots.

## Usage
Simply drop the HTML structure into your messaging app UI and include the CSS. The animation loops infinitely automatically.
