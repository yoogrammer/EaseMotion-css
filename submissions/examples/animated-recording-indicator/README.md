# Animated Recording Indicator

## What does this do?

Creates a recording status indicator with a pulsing red dot animation.

## How is it used?

```html
<div class="recording-indicator">
    <span class="record-dot"></span>
    <span class="record-text">REC</span>
</div>
```

## Why is it useful?

Recording indicators are commonly used in streaming platforms, video recording tools, and online meeting applications. This component provides a simple and reusable visual cue while fitting EaseMotion CSS's animation-first philosophy.
