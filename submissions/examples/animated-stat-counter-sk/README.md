# Animated Stat Counter

## What does this do?

A responsive grid of 4 stat counters (Projects, Clients, Team Members, Satisfaction) that animate counting up on page load using CSS `@property` for integer animation with staggered delays.

## How is it used?

Open `demo.html` in a modern browser. Each stat box displays a large gradient number that counts from 0 to its target value. Hover over any stat to see a decorative underline expand beneath it. The grid adapts from 4 columns down to a single column on small screens.

## Why is it useful?

Showcases CSS `@property` custom property animation to create number counters without JavaScript. The staggered delays add visual interest, and the hover underline provides interactive feedback. Fully responsive and respects `prefers-reduced-motion`.
