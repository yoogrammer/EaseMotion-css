# Ease Dropcap Zd

## What does this do?
An animated drop cap / initial letter component with 6 style variants (simple, border, circle, gradient, outline, serif) and entrance animation — pure HTML and CSS.

## How is it used?
```html
<p class="dc-p">
  <span class="dc-cap dc-cap-simple">T</span>ext content here...
</p>
```

Variants: dc-cap-simple, dc-cap-border, dc-cap-circle, dc-cap-gradient, dc-cap-outline, dc-cap-serif

## Why is it useful?
Drop caps are a classic typographic treatment that adds visual interest to articles, blog posts, documentation, and editorial content. This component provides 6 distinct styles with smooth drop-in animation.
