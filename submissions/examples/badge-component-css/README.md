# Badge Components — CSS

Dot badges, counter badges, and icon badges built entirely with CSS. Position and color variants for every use case.

## Features

- Dot badges with color and pulse animation variants
- Counter badges positioned top-right, top-left, bottom-right, bottom-left
- Icon badges with tinted background variants
- Color variants: default, success, warning, error
- Dark theme with self-contained CSS

## Usage

```html
<!-- Dot -->
<span class="badge-dot badge-success"></span>

<!-- Counter -->
<div class="badge-container">
  <span class="badge-counter badge-top-right">3</span>
  <span class="badge-base">Label</span>
</div>

<!-- Icon -->
<span class="badge-icon badge-error">⚙</span>
```
