# ease-underline-offset Utilities

Controls `text-underline-offset` — the distance of the underline from the text.

## Usage

```html
<span class="ease-decoration-underline ease-underline-offset-4"
  >Offset underline</span
>
```

## Classes

| Class                        | CSS Property                |
| ---------------------------- | --------------------------- |
| `ease-underline-offset-auto` | text-underline-offset: auto |
| `ease-underline-offset-1`    | text-underline-offset: 2px  |
| `ease-underline-offset-2`    | text-underline-offset: 4px  |
| `ease-underline-offset-4`    | text-underline-offset: 6px  |
