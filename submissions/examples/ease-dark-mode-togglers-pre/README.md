## ease-dark-mode-togglers

**What does this do?**
A collection of 7 animated dark mode togglers with smooth
transitions for switching between light and dark themes.

**How is it used?**
<button class="sun-moon-toggle" onclick="toggle(this, 'body')"></button>
<button class="sliding-switch" onclick="toggle(this)"></button>
<button class="neon-toggle" onclick="toggle(this)"></button>

Add class `dark-on` to activate dark state.

**Variants included:**
1. Sun & Moon Toggle
2. Sliding Theme Switch
3. Material-Inspired Toggle
4. Neon Glow Toggle
5. Minimal Toggle
6. Morphing Icon Toggle
7. Floating Theme Switch

**Why is it useful?**
Provides ready-to-use animated theme togglers — one of the most
commonly used UI elements in modern web apps, consistent with
EaseMotion CSS's animation-first philosophy.
