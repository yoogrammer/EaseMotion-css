Delay utilities

Animation Delay Utilities

Reuse small delay classes to stage entrance motion without writing one-off animation-delay values for each element.

.em-delay-1 0.5s

Starts first for the lead item in a staggered group.

.em-delay-2 1s

Creates a clear second beat for supporting content.

.em-delay-3 1.5s

Lets secondary cards arrive after the primary action.

.em-delay-4 2s

Works for final callouts, badges, and helper panels.

Custom property timing

Use .em-delay-custom with --em-delay when a sequence needs a value outside the default utility scale.
