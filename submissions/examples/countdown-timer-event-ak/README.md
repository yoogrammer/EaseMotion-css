# Countdown Timer — New Year 2027

## What
A countdown timer displaying days, hours, minutes, and seconds remaining until New Year 2027.

## How
Built with JavaScript using `setInterval` to update every second. Calculates the difference between the target date and current time, then converts milliseconds into days, hours, minutes, and seconds with zero-padded display.

## Why
Provides a real-time countdown experience useful for event anticipation, deadline tracking, or celebration countdowns.
