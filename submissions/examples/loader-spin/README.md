# ease-loader-spin

Classic spinning border loader. The most widely recognized loading indicator pattern — a rotating semi-transparent bordered circle.

## Usage

```html
<div class="ease-loader-spin"></div>
```

## Variants

| Class | Description |
|---|---|
| `ease-loader-spin` | Default spinning loader |
| `ease-loader-spin-sm` | Small variant |
| `ease-loader-spin-lg` | Large variant |

## Notes

- Add inline size: `width: 40px; height: 40px;` for custom sizing
- Uses `--ease-color-primary` for the active border color
- Respects `prefers-reduced-motion`
