# ease-shimmer-sweep

Shimmer sweep overlay animation using a translucent gradient with sweeping background-position. Creates a glossy scanning effect.

## Usage

```html
<div class="ease-shimmer-sweep">Shimmer surface</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-shimmer-sweep` | Default shimmer sweep (3s loop) |

## Notes

- Sweeping gradient overlay for glossy/luxury effect
- Great for premium cards, product images, and hero sections
- Respects `prefers-reduced-motion`
