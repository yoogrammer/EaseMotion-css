# Feature Comparison Table

## What does this do?

A responsive feature comparison table that allows users to compare plans, products, or services side-by-side.

## How is it used?

```html
<table class="comparison-table">
  <thead>
    <tr>
      <th>Features</th>
      <th>Free</th>
      <th>Pro</th>
      <th>Enterprise</th>
    </tr>
  </thead>
</table>
```

## Why is it useful?

Feature comparison tables are widely used on SaaS pricing pages, product websites, subscription plans, and service comparison platforms. They help users quickly understand differences between available options.
