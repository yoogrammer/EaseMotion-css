Dark Mode Token Overrides

This card adapts automatically

No extra classes needed. Because every EaseMotion component references the four semantic tokens — --ease-color-bg, --ease-color-surface, --ease-color-text, and --ease-color-muted — flipping those four values in a single @media block is enough to theme the entire framework at once.
