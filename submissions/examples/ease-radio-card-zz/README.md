# Radio Card

## What does this do?
Radio Card component for EaseMotion CSS framework with primary, secondary, and outline style variants. Dark theme with CSS custom properties.

## How is it used?
```html
<div class=\"rd-primary\">Radio Card Primary</div>
```

## Why is it useful?
Provides a reusable radio card UI element without JavaScript. Dark theme CSS custom properties, multiple variants, and hover/active states make it suitable for modern web applications.
