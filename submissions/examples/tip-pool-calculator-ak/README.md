# Tip Pool Calculator

## What does this do?
A tip pool calculator with hours-based proportional distribution, per-person earnings breakdown, hourly rate calculation, and share percentage display.

## How is it used?
Enter total tips for the pool. Add staff members with their hours worked. Tips are distributed proportionally by hours. View each person's earnings, hourly rate, and percentage of the pool.

## Why is this useful?
This helps restaurants and service teams fairly distribute pooled tips based on hours worked, ensuring equitable compensation.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to use the calculator.
