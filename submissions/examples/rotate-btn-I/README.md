 
# Rotate Button Entrance Animation

Smooth rotate entrance animation for buttons on page load.

## Files

- demo.html - Interactive demo
- style.css - Rotate animation styles
- README.md - Documentation

## Classes

- ease-rotate-btn - 360° rotate entrance
- ease-rotate-btn-5/10/15/20/30 - Angle variants
- ease-rotate-left/right/up/down - Direction variants
- ease-rotate-btn-fast/slow - Speed variants
- ease-rotate-btn-bounce/elastic/back - Easing variants
- ease-rotate-btn-delay-1/2/3 - Staggered delays
- ease-rotate-replay - Hover to replay

## Usage

```html
<button class="ease-rotate-btn">Click Me</button>

<!-- Different angles -->
<button class="ease-rotate-btn-10">10° Rotate</button>

<!-- Staggered entrance -->
<button class="ease-rotate-btn-delay-1">First</button>
