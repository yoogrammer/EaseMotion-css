Usage

<button class="ease-rotate-btn">Click Me</button>

<!-- Different angles -->
<button class="ease-rotate-btn-10">10° Rotate</button>
<button class="ease-rotate-btn-20">20° Rotate</button>

<!-- Different directions -->
<button class="ease-rotate-left">Rotate Left</button>
<button class="ease-rotate-right">Rotate Right</button>

<!-- Different speeds -->
<button class="ease-rotate-btn-fast">Fast</button>
<button class="ease-rotate-btn-slow">Slow</button>

<!-- Staggered entrance -->
<button class="ease-rotate-btn-delay-1">First</button>
<button class="ease-rotate-btn-delay-2">Second</button>

🔄 Rotate Button Features

  • ✨ Smooth rotate entrance animation on page load
  • 🎯 Multiple rotation angles (5°, 10°, 15°, 20°, 30°)
  • ➡️ Direction variants (left, right, up, down)
  • ⚡ Speed variations (fast, normal, slow)
  • 🎭 Easing variations (bounce, elastic, back)
  • 🎨 Color variants (primary, success, danger, warning)
  • 📏 Size variants (small, normal, large)
  • ⏱️ Staggered entrance delays
  • 🖱️ Hover to replay option
