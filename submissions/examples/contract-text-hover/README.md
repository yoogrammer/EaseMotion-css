# Contract Text Hover

A smooth contract-style hover animation for text components.

## Features

- Lightweight pure CSS animation
- Smooth contraction effect
- Cross-browser compatible
- Reusable utility class

## Usage

```html
<h1 class="ease-contract-text-hover">
  Hover Over Me
</h1>
```

## Animation Details

- Text slightly contracts on hover
- Uses scale and letter-spacing animation
- Smooth easing interaction
