# Expand Text Hover

1. **What does this do?** Expands text on hover using letter-spacing, scale, or word-spacing transitions.
2. **How is it used?**
```html
   <p class="expand-text">Hover me</p>
   <p class="expand-text-scale">Hover me</p>
   <p class="expand-text-words">Hover me</p>
```
3. **Why is it useful?** Adds subtle, elegant motion to text elements with zero JavaScript — fits EaseMotion CSS's philosophy of pure CSS interactions.
