Usage

<!-- Basic typing -->
<div class="ease-typing">Your text here</div>

<!-- Different speeds -->
<div class="ease-typing-fast">Fast typing</div>
<div class="ease-typing-slow">Slow typing</div>

<!-- Hover to type -->
<div class="ease-typing-hover">Hover to type</div>

<!-- Cursor variants -->
<div class="ease-typing-blink">Blinking cursor</div>
<div class="ease-typing-no-cursor">No cursor</div>

<!-- With delay -->
<div class="ease-typing-delay-1">Appears after 1s</div>

⚡ CSS Typing Effect Features

  • ⌨️ Character-by-character typing animation
  • 💡 Blinking cursor at the end
  • ⚡ Speed variations (fast, normal, slow)
  • 🖱️ Hover trigger option
  • ⏱️ Delay variants for staggered typing
  • 🎨 Cursor style variants (blinking, none)
  • 📏 Works with any text length
