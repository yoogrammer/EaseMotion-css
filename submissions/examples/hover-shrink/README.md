# ease-hover-shrink

Subtle press/click-feedback hover effect that scales the element down to `0.95`.

## Usage

```html
<button class="ease-btn ease-hover-shrink">Click me</button>
```

## Variants

| Class | Description |
|---|---|
| `ease-hover-shrink` | Default shrink to 0.95 |
| `ease-hover-shrink-sm` | Subtle shrink to 0.98 |
| `ease-hover-shrink-lg` | Strong shrink to 0.90 |

## Notes

- Great for buttons, cards, and interactive elements
- Provides gentle tactile feedback
- Respects `prefers-reduced-motion`
