# ease-decoration Utilities

Controls the `text-decoration` property.

## Usage

```html
<a href="#" class="ease-decoration-none">No underline</a>
<span class="ease-decoration-underline">Underlined</span>
<span class="ease-decoration-line-through">Strikethrough</span>
```

## Classes

| Class                          | CSS Property                  |
| ------------------------------ | ----------------------------- |
| `ease-decoration-none`         | text-decoration: none         |
| `ease-decoration-underline`    | text-decoration: underline    |
| `ease-decoration-line-through` | text-decoration: line-through |
