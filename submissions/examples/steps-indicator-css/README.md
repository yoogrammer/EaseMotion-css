# Steps Indicator

Multi-step progress stepper with numbered circles, connector lines, and animated state transitions.

## Features

- Horizontal layout with numbered circular step indicators
- Three states: active (blue glow), completed (green), pending (muted)
- Animated connector line fill between completed steps
- Checkmark icon replaces number on completion via pseudo-element
- Interactive Previous / Next / Reset navigation buttons
- Responsive with label handling on small screens
- `prefers-reduced-motion` support

- `demo.html` — Interactive stepper with navigation controls
