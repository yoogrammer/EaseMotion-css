ease-wobble

Horizontal shake with rotation — combines translateX and rotate for a natural wobbly effect. Ideal for error states, invalid inputs, and playful icon hover effects.

Use cases

Error State

⚠️

Infinite wobble for persistent error state.

Invalid Input

✖️

Trigger wobble on form validation error.

Fast Wiggle

🌀

Fast (.ease-wobble-fast) at 0.6s.

Playful Hover

🎈

One-shot wobble replay.

CSS source

@keyframes ease-kf-wobble {
  0% { transform: translateX(0%) rotate(0deg); }
  15% { transform: translateX(-25%) rotate(-5deg); }
  30% { transform: translateX(20%) rotate(3deg); }
  45% { transform: translateX(-15%) rotate(-3deg); }
  60% { transform: translateX(10%) rotate(2deg); }
  75% { transform: translateX(-5%) rotate(-1deg); }
  100% { transform: translateX(0%) rotate(0deg); }
}

.ease-wobble {
  --ease-wobble-duration: 1s;
  animation: ease-kf-wobble ... both;
  animation-iteration-count: var(--ease-animation-iterations, 1);
}

Usage

<!-- One-shot wobble -->
<div class="ease-wobble">⚠️</div>

<!-- Infinite looping -->
<div class="ease-wobble ease-wobble-infinite">🌀</div>

<!-- Fast variant -->
<div class="ease-wobble ease-wobble-fast">🎈</div>

<!-- Custom duration -->
<div style="--ease-wobble-duration:2s;" class="ease-wobble">✖️</div>
