# CSS Pulse Active Modal

Pure CSS animated modal utilizing a Pulse Active interaction designed for Glassmorphism UI layouts.

## Features

- Pure CSS
- No JavaScript
- Responsive
- Keyboard accessible
- Glassmorphism styling
- Pulse animation
- CSS custom properties
- Modern UI aesthetics
- Smooth 60fps animations

## Variables

```css
--primary
--secondary
--duration
--ease
```
