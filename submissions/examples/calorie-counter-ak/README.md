# Calorie Counter

## What does this do?
A daily calorie counter with separate meal sections (breakfast, lunch, dinner, snacks), food item logging with calorie values, progress bar against daily goal, macronutrient breakdown, and weekly summary chart.

## How is it used?
Set your daily calorie goal, then add food items to each meal section with name and calorie value. The progress bar updates in real time. Use the macro sliders to set carb/protein/fat distribution.

## Why is this useful?
This provides a complete nutrition tracking tool for fitness apps, diet planning, or health dashboards, helping users monitor their daily intake and macronutrient balance.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to use the counter.
