# Neon Glow Text Demo

This demo showcases advanced neon glow text effects using CSS `text-shadow` with animated pulsing and color-shifting transitions.

## Neon Glow Technique
The neon glow effect is achieved using multiple layers of `text-shadow` to create a glowing appearance. The colors can be customized to create different neon effects.

## Text-shadow Layering
By layering multiple shadows with varying blur radii and colors, we can create a more dynamic and visually appealing glow.

## Animation Approach
The animations are created using CSS keyframes, allowing for smooth transitions and effects such as pulsing and flickering.

## Customization Options
You can customize the colors, font sizes, and animation speeds to fit your design needs. Simply modify the CSS variables in `style.css`.
