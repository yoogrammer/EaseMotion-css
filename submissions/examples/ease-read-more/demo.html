A pure CSS expandable text component using semantic HTML.

Product Description

Experience the future of UI animation with EaseMotion CSS. Our framework is designed from the ground up to be lightweight, human-readable, and incredibly powerful. Whether you're building a simple landing page or a complex SaaS dashboard, EaseMotion provides the tools you need to create delightful user experiences.

This component uses the semantic <details> and <summary> elements to provide a robust, keyboard-accessible way to toggle content. By leveraging modern CSS features like interpolate-size and CSS variables, we achieve smooth animations without a single line of JavaScript.

The gradient fade-out in the collapsed state subtly hints to the user that there is more content to explore, while the rotating chevron provides clear visual feedback on the current state.
