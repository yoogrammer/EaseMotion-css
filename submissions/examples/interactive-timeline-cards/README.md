# Animation: Interactive Timeline Cards

An animated vertical timeline component that highlights milestones with glowing nodes, expanding cards, and smooth content reveals. Designed for product roadmaps, project histories, changelogs, and achievement showcases.

## Features

* Pure CSS timeline interaction
* Animated glowing timeline nodes
* Expanding milestone cards
* Smooth content reveal
* Vertical roadmap layout
* Responsive design
* Reduced motion support
* No JavaScript required

## Usage

```html
<div class="timeline-item">
    <div class="timeline-dot"></div>

    <div class="timeline-card">
        <h3>Development Phase</h3>
        <p>Core features implemented.</p>

        <div class="timeline-extra">
            Additional milestone information.
        </div>
    </div>
</div>
```

Hovering a milestone expands the card, highlights the timeline node, and reveals additional information.

## Why is it useful?

Timelines help users visualize progress, history, and future plans. The expandable interaction allows more information to be displayed without cluttering the layout, making it ideal for documentation pages, roadmaps, portfolios, and product launch histories.
