# macOS-inspired Animated Dock Navigation (`ease-dock-nav-pb`)

A self-contained, responsive desktop-style dock navigation layout built using advanced structural CSS layout metrics, seamless backdrop blur filter styling, and responsive proximity scaling transformations.

## Features
- **Fluid Proximity Magnification:** Uses cubic-bezier transformations to elevate and scale dock items smoothly on hover.
- **Glassmorphic Filter Base:** Implements frosted backdrop glass layers matching premium operating system specs.
- **CSS-Only Tooltips:** Uses data attributes and absolute pseudo-elements to trigger responsive micro-labels conditionally without JS.
