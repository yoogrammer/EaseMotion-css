# CSS-only Neumorphic Toggle Switch

### 1. What does this do?
A purely CSS-driven toggle switch featuring soft UI (neumorphism) aesthetics with complex box-shadows simulating a tactile 3D physical switch mechanism.

### 2. How is it used?
```html
<label class="neumorphic-toggle">
  <input type="checkbox">
  <span class="slider"></span>
</label>
```

### 3. Why is it useful?
Neumorphism remains a popular design choice for premium, minimalist interfaces. By providing this as a strictly CSS-only snippet, it fits EaseMotion CSS's philosophy of offering beautifully crafted, plug-and-play UI components.
