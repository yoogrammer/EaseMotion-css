 
# Glassmorphism Card (ease-card-glass)

Glassmorphism-style card using backdrop-filter blur, semi-transparent background, and frosted glass border.

## Files

- demo.html - Interactive demo
- style.css - Card styles
- README.md - Documentation

## Classes

- ease-card-glass - Base glass card
- glass-blue - Blue tint variant
- glass-purple - Purple tint variant
- glass-green - Green tint variant
- glass-pink - Pink tint variant
- ease-card-glass-hover - Hover effect variant

## Usage

```html
<div class="ease-card-glass">
    <h3>Card Title</h3>
    <p>Card content here</p>
</div>

<!-- With color variant -->
<div class="ease-card-glass glass-blue">
    <h3>Blue Glass Card</h3>
</div>
