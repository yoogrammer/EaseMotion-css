# ease-heartbeat

## 1. What does this do?
A looping double-pump heartbeat animation using CSS
transform: scale() keyframes — no JS, no SVG.

## 2. How is it used?

<!-- Basic -->
<div class="ease-heartbeat">❤️</div>

<!-- On a button -->
<button class="ease-heartbeat">Like</button>

<!-- Slow variant -->
<div class="ease-heartbeat ease-heartbeat-slow">🩷</div>

<!-- Fast variant -->
<div class="ease-heartbeat ease-heartbeat-fast">❤️</div>

## 3. Why is it useful?
Heartbeat animations are one of the most needed micro-animations
in modern UI — like buttons, health apps, live indicators.
EaseMotion had no pulse animation. This gives developers a
powerful result from a single readable class, fully composable
with existing EaseMotion utilities.
