# Hover Buzz Example

A high-frequency, subtle positional micro-interaction that triggers on hover or focus states. 

## How it works
It alternates small translations (`1px` to `-1px`) rapidly across the X and Y axes to simulate an energetic "buzzing" state. This is ideal for drawing immediate user attention to alert states, warnings, or destructive action indicators (like delete or cancel buttons).

## Usage
Apply the class to an interactive element:
```html
<button class="hover-buzz">Delete</button>
```
