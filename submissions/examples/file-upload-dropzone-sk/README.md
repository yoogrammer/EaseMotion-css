# File Upload Dropzone with Drag Feedback

A premium, reusable file upload dropzone component built with **EaseMotion CSS**. This example features interactive drag-and-drop states, animated upload progress, and clear validation feedback.

## Key Features
- **Interactive Drag Feedback**: Visual transformation of the dropzone area when files are dragged over.
- **Animated Progress Visualization**: Simulated upload progress bar with smooth transitions.
- **Validation Feedback**: Distinct visual states for success and error conditions.
- **Mobile Support**: Fully responsive layout that supports touch interactions.
- **Accessibility**: Semantic HTML, ARIA roles, and keyboard navigation support.

## How to Use
1. Include `easemotion.css` in your project.
2. Use the `.ease-dropzone` wrapper structure provided in `demo.html`.
3. Apply the custom styles from `style.css`.
4. (Optional) Use the provided JavaScript in `demo.html` to handle drag-and-drop events and progress simulation.

## Directory Structure
- `demo.html`: Interactive demo showcasing the dropzone and its states.
- `style.css`: Styles for the dropzone, animations, and feedback states.
