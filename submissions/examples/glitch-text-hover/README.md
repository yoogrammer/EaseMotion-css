# Ultimate Glitch Text Hover Collection

**What does this do?**
This provides **four completely distinct** pure CSS glitch effects. Because "glitch" can mean many different things, this submission gives developers an entire suite of corruption aesthetics to choose from:
1. `glitch-cyberpunk`: Aggressive horizontal slicing with Cyan/Red chromatic aberration (Cyberpunk 2077 style).
2. `glitch-tiktok`: Smooth, continuous intense RGB split jittering.
3. `glitch-vhs`: Retro tracking error with vertical bouncing and ghosting blur.
4. `glitch-matrix`: Drops the main text into pure blackness and replaces it with neon green binary code rain.

**Why is it useful?**
It provides maximum flexibility for developers looking for high-end digital corruption animations, all executed smoothly via CSS without a single line of JavaScript.

**How is it used?**
```html
<!-- Add the base wrapper class and pick your variant! -->
<div class="glitch-wrapper glitch-cyberpunk" data-text="CYBERPUNK">CYBERPUNK</div>

<div class="glitch-wrapper glitch-tiktok" data-text="TIKTOK">TIKTOK</div>

<div class="glitch-wrapper glitch-vhs" data-text="VHS">VHS</div>

<div class="glitch-wrapper glitch-matrix" data-text="MATRIX" data-scramble="101100101">MATRIX</div>
```
