# Animated Skill Bars

## What it does

Animated progress bars that fill to target percentage with smooth transitions. Perfect for skills showcase, statistics, or progress indicators.

## How to use it

<div class="ease-skill-bar">
    <div class="ease-skill-label">Skill Name</div>
    <div class="ease-skill-track">
        <div class="ease-skill-fill" data-width="90">90%</div>
    </div>
</div>

## Variants

- ease-skill-fast - 0.4s animation
- ease-skill-slow - 2s animation
- ease-skill-gradient - Animated shimmer effect
- ease-skill-striped - Striped pattern overlay
- ease-skill-rounded - Pill-shaped corners

## Why it fits EaseMotion CSS

- Pure CSS transitions
- Smooth cubic-bezier easing
- Multiple speed variants
- Shimmer effect option
