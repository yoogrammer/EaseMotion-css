# Scale Icon Emphasis

## What does this do?

Adds a hover animation that scales icons to create emphasis and visual feedback.

## How is it used?

```html
<div class="scale-icon-emphasis">⭐</div>
```

## Why is it useful?

Scale icon emphasis is useful for interactive buttons, feature icons, navigation items, and action controls. It helps draw attention to clickable elements while keeping the interaction lightweight and readable.
