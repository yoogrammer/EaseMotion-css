# Animated Vertical Timeline

## What does this do?

A vertical timeline with 5 events alternating left and right of a center line. Each event shows a year badge, title, and description inside a styled card. Events animate in with staggered delays on page load.

## How is it used?

Open `demo.html` in a browser. The timeline renders automatically with a center gradient line, circular dot markers, and content cards. Hover over any card or dot for interactive feedback. On screens narrower than 640px, all events collapse to the left side for better readability.

## Why is it useful?

Presents chronological information in a visually engaging, scan-friendly layout. The staggered entry animation draws the reader's eye through the sequence naturally. Fully responsive and respects `prefers-reduced-motion` for accessibility.
