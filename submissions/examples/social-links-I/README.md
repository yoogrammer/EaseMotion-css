 
# Social Links Component (ease-social-links)

Horizontal row of social icon links with hover color animations per platform.

## Files

| File | Description |
|------|-------------|
| 📄 [demo.html](./demo.html) | Interactive demo |
| 🎨 [style.css](./style.css) | Social links styles |
| 📖 [README.md](./README.md) | Documentation |

## Classes

| Class | Description |
|-------|-------------|
| `ease-social-links` | Container for text-style links |
| `ease-social-links-icon` | Container for icon-only links |
| `ease-social-links-rounded` | Container for rounded icon links |
| `social-github` / `github-icon` | GitHub link style |
| `social-twitter` / `twitter-icon` | Twitter link style |
| `social-linkedin` / `linkedin-icon` | LinkedIn link style |
| `social-facebook` / `facebook-icon` | Facebook link style |
| `social-instagram` / `instagram-icon` | Instagram link style |

## Usage

```html
<!-- Basic text social links -->
<div class="ease-social-links">
    <a href="#" class="social-link social-github">GitHub</a>
    <a href="#" class="social-link social-twitter">Twitter</a>
    <a href="#" class="social-link social-linkedin">LinkedIn</a>
</div>

<!-- Icon only social links -->
<div class="ease-social-links-icon">
    <a href="#" class="social-icon github-icon">🐙</a>
    <a href="#" class="social-icon twitter-icon">🐦</a>
</div>
