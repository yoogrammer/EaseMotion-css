# ease-no-select

A utility class that prevents users from selecting text.

## Features

- Prevents accidental text selection
- Useful for buttons, draggable items, interactive UI elements
- Lightweight and reusable

## Usage

```html
<div class="ease-no-select">Example Content</div>
```

## CSS

```css
.ease-no-select {
  -webkit-user-select: none;
  -moz-user-select: none;
  -ms-user-select: none;
  user-select: none;
}
```
