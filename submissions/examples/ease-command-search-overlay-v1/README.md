# Ease Command Search Overlay

## Overview

Ease Command Search Overlay is a reusable command palette inspired by modern productivity applications. It provides a searchable command interface with animated results using pure HTML and CSS.

---

## Features

- 🔍 Command search overlay
- ⌨️ Keyboard shortcut hint
- 📄 Search results list
- ✨ Hover animation
- 📱 Responsive layout
- ⚡ Pure HTML & CSS

---

## Demo

The demo includes:

- Keyboard shortcut badge
- Command list
- Hover interactions
