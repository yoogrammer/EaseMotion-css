# Cyberpunk Neon Checkbox

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A futuristic cyberpunk-inspired checkbox collection featuring glowing neon borders, animated checkmark reveals, pulse effects, and multiple color variants using pure HTML and CSS.

## Features

- Neon checkbox design
- Animated checkmark reveal
- Pulse animation on selection
- Cyan, purple, pink, and orange variants
- Glassmorphism settings panel
- Hover interactions
- Responsive layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<label class="cyber-checkbox">
  <input type="checkbox">
  <span class="checkmark"></span>
  Enable Feature
</label>
```

## Main Classes

```css
.cyber-checkbox
.checkmark
.cyan
.purple
.pink
.orange
```

## Why It Fits EaseMotion CSS

- Animation-first interaction
- Human-readable class names
- Reusable form component
- Modern visual effects
- Lightweight implementation
