 
# Chip Group Multi-Select Component

Multi-select chip component using CSS checkbox pattern — selected chips change color and show checkmark.

## Files

- demo.html - Interactive demo
- style.css - Chip styles
- README.md - Documentation

## Classes

- ease-chip-group - Base chip group
- ease-chip-group-primary/success/warning/info - Color variants
- ease-chip-group-small/large - Size variants
- ease-chip-group-rounded - Rounded chips
- ease-chip-group-tags - Tag style chips

## Usage

```html
<div class="ease-chip-group">
    <label class="chip">
        <input type="checkbox" name="option" value="value">
        <span>Label</span>
    </label>
</div>
