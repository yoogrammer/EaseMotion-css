Accessible Skip Link

Press Tab to reveal the skip link. It lets keyboard users bypass navigation and jump straight to content.

class="skip-link"

What is a Skip Link?

A skip link is a hidden anchor element that becomes visible when focused via keyboard navigation (usually by pressing Tab). It allows users to bypass repetitive navigation links and jump directly to the main content of the page.

Try it: Press Tab right now. The skip link will appear at the top of the page. Press Enter to skip the navigation.

Why It Matters

Screen reader users and keyboard-only users often have to tab through dozens of navigation links before reaching the main content. A skip link provides a one-keystroke shortcut to the content area, dramatically improving the experience for users with motor disabilities.

WCAG 2.1 Compliance

Skip links satisfy WCAG 2.1 Success Criterion 2.4.1 (Bypass Blocks), which requires a mechanism to bypass blocks of content that are repeated on multiple pages. This is a Level A requirement — the most basic level of accessibility compliance.

How This Implementation Works

The skip link is positioned off-screen by default (top: -100%). When it receives keyboard focus, CSS transitions move it into view at the top of the viewport. The link targets an element with a matching id attribute, so pressing Enter jumps directly to that element.

Variant: Pill Shape

Add the skip-link--pill class for a rounded pill shape instead of the default rectangular appearance with bottom border radius.

Reduced Motion

The slide-in animation is automatically disabled when the user has prefers-reduced-motion: reduce enabled in their OS settings. The link still appears on focus, just without the transition effect.
