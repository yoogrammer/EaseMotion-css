\# ease-font-weight utilities



Font weight classes from Thin (100) to Black (900).



\## Available Classes



| Class | Value |

|-------|-------|

| `ease-font-thin` | 100 |

| `ease-font-extralight` | 200 |

| `ease-font-light` | 300 |

| `ease-font-regular` | 400 |

| `ease-font-medium` | 500 |

| `ease-font-semibold` | 600 |

| `ease-font-bold` | 700 |

| `ease-font-extrabold` | 800 |

| `ease-font-black` | 900 |



\## Usage



```html

<p class="ease-font-thin">Thin text (100)</p>

<p class="ease-font-bold">Bold text (700)</p>

<p class="ease-font-black">Black text (900)</p>
