# Motion Curve Gallery

## What does this do?
A visual gallery comparing different CSS easing curves — each card shows a box that slides on hover using a different timing function.

## How is it used?
Add the class to any HTML element:

    <div class="curve-card">
      <div class="curve-box ease-in-out"></div>
      <span class="curve-label">ease-in-out</span>
    </div>

## Why is it useful?
Helps developers visually compare easing curves to choose the right motion feel for their UI. Great for design system documentation and prototyping.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
