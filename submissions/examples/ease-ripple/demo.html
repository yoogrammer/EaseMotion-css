ease-ripple

Composable click ripple effect — add to any interactive element

Buttons

Add class="ease-ripple" to any button.

Cards

Ripple also works on larger elements like cards.

Clickable Card

Tap or click to see the ripple effect emanate from the center.

Custom Colour

Uses --ease-ripple-color for a green ripple.

Ripple center is fixed at 50% / 50% — adjust --ease-ripple-color to match the element.

Customisation

Customise via --ease-ripple-color (default: currentColor) and --ease-ripple-duration (default: 0.5s).

CSS Variables

--ease-ripple-color
Ripple colour — defaults to currentColor of the element
--ease-ripple-duration
Animation duration — defaults to 0.5s

Usage

<button class="ease-btn ease-ripple">Click me</button>
<div class="ease-card ease-ripple">Clickable card</div>

Works with any element that has position: relative context. overflow: hidden clips the ripple to the element bounds.
