# Text Transform Utilities

CSS utility classes for text transformation.

## Classes

- `.text-uppercase` — `text-transform: uppercase`
- `.text-lowercase` — `text-transform: lowercase`
- `.text-capitalize` — `text-transform: capitalize`
- `.text-normal-case` — `text-transform: none`

## Why

Quickly change the capitalization of text without modifying the source content. Useful for headings, labels, buttons, and UI copy where consistent casing is needed regardless of the actual written content.
