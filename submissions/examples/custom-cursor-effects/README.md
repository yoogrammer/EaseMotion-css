# Custom Cursor Effects

## What does this do?
Replaces the default cursor with a custom follower that changes shape/size on hover targets.

## How is it used?
Add a `.cursor-follower` div and target elements with `data-effect`:

    <div class="cursor-follower"></div>
    <div class="cursor-target" data-effect="glow">Glow</div>

## Why is it useful?
Enhances site interactivity with unique cursor feedback on interactive elements.

## Tech Stack
- HTML
- CSS
- Vanilla JS

## Preview
Open demo.html directly in your browser to see the effect.
