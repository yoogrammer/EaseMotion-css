# Ease Card Dark Variant

1. **What does this do?** Adds a dark-themed card variant using CSS variables for background, text, and border colors with hover animations.

2. **How is it used?**
```html
   <div class="ease-card-dark">
     <div class="card-title">Title</div>
     <div class="card-body">Card content here.</div>
     <div class="card-footer">Footer text</div>
   </div>

   <!-- Accent variant -->
   <div class="ease-card-dark accent">...</div>

   <!-- Elevated variant -->
   <div class="ease-card-dark elevated">...</div>
```

3. **Why is it useful?** A fully customizable dark card using CSS variables with smooth hover transitions — zero JavaScript, fits EaseMotion CSS's animation-first philosophy.
