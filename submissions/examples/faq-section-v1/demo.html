Frequently Asked Questions

How does EaseMotion CSS differ from Tailwind?

EaseMotion uses human-readable class names like ease-fade-in and is animation-first. No build step or config needed — just link one file and go.

Can I use it with React or Vue?

Yes. Install via npm and import the stylesheet in your root component. It works flawlessly with any framework that renders standard HTML class attributes.

Does it work without a build step?

Yes. Link the CDN stylesheet inside your document head and start using classes immediately. No bundler, compiler, or node modules required.

How do I submit a new component?

Add your demo inside submissions/examples/your-feature/ containing your standalone demo.html, style.css, and an instructional README.md. The core repository maintainer will review and integrate it.

Can I override the default colors and tokens?

Yes. Simply override any --ease-* CSS custom properties inside your local :root declaration to cleanly retheme the entire framework without altering source files.
