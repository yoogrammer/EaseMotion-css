# ease-shake-card-exit

Card exit animation that oscillates left-right with decreasing amplitude while fading out. Ends at opacity 0 with slight rotation.

## Usage

```html
<div class="ease-shake-card-exit">Shakes & fades out</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-shake-card-exit` | Default shake exit animation |

## Notes

- Oscillates left-right with decreasing amplitude
- Ends fully transparent
- Respects `prefers-reduced-motion`
