Fix: Tabs Layout Shift (Issue #4783)

In .ease-tabs-auto, the active indicator adds a border-bottom: 2px solid only to the checked label — causing a 2-pixel vertical layout shift. The fix pre-reserves the border space on all labels using a transparent border, so only the color changes when a tab is selected — no shift.
