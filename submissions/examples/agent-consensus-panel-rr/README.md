# Agent Consensus Voting Panel Component

## What does this do?

Provides a voting interface where multiple AI agents evaluate a decision and contribute to a final consensus outcome.

## How is it used?

```html
<div class="agent-card approve">
    <h3>Research Agent</h3>
</div>
```

## Why is it useful?

Modern multi-agent AI systems often rely on consensus-based decision making to improve reliability and transparency. This component provides a reusable interface for displaying agent votes, confidence levels, and final outcomes while demonstrating modern AI collaboration patterns.
