Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A modern infrastructure governance dashboard showcasing policy enforcement, compliance monitoring, audit readiness, infrastructure inventory management, and governance analytics using pure HTML and CSS.

## Features

- Executive governance overview
- Compliance scorecard
- Policy enforcement metrics
- Infrastructure inventory panel
- Compliance status badges
- Audit readiness tracker
- Governance event timeline
- Responsive dashboard layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="overview-card">
  Governance Metric
</div>
```

## Main Classes

```css
.overview-card
.panel
.status
.resource-card
.audit-progress
.timeline-item
```
