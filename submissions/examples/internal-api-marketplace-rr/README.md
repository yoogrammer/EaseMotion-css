# Internal API Marketplace Component

## What does this do?

Provides a centralized interface for discovering, cataloging, evaluating, and consuming internal APIs across an organization.

## How is it used?

```html
<div class="api-card">
    <h3>Customer Profile API</h3>
</div>
```

## Why is it useful?

As organizations scale, hundreds of APIs are created across teams. This component provides a reusable marketplace experience that improves API discoverability, adoption, governance, and developer productivity while demonstrating modern platform engineering and developer portal patterns.
