# Animated Pricing Card

## What it does

Premium pricing cards with monthly/yearly toggle, animated price transitions, and hover effects.

## How to use it

<div class="ease-pricing-card">
    <h3>Plan Name</h3>
    <div class="ease-price">$<span class="amount">19</span><span class="period">/mo</span></div>
    <ul class="ease-features">...</ul>
    <button class="ease-pricing-btn">Get Started</button>
</div>

## Variants

- ease-pricing-popular - Highlighted card with badge
- Billing toggle - Monthly/Yearly with animation

## Features

- Smooth price transition animation
- Hover lift effect
- Popular plan highlighting
- Feature list with icons
- Toggle switch for billing cycle

- Animated price changes
