# Team Goals Tracking Dashboard Showcase

## What does this do?

Provides a dashboard for monitoring team objectives, key results, KPI metrics, and goal completion progress.

## Features

- Objectives overview
- Key results metrics
- Goal tracking cards
- Animated progress bars
- KPI dashboard widgets
- Responsive layout

## How is it used?

Open `demo.html` directly in a browser.

## Why is it useful?

It demonstrates modern OKR and KPI tracking patterns commonly used in productivity and performance-management applications.
