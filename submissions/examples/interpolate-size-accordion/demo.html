CSS interpolate-size: allow-keywords

Smooth height: auto accordion transitions

interpolate-size is a CSS property that allows keyword values like auto, fit-content, max-content, and min-content to participate in CSS transitions and animations. By default, CSS cannot animate to or from these keyword values.

When you set interpolate-size: allow-keywords on an element, the browser can interpolate between a numeric length (like 0) and a keyword value (like auto). This enables smooth open/close animations for elements with dynamic content sizes, without needing JavaScript to calculate exact pixel heights.

As of 2024, interpolate-size: allow-keywords is supported in Chromium-based browsers and Firefox. Safari support is coming. This demo uses @supports to provide a graceful fallback—instant expand/collapse without animation—for browsers that don't support it yet.
