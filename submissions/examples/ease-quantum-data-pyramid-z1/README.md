# Ease Quantum Data Pyramid

## Overview

A futuristic layered pyramid animation featuring floating geometric levels and a glassmorphism information panel built entirely with HTML and CSS.

## Features

- Layered pyramid design
- Floating animation
- Glassmorphism center panel
- Responsive layout
- Lightweight CSS
- Pure HTML & CSS
