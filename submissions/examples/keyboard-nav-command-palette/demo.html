CSS Classes Reference

Available CSS classes for the command palette component:

Class Name Purpose
.ease-command-palette Main container for the command palette
.ease-command-search Search input container
.ease-command-list Container for command items list
.ease-command-item Individual command item
.ease-command-active Marks the currently active/selected command
.ease-command-shortcut Keyboard shortcut badge styling

Quick Start

Here's how to implement the command palette in your project:

<div class="ease-command-palette" role="dialog"> <div class="ease-command-search"> <input type="text" placeholder="Type a command..."> </div> <div class="ease-command-list"> <div class="ease-command-item ease-command-active"> <div class="command-icon">📄</div> <div class="command-content"> <div class="command-title">New File</div> </div> <div class="ease-command-shortcut">Ctrl+N</div> </div> </div> </div>

Customization with CSS Variables

Customize the command palette appearance using CSS variables:

:root { --palette-primary-color: #667eea; --palette-bg-color: #ffffff; --palette-text-color: #333333; --palette-border-color: #e0e0e0; --palette-hover-bg: #f5f7ff; --palette-active-bg: #eef0ff; --palette-animation-speed: 0.3s; --palette-border-radius: 8px; }

Available Variables:

--palette-primary-color

Primary accent color (default: #667eea)

--palette-bg-color

Background color (default: #ffffff)

--palette-text-color

Text color (default: #333333)

--palette-animation-speed

Animation duration (default: 0.3s)
