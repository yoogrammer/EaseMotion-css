A11y Reduced Motion Test

This demo tests the global prefers-reduced-motion override for easemotion-css.

How to test this:

  1. Right now, you should see a bouncing blue box that rotates when hovered.
  2. Turn on Reduce Motion in your Windows settings (or emulate it via browser DevTools).
  3. The box should instantly stop bouncing and hovering should feel instantaneous, without motion.
