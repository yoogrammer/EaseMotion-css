Bundler CSS Import Resolution Fix

By declaring a "style" field inside package.json, bundlers can automatically map direct package imports to the minified style file.

{
  "name": "easemotion-css",
  "style": "easemotion.css",
  "exports": {
    ".": {
      "import": "./easemotion.js",
      "require": "./easemotion.js",
      "style": "./easemotion.css"
    }
  }
}
