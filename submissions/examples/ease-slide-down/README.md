# Slide Down Entrance Animation

## What does this do?

Adds a smooth slide-down entrance animation for elements appearing on the page.

## How is it used?

```html
<div class="slide-down">
  Content
</div>
```

## Why is it useful?

Slide-down animations help introduce content smoothly and create a more polished user experience for cards, banners, alerts, headings, and sections.
