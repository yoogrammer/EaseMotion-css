# Ease Nav Underline AK

## What does this do?

Adds an animated underline hover effect for navigation links with smooth transitions and active states.

---

## How is it used?

```html
<nav class="navbar">
  <a href="#" class="nav-link active">Home</a>
  <a href="#" class="nav-link">About</a>
  <a href="#" class="nav-link">Contact</a>
</nav>
