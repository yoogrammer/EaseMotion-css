# Animation Speed Controller

## Description

This example demonstrates a live animation speed controller.

## Features

- Interactive speed slider
- Instant animation updates
- Current duration display
- Live animation preview

## How It Works

Users can adjust the slider to change animation duration.

Changes are applied immediately to the animation preview.
