# Text Underline Draw Hover

**What does this do?**  
Animates an underline under text/links on hover using three distinct animation presets: left-to-right, center-out, and continuous slide-through.

**How is it used?**  
```html
<!-- Left to right underline -->
<a class="underline-link underline-left">Discover More</a>

<!-- Expand from center -->
<a class="underline-link underline-center">Explore Portfolio</a>

<!-- Slide through enter-left exit-right -->
<a class="underline-link underline-slide-through">Contact Us</a>
```

**Why is it useful?**  
Underline draw animations are a clean, lightweight, and modern standard for web navigation and header interactions. Because they rely on CSS transforms (`scaleX`), they execute without triggering layout reflow, offering extreme responsiveness.
