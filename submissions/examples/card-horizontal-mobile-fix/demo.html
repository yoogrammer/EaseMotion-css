ease-card-horizontal Mobile Fix

Fixes overflow on mobile by adding responsive flex-wrap and column stacking

BEFORE — Buggy No flex-wrap — overflows on mobile

Resize your browser to under 600px wide. The card below overflows or squeezes content because flex-direction: row is stuck with no responsive breakpoint.

The Fix — CSS

css
/* BEFORE (buggy) */
.ease-card-horizontal {
  display: flex;
  flex-direction: row;
  gap: var(--ease-space-4);
  align-items: flex-start;
  padding: var(--ease-space-4);
  /* ← no flex-wrap, no responsive breakpoint */
}

/* AFTER (fixed) — add this media query */
@media (max-width: 600px) {
  .ease-card-horizontal {
    flex-direction: column;  /* stack vertically */
  }

  .ease-card-horizontal .ease-card-img,
  .ease-card-horizontal img {
    width: 100%;             /* full width image */
    height: auto;
    border-radius:
      var(--ease-radius-lg)
      var(--ease-radius-lg)
      0 0;
  }
}
