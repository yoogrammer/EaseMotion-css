Bug: color-mix() without @supports

In browsers without color-mix() support, the border between grouped buttons disappears entirely (no fallback).

Line 299 (bug)

.ease-btn-group .ease-btn:not(:last-child) {
  border-right-color: color-mix(in srgb, currentColor 30%, transparent);
}

Fix

@supports (color: color-mix(in srgb, red 50%, transparent)) {
  .ease-btn-group .ease-btn:not(:last-child) {
    border-right-color: color-mix(in srgb, currentColor 30%, transparent);
  }
}
