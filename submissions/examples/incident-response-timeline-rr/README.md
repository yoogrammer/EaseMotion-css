# Incident Response Timeline Component

## What does this do?

Provides a chronological timeline for displaying incidents, investigations, mitigations, and resolutions.

## How is it used?

```html
<div class="event critical">
    <div class="dot"></div>
    <div class="content">
        <h3>API Outage Detected</h3>
    </div>
</div>
```

## Why is it useful?

Incident timelines are widely used by SaaS platforms, cloud providers, and engineering teams to communicate outages and recovery progress. This component offers a clean and reusable way to present incident reports using pure HTML and CSS.
