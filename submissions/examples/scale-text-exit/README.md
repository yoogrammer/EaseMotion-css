# ease-scale-text-exit

Text exit animation that scales text down while fading out. A clean dismiss effect for text elements.

## Usage

```html
<span class="ease-scale-text-exit">Text shrinks away</span>
```

## Variants

| Class | Description |
|---|---|
| `ease-scale-text-exit` | Default scale text exit |

## Notes

- Scales text to 0.6x while fading to opacity 0
- Ideal for labels, badges, and inline text removal
- Respects `prefers-reduced-motion`
