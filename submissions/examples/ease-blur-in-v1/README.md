# Ease Blur In Entrance Animation

## What does this do?
A premium, responsive, pure CSS entrance animation utility. Elements fade in and transition smoothly from an opaque blurred state to sharp visual clarity and default size.

## How is it used?
Add the utility class to any HTML element to animate its entrance:

```html
<h1 class="ease-blur-in">Your Title</h1>

<p class="ease-blur-in-delayed">Your description text here.</p>
```

Ensure `style.css` is links-loaded to apply the staggered timing offsets, scale matrices, and keyframe animations.

## Why is it useful?
It provides a high-fidelity, polished, and modern landing page entrance interaction (reminiscent of landing pages from Vercel, Linear, and Stripe). It runs smoothly via CSS hardware acceleration and contains accessible fallback behaviors for users requesting reduced motion.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)
