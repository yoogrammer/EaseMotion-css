Section 01

How Dark Mode Works in EaseMotion

💡
EaseMotion CSS stores every visual value as a CSS custom property (--ease-*) inside :root. Dark mode works by overriding only the visual tokens — colors, surfaces, and shadows. Spacing, radii, fonts, z-index, and motion values stay completely untouched.

Two strategies are supported. Both override the exact same variables — choose whichever fits your project.

🤖
Strategy 1 — Automatic

Uses @media (prefers-color-scheme: dark). Follows the user's OS setting. Zero JavaScript.

🎛️
Strategy 2 — Manual Toggle

Uses [data-theme="dark"] on the <html> element. Controlled by a button — works regardless of OS preference.

Section 02

Strategy 1 — Automatic (OS Preference)

Add this to your CSS file (after linking easemotion.min.css). The browser automatically applies it when the user's system is in dark mode.

/* style.css — load AFTER easemotion.min.css */ @media (prefers-color-scheme: dark) { :root { /* Backgrounds */ --ease-color-bg: #0f172a; --ease-color-surface: #1e293b; /* Text */ --ease-color-text: #f1f5f9; --ease-color-muted: #94a3b8; /* Primary accent — brightened for dark bg */ --ease-color-primary: #817cf8; --ease-color-primary-light: #c4c0fb; --ease-color-primary-dark: #6c63ff; /* Shadows — deeper on dark surfaces */ --ease-shadow-md: 0 4px 6px -1px rgba(0,0,0,0.50), 0 2px 4px -1px rgba(0,0,0,0.35); /* Glows — intensified */ --ease-glow-primary: 0 0 20px rgba(129, 124, 248, 0.60); } }

Section 03

Strategy 2 — Manual Toggle (data-theme)

Use this when you want users to control the theme themselves — a toggle button in your navbar, for example. It also lets you persist their preference to localStorage.

Step 1 — CSS: scope overrides to the attribute selector

/* style.css */ [data-theme="dark"] { --ease-color-bg: #0f172a; --ease-color-surface: #1e293b; --ease-color-text: #f1f5f9; --ease-color-muted: #94a3b8; /* ... all other token overrides ... */ } /* Optional: smooth color transition on switch */ *, *::before, *::after { transition: background-color 300ms ease, color 300ms ease, border-color 300ms ease, box-shadow 300ms ease; }

Step 2 — HTML: set the attribute on <html>

<!-- Start in light mode --> <html lang="en" data-theme="light"> <!-- Toggle button anywhere in your page --> <button onclick="toggleTheme()">Toggle Dark Mode</button>

Step 3 — JavaScript: toggle + persist preference

// Reads saved preference on page load const saved = localStorage.getItem('easemotion-theme'); if (saved) { document.documentElement.setAttribute('data-theme', saved); } // Toggle function function toggleTheme() { const current = document.documentElement.getAttribute('data-theme'); const next = current === 'dark' ? 'light' : 'dark'; document.documentElement.setAttribute('data-theme', next); localStorage.setItem('easemotion-theme', next); }
