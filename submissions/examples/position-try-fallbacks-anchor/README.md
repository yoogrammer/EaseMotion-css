# CSS position-try-fallbacks Anchor Example

This example demonstrates how to use CSS `position-try-fallbacks`
with CSS Anchor Positioning.

## Features

- Tooltip attached to a button
- Preferred placement above the button
- Automatic fallback positions:
  - Right
  - Bottom
  - Left

## Files

- `demo.html` – HTML structure
- `style.css` – Styling and fallback positioning
- `README.md` – Documentation
