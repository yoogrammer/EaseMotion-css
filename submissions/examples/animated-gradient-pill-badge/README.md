# Animated Gradient Pill Badge

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A reusable animated pill badge utility featuring moving gradient borders, hover glow effects, status variants, and responsive layouts using pure HTML and CSS.

## Features

- Animated gradient border
- Reusable pill badge component
- Success variant
- Warning variant
- Danger variant
- Notification badge example
- Hover glow effect
- Responsive layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<span class="gradient-pill">
  New Feature
</span>
```

## Main Classes

```css
.gradient-pill
.success
.warning
.danger
.notification
```

## Why It Fits EaseMotion CSS

- Animation-first design
- Human-readable class names
- Reusable utility component
- Lightweight implementation
- Modern UI effects
