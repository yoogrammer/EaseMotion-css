# Button Gradient & Soft Color Variants

## What does this do?

Adds gradient and soft color button variants that provide more visual styling options for user interfaces.

## How is it used?

```html
<button class="btn-gradient-purple">Gradient Purple</button>
<button class="btn-gradient-green">Gradient Green</button>
<button class="btn-gradient-red">Red Yellow Gradient</button>

<button class="btn-soft-purple">Soft Purple</button>
<button class="btn-soft-green">Soft Green</button>
<button class="btn-soft-red">Soft Red</button>
```

## Why is it useful?

These button variants expand EaseMotion CSS's button styling options by providing visually appealing gradient buttons and subtle soft-color buttons. They help developers create more engaging interfaces while maintaining a clean and readable design style.
