# Animation: Bento Grid Expansion

A modern Bento Grid layout with expandable information panels. Hovering a tile highlights it with a gradient glow while revealing additional content.

## Features

- Pure CSS interaction
- Bento-style dashboard layout
- Gradient hover glow
- Content reveal animation
- Responsive design
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="tile">
    <h3>Performance</h3>

    <div class="extra">
        Additional information.
    </div>
</div>
