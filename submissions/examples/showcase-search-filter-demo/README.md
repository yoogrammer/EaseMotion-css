# Showcase Search & Filter Demo

This example demonstrates how search and category filtering can improve animation discovery in a growing animation showcase.

## Features

- Real-time search
- Category filters
- Mobile-friendly layout
- Animation cards
- Responsive grid

## Categories

- Fade Animations
- Slide Animations
- Scale Animations
- Rotation Animations
- Entrance Animations
- Exit Animations

## Benefits

- Improved discoverability
- Better user experience
- Faster animation selection
