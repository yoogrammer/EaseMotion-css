# CSS-Only Floating Ghosts

A fun and spooky animated component featuring two floating ghosts. Built entirely with CSS, featuring a wavy skirt, blinking eyes, an animated mouth, and dynamic shadows.

## Features
- ✨ **100% CSS-only**: No JavaScript required for the floating or facial animations.
- 👻 **Procedural Wavy Skirt**: The ghost's wavy bottom is created cleverly using complex `radial-gradient` background images instead of relying on SVGs.
- 👁️ **Facial Animations**: Keyframe animations make the ghosts periodically blink their eyes and open their mouths.
- 🌒 **Dynamic Shadow**: As the ghosts float up and down, their drop shadows scale and change opacity accordingly.

## Files
- `demo.html`: The HTML structure of the two ghosts, their faces, and shadows.
- `style.css`: The styling rules, gradients, and multiple keyframe animations (`float`, `blink`, `o-mouth`, `shadow-scale`).

## Usage
Simply drop the HTML structure into your project and link the CSS. The animation loops infinitely automatically.
