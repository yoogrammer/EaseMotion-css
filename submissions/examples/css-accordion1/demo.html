★ What is EaseMotion CSS? ›

EaseMotion CSS is an animation-first CSS framework that provides human-readable, composable utility classes for modern UI animations. It requires zero dependencies and no build step.

💡 How do I install it? ›

Simply include the CDN link in your HTML head tag. No npm, no bundler, no configuration required. Just drop it in and start using the animation classes immediately.

⚙ Can I customize the animations? ›

Yes! EaseMotion CSS exposes CSS custom properties like --ease-duration and --ease-delay so you can override timing and easing on any element without writing new CSS.

📈 Is it production ready? ›

EaseMotion CSS is actively maintained, well tested across modern browsers, and used in production by developers building dashboards, landing pages, and portfolio sites around the world.

🔗 Does it support dark mode? ›

Yes. EaseMotion CSS includes a dark mode token layer that automatically adapts component colors based on the user's system preference using the prefers-color-scheme media query.
