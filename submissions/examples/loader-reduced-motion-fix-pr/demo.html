🐛 Loader Reduced Motion Fix

Adds missing prefers-reduced-motion: reduce support to the loaders component.

All Loader Variants (animated)

.ease-loader-spin
.ease-loader-pulse
.ease-loader-ping
.ease-loader-dots
How to test:
1. Open this page — all four loaders should be animating normally.
2. Enable prefers-reduced-motion: reduce in your OS:
    • Windows: Settings → Accessibility → Visual effects → turn off "Animation effects"
    • macOS: System Settings → Accessibility → Display → "Reduce motion"
    • Chrome DevTools: Ctrl+Shift+P → "Emulate CSS prefers-reduced-motion: reduce"
3. Reload the page — all four loaders should be frozen/static.
4. Without the fix (comment out style.css), the loaders continue animating even with reduced motion enabled.
