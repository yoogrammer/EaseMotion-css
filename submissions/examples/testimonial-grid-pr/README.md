# Responsive Testimonial Grid

## What does this do?

A responsive testimonial grid component that displays customer reviews in clean, modern cards.

## How is it used?

```html
<div class="testimonial-card">
  <div class="avatar">A</div>
  <h3>Alice Johnson</h3>
  <div class="rating">★★★★★</div>
  <p>Customer review goes here.</p>
</div>
```

## Why is it useful?

Testimonials are widely used on landing pages, SaaS products, portfolios, and business websites to build trust and showcase customer feedback. This component provides a lightweight, responsive solution using only HTML and CSS.
