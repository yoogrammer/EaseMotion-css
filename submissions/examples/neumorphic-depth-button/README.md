# Neumorphic Depth Button

## What does this do?
Neumorphic buttons that simulate physical depth using inset/outset box-shadows, with a compressed depth effect on click or press.

## How is it used?
Add the class `neu-btn` to a `<button>` element and optionally a variant modifier:

    <button class="neu-btn neu-btn--primary">Primary</button>
    <button class="neu-btn neu-btn--secondary">Secondary</button>
    <button class="neu-btn neu-btn--accent">Accent</button>

Size modifiers `neu-btn--sm` and `neu-btn--lg` are also available.

## Why is it useful?
Provides a tactile, skeuomorphic interaction that communicates press state through visual depth — ideal for dashboards, control panels, and modern UI toolkits.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
