Mobile Sidebar Transition Fix

To test this fix, switch your browser view to a mobile device emulation (width below 768px) and click the hamburger menu button in the topbar.

The sidebar will now **slide in smoothly** from the left side. Previously, it snapped onto the screen instantly because toggling display: none in the same frame as the transform change blocked the transition.
