# Flashcard App

## What
A flashcard study app with click-to-flip animation, next/previous navigation, and progress tracking.

## How
Uses CSS 3D transforms with `perspective` and `rotateY` for card flipping. JavaScript manages an array of flashcard objects, tracking the current index and updating the DOM on navigation.

## Why
Demonstrates a practical study tool with smooth 3D flip transitions, making interactive learning engaging and accessible.
