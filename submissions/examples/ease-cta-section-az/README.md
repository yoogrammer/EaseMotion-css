# CTA Section

Call-to-action banner section with icon, headline, description, and action buttons. Available in dark (default), light, and gradient variants with small size option.

## Usage

```html
<div class="ease-cta-section-az">
  <div class="ease-cta-icon-az">&#x1f680;</div>
  <h2 class="ease-cta-title-az">Ready to get started?</h2>
  <p class="ease-cta-desc-az">Join thousands of developers.</p>
  <div class="ease-cta-buttons-az">
    <button class="ease-cta-btn-az --primary">Get started</button>
    <button class="ease-cta-btn-az --secondary">Learn more</button>
  </div>
</div>
```

## Variants

- **Default**: Dark background with white text
- **`--light`**: Light background with dark text
- **`--gradient`**: Purple gradient background
- **`--sm`**: Reduced padding and font sizes
- **`--primary`** / **`--secondary`** on `.ease-cta-btn-az`

## CSS Variables

| Variable | Default |
|----------|---------|
| `--ease-space-*` | spacing |
| `--ease-radius-lg` | 12px |
| `--ease-radius-md` | 8px |
| `--ease-color-primary` | #6366f1 |
| `--ease-color-primary-dark` | #4f46e5 |
| `--ease-color-neutral-*` | grays |
| `--ease-font-sans` | system-ui |
