# Pricing Calculator

An interactive pricing estimator with sliders for users and storage, plus a support tier dropdown. The monthly total updates in real time. Base price is $5/user + $0.10/GB + optional support surcharge.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser. Adjust the sliders and dropdown to see the price update.

## Accessibility notes

All controls have visible labels. Range inputs display the current value as text. The selected support tier shows its dollar amount in the option text.
