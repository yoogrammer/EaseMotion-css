ease-line-clamp utilities

Truncate text at a specific number of lines with an ellipsis.

ease-line-clamp-1-za

This is a very long text that should be truncated after the first line so that the user only sees one line of content and the rest is hidden behind an ellipsis.

ease-line-clamp-2-za

This text should be clamped to two lines maximum. Any content beyond the second line will be hidden and replaced with an ellipsis. This is useful for card descriptions and previews where you want to show a concise snippet.

ease-line-clamp-3-za

This text allows up to three lines before being truncated. It gives more room for content while still maintaining a clean, uniform card layout. Great for article previews and feed items where you want a consistent block height without overflowing text.

ease-line-clamp-4-za

This variant allows four lines of text before truncation. It provides even more content visibility while still enforcing a limit. Suitable for longer descriptions in product cards or blog post excerpts where you want to give the reader a substantial preview before they click through to read more.

ease-line-clamp-5-za

With five lines allowed, this variant offers the most generous truncation point. It is ideal for content-heavy sections like comment previews, bio snippets, or multi-line summaries where you want to show as much text as possible while still maintaining a predictable layout and preventing excessive overflow that could break your design.

ease-line-clamp-none-za

This disables line clamping entirely, allowing the full text to be displayed without any truncation. It restores the default overflow behavior.

Usage

Add the class to any block element: <p class="ease-line-clamp-2-za">...</p>
