# Zero Trust Security Architecture Component

## What does this do?

Visualizes a Zero Trust security workflow where every request must pass through verification and policy checks before accessing protected resources.

## How is it used?

```html
<div class="layer identity">
    <h3>Identity Verification</h3>
</div>

<div class="layer policy">
    <h3>Policy Engine</h3>
</div>
```

## Why is it useful?

Zero Trust Architecture is a widely adopted cybersecurity model that requires continuous verification of users, devices, and policies. This component provides a clear visual representation of that process using pure HTML and CSS.
