CSS Grid Auto-Placement Refactor

By enforcing grid-auto-rows and grid-auto-flow: dense across the framework, we guarantee that when dynamic database content injects extra items into a CSS Grid component, the implicit layout remains perfectly uniform and mathematically unbreakable.

❌ Legacy Grid (Distorted Implicit Row)

This grid explicitly defines only 1 row. When the CMS dynamically injects a 4th item, it falls into an undefined "implicit" row. Because grid-auto-rows is missing from the CSS, the 4th box collapses and completely breaks the layout uniformity.
