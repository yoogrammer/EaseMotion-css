# Swing Animation

This example demonstrates the usage of the `ease-swing` animation utility.

## Description

The `ease-swing` utility class applies a rotational animation from the `top center` origin, causing the element to swing back and forth before coming to a rest. 

This animation is ideal for notification bells when a new message arrives, or for bringing life to hanging UI elements like badges or tags.

## Usage

Include the `style.css` containing the animation, then apply the class to your HTML element.

```html
<div class="ease-swing">
  🔔 New Notification!
</div>
```

## Preview

Open `demo.html` in your browser to test out the swing animation on a sample notification bell.
