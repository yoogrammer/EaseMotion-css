 
# Responsive Container Utility (ease-container)

Centered max-width container with responsive padding — SM, MD, LG, XL, and Fluid sizes.

## Files

| File | Description |
|------|-------------|
| 📄 [demo.html](./demo.html) | Interactive demo |
| 🎨 [style.css](./style.css) | Container styles |
| 📖 [README.md](./README.md) | Documentation |

## Classes

| Class | Max Width | Best For |
|-------|-----------|----------|
| `ease-container-sm` | 640px | Mobile content |
| `ease-container-md` | 768px | Tablets |
| `ease-container-lg` | 1024px | Desktop |
| `ease-container-xl` | 1280px | Large screens |
| `ease-container-fluid` | 100% | Full-width layouts |

## Usage

```html
<!-- Small container -->
<div class="ease-container-sm">
    <p>Centered content max-width 640px</p>
</div>

<!-- Medium container -->
<div class="ease-container-md">
    <p>Centered content max-width 768px</p>
</div>

<!-- Large container -->
<div class="ease-container-lg">
    <p>Centered content max-width 1024px</p>
</div>

<!-- Fluid container -->
<div class="ease-container-fluid">
    <p>Full width with responsive padding</p>
</div>
