# A11y Hover Card

An accessible hover card component that gracefully degrades its animations when the user has `prefers-reduced-motion` enabled on their device.

## Features
- Smooth hover lift effect
- Uses CSS variables for theming
- Strictly respects `prefers-reduced-motion: reduce` to prevent motion sickness.

## Usage
Simply include the CSS and HTML structures to provide an accessible component for users prone to vestibular motion sickness.
