# Animated Shimmer Skeleton Card

**What does this do?**  
Creates a shimmer loading skeleton for card layouts with animated gradient sweep, simulating content loading state.

**How is it used?**  
Use `.skeleton-card` with `.skeleton-img` for image placeholder and `.skeleton-line` with width classes (`.w-80`, `.w-60`, `.w-40`) for text lines. Group multiple cards in a `.skeleton-grid`.

**Why is it useful?**  
Provides a polished loading experience that reduces perceived wait time. Pure CSS, responsive grid, respects `prefers-reduced-motion`.
