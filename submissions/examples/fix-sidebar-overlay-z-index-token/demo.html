Sidebar Overlay z-index Token Fix

Mobile sidebar overlay now uses var(--ease-z-overlay, 100) instead of hardcoded z-index: 100 — consistent with the z-index token scale defined in variables.css.

Resize to mobile viewport (<768px) to see the sidebar overlay. The z-index is driven by --ease-z-overlay.

Token values from variables.css

--ease-z-base: 0
--ease-z-raised: 10
--ease-z-overlay: 100 ← sidebar mobile overlay now uses this
--ease-z-modal: 200

Override globally: :root { --ease-z-overlay: 200; }
