🌀 Liquid Border Trace Features

  • ✨ Glowing border trace around elements on hover
  • 🎨 Multiple color variants (primary, success, danger, warning, purple, neon)
  • ⚡ Speed variations (fast, normal, slow)
  • 📏 Thickness variants (thin, normal, thick)
  • 💫 Smooth liquid-like motion around perimeter
  • 🎯 Works on buttons, cards, links, and any element
  • 🚀 Pure CSS, no JavaScript
