# Text Selection Showcase

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A collection of themed text selection examples demonstrating the CSS `::selection` pseudo-element.

Users can highlight text across different sections to see custom selection colors and visual styles.

## Features

- Custom text selection themes
- Neon selection effect
- Sunset selection effect
- Quote block styling
- Code block selection styling
- Article content example
- Responsive layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```css
::selection{
  background:#3b82f6;
  color:white;
}
```

## Main Demonstrations

- Default Theme
- Neon Theme
- Sunset Theme
- Quote Theme
- Code Theme
- Article Theme
