Smart Quotation Marks with CSS quotes

The same markup produces correct, language-specific quotation marks. The marks below are generated entirely by CSS using the quotes property together with the open-quote / close-quote keywords — no characters are typed into the text.

  • English

    The web is for everyone, everywhere.

  • French

    Le web est pour tout le monde, partout.

  • German

    Das Web ist für alle, überall.

  • Russian

    Сеть для всех, повсюду.

  • Japanese

    ウェブはみんなのもの
