 
# Underline Expansion Animation

Animated underline that expands on hover — from center, left, or right.

## Files

| File | Description |
|------|-------------|
| 📄 [demo.html](./demo.html) | Interactive demo |
| 🎨 [style.css](./style.css) | Underline styles |
| 📖 [README.md](./README.md) | Documentation |

## Classes

| Class | Direction | Color | Thickness | Speed |
|-------|-----------|-------|-----------|-------|
| `ease-underline-center` | Center | Default | 2px | 0.3s |
| `ease-underline-left` | Left to right | Default | 2px | 0.3s |
| `ease-underline-right` | Right to left | Default | 2px | 0.3s |
| `ease-underline-center-primary` | Center | #667eea | 2px | 0.3s |
| `ease-underline-center-success` | Center | #10b981 | 2px | 0.3s |
| `ease-underline-center-danger` | Center | #ef4444 | 2px | 0.3s |
| `ease-underline-center-warning` | Center | #f59e0b | 2px | 0.3s |
| `ease-underline-center-thin` | Center | Default | 1px | 0.3s |
| `ease-underline-center-thick` | Center | Default | 4px | 0.3s |
| `ease-underline-center-fast` | Center | Default | 2px | 0.15s |
| `ease-underline-center-slow` | Center | Default | 2px | 0.6s |

## Usage

```html
<!-- Center underline -->
<a href="#" class="ease-underline-center">Link</a>

<!-- Left to right -->
<a href="#" class="ease-underline-left">Link</a>

<!-- Right to left -->
<a href="#" class="ease-underline-right">Link</a>

<!-- With color -->
<a href="#" class="ease-underline-center-primary">Primary</a>
