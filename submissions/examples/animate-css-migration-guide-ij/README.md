# Component: Animate.css → EaseMotion Migration Guide

This submission implements a migration guide from Animate.css to EaseMotion for issue **#13629**.

## Description

A migration guide table mapping common Animate.css classes to their EaseMotion equivalents. Includes a checklist to ensure a smooth transition between libraries. Covers entrance effects, emphasis effects, delay classes, and speed modifiers.

## Acceptance Criteria

- Migration table with 14+ class mappings
- Notes explaining naming differences (e.g., ms vs s delays)
- Migration checklist with actionable steps
- Covers prefixes, delays, base classes, and CDN replacement
