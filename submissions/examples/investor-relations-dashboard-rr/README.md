# Investor Relations Dashboard Component

## What does this do?

Provides a professional investor-focused dashboard displaying funding, valuation, growth metrics, investor activity, and business performance indicators.

## How is it used?

```html
<div class="card valuation">
    <h2>$85M</h2>
    <span>Company Valuation</span>
</div>
```

## Why is it useful?

Investor dashboards help startups and businesses communicate performance, funding status, and growth metrics to stakeholders. This component provides a clean and responsive investor relations interface built entirely with HTML and CSS.
