Global Reduced Motion — EaseMotion CSS

How to test: Enable "Reduce motion" in your operating system accessibility settings, then refresh this page. Animations will automatically be minimized while content remains fully functional.

Windows: Settings → Accessibility → Visual effects → Animation effects (OFF)
macOS: System Settings → Accessibility → Display → Reduce motion (ON)
Android/iOS: Accessibility → Motion → Reduce Motion (ON)
