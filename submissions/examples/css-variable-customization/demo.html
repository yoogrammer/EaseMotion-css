EaseMotion Animation Playground

Demonstration of animation customization using CSS custom properties. Change duration, delay and easing globally or per component without modifying animation definitions.

Overview

Why CSS Variables?

EaseMotion traditionally relies on predefined animation values. CSS custom properties provide a scalable mechanism for globally controlling timing and easing behavior while preserving existing animation APIs.

Global Configuration

Configure animation timing once and apply it across the entire application.

Component Overrides

Customize behavior for specific components using local variable overrides.

Maintainability

Centralized animation control improves consistency and long-term maintainability.

Variables

Supported CSS Custom Properties 

:root {
  --easemotion-duration: 500ms;
  --easemotion-delay: 100ms;
  --easemotion-easing: ease-in-out;
}

--easemotion-duration

Controls overall animation duration.

--easemotion-delay

Controls animation start delay.

--easemotion-easing

Controls animation timing function.
