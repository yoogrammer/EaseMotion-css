# Font Weight Utilities

## What does this do?

Provides reusable utility classes for different font weights.

## How is it used?

```html
<p class="font-thin">Thin text</p>
<p class="font-light">Light text</p>
<p class="font-normal">Normal text</p>
<p class="font-medium">Medium text</p>
<p class="font-semibold">Semi Bold text</p>
<p class="font-bold">Bold text</p>
<p class="font-extrabold">Extra Bold text</p>
```

## Why is it useful?

Font weight utilities make typography easier to control using clear and readable class names. They align with EaseMotion CSS's goal of creating human-friendly and reusable utility classes.
