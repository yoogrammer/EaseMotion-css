# 💵 Tip Calculator

A simple, fast tip calculator that helps you split bills fairly among friends.

## 🚀 Features

- **Bill Amount**: Enter the total bill
- **Tip Percentage**: Adjust from 0% to 30% using slider
- **Number of People**: Split among any number of people
- **Real-time Results**: Updates automatically as you change values
- **Clear Display**: Shows tip amount, total bill, and per-person costs

## 📊 How It Works

### Formula
Tip Amount = Bill × (Tip Percentage / 100)
Total Bill = Bill + Tip Amount
Per Person = Total Bill / Number of People
Tip Per Person = Tip Amount / Number of People
