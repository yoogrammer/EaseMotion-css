# ease-list-disc / ease-list-decimal / ease-list-square / ease-list-none

Controls the `list-style-type` on ordered and unordered lists.

## Usage

```html
<ul class="ease-list-disc">
  <li>Bullet list</li>
</ul>
<ol class="ease-list-decimal">
  <li>Numbered list</li>
</ol>
<ul class="ease-list-none">
  <li>No marker</li>
</ul>
```

## Classes

| Class               | CSS Property             |
| ------------------- | ------------------------ |
| `ease-list-disc`    | list-style-type: disc    |
| `ease-list-decimal` | list-style-type: decimal |
| `ease-list-square`  | list-style-type: square  |
| `ease-list-none`    | list-style-type: none    |
