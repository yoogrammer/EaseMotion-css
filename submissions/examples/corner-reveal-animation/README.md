# Corner Reveal Animation

## What does this do?

Creates an animated corner framing effect on hover. Instead of animating a full border, only the corner segments appear and expand into place.

## Features

* Pure CSS
* No JavaScript
* Lightweight
* Responsive
* Reusable across cards, buttons, and containers
* Customizable corner size

## Usage

```html
<div class="corner-reveal">
  Content
</div>
```

## Included Examples

* Feature Card
* Button
* Container
* Large Corner Variant

## Why is it useful?

This effect provides a modern alternative to traditional border hover animations while maintaining a clean and minimal design style.
