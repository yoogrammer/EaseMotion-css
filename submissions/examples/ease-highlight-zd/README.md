# Ease Highlight Zd

## What does this do?
An animated text highlight component with inline mark highlights (sweep reveal animation) and block highlight cards (colored left border) in 6 colors — pure HTML and CSS.

## How is it used?
```html
<!-- Inline highlight -->
<p>This is <span class="hl-mark hl-yellow">highlighted text</span> in a sentence.</p>

<!-- Block highlight -->
<div class="hl-block hl-block-blue">
  <div class="hl-block-title">Note</div>
  <div class="hl-block-desc">Content here...</div>
</div>
```

Inline colors: hl-yellow, hl-green, hl-blue, hl-pink, hl-purple, hl-orange
Block variants: hl-block-yellow, hl-block-green, hl-block-blue, hl-block-pink

## Why is it useful?
Text highlights are essential for drawing attention to key information in documentation, blog posts, feature announcements, and educational content. This component provides inline sweep-reveal highlights and block highlight cards with 6 color variants, entrance animations, hover effects, and prefers-reduced-motion support — all pure CSS.
