# Interactive Holographic Event Ticket

## What does this do?
This is a premium, interactive 3D event ticket featuring a dynamic holographic foil shine that tracks mouse movement, a frosted glass backdrop, and authentic perforated edges.

## How is it used?
Copy the HTML structure and CSS into your project. Include the brief inline JavaScript snippet attached to the `demo.html` to enable the 3D mouse tracking and gradient angle calculation.

```html
<div class="holographic-ticket-container">
  <div class="holographic-ticket" id="holoTicket">
    <div class="ticket-foil"></div>
    <div class="ticket-content">
      <!-- Your ticket content here -->
    </div>
  </div>
</div>
```

## Why is it useful?
EaseMotion CSS prioritizes visually stunning, premium motion graphics. This component leverages advanced CSS `perspective`, `transform-style: preserve-3d`, and `mix-blend-mode: color-dodge` to achieve an ultra-realistic holographic foil material effect that elevates any event, ticketing, or VIP dashboard UI.
