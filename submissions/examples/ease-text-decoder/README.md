# ease-text-decoder

Hacker-style decoding text reveal — characters randomise then resolve into the final text on hover.

## Usage

```html
<span class="ease-decoder">Hello World</span>
```

Include the companion script. It stores the original text in `data-text` and randomises characters progressively on `mouseenter`.

## Why it fits EaseMotion CSS

CSS provides the font and cursor style. The script handles the character cycling — clean separation of visual style and behaviour.
