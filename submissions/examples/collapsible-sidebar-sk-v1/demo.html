Reusable Sidebar Pattern

This example demonstrates a complete dashboard shell with a collapsible sidebar. Key features include:

  • Smooth Transitions: Uses EaseMotion's medium speed and standard easing for fluid expand/collapse.
  • Icon-only State: Automatically hides labels and adjusts spacing when collapsed.
  • Nested Menus: Supports hierarchical navigation with animated dropdowns.
  • Responsive Design: Switches to an overlay drawer on mobile devices with a floating toggle.
  • Reduced Motion: Respects prefers-reduced-motion media query by disabling transitions.
