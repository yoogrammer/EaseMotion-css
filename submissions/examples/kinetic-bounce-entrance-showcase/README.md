# Kinetic Bounce Entrance Showcase

Issue: #10388

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Features

- Pure CSS
- No JavaScript required
- Multiple bounce variants
- Staggered animation support
- Responsive design
- Lightweight implementation
- Reusable utility classes

## Usage

```html
<div class="ease-bounce-in">
  Content
</div>
```

## Available Variants

```html
<div class="ease-bounce-in"></div>
<div class="ease-bounce-in-up"></div>
<div class="ease-bounce-in-left"></div>
<div class="ease-bounce-in-right"></div>
```

## Delay Utilities

```html
<div class="ease-bounce-in delay-1"></div>
<div class="ease-bounce-in delay-2"></div>
<div class="ease-bounce-in delay-3"></div>
```

## Why It Fits EaseMotion CSS

- Human-readable class names
- Animation-first approach
- Easy integration
- Lightweight and reusable
- Practical real-world UI animations
