# ⚡ Ease Neon Text Flicker Animation

Realistic neon sign style text animations for EaseMotion CSS.

## ✨ What it does
Transforms any text element into a glowing neon sign with a 
realistic flicker animation. Supports 6 color variants and 
3 speed variants.

## 🚀 How to Use
```html
<h1 class="ease-neon-flicker ease-neon-flicker--pink">
  Neon Text
</h1>
```

## 🎨 Color Variants
| Class | Color |
|-------|-------|
| `ease-neon-flicker--pink` | Pink |
| `ease-neon-flicker--cyan` | Cyan |
| `ease-neon-flicker--green` | Green |
| `ease-neon-flicker--purple` | Purple |
| `ease-neon-flicker--yellow` | Yellow |
| `ease-neon-flicker--red` | Red |

## ⚡ Speed Variants
| Class | Speed |
|-------|-------|
| `ease-neon-flicker--slow` | 6s |
| *(default)* | 3s |
| `ease-neon-flicker--fast` | 1.5s |
