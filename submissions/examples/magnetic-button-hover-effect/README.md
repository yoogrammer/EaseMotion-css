# Magnetic Button Hover Effect

## Description

A premium micro-interaction that creates a magnetic attraction effect between the cursor and button.

## Features

- Cursor-following movement
- Smooth hover animation
- Lightweight implementation
- Interactive user feedback

## Files

- demo.html
- style.css
- script.js

## Usage

Open `demo.html` and move your cursor over the button.

- Chrome
- Firefox
- Edge
