# ease-btn-pill

Pill/capsule shaped button variant using `border-radius: 9999px`. Composable with all button variants and sizes.

## Usage

```html
<button class="ease-btn ease-btn-primary ease-btn-pill">Pill Button</button>
```

## Variants

| Class | Description |
|---|---|
| `ease-btn-pill` | Fully rounded button (9999px radius) |
| `ease-btn-pill-icon` | Square pill for icon-only buttons |
| `ease-btn-pill-group` | Grouped pill buttons joined together |

## Compatible classes

| Class | Works with |
|---|---|
| `ease-btn-primary/success/danger/outline/ghost/link` | All variants |
| `ease-btn-sm/lg/xl` | All sizes |
| `ease-btn-hover` | Hover lift effect |
| `ease-btn-block` | Full width |

## Notes

- Combine with any button variant and size
- Pairs well with `ease-btn-hover` for lift effect
- Icon buttons: use `ease-btn-pill-icon` for equal width/height
