Navigation Button Vertical Alignment Fix

This page demonstrates the misalignment issue affecting the Live Demo button in the documentation header, alongside the applied CSS flexbox fix.

❌ Misaligned Button (Before Fix)

Without flexbox centering on the list items (li), block-level list-item layout aligns on text baselines. The large height of the inline-flex button pushes its box layout down relative to text links, causing it to sit slightly below the centerline.

✅ Aligned Button (After Fix)
