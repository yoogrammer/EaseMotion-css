# EaseOTP Input Component

An interactive, responsive 4-6 digit OTP/PIN confirmation dashboard component built with accessible keyboard navigation guidelines.

## Features Included
- **Auto-Advance:** Focus automatically moves to the next block on input entry.
- **Smart Backspacing:** Smooth backward focus shifting when hitting backspace on empty entries.
- **Clipboard Paste Routing:** Splitting and populating numeric clipboard text blocks across distinct input boxes natively.
- **Dynamic CSS Focus Enhancements:** Interactive scale modifications paired with smooth highlight accents upon manual targeting.
- **Visual Alert Modules:** Keyframe shake transitions mapping to error validations alongside subtle filled pulse prompts.
