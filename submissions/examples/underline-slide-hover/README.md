# Underline Slide Hover

A lightweight hover effect that animates an underline from left to right using pure CSS.

## Features

- Pure CSS
- No JavaScript required
- Works on links
- Works on buttons
- Keyboard accessible using `:focus-visible`
- Easy to customize

## Basic Usage

```html
<a href="#" class="ease-underline-slide">
  Link
</a>
```

## Available Variants

### Default

```html
<a href="#" class="ease-underline-slide">
  Link
</a>
```

### Center Origin

```html
<a href="#" class="ease-underline-slide-center">
  Link
</a>
```

### Accent Underline

```html
<a href="#" class="ease-underline-slide-accent">
  Link
</a>
```

## Why It Fits EaseMotion CSS

- Human-readable class names
- Animation-first design
- Lightweight implementation
- Easy integration into existing projects
- Demonstrates practical hover interactions
