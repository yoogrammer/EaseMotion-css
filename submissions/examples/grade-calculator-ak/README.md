# Grade Calculator

## What does this do?
A weighted grade calculator that organizes scores by category, calculates current overall grade, and provides what-if projection for future assignments.

## How is it used?
Add categories with weights, enter assignment scores within each, and see real-time grade calculation. Use the what-if section to project future scores.

## Why is this useful?
Students can track their current grade across weighted categories and plan what scores they need on remaining assignments to hit their target grade.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to use the calculator.
