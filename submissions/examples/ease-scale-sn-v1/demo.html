Scale Animation Utilities

Entry animations and hover scale effects for interactive elements like cards, buttons, and images.

Entry Animations

.animate-scale-up and .animate-scale-down play on page load. Refresh to replay.

+
.animate-scale-up Scale from 0 to 1
-
.animate-scale-down Scale from 1.2 to 1
!
.animate-scale-up.scale-fast Fast entrance (0.2s)
#
style="--scale-origin: bottom left" Slow entrance from corner

Hover Scale

Hover over the cards to see scale effects.

1
Hover to scale up (1.1x)
.hover-scale-up
2
Hover to scale down (0.9x)
.hover-scale-down
3
Custom 1.2x scale up
style="--scale-amount: 1.2"

CSS Custom Properties

--scale-origin

Default: center

Sets the transform-origin for entry and hover animations.

--scale-amount

Default: 1.1 / 0.9

Controls the scale factor on hover for hover-scale classes.
