 
# Width Utilities

Width utilities: full, half, third, quarter, three-quarter, screen, auto.

## Files

- demo.html - Interactive demo
- style.css - Width utilities styles
- README.md - Documentation

## Classes

| Class | Width |
|-------|-------|
| `ease-w-full` | 100% |
| `ease-w-half` | 50% |
| `ease-w-third` | 33.333% |
| `ease-w-quarter` | 25% |
| `ease-w-three-quarter` | 75% |
| `ease-w-screen` | 100vw |
| `ease-w-auto` | auto |

## Usage

```html
<div class="ease-w-full">Full width</div>
<div class="ease-w-half">Half width</div>
<div class="ease-w-third">Third width</div>

<!-- Three column layout -->
<div class="row">
    <div class="ease-w-third">Column 1</div>
    <div class="ease-w-third">Column 2</div>
    <div class="ease-w-third">Column 3</div>
</div>
