CSS Custom Properties

Customize underline color and thickness with CSS custom properties. Defaults: color #f97316, thickness 2px.

--underline-color

Default: #f97316

Sets the color of the underline decoration for all utility classes.

--underline-thickness

Default: 2px

Controls the thickness of the underline. Accepts any valid CSS length value.

Custom Styling Examples

Use inline style attributes to customize --underline-color and --underline-thickness.

Cyan Thick --underline-color: #06b6d4; --underline-thickness: 3px Custom cyan color, 3px thick
Purple Thin --underline-color: #a855f7; --underline-thickness: 1px Purple center draw, thin
Green Fade --underline-color: #22c55e Green opacity fade
Pink Wave --underline-color: #ec4899; --underline-thickness: 3px Pink wavy underline
