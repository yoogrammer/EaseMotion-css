# Mask Image Gradient Fade

A CSS demo of mask-image with linear-gradient to create smooth fade-out effects. Elements fade into the background without affecting their internal opacity.

## Files

- `demo.html` - Three mask fade variants
- `style.css` - mask-image with linear-gradient and radial-gradient, -webkit-mask-image prefix

## Usage

Open `demo.html` to see horizontal, vertical, and radial mask fade effects.
