# Ease Feature Grid Zd

## What does this do?
An animated feature/service card grid component with icon containers, title, description, link, hover lift, icon pop animation, and 6 color variants — pure HTML and CSS.

## How is it used?
```html
<div class="fg-grid">
  <div class="fg-card">
    <div class="fg-icon-wrap fg-icon-purple" aria-hidden="true">⚡</div>
    <div class="fg-title">Feature Title</div>
    <div class="fg-desc">Feature description text...</div>
    <a class="fg-link" href="#">Learn more →</a>
  </div>
</div>
```

Icon colors: fg-icon-purple, fg-icon-pink, fg-icon-green, fg-icon-yellow, fg-icon-blue, fg-icon-red

## Why is it useful?
Feature grids are a staple of landing pages, marketing sites, and product showcases. This component provides a cohesive, animated grid with icon pop, hover lift, and link hover effects.
