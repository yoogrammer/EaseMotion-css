# ease-items-baseline / ease-self-auto

Controls `align-items` and `align-self` for flexbox/grid layout alignment.

## Usage

```html
<div class="ease-flex ease-items-baseline">
  <span>Items align by baseline</span>
</div>
<div class="ease-self-auto">Override individual item alignment</div>
```

## Classes

| Class                 | CSS Property            |
| --------------------- | ----------------------- |
| `ease-items-baseline` | `align-items: baseline` |
| `ease-self-auto`      | `align-self: auto`      |
