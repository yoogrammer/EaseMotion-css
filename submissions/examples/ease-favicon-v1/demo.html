Favicon Fix Demo

This demo shows how to correctly add a favicon to all EaseMotion CSS demo pages so browser tabs display a branded icon instead of a blank default.

❌

Before Fix

Browser tab shows a blank/default icon. No favicon is linked in the HTML head.

<!-- No favicon link -->
✅

After Fix

Browser tab now shows the EaseMotion branded favicon icon correctly.

<link rel="icon" href="favicon.svg" />
📋

How to Use

Add this line inside the <head> tag of every demo HTML file in the project.

<link rel="icon" type="image/svg+xml" href="favicon.svg" />

📄 Full Implementation

Add the following snippet to the <head> of every HTML file:

<!-- Favicon for browser tab branding -->
<link rel="icon" type="image/svg+xml" href="/favicon.svg" />
<link rel="icon" type="image/png" href="/favicon.png" />
<link rel="shortcut icon" href="/favicon.ico" />
