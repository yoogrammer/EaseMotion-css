# Animation: Interactive Kanban Board

A Kanban-style task board inspired by modern project management tools. Columns and task cards respond to hover interactions with subtle elevation and emphasis animations.

## Features

- Pure CSS interaction
- Kanban board layout
- Interactive task cards
- Column highlight effect
- Responsive design
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="column">

    <h2>Todo</h2>

    <div class="task">
        Design landing page
    </div>

</div>
```

Hovering task cards elevates them while highlighting their current workflow state.

## Why is it useful?

Kanban boards are widely used in project management, product planning, issue tracking, and team collaboration tools. This component provides a visually engaging way to present workflow progress while maintaining clarity and organization.
