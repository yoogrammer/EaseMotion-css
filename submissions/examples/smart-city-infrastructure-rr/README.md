# Smart City Infrastructure Visualization Component

## What does this do?

Visualizes a connected smart city ecosystem consisting of transportation systems, energy grids, public safety services, environmental monitoring, IoT sensors, and analytics platforms.

## How is it used?

```html
<div class="node center">
    City Operations Center
</div>

<div class="node iot">
    IoT Sensors
</div>
```

## Why is it useful?

Smart cities rely on interconnected infrastructure and real-time data systems to improve efficiency, sustainability, and public services. This component provides a clean visualization of those relationships using pure HTML and CSS.
