## What does this do?

Provides `ease-float` utility classes for floating elements left, right, or clearing floats.

## How is it used?

```html
<img class="ease-float-left-za" src="..." />
<img class="ease-float-right-za" src="..." />
<div class="ease-clear-both-za">Cleared content</div>
```

## Why is it useful?

EaseMotion core has no float or clear utility classes. While modern layout uses flexbox/grid, floats are still needed for text wrapping around images, print stylesheets, and legacy content integration.
