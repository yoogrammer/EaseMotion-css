 
# Hover Line Height Grow

Line height expands on hover giving text more room to breathe — line-height: normal → 1.8 with smooth 0.3s transition.

## Files

- demo.html - Interactive demo
- style.css - Line height styles
- README.md - Documentation

## Classes

| Class | Target Line Height | Speed |
|-------|-------------------|-------|
| `ease-hover-line-height-grow` | 1.8 | 0.3s |
| `ease-hover-line-height-1-5` | 1.5 | 0.3s |
| `ease-hover-line-height-2-0` | 2.0 | 0.3s |
| `ease-hover-line-height-2-5` | 2.5 | 0.3s |
| `ease-hover-line-height-fast` | 1.8 | 0.1s |
| `ease-hover-line-height-slow` | 1.8 | 0.6s |

## Usage

```html
<div class="ease-hover-line-height-grow">
    <p>Text with line height that expands on hover</p>
</div>
