# Calculator App

## What does this do?
A fully functional calculator supporting addition, subtraction, multiplication, division, percentage, sign toggle, and decimal inputs with a clean grid layout.

## How is it used?
Open demo.html in a browser. Click number and operator buttons to perform calculations. Press C to clear, = to evaluate.

## Why is this useful?
Demonstrates a classic calculator UI with button grid, expression evaluation logic, and display formatting.

## Tech Stack
- HTML
- CSS
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
