# CSS Outline Creative Border Effects Example

Creative border effects using CSS `outline` and `outline-offset` with hover-triggered animations.

## Features

- Four border styles: double outline, offset glow, dashed reveal, corner brackets
- Uses `outline`, `outline-offset`, and `border` combinations
- Hover animations for interactive effects
- Dark theme with `prefers-reduced-motion` support

## Files

- `demo.html` — Border effects card layout
- `style.css` — Self-contained CSS (85 lines)
