# Animated Grid Background

## What it does

Dynamic canvas-based grid background with sine wave distortion on both axes. Creates a flowing, organic grid effect.

## How to use it

<canvas id="gridCanvas" class="ease-grid-canvas"></canvas>

## Features

- Horizontal and vertical grid lines
- Sine wave distortion on both axes
- Neon cyan glow effect
- 60fps smooth animation
- Full responsive

## Why it fits EaseMotion CSS

- Pure Canvas animation
- Organic flowing movement
- Cyberpunk aesthetic
- Performance optimized
