ease-modal

CSS-only modal dialog — checkbox toggle with animated backdrop + panel

Confirm Action

This modal is powered entirely by CSS — no JavaScript required. The hidden checkbox toggles the overlay and panel visibility via the :checked pseudo-class.

Delete Item

Are you sure you want to delete this item? This action cannot be undone.

Uses <input type="checkbox"> + :checked sibling selector. Click the backdrop or close button to dismiss. Works without JavaScript.

Bonus: :target variant

An alternative approach uses the :target pseudo-class with URL hashes. Replace <input> + <label> with <a href="#modal-id"> triggers and <div id="modal-id" class="ease-modal-overlay">. See the README for both patterns.
