# Skeleton Loading Screens

## What does this do?
Skeleton loading screens with shimmer animation that simulate content loading. Shows skeleton placeholders for 2 seconds before revealing the actual content.

## How is it used?
Open demo.html in a browser. Watch the skeleton placeholders with shimmer effect for 2 seconds, then the actual content fades in.

## Why is this useful?
Demonstrates a skeleton loading pattern that improves perceived performance by showing placeholder content while real content loads.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
