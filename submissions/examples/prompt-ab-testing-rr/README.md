# Prompt A/B Testing Component

## What does this do?

Provides a dashboard interface for comparing two prompt variations and evaluating their performance metrics.

## How is it used?

```html
<div class="prompt-card">
    <div class="badge">Prompt A</div>
</div>
```

## Why is it useful?

Prompt engineering often requires testing multiple prompt variations to determine which performs better. This component provides a structured and visually appealing comparison interface while demonstrating responsive layouts and modern AI-focused design patterns.
