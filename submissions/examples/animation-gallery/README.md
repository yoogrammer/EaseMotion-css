# Visual Class Reference Gallery

## 1. What does this do?
Provides an interactive, categorized visual dashboard to preview animation utility classes without checking the underlying source CSS files.

## 2. How is it used?
Developers browse categories (entrance, hover, continuous, exit) and click on any utility class badge snippet to instantly copy it to their clipboard:
```html
<div class="demo-box animate-pop-hover">
    <!-- Component Element -->
</div>
