# Event-Driven Architecture Visualization Component

## What does this do?

Visualizes how services communicate through an event-driven architecture using producers, an event bus, and consumers.

## How is it used?

```html
<div class="node producer">
    User Service
</div>

<div class="bus">
    Event Bus
</div>

<div class="node consumer">
    Email Service
</div>
```

## Why is it useful?

Event-driven systems are widely used in modern cloud-native applications. This component helps visualize asynchronous communication patterns, making it useful for architecture documentation, engineering blogs, technical portfolios, and educational content.
