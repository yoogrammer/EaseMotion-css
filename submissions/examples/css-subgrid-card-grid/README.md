# CSS Subgrid Card Grid Example

A card grid layout using CSS `subgrid` to align content across rows and columns within a nested grid.

## Features

- Parent grid with `grid-template-rows: subgrid` for aligned card content
- Cards with header, body, and footer that stay vertically aligned across rows
- Responsive column count
- Dark theme with `prefers-reduced-motion` support

## Files

- `demo.html` — Subgrid-aligned card grid
- `style.css` — Self-contained CSS (85 lines)
