# Ease Fern Unfurl Animation

An elegant CSS-based animation that simulates the natural, organic unfurling of a fern leaf using smooth custom easing curves.

## How to View
1. Open `demo.html` in any modern web browser.
2. Click the "Replay" button to watch the animation trigger again.

## Technical Details
- **Tech Stack:** HTML5, CSS3 (Keyframes & Transforms)
- **Easing:** `cubic-bezier(0.25, 1, 0.5, 1)` for an organic, lifelike motion profile.
