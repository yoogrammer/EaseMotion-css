# Truth or Dare

## What does this do?
A fun Truth or Dare party game with flip-animation cards, 22+ pre-loaded questions for each category, player turn tracking, and shuffle functionality.

## How is it used?
Click a Truth or Dare card to flip it and reveal a random question. Use Next Player to advance turns. Shuffle to re-randomize all questions.

## Why is this useful?
Perfect for parties, sleepovers, or icebreaker activities — no setup required, just open and play.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to start playing.
