# Corner Ribbon

## What it does

Adds an animated ribbon badge to the corner of cards or containers.

## How to use it

<div class="ease-corner-ribbon">NEW</div>

## Variants

- ease-ribbon-sale - Red/pink gradient
- ease-ribbon-popular - Yellow/gold gradient
- ease-ribbon-limited - Green gradient
- ease-ribbon-glow - Pulsing glow effect
- ease-ribbon-pulse - Subtle shaking animation
- ease-corner-ribbon-left - Positions on left corner
- ease-ribbon-small/large - Size variants

## Why it fits EaseMotion CSS

- Pure CSS
- Multiple color variants
- Animated glow and pulse options
- Easy to use
