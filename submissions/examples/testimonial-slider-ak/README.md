## What does this do?

This component renders a testimonial slider with quote cards, author names, star ratings, and dot navigation to cycle through multiple testimonials. It auto-advances every 5 seconds.

## How is it used?

Include the `style.css` and the HTML structure. Each `.slide` contains a quote, star rating, and author details. Dot navigation is generated dynamically by the JavaScript. Click a dot to jump to a specific testimonial.

## Why is this useful?

Testimonial sliders are a staple of marketing pages and product landing sites. This component provides a polished, accessible way to showcase customer feedback without overwhelming the layout.

## Tech Stack

- HTML
- CSS
- JavaScript (vanilla)
