# Section Dividers with Animated Variants

## 1. What does this do?
Provides section divider/separator components in multiple visual styles: horizontal rule with optional centered text, wave, arrow/chevron, dots, and an animated gradient shimmer divider.

## 2. How is it used?

```html
<!-- Basic horizontal rule -->
<div class="divider"></div>

<!-- With centered label -->
<div class="divider">Continue Reading</div>

<!-- Wave separator -->
<div class="divider-wave"></div>

<!-- Arrow/chevron separator -->
<div class="divider-arrow"></div>

<!-- Dot pattern -->
<div class="divider-dots">
  <span></span><span></span><span></span>
</div>

<!-- Animated gradient line -->
<div class="divider-animated"></div>
```

## 3. Why is it useful?
EaseMotion CSS had no section divider component. Dividers are a fundamental layout element for landing pages and content-heavy sites. This provides multiple visual styles — from simple lines to animated gradients — so developers can choose the right visual separation without custom SVG or CSS.
