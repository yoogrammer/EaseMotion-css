# ease-rating

A CSS-only star rating component built using radio inputs and reverse sibling selectors.

## Features

- Pure CSS implementation
- Radio input based
- Hover states
- Selected states
- Reverse sibling selector technique
- Half-star variant support
- No JavaScript required

## Usage

```html
<div class="ease-rating">
    <input type="radio" id="star5" name="rating">
    <label for="star5">★</label>

    <input type="radio" id="star4" name="rating">
    <label for="star4">★</label>

    <input type="radio" id="star3" name="rating">
    <label for="star3">★</label>

    <input type="radio" id="star2" name="rating">
    <label for="star2">★</label>

    <input type="radio" id="star1" name="rating">
    <label for="star1">★</label>
</div>
```

## Why it fits EaseMotion CSS

This component provides a common UI interaction pattern using only HTML and CSS. It combines hover feedback, selection states, and smooth transitions while remaining lightweight and framework-independent.
