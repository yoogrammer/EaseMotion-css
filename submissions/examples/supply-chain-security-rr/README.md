# Software Supply Chain Security Dashboard Component

## What does this do?

Provides a centralized dashboard for monitoring dependency health, software vulnerabilities, artifact verification, release integrity, and supply chain security posture.

## How is it used?

```html
<div class="package-card healthy">
    <h3>Frontend Dependencies</h3>
</div>
```

## Why is it useful?

Modern software delivery depends on secure dependencies, trusted artifacts, and verified releases. This component provides a reusable DevSecOps dashboard while demonstrating software supply chain security and secure SDLC design patterns.
