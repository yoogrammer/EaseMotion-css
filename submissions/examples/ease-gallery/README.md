\# ease-gallery



Responsive image gallery grid component with hover zoom effect.



\## Classes



| Class | Description |

|-------|-------------|

| `ease-gallery` | CSS grid wrapper for gallery |

| `ease-gallery-masonry` | Masonry (Pinterest-style) variant |

| `ease-gallery-item` | Individual gallery item |

| `ease-gallery-image` | Image container |

| `ease-gallery-caption` | Image caption |

| `ease-gallery-overlay` | Overlay effect on hover |

| `ease-gallery-tall` | Tall item for masonry |

| `ease-gallery-wide` | Wide item for masonry |



\## Features



\- 🖼️ Responsive CSS grid layout

\- 🔍 Hover zoom effect on images

\- 🧩 Masonry variant (Pinterest-style)

\- 🎨 Overlay effect variant

\- 📱 Mobile responsive



\## Usage



```html

<!-- Basic Gallery -->

<div class="ease-gallery">

&#x20;   <div class="ease-gallery-item">

&#x20;       <div class="ease-gallery-image">🖼️</div>

&#x20;       <div class="ease-gallery-caption">Caption</div>

&#x20;   </div>

</div>



<!-- Masonry Gallery -->

<div class="ease-gallery-masonry">

&#x20;   <div class="ease-gallery-item ease-gallery-tall">...</div>

</div>
