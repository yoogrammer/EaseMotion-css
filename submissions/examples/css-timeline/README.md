# CSS Timeline

## What does this do?
A vertical timeline with alternating left/right cards connected by a central line, with hover animations on each card.

## How is it used?
Add the timeline markup to any page:

    <div class="timeline">
      <div class="timeline-item timeline-item--left">
        <div class="timeline-dot"></div>
        <div class="timeline-card">
          <h3>Year</h3>
          <p>Description</p>
        </div>
      </div>
    </div>

## Why is it useful?
Presents chronological events or milestones in a visually structured, engaging format ideal for company histories, project roadmaps, and career timelines.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
