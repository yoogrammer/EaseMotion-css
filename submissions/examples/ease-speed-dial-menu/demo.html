Speed Dial Menu

A floating action button that expands into multiple action buttons with smooth staggered animations. Click the button at the bottom right!

🎨 Variants

🔵

Default (Vertical)

Buttons fan out vertically upward from the trigger button.

ease-speed-dial
🟣

Horizontal

Buttons fan out horizontally to the left of the trigger.

ease-speed-dial--horizontal
🟢

Color Variants

Trigger button supports indigo, rose, emerald color themes.

ease-speed-dial--rose

📄 How to Use

Add ease-speed-dial anywhere in your HTML:

<div class="ease-speed-dial" id="myDial">
  <button class="ease-speed-dial__trigger">+</button>
  <div class="ease-speed-dial__menu">
    <button class="ease-speed-dial__item" title="Share">📤</button>
    <button class="ease-speed-dial__item" title="Edit">✏️</button>
    <button class="ease-speed-dial__item" title="Delete">🗑️</button>
    <button class="ease-speed-dial__item" title="Save">💾</button>
  </div>
</div>
