Technical Specifications

  • animation-timeline: scroll(root) — Maps keyframe progression directly to the document root element's scroll percentage.
  • animation-range: 0px 150px — Locks the transition start to exactly 0px scroll position and ends it at 150px, preventing layout shifting on deeper scrolls.
  • Progressive Enhancement — Employs @supports queries so browsers without scroll-driven animation support fall back to a static glassmorphic navbar.
  • Accessibility Compliant — Detects prefers-reduced-motion: reduce and disables size morphs, presenting a static header to prevent scrolling motion sickness.
