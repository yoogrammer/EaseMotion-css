@property Registered Custom Properties

Register typed CSS custom properties with @property so the browser can interpolate them in animations.

Registered Angle

Gradient Spin

A registered <angle> property animates the gradient rotation smoothly. Without @property, the browser cannot interpolate custom properties used in conic-gradient().

Registered Length

Pulse

A registered <length-percentage> property controls border-radius — animating from rounded to circular.

Registered Color

Color Shift

A registered <color> property transitions smoothly between colors in a gradient background.
