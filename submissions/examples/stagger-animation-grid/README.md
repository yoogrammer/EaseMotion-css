# Stagger Animation Grid

## What does this do?
Grid items animate in with staggered delays creating a wave effect using CSS custom properties.

## How is it used?
Set `--i` index on each grid item:

    <div class="stagger-item" style="--i:0">1</div>

## Why is it useful?
Adds polished entrance animations for grids, galleries, and card layouts.

## Tech Stack
- HTML
- CSS
- Vanilla JS (replay)

## Preview
Open demo.html directly in your browser to see the effect.
