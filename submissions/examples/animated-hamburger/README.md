# Animated Hamburger to Close 'X' Toggle

A pure CSS hamburger menu icon that smoothly morphs into a cross ('X') when clicked. Built for the EaseMotion CSS library.

## Features
- Pure CSS logic using the invisible `:checked` pseudo-class hack.
- Smooth `transform` and `opacity` transitions.
- Fully adheres to the `ease-` class naming convention.
- Zero JavaScript required.
