# Card with Gradient Image Overlay

A card with a full-bleed gradient background and a linear gradient overlay that fades from transparent to dark at the bottom. The overlay ensures text readability on any image. Includes a tag badge, title, and description overlaid on the gradient.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser to view the gradient overlay cards.

## Accessibility notes

The gradient backgrounds are decorative. Text content has sufficient contrast against the dark gradient overlay.
