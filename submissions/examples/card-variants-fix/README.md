# EaseMotion CSS - Card Variants Showcase

This demo showcases different card UI variants built using EaseMotion CSS utilities.

---

## ✨ Variants Included

### 1. Base Card
Minimal card using only core `.ease-card` class.

### 2. Shadow Card
Adds elevation using `.ease-card-shadow`.

### 3. Hover Card
Interactive motion effect using `.ease-card-hover`.

### 4. Neumorphic Card
Soft UI style using `.ease-card-neumorphic`.

---

## 🚀 Features

- Clean grid layout
- Responsive auto-fit cards
- Multiple UI elevation styles
- Hover-based motion interaction
- Soft UI (neumorphism) support
