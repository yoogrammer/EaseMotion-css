EaseMotion Dark Mode Demo

Automatic Dark Mode

This page automatically switches to dark mode based on your system settings using @media prefers-color-scheme.

Manual Toggle

You can also manually toggle dark mode using the button above. It adds or removes the data-theme attribute on the wrapper.

CSS Variables Used

--ease-bg controls background color.

--ease-text controls text color.

--ease-primary controls button color.

--ease-surface controls card color.
