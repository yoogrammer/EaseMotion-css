# Fluid Typography Scale

## What does this do?
Demonstrates responsive type scaling using CSS clamp() for fluid typography.

## How is it used?
Apply clamp-based font-size classes:

    <p class="type-h1">Heading 1</p>
    <p class="type-body">Body text</p>

## Why is it useful?
Ensures text scales smoothly across all viewport sizes without media query breakpoints.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
