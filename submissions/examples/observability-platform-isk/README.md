# Tech — Observability Platform

## What does this do?
A complete, copy-paste ready full-page website for an Observability and APM SaaS named `OmniSentry`, featuring system utilization dashboard mockups, feature grids (APM, infra, logs), MTTR/MTTD performance counters, integration grids, interactive data-volume pricing calculators, and free trial forms.

## How is it used?
Open `demo.html` directly in any web browser to view the page. No build step is required. The page transition, reveal animations, and hover components use the project's standard CSS classes.

## Why is it useful?
This showcases how EaseMotion CSS can be applied to design premium Datadog-style DevOps SaaS dashboards with green progress bars, slider calculators, and clean monospace statistics grids.

## EaseMotion CSS Classes Showcased
- **Entrance & Hover Animations**:
  - `.ease-slide-in-left` / `.ease-slide-in-right`: Used for the hero content entrances.
  - `.ease-hover-grow`: Applied to header logo elements and button states.
  - `.ease-hover-lift`: Used for interactive pricing calculators.
  - `.ease-hover-underline`: Applied to nav menu links.
  - `.ease-card-lift`: Used for telemetry feature cards.
- **Scroll Reveal Animations**:
  - `.ease-reveal`: Applied to sections to hide elements offscreen.
  - `.ease-reveal-up` / `.ease-reveal-scale`: Sliding directional reveals.
  - `.ease-reveal-delay-1` / `.ease-reveal-delay-2` / `.ease-reveal-delay-3`: Staggered timing delays for grid columns or stats blocks.
