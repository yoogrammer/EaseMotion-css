# Animated Hamburger to Cross Menu Icon

Issue: [#9668](https://github.com/SAPTARSHI-coder/EaseMotion-css/issues/9668)

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A collection of animated hamburger menu icons that smoothly transform into a close (X) icon using pure CSS transitions and transforms.

## Features

- Hamburger to cross animation
- Pure CSS implementation
- Multiple visual variants
- Neon version
- Gradient version
- Glassmorphism version
- Responsive layout
- Hover interactions
- No JavaScript required

## Usage

```html
<label class="menu-card">

  <input type="checkbox">

  <span class="hamburger">
    <span></span>
    <span></span>
    <span></span>
  </span>

</label>
```

## Main Classes

```css
.hamburger
.neon
.gradient
.glass
.menu-card
```

## Why It Fits EaseMotion CSS

- Animation-first interaction
- Human-readable class names
- Reusable navigation pattern
- Lightweight implementation
- Pure CSS transitions
