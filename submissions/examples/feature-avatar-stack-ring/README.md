# Animated Avatar Stack

A CSS-only avatar group component with overlapping avatars, hover expansion, and animated online status indicators.

## Features

- ✨ Smooth hover animation
- 👥 Overlapping avatar layout
- ⬆️ Lift effect on hover
- 🟢 Animated online status
- 📱 Responsive design
- ⚡ Pure CSS
- 🚫 No JavaScript required

## How It Works

The avatars are partially overlapped using negative margins. When the stack is hovered, the avatars spread apart smoothly. Hovering an individual avatar raises it above the others for better emphasis.
