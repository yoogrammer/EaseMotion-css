# ease-fullscreen-icon-morph

Animated fullscreen control icon that morphs between expand and collapse states.

## Features

- Expand icon state
- Collapse icon state
- Smooth morph transition
- Hover animation
- Media player UI example
- Responsive design

## Acceptance Criteria Covered

### Icon Morphing

The icon transitions between:

- Expand state
- Collapse state

using CSS transforms.

### Fullscreen State Toggle

A fullscreen class changes the icon appearance to simulate entering and exiting fullscreen mode.

### Use Cases

- Video players
- Media streaming apps
- Presentation software
- Image galleries
