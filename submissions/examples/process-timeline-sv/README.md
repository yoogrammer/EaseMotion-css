# Animated Horizontal Process Timeline

## What does this do?

Creates a responsive animated timeline component for visualizing workflows, onboarding journeys, project milestones, learning paths, and multi-step processes.

## How is it used?

```html
<div class="timeline">
  <div class="step">
    <div class="circle">1</div>
    <h3>Research</h3>
  </div>
</div>
