# ease-squish-button

Squish/press animation on hover, focus, and active states. Scales X-axis up and Y-axis down for a playable squish effect.

## Usage

```html
<button class="ease-btn ease-squish-button">Squish me</button>
```

## Variants

| Class | Description |
|---|---|
| `ease-squish-button` | Default squish animation |
| `ease-squish-button-sm` | Subtle squish |
| `ease-squish-button-lg` | Strong squish |

## Notes

- Applies `scaleX(1.12)` + `scaleY(0.88)` on hover
- Active state adds stronger squish
- Great for playful button interactions
- Respects `prefers-reduced-motion`
