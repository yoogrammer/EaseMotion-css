## What does this do?

Provides extended cursor utility classes beyond the basic set in core (pointer, default, not-allowed). Covers grab, zoom, wait, progress, help, move, text, and crosshair cursors.

## How is it used?

```html
<div class="ease-cursor-grab-za">Draggable element</div>
<div class="ease-cursor-zoom-in-za">Zoomable image</div>
```

## Why is it useful?

EaseMotion core only has cursor-pointer, cursor-default, and cursor-not-allowed. Common interactive patterns like draggable elements (grab), image viewers (zoom-in/out), and loading states (wait/progress) require additional cursor variants.
