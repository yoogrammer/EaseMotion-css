animation-direction

Controls whether an animation plays forward, reverse, or alternates.

normal

Plays forward each cycle (0% → 100%).

●

reverse

Plays backward each cycle (100% → 0%).

●

alternate

Forward, then reverse, then forward... (0% → 100% → 0%).

●

alternate-reverse

Reverse, then forward, then reverse... (100% → 0% → 100%).

●

All run the same 2s animation, 2 iterations. Only animation-direction differs.
