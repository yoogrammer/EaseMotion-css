object-view-box

Crop and pan across replaced elements (images/videos) using CSS — no clipping.

Pan Across (inset pan)

object-view-box: inset(0 0 0 0) animates to inset(0 50% 0 0) — like a camera panning across a scene, but on a gradient element.

Zoom In/Out

object-view-box animates from showing the full element to zooming in on the center — a "Ken Burns" style effect using pure CSS.
