CSS Fluid Sizing

min() · max() · clamp()

A visual exploration of modern CSS mathematical functions for fluid typography, responsive containers, and adaptive layouts — with zero media queries.

min()

The Minimum Value

min() selects the smallest value from a comma-separated list. Perfect for capping element widths to prevent overflow on large screens.

width: min(90%, 600px)
This box caps at 600px

On wide screens the box stops at 600px. On narrow screens it fills 90% of the container.

padding: min(4vw, 2rem)
Fluid padding

Padding scales with the viewport but never exceeds 2rem.

Syntax
width: min(90%, 600px);

Use when you want something to be at most a certain size while staying responsive.

max()

The Maximum Value

max() selects the largest value from a comma-separated list. Ideal for enforcing minimum sizes while allowing growth.

width: max(320px, 50%)
This box never shrinks below 320px

The element will always be at least 320px wide, and expands to 50% of the parent when space allows.

font-size: max(1rem, 2.5vw)
Scaling text

The font-size grows with the viewport but is never smaller than 1rem.

Syntax
width: max(320px, 50%);

Use when you want something to be at least a certain size while remaining fluid.

clamp()

The Clamped Value

clamp(min, preferred, max) combines both min and max into one function. The go-to tool for fluid typography and truly responsive sizing.

font-size: clamp(1rem, 4vw, 3rem)

Fluid Typography

Headings that scale smoothly between 1rem and 3rem based on viewport width.

width: clamp(280px, 60%, 800px)
Fluid container

The element stretches from 280px to 800px, preferring 60% of the parent.

padding: clamp(0.5rem, 3vw, 2rem)
Adaptive spacing

Padding that feels spacious on large screens and compact on mobile, all without breakpoints.

gap: clamp(0.5rem, 2vw, 1.5rem)

Grid and flex gaps that scale naturally across breakpoints.

Syntax
font-size: clamp(1rem, 4vw, 3rem);

Use when you need a value that stays between a minimum and maximum, with a preferred value in between.

Best Practices

⬇ Use min() when You want to prevent an element from exceeding a maximum dimension while staying responsive below that threshold.
⬆ Use max() when You need to guarantee a minimum dimension for usability while allowing growth on larger viewports.
⬍ Use clamp() when You want true fluid behavior — a value that scales between a minimum and maximum with a preferred middle value.
