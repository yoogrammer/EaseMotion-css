# Live Digital Clock

A real-time digital clock display with a sleek dark glassmorphism design. Updates every second with current time (12-hour format) and full date.

## Usage

Open `demo.html` directly in any browser. No build step or server required.

## Why it fits EaseMotion CSS

Showcases dark-theme design tokens, glassmorphism surfaces, and monospaced typography — aligned with EaseMotion CSS's animation-first design system.
