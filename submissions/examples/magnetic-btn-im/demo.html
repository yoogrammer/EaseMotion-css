Magnetic Button

Hover near any button below — it elastically pulls toward your cursor. Move away and it snaps back.

Primary

Outline Variants

Link Button

Sizes

Magnetic Strength

How to Use

html
<!-- Add data-magnetic attribute to enable the effect -->
<button class="magnetic-btn" data-magnetic>Get Started</button>

<!-- Control strength (default: 0.4) -->
<button class="magnetic-btn" data-magnetic data-magnetic-strength="0.6">
  Strong Pull
</button>

<!-- Works on links too -->
<a href="#" class="magnetic-btn" data-magnetic>Link</a>
