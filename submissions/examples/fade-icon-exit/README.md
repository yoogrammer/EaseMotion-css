# ease-fade-icon-exit

Icon exit animation that simultaneously scales down and fades out. Perfect for icon dismiss/remove transitions.

## Usage

```html
<span class="ease-fade-icon-exit">🔔</span>
```

## Variants

| Class | Description |
|---|---|
| `ease-fade-icon-exit` | Default icon fade exit |

## Notes

- Scales to 0.75 while fading to opacity 0
- Ideal for notification icons, close buttons, status indicators
- Respects `prefers-reduced-motion`
