# Easing Playground

Interactive playground to preview and compare CSS easing animations in real time.

## Features

- Select from 9 easing functions (linear, ease, ease-in/out, spring, elastic, expo, steps)
- Adjustable duration slider (200ms – 2000ms)
- Animated preview card on a track
- Live easing curve graph with bezier control point handles
- Quick easing grid — click any chip to preview instantly
- Compare all easings side-by-side in one click

## Files

| File | Purpose |
|------|---------|
| `demo.html` | Full interactive playground |
| `style.css` | All styles (dark theme, controls, track, canvas, grid) |
| `README.md` | This file |

## Usage

Open `demo.html` directly in a browser. No build step needed.

## Suggested EaseMotion CSS Class
ease-playground

## Preview

- Select easing from dropdown
- Drag duration slider
- Click ▶ Play to animate
- Click any chip in the grid for quick preview
- Click ▶ Run All to compare all easings simultaneously
