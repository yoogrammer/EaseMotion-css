This is a demo page showing the responsive navbar component. Resize your browser window to see the mobile hamburger menu in action. The menu uses a pure CSS checkbox hack — no JavaScript required.

The navbar includes a brand logo, navigation links with active state indicator, a CTA button, and a fully functional mobile menu that slides down with a smooth animation.

Sticky Navbar Demo

This navbar is sticky — it stays at the top of the viewport as you scroll down the page. Try scrolling to see it in action.

The sticky variant uses position: sticky and top: 0 to keep it fixed during scroll. Combine it with the mobile menu for a complete responsive navigation solution.
