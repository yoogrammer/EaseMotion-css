Modal Backdrop Token Fix

The modal backdrop now uses --ease-modal-backdrop. Override it per modal or globally.

Default Backdrop

rgba(15, 23, 42, 0.6) — the default value via fallback

Light Backdrop

--ease-modal-backdrop: rgba(255, 255, 255, 0.7)

Heavy Backdrop

--ease-modal-backdrop: rgba(15, 23, 42, 0.9)

Tinted Backdrop

--ease-modal-backdrop: rgba(108, 99, 255, 0.5)
