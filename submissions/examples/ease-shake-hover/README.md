# ease-shake-hover

CSS-only shake animation on hover — perfect for buttons, cards, and interactive elements.

## Files

| File | Description |
|------|-------------|
| 📄 [demo.html](./demo.html) | Interactive demo |
| 🎨 [style.css](./style.css) | Shake animation styles |
| 📖 [README.md](./README.md) | Documentation |

## Classes

| Class | Effect | Duration |
|-------|--------|----------|
| `ease-shake-hover` | Horizontal shake | 0.4s |
| `ease-shake-subtle` | Subtle shake | 0.3s |
| `ease-shake-strong` | Strong shake | 0.5s |
| `ease-shake-horizontal` | Left-right shake | 0.4s |
| `ease-shake-vertical` | Up-down shake | 0.4s |
| `ease-shake-diagonal` | Diagonal shake | 0.4s |
| `ease-shake-fast` | Fast shake | 0.2s |
| `ease-shake-slow` | Slow shake | 0.8s |

## Usage

```html
<!-- Basic shake on hover -->
<button class="ease-shake-hover">Shake Me!</button>

<!-- Different intensities -->
<div class="ease-shake-subtle">Subtle shake</div>
<div class="ease-shake-strong">Strong shake</div>

<!-- Different directions -->
<div class="ease-shake-horizontal">Horizontal shake</div>
<div class="ease-shake-vertical">Vertical shake</div>
<div class="ease-shake-diagonal">Diagonal shake</div>

<!-- Different speeds -->
<div class="ease-shake-fast">Fast shake</div>
<div class="ease-shake-slow">Slow shake</div>
