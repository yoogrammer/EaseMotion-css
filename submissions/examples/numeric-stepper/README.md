# Numeric Stepper Input

A simple increment/decrement input with +/- buttons. Clamps to min/max range (default -10 to 10). Dark theme matching EaseMotion CSS design language.

## Usage

```html
<div class="stepper">
  <button class="stepper-btn" aria-label="Decrease">&minus;</button>
  <input class="stepper-input" type="number" value="0" readonly />
  <button class="stepper-btn" aria-label="Increase">+</button>
</div>
```

## Why it fits EaseMotion CSS

Demonstrates interactive input controls with hover/active transitions, dark-theme tokens, and minimal JavaScript — consistent with EaseMotion's utility-first approach.
