## What does this do?

Provides `ease-columns` utility classes for CSS multi-column layout — controlling column count, gap, width, and column rule styling.

## How is it used?

```html
<div class="ease-columns-2-za ease-column-gap-6-za">
  <p>Text flows into 2 columns...</p>
</div>
```

## Why is it useful?

EaseMotion core has no multi-column utilities. CSS columns are useful for magazine-style layouts, long-form content, and glossary lists without needing flexbox or grid.
