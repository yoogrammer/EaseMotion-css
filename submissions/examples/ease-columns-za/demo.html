ease-columns utilities

CSS multi-column layout without flexbox or grid.

ease-columns-2-za

CSS columns let content flow naturally from one column to the next. Unlike flexbox or grid, columns are ideal for long-form reading where you want text to break into equal vertical strips and continue at the bottom of the next column.

Column layout is also great for lists of short items like features, glossary terms, or navigation links where you want to minimize vertical scrolling.

ease-columns-3-za

Three columns work well for wider containers. The browser automatically balances the content height across columns.

Use column-gap to control spacing between columns. Combine with column-rule for a visual divider.

ease-columns-2-za with column-rule

Add a vertical rule between columns to visually separate them. The rule only appears between columns, not at the edges.

Column rules respect the column gap — they float in the middle of the gap space.
