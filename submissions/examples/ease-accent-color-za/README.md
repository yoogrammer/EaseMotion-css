## What does this do?

Provides `ease-accent` utility classes for setting the accent color on form controls — checkboxes, radio buttons, range sliders, and progress elements.

## How is it used?

```html
<input type="checkbox" class="ease-accent-primary-za" checked />
<input type="radio" class="ease-accent-success-za" />
<input type="range" class="ease-accent-danger-za" />
```

## Why is it useful?

EaseMotion core has no accent-color utilities. The accent-color property lets you theme form controls to match your brand without custom CSS for each browser's pseudo-elements.
