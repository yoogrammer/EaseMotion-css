# ease-hover-lift-shadow

Classic card lift effect combining `translateY(-4px)` lift with a growing box-shadow on hover.

## Usage

```html
<div class="ease-card ease-hover-lift-shadow">Lifts on hover</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-hover-lift-shadow` | Default lift with shadow grow |
| `ease-hover-lift-shadow-sm` | Subtle lift |
| `ease-hover-lift-shadow-lg` | Strong lift |

## Notes

- One of the most commonly used card hover patterns in UI design
- Respects `prefers-reduced-motion`
- Works well with cards, buttons, and panels
