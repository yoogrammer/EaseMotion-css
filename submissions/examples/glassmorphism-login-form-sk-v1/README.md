# Glassmorphism Login Form Example

A premium, reusable glassmorphic login form built using **EaseMotion CSS**. This example demonstrates how to create a modern authentication interface with frosted glass effects, animated input states, and a password visibility toggle.

## Key Features
- **Frosted Glass Effect**: Utilizes `.ease-card-glass` for a sophisticated translucent backdrop.
- **Animated Input States**: Custom focus animations that elevate the input field on focus.
- **Password Visibility Toggle**: Integrated JavaScript toggle for better user experience.
- **Social Login Integration**: Stylized buttons for Google and GitHub authentication.
- **Responsive Design**: Fluid layout that adapts to different screen sizes.
- **Accessibility**: Includes `aria-label` for icons and respects `prefers-reduced-motion`.

## How to Use
1. Include `easemotion.css` in your project.
2. Use the `.ease-card-glass` class for your form container.
3. Apply the custom styles provided in `style.css` for the specific glassmorphic look.

## Directory Structure
- `demo.html`: The markup for the login form.
- `style.css`: Custom glassmorphism and animation styles.
