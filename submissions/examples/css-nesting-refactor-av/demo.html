Native CSS Nesting Refactor

By actively leveraging the modern native CSS Nesting feature (&), we can completely eliminate bloated, deeply chained descendant selectors inside the component files.

This drastically improves the maintainability of the EaseMotion codebase, significantly reduces the final file size by keeping class names perfectly DRY, and marginally improves CSS parsing performance in the browser by flattening the BEM hierarchy.
