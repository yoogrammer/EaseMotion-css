# Four Direction Popovers

**What does this do?**  
Creates lightweight popovers that appear around a trigger element in four directions: top, bottom, left, and right. Each popover smoothly fades and slides into view when the user hovers over the trigger.

**How is it used?**

```html
<div class="popover-wrapper">

  <button class="btn">
    Hover Me
  </button>

  <div class="popover popover-top">
    Appears above the button
  </div>

</div>
