# Animated Stat Counter

## What does this do?
Statistics display with animated counting-up number effect, triggering when the section scrolls into view.

## How is it used?
Add the stat card markup to any page:

    <div class="stat-card">
      <div class="stat-icon">&#9733;</div>
      <div class="stat-number" data-target="500">0</div>
      <div class="stat-label">Projects</div>
    </div>

## Why is it useful?
Creates engaging animated statistics that capture attention and make data more memorable, with scroll-triggered animation.

## Tech Stack
- HTML
- CSS (minimal JS for count-up animation)

## Preview
Open demo.html directly in your browser to see the effect.
