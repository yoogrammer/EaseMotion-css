# Card Tilt Hover

1. **What does this do?** Adds a 3D tilt effect to card elements on hover using CSS `perspective` and `rotateX`/`rotateY` transforms тАФ no JavaScript required.
2. **How is it used?**
```html
<div class="tilt-card">Default tilt</div>
<div class="tilt-card-gentle">Gentle tilt</div>
<div class="tilt-card-left">Tilt left</div>
<div class="tilt-card-right">Tilt right</div>
```
3. **Why is it useful?** Creates a premium interactive feel for product cards, feature cards, or content grids using only class toggles тАФ aligns with EaseMotion CSS's animation-first, zero-JavaScript philosophy.
