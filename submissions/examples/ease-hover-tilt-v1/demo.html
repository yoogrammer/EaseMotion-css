Usage

<!-- Basic 3D tilt effect -->
<div class="ease-hover-tilt">
    <h3>Card Title</h3>
    <p>Hover to see the 3D tilt effect</p>
</div>

<!-- Different intensities -->
<div class="ease-hover-tilt-subtle">Subtle tilt (5°)</div>
<div class="ease-hover-tilt-strong">Strong tilt (15°)</div>

<!-- Different directions -->
<div class="ease-hover-tilt-left">Tilts left</div>
<div class="ease-hover-tilt-right">Tilts right</div>
<div class="ease-hover-tilt-up">Tilts up</div>
<div class="ease-hover-tilt-down">Tilts down</div>

⚡ 3D Tilt Effect Features

  • 🎮 Pure CSS 3D perspective tilt
  • 🎯 Multiple intensity levels (subtle, normal, strong)
  • ➡️ Directional variants (left, right, up, down)
  • 💫 Smooth transition with perspective
  • 🚀 No JavaScript required
  • 📱 Works on any card or container
