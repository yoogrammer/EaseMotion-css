# Stacked Cards Hover Spread

## Description

A layered card animation where stacked cards spread apart on hover.

## Features

- Smooth hover animation
- CSS-only implementation
- Modern layered UI effect
- Lightweight and reusable

## Files

- demo.html
- style.css
- script.js

## Usage

Open `demo.html` in a browser and hover over the card stack.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
