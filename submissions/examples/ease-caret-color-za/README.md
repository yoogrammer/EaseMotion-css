## What does this do?

Provides `ease-caret` utility classes for setting the caret (text cursor) color on input fields, textareas, and contenteditable elements.

## How is it used?

```html
<input class="ease-caret-primary-za" placeholder="Primary caret" />
<textarea class="ease-caret-danger-za">Red caret</textarea>
```

## Why is it useful?

EaseMotion core has no caret-color utilities. Customizing the caret color helps reinforce brand identity in form fields and can improve accessibility by making the cursor more visible against dark backgrounds.
