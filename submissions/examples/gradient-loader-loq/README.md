# Rotating Gradient Loader

A premium, purely CSS-driven circular loader component utilizing conic-gradient rotation effects.

## Questions

### 1. What does this do?

This component renders high-performance circular loaders using infinite rotating gradient effects and glowing drop-shadows.

### 2. How is it used?

Use the following HTML structure depending on your preferred variation:

```html
<!-- 1. Aurora Glow Ring Loader -->
<div class="ease-loader-aurora"></div>

<!-- 2. Cosmic Dual Orbit Loader -->
<div class="ease-loader-dual-orbit">
  <div class="outer-orbit"></div>
  <div class="inner-orbit"></div>
</div>

<!-- 3. Cyberpunk Neon Loader -->
<div class="ease-loader-cyberpunk"></div>
```

### 3. Why is it useful?

It offers a modern, visual-first loading indicator that avoids heavy JavaScript dependency. It is responsive, leverages GPU acceleration for layout performance, and honors user accessibility preferences with a smooth `prefers-reduced-motion` pulse fallback.
