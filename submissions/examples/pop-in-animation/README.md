# Pop-in Animation

1. What does this do?

Adds a smooth scale-and-fade entrance animation that makes elements appear with a subtle pop-in effect.

2. How is it used?

Apply the animation class to any element:

```html
<div class="pop-in">
  Content
</div>
```

3. Why is it useful?

This animation improves visual feedback and makes interfaces feel smoother and more modern. It can be reused across cards, buttons, modals, tooltips, notifications, and other UI elements without rewriting animation keyframes.
