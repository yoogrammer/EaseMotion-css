# Split Screen Hero

## What does this do?
A hero section split into two panels that expand on hover with content reveals.

## How is it used?
Structure two `.split-panel` inside a `.split-wrapper`:

    <div class="split-wrapper">
      <div class="split-panel panel-left">...</div>
      <div class="split-panel panel-right">...</div>
    </div>

## Why is it useful?
Dramatic landing page hero for brand showcases and product comparisons.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
