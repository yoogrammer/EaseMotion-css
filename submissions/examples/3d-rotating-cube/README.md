# 3D Rotating Cube

## What it does

A fully 3D cube made with pure CSS transforms that rotates continuously in 3D space.

## How to use it

<div class="ease-cube-container">
    <div class="ease-cube">
        <div class="ease-cube-face ease-cube-front">Front</div>
        <div class="ease-cube-face ease-cube-back">Back</div>
        <div class="ease-cube-face ease-cube-right">Right</div>
        <div class="ease-cube-face ease-cube-left">Left</div>
        <div class="ease-cube-face ease-cube-top">Top</div>
        <div class="ease-cube-face ease-cube-bottom">Bottom</div>
    </div>
</div>

## Variants

- ease-cube-fast - 4s rotation
- ease-cube-slow - 16s rotation
- ease-cube-reverse - Reverse direction
- ease-cube-pause - Paused state

## Why it fits EaseMotion CSS

- Pure CSS 3D transforms
- GPU-accelerated
- Multiple speed variants
- Glass morphism faces
