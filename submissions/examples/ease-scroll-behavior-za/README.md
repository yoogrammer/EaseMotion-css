## What does this do?

Provides `ease-scroll-behavior` utility classes to control smooth scrolling — `smooth` for animated scroll and `auto` for instant jump.

## How is it used?

```html
<a href="#section" class="ease-scroll-smooth-za">Smooth jump</a>
<div class="ease-scroll-smooth-za">Scroll container with smooth behavior</div>
```

## Why is it useful?

EaseMotion core has no scroll-behavior utilities. `scroll-behavior: smooth` is commonly desired for anchor links, scrollable containers, and "back to top" buttons to provide a polished user experience.
