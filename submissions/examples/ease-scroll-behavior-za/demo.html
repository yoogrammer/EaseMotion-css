ease-scroll-behavior utilities

Control smooth vs instant scrolling.

ease-scroll-smooth-za

Scrolling inside this box is animated — it moves smoothly rather than jumping instantly.

Useful for polished navigation and scrollable containers where a smooth feel is desired.

ease-scroll-auto-za (default)

Scrolling here jumps instantly to the target position — the browser default behavior.

Restores default if smooth was previously applied.
