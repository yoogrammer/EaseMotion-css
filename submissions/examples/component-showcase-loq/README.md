# Component Showcase Showcase

## What does this do?

This provides a fully responsive component showcase page displaying all 17 button variants and 27 card variants of EaseMotion CSS with light/dark theme toggling and copyable code snippets.

## How is it used?

Simply open `demo.html` in any web browser to view the interactive gallery of components. You can click on the "Copy" buttons next to any button or card to instantly copy the required HTML markup to your clipboard.

## Why is it useful?

It acts as a live interactive documentation and testing reference for developers, allowing them to rapidly view, test, and copy various style combinations directly into their projects.
