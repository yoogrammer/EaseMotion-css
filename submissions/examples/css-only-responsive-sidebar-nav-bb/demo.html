Dashboard

This is a CSS-only responsive sidebar navigation. On desktop, the sidebar is fixed to the left side. On mobile (below 768px), it slides in as an overlay triggered by the hamburger menu.

The toggle uses a hidden checkbox and the :checked selector to control the sidebar visibility. No JavaScript is needed for the open/close behavior.

Features

The sidebar includes section titles, active link indicators with a left border, SVG icons for each navigation item, and a brand header. The mobile overlay includes a semi-transparent backdrop and a close button.

All transitions respect prefers-reduced-motion: reduce. Focus styles are provided for keyboard navigation via :focus-visible.
