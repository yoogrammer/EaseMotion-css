# Slide-in Panel

## What it does

Off-canvas panel that slides in from any edge (top, bottom, left, right).

## How to use it

<div id="myPanel" class="ease-slide-panel ease-slide-panel-right">
    <div class="ease-slide-panel-content">Menu content</div>
</div>

<script>
document.getElementById('myPanel').classList.add('ease-slide-panel-open');
</script>

## Variants

- ease-slide-panel-left - Slides from left
- ease-slide-panel-top - Slides from top
- ease-slide-panel-bottom - Slides from bottom
- ease-slide-panel-fast - 0.15s transition
- ease-slide-panel-slow - 0.6s transition
- ease-slide-panel-blur - Backdrop blur effect

## Why it fits EaseMotion CSS

- Pure CSS transitions
- 4 directional variants
- Speed control options
- Easy to integrate
