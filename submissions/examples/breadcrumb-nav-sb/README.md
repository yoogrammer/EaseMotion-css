# CSS-only Breadcrumb Navigation

A lightweight, CSS-only horizontal breadcrumb component for multi-level navigation.

This feature includes:

- Horizontal breadcrumb layout with chevron-style separators
- Link styles with distinct hover and focus states
- A disabled/current item style to indicate the active page
- A responsive design that handles wrapping for long paths
- A compact/collapsed variant for mobile interfaces
- Dark mode compatibility

## Why is this useful for EaseMotion CSS?

Breadcrumb navigation improves usability by helping users understand where they are in a page's hierarchy and quickly navigate back to parent sections. It is a critical component for documentation sites, dashboards, admin panels, and e-commerce product pages.

Adding this to EaseMotion CSS expands the library's utility beyond standard buttons and cards, making it more practical for content-heavy and multi-level interfaces without adding JavaScript dependencies.
