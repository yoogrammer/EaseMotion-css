# ease-overflow-utilities

Overflow control utilities for managing content scroll and clipping. Part of `core/utilities.css`.

## Usage

```html
<div class="ease-overflow-auto">Scrollable content</div>
<div class="ease-overflow-hidden">Clipped content</div>
```

## Classes

| Class | Description |
|---|---|
| `ease-overflow-hidden` | `overflow: hidden` |
| `ease-overflow-auto` | `overflow: auto` |
| `ease-overflow-scroll` | `overflow: scroll` |
| `ease-overflow-x-auto` | `overflow-x: auto` |
| `ease-overflow-y-auto` | `overflow-y: auto` |

## Notes

- Axis-specific variants for horizontal/vertical scrolling
- Pairs with `ease-snap-x`/`ease-snap-y` for scroll snap
- Combine with `ease-scrollbar-thin` for styled scrollbars
