Tooltip Centering Fix Demo

This page demonstrates the prefers-reduced-motion override fix. Hovering on any button will show tooltips that remain perfectly centered even when motion reduction is active.

How to Verify:

  1. Open browser DevTools (F12).
  2. Open the Command Menu (Ctrl+Shift+P or Cmd+Shift+P).
  3. Search for "Emulate CSS prefers-reduced-motion: reduce" and press Enter.
  4. Hover over the buttons. The tooltips will fade in instantly without sliding, but they will stay perfectly aligned and centered.
