# Shipping Cost Calculator

## What does this do?
A shipping cost calculator that computes rates based on package weight, shipping zone, and carrier selection with carrier price comparison.

## How is it used?
Enter package weight, select weight unit (kg/lbs), shipping zone (local/national/international), and carrier (standard/express/overnight). View calculated rate, estimated delivery time, and total cost. The carrier comparison section shows all rates and highlights the cheapest option.

## Why is this useful?
This helps users compare shipping costs across carriers and zones to find the most cost-effective option for their packages.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to use the calculator.
