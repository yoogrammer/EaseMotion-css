# Animation: Morphing Search Bar

A search component that smoothly expands when hovered or focused. The interaction reveals the input field while maintaining a clean compact appearance when inactive.

## Features

- Pure CSS interaction
- Smooth width expansion
- Focus-within support
- Glassmorphism-inspired styling
- Responsive layout
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="search-wrapper">

    <span class="search-icon">🔍</span>

    <input
        type="text"
        placeholder="Search documentation..."
    >

</div>
```

Hovering or focusing the component expands the search bar and reveals the input field.

## Why is it useful?

Expandable search inputs save space while remaining easily accessible. They are commonly used in documentation websites, dashboards, admin panels, and modern SaaS applications.
