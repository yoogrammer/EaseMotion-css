Scroll Progress Bar

A pure-CSS page scroll progress indicator. Scroll down to see the gradient bar fill across the top of the viewport.

CSS Scroll-Driven Animation

How It Works

This component uses the modern CSS animation-timeline: scroll() property to tie a scaleX transform directly to the page scroll position. As you scroll from top to bottom, the bar fills from 0% to 100% — no JavaScript, no scroll event listeners, no requestAnimationFrame.

The progress bar is a single <div> element with two pseudo-elements: one for the gradient fill and one for a subtle glow effect underneath, creating a polished, modern feel.

Features

⚡ Zero JavaScript

Entirely powered by CSS scroll-driven animations.

🎨 Gradient Fill

Customizable multi-stop gradient via CSS custom properties.

✨ Glow Effect

Subtle blurred glow underneath for a premium look.

♿ Accessible

Respects prefers-reduced-motion automatically.

Customization

All visual properties are controlled through CSS custom properties on :root. Change the gradient colors, bar height, or glow intensity without touching the component CSS itself.

The bar height defaults to 4px for a subtle, non-intrusive indicator. Increase to 6px or 8px for a more prominent display.

Browser Support

The animation-timeline: scroll() property is supported in Chrome 115+, Edge 115+, and Opera 101+. Firefox and Safari have partial or no support as of 2024 — the bar gracefully degrades to invisible in unsupported browsers.

Usage

Add a single element at the top of your <body>. That's it. The component positions itself fixed at the viewport top and requires no wrapper, no container queries, and no layout considerations.
