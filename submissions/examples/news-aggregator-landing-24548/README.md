# News Aggregator Landing Template
A professional, editorial-focused, and high-trust landing page template for news aggregators, briefing apps, and media curation tools.

- **Design**: Uses a "Modern-Editorial" palette with headline-red accents, paper-offwhite backgrounds, and clean, typography-led layouts designed to communicate credibility and clarity.
- **Use Case**: News briefing apps, automated media curation, and source-verified information platforms.
