# ease-bg-repeat / ease-bg-no-repeat / ease-bg-repeat-x / ease-bg-repeat-y

Controls the `background-repeat` property on elements with background images.

## Usage

```html
<div class="ease-bg-no-repeat" style="background-image: url(...)">
  No repeat
</div>
<div class="ease-bg-repeat-x" style="background-image: url(...)">
  Repeat horizontally
</div>
```

## Classes

| Class               | CSS Property                 |
| ------------------- | ---------------------------- |
| `ease-bg-repeat`    | background-repeat: repeat    |
| `ease-bg-no-repeat` | background-repeat: no-repeat |
| `ease-bg-repeat-x`  | background-repeat: repeat-x  |
| `ease-bg-repeat-y`  | background-repeat: repeat-y  |
