## Text-Decoration-Thickness Utility

**What does this do?**  
Provides utility classes for the CSS `text-decoration-thickness` property, which controls the thickness (stroke width) of underline, overline, and line-through decorations applied via `text-decoration`.

**How is it used?**  
Add a class to any text element to control its underline thickness:

```html
<p class="text-decoration-thickness-2">This text has a 2px underline.</p>
<p class="text-decoration-thickness-4">This has a 4px underline.</p>
```

**Why is it useful?**  
Default underline thickness varies by browser and font size. These utilities give designers precise control over underline weight for better visual hierarchy and brand consistency — especially important for large headings where default underlines can look too thin.
