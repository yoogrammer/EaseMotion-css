# ease-contract-image-entrance

Image entrance animation starting zoomed in at scale(1.25) with opacity 0, then contracting to scale(1) with full opacity. Great for hero images and galleries.

## Usage

```html
<img class="ease-contract-image-entrance" src="..." />
```

## Variants

| Class | Description |
|---|---|
| `ease-contract-image-entrance` | Default contract entrance (zoom in → settle) |
| `ease-contract-image-entrance-sm` | Subtle zoom (1.1x) |
| `ease-contract-image-entrance-lg` | Strong zoom (1.5x) |

## Notes

- Creates a dramatic "zoom in and settle" effect
- Perfect for hero images, profile photos, and gallery thumbnails
- Respects `prefers-reduced-motion`
