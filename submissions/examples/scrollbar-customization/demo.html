Scrollbar Customization

Styling scrollbars with scrollbar-width and scrollbar-color — no JavaScript required.

Default (auto)

scrollbar-width: auto

Scroll down to see the default scrollbar behavior.

These properties let you customize the scrollbar without browser-specific pseudo-elements like ::-webkit-scrollbar.

scrollbar-width accepts auto, thin, or none.

scrollbar-color sets the thumb and track colors in one declaration.

Browser support is growing — Firefox and Chromium 121+ support both properties.

Thin with Custom Colors

scrollbar-width: thin; scrollbar-color: #a78bfa #2a2a2a;

A thin scrollbar with custom purple thumb and dark track.

The thin value reduces the scrollbar width significantly.

Pairing with scrollbar-color gives a complete custom look.

This is much cleaner than maintaining separate webkit and Firefox styles.

Fallback: browsers without support show the default scrollbar.

Custom Colors, Auto Width

scrollbar-width: auto; scrollbar-color: #f59e0b #1e1e1e;

Default width with amber thumb and near-black track.

Sometimes you want the full-width scrollbar with just a color change.

The scrollbar-color property takes two values: thumb color and track color.

You can also use system colors like Scrollbar and ButtonFace.
