# Crypto Trading Terminal

## What does this do?
A dark-themed crypto trading terminal UI with animated price ticker, order book, and a chart placeholder.

## How is it used?
Simply structure the terminal markup as shown in demo.html. The ticker, order book, and trade panels are pure HTML + CSS.

    <div class="terminal">
      <header class="terminal-header">...</header>
      <div class="ticker">...</div>
      <div class="terminal-grid">
        <section class="panel chart-panel">...</section>
        <section class="panel orderbook-panel">...</section>
        <section class="panel trades-panel">...</section>
      </div>
    </div>

## Why is it useful?
Provides a realistic trading interface mockup using pure CSS — ideal for financial dashboards, crypto portfolio UIs, and dark-themed applications.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
