# Tactile Active Press

## 1. What does this do?

This adds a physical, springy tactile press effect to clickable elements (like buttons or cards) when in the `:active` state. The element scales down slightly and drops its shadow weight to simulate depth compression.

## 2. How is it used?

Apply the `.click-press-btn` class to any button, link, or card:

```html
<button class="click-press-btn">Primary Action</button>
```

## 3. Why is it useful?

It aligns with EaseMotion CSS's focus on micro-interactions by providing instant physical feedback to user clicks, enhancing the perceived responsiveness of interactive components with pure CSS.
