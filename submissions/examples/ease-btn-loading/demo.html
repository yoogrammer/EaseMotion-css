Button Loading State

A CSS-only loading state indicator for buttons using pseudo-elements, disabling user click interactions automatically.

Primary Loading

Appends a spinning wheel indicator on a solid primary filled button. Disables hover lifts and clicks.

Outline Loading

Appends the loader to outline buttons. The spinner automatically inherits text color via CSS `currentColor`.

Size Variations

The spinner size scales proportionally based on font size (`0.85em`), matching sm, md, or lg button rules.

State Comparison

Side-by-side comparison of a standard primary button and an active loading button state.

Developer Guide

Apply the loading state by adding `.ease-btn-loading` dynamically (e.g. during form submission): 

<button class="ease-btn ease-btn-primary ease-btn-loading">
  Processing
</button>
