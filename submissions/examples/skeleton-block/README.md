# ease-skeleton-block

A rounded skeleton placeholder block (`display: block` + `border-radius`). The structural building block for creating skeleton screen layouts.

## Usage

```html
<div class="ease-skeleton-block" style="width:100%;height:16px;"></div>
```

## Variants

| Class | Description |
|---|---|
| `ease-skeleton-block` | Default rounded skeleton block |

## Notes

- Combine with `ease-skeleton-pulse` for animated loading
- Set inline width/height for custom shapes
- Pairs with other skeleton utilities for full skeleton screens
