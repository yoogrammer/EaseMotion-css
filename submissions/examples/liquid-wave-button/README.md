# Liquid Wave Button

## What does this do?
Buttons with a liquid/wave filling animation on hover — the button fills from below like rising water using a pseudo-element with a circular overflow effect.

## How is it used?
Add the class to any HTML element:

    <button class="liquid-btn liquid-primary">Submit</button>

## Why is it useful?
Adds a tactile, organic hover effect to buttons that feels more natural than a standard background color swap. Great for call-to-action buttons, primary actions, and onboarding flows.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
