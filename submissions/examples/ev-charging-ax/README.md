# ChargeUp – EV Charging Network App Landing Page

A complete, ready‑to‑copy landing page for an EV infrastructure mobile app built entirely with **EaseMotion CSS** utility classes – no custom keyframes.

## Sections
- **Header** – clean navigation
- **Hero** – app mockup with download CTA
- **Features** – real‑time availability, smart routing, seamless payment
- **Charger Network Stats** – stations, cities, charges, uptime
- **EV Brand Compatibility Logos** – works with all major brands
- **Pricing** – per kWh for standard, fast, and ultra‑fast charging
- **Testimonials** – what EV drivers say
- **App Store CTAs** – green call‑to‑action with iOS and Android buttons
- **Footer**

## EaseMotion Classes Showcased
- **Layout:** `ease-container`, `ease-flex`, `ease-grid`, `ease-gap-*`, `ease-justify-between`, `ease-items-center`, `ease-max-w-*`, `ease-mx-auto`
- **Typography:** `ease-text-*`, `ease-font-bold`, `ease-font-semibold`, `ease-text-gray-*`, `ease-text-green-*`, `ease-text-white`
- **Spacing:** `ease-py-*`, `ease-px-*`, `ease-p-*`, `ease-mb-*`, `ease-mt-*`
- **Components:** `ease-btn`, `ease-btn-primary`, `ease-btn-lg`, `ease-card`
- **Hover Effects:** `ease-hover-scale-105`, `ease-hover-shadow-xl`, `ease-hover-text-green-700`, `ease-hover-opacity-100`
- **Animations:** `ease-fade-in`, `ease-slide-up`, `ease-delay-*`, `ease-transition`
- **Utilities:** `ease-rounded-xl`, `ease-shadow-2xl`, `ease-bg-*`, `ease-w-full`, `ease-opacity-80`

## How to Use
1. Copy `demo.html` into your project.
2. Make sure the path to `easemotion.css` is correct.
3. Open in any modern browser – no build step required.
