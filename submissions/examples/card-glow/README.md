# ease-card-glow

Glow hover effect for cards. Adds a colored box-shadow glow and border highlight on hover.

## Usage

```html
<div class="ease-card ease-card-glow">Glow on hover</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-card-glow` | Default primary glow |
| `ease-card-glow-success` | Success green glow |
| `ease-card-glow-danger` | Danger red glow |
| `ease-card-glow-warning` | Warning amber glow |

## Notes

- Composable with `ease-card-hover`, `ease-card-shadow`, `ease-card-accent`
- The glow color is controlled via `--ease-glow-primary` token
- Respects `prefers-reduced-motion`
