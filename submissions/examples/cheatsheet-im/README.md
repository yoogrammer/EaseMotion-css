# EaseMotion CSS — Class Cheatsheet

## What does this do?
A fully self-contained cheatsheet page listing every `ease-*` class with:
- Live mini animation preview per card (click any card to replay)
- Category filter — Entrance, Hover, Looping, Exit, Skeleton, Scroll, Modifier
- Real-time search across class names and descriptions
- One-click copy of any class name
- Dark/light mode matching the docs site
- Responsive grid layout

## How to use it?
Open `demo.html` directly in a browser — no server required.
Links back to `../../docs/index.html` and `../../easemotion.css` relative to the submissions folder.

## Why it fits EaseMotion CSS?
Right now developers have to dig through the docs page, the animation preview page, and the demo page separately to find what they need. This page is the single quick reference — the tab you keep open while building. Every class is visible, previewable, and copyable in seconds. Fits the human-readable philosophy perfectly: see the name, see what it does, copy it.

## Files
