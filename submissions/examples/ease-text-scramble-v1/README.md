# Ease Text Scramble Decode Effect

A text animation that rapidly cycles through random characters before revealing the final text.

## Features

- Matrix / hacker style text reveal
- Automatic animation on page load
- Hover to replay animation
- Lightweight implementation
- Suitable for hero sections and headings

## Example

```html
<h1 class="ease-scramble" data-text="Hello World">
  Hello World
</h1>
