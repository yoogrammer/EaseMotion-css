# Dark Mode Design System Showcase

A premium dark mode design system showcase demonstrating cards, buttons, forms, badges, and navigation components in a cohesive dark theme.

## Features

- Dark theme with proper color hierarchy (surface, elevated, interactive)
- Component gallery: cards, buttons (primary/secondary/ghost), form inputs, badges, toggles, navigation tabs
- Smooth hover and focus transitions
- CSS custom properties for consistent theming
- `prefers-reduced-motion` support

## Files

- `demo.html` — Component gallery layout
- `style.css` — Self-contained CSS (200 lines)
