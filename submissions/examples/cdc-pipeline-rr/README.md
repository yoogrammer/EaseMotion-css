# Change Data Capture Pipeline Component

## What does this do?

Visualizes how database changes are captured and streamed through a CDC pipeline into analytics and reporting systems.

## How is it used?

```html
<div class="node cdc">
    CDC Engine
</div>

<div class="node broker">
    Event Broker (Kafka)
</div>
```

## Why is it useful?

Change Data Capture (CDC) enables real-time synchronization between operational databases and analytics platforms. This component helps explain modern streaming architectures and data engineering workflows using pure HTML and CSS.
