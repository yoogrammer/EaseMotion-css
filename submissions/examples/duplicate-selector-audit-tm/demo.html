Duplicate Selector Audit

A compact review pattern for spotting repeated public classes, stale allowlists, and split reduced-motion ownership before a stylesheet is promoted into the main bundle.

Goal 0 duplicates

Suggested Review Checklist

  1. Keep component-specific reduced-motion rules with the component.
  2. Remove stale duplicate allowlist entries after each scan.
  3. Check multi-line selector lists, not only single-line class rules.
  4. Publish modular animation files only after they share one source of truth.
