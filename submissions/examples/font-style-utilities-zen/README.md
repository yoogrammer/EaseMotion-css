## What does this do?

Provides `font-style` utility classes: italic, normal, and oblique.

## How is it used?

```html
<p class="font-italic">This text is italicized.</p>
<p class="font-normal">Normal text.</p>
```

## Why is it useful?

EaseMotion has font-weight utilities but no font-style classes. Italic styling is a fundamental typographic tool needed for emphasis, quotes, and semantic formatting.
