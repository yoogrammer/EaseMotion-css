# Animated Dashboard Statistic Widgets

A responsive animated dashboard statistic widget component built using pure HTML and CSS.

## Features

- Animated metric cards
- Hover glow effects
- Trend indicators
- Responsive dashboard layout
- Smooth transitions and hover animations
- Pure CSS implementation

## Usage

Open `demo.html` in a browser to view the animated dashboard statistic widgets.

## Files

- `demo.html` → Dashboard widget structure
- `style.css` → Widget animations and styles
- `README.md` → Documentation
