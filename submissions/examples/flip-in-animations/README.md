# flip-in animation

## What does this do?
This adds a flip-in animation that rotates an element along the X or Y axis while fading it into view.

```
flip-in-x :- flips the element from top to front
```

```
flip-in-y :- flips the element from left to front
```

## How is it used?
Apply the class directly to any HTML element:

```html
<div class="flip-in-x">
  Flip X
</div>
```
```html
<div class="flip-in-y">
  Flip Y
</div>
```

## Why is it useful?
It creates a smooth entrance effect where elements appear with a flipping motion instead of just fading in.

It is useful for cards, buttons, UI sections, icons, content blocks

It improves visual feedback and makes UI feel more interactive.
