# Bounce Animation Guide

## 1. What does this do?
A demo showing four ease-bounce CSS animations: continuous bounce, hover bounce, bounce-in on load, and a click bounce button.

## 2. How is it used?

```html
<!-- Continuous bounce -->
<div class="bounce-basic">Bounce</div>

<!-- Hover to bounce -->
<div class="bounce-hover">Hover Me</div>

<!-- Bounces in on page load -->
<div class="bounce-in">Hello!</div>

<!-- Click bounce button -->
<button class="bounce-btn">Click Me</button>
