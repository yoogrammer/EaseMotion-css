\# ease-checkbox



Custom-styled checkbox component with animated checkmark.



\## Classes



| Class | Description |

|-------|-------------|

| `ease-checkbox` | Base custom checkbox |

| `ease-checkbox-primary` | Primary color variant |

| `ease-checkbox-success` | Success color variant |

| `ease-checkbox-danger` | Danger color variant |

| `ease-checkbox-warning` | Warning color variant |

| `ease-checkbox-small` | Small size |

| `ease-checkbox-large` | Large size |



\## Features



\- ✅ Custom styled checkbox

\- ✨ Animated checkmark using CSS ::after

\- 🔲 Indeterminate state support

\- 🎨 Multiple color variants

\- 📏 Multiple size variants

\- ⌨️ Keyboard accessible



\## Usage



```html

<label class="ease-checkbox">

&#x20;   <input type="checkbox">

&#x20;   <span class="ease-checkbox-label">Label text</span>

</label>
