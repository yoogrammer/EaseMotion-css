# Text Overflow Utilities

CSS utility classes for controlling text overflow behavior.

## Classes

- `.truncate` — `overflow: hidden; text-overflow: ellipsis; white-space: nowrap`
- `.text-clip` — `overflow: hidden; text-overflow: clip; white-space: nowrap`

## Why

Text overflow handling is essential for UI elements where content must fit within a fixed width — such as table cells, card titles, navigation items, and breadcrumbs. The `.truncate` class is a common shorthand for single-line text truncation with ellipsis.
