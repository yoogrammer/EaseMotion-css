# Project Milestone Tracker UI Component

## What does this do?

Provides a timeline-based milestone tracker for monitoring project progress through various stages such as planning, design, development, testing, and launch.

## How is it used?

```html
<div class="milestone active">
    <div class="circle"></div>
    <div class="content">
        <h3>Development Phase</h3>
    </div>
</div>
```

## Why is it useful?

Milestone trackers help teams visualize project progress and upcoming goals. This component demonstrates timeline layouts, status indicators, hover interactions, and animations using pure HTML and CSS.
