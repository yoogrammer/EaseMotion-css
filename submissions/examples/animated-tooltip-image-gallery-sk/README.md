# Animated Tooltip Image Gallery

**What does this do?**  
Creates an image gallery with hover-triggered glassmorphism tooltips that slide up with a spring animation.

**How is it used?**  
Use `.gallery` container with `.gallery-item` elements. Each item has a `.gallery-img` (gradient placeholder) and a `.tooltip` that appears on hover.

**Why is it useful?**  
Adds interactive image captions with a modern glass aesthetic, zero dependencies, and smooth spring animations.
