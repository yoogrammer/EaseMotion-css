# Ease 3D Card Flip

### What does this do?
Executes a modern 3D rotation flip effect for content cards on hover.

### How is it used?
Wrap cards inside `.ease-card-container` containing a `.ease-card-flip-pr` element with `.ease-card-front` and `.ease-card-back` children.

### Why is it useful?
Great for space-saving designs, feature explanations, or interactive team directories.
