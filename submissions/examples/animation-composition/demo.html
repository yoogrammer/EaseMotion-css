animation-composition

Controls how multiple animations combine when they target the same property.

replace (default)

The last animation wins. The scale fades out because the second animation's opacity overrides the first.

R

add

Values are added together. The box both scales AND fades — both animations contribute simultaneously.

A

accumulate

Like add, but smarter — it accumulates values based on their type (adds transforms, blends colors, etc.).

C

Each box has two animations running simultaneously: one scales, one fades. Only animation-composition changes how they combine. Open DevTools → Computed to inspect the active values.
