# Cinematic Parallax Gallery

## What does this do?
An image gallery grid where cards respond to hover with a cinematic parallax depth effect — the image scales and tilts while the label lifts independently.

## How is it used?
Add the class to any HTML element:

    <div class="parallax-card">
      <div class="parallax-image" style="background-image: url(...);"></div>
      <div class="parallax-label">Label</div>
    </div>

## Why is it useful?
Adds a polished, immersive feel to image galleries, portfolios, and media grids without any JavaScript. The parallax effect gives each card a sense of depth.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
