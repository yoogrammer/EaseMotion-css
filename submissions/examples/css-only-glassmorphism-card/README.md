# CSS-only Glassmorphism Credit Card

### 1. What does this do?
A sleek, realistic CSS-only glassmorphism credit card UI featuring frosted glass effects (`backdrop-filter`), dynamic hover transformations, and a glossy reflection effect.

### 2. How is it used?
```html
<div class="glass-card">
  <div class="card-chip"></div>
  <div class="card-number">•••• •••• •••• 1234</div>
  <div class="card-details">
    <div class="card-holder">
      <span>Card Holder</span>
      <div class="card-name">JOHN DOE</div>
    </div>
    <div class="card-expiry-wrap">
      <span>Expires</span>
      <div class="card-expiry">12/28</div>
    </div>
  </div>
</div>
```

### 3. Why is it useful?
Glassmorphism is a prominent design trend that provides a premium, immersive aesthetic. This component allows developers to implement a beautiful, interactive card instantly without requiring JavaScript or complex assets, fitting perfectly with EaseMotion CSS.
