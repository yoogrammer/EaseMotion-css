## ease-bounce-shadow-entrance

**What does this do?**
Animates an element's entrance with a bouncing drop effect where the box-shadow dynamically grows and shrinks to simulate depth on landing.

**How is it used?**
<div class="bounce-shadow-entrance">Content</div>
<div class="bounce-shadow-slow">Content</div>

**Why is it useful?**
Adds a satisfying physical feel to entrance animations by tying shadow depth to motion, consistent with EaseMotion CSS's keyframe-driven animation philosophy.
