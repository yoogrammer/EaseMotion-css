# Floating Dock Navigation

A macOS-inspired floating navigation dock where icons scale up smoothly on hover.

## Usage
```html
<nav class="floating-dock">
  <button class="dock-item">🏠</button>
  <button class="dock-item">🔍</button>
</nav>
```

## Why EaseMotion CSS?
Animation-first, lightweight, elegant micro-interaction with zero dependencies.
