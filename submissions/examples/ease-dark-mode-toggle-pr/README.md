# Ease Dark Mode Toggle PR

## What does this do?

Creates a smooth rotating transition between light mode and dark mode icons.

The sun icon rotates and fades out while the moon icon rotates and fades into view.

## Features

- Pure CSS implementation
- Light/Dark mode icon switch
- Rotation animation
- Scale transition
- Opacity transition
- No JavaScript required

## How is it used?

Click the icon to toggle between:

- ☀️ Light Mode
- 🌙 Dark Mode

The transition is handled entirely with CSS transforms and opacity animations.

## Why is it useful?

Theme toggles are a common UI pattern in modern interfaces. This example demonstrates how to create a clean and reusable icon-switching animation using only CSS.
