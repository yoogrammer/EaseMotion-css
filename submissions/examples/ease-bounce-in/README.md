\# ease-bounce-in



Playful bounce-in entrance animation with spring-like effect.



\## Classes



| Class | Description | Speed | Direction |

|-------|-------------|-------|-----------|

| `ease-bounce-in` | Standard bounce | 0.8s | Center |

| `ease-bounce-in-fast` | Fast bounce | 0.4s | Center |

| `ease-bounce-in-slow` | Slow bounce | 1.2s | Center |

| `ease-bounce-in-up` | Bounce from bottom | 0.8s | Up |

| `ease-bounce-in-down` | Bounce from top | 0.8s | Down |

| `ease-bounce-in-left` | Bounce from right | 0.8s | Left |

| `ease-bounce-in-right` | Bounce from left | 0.8s | Right |

| `ease-bounce-in-delay-1` | Delay 0.2s | 0.8s | Center |

| `ease-bounce-in-delay-2` | Delay 0.4s | 0.8s | Center |

| `ease-bounce-in-delay-3` | Delay 0.6s | 0.8s | Center |



\## Usage



```html

<div class="ease-bounce-in">

&#x20;   Content bounces in

</div>
