# Animated Notification Stack

A responsive animated notification stack component built using pure HTML and CSS.

## Features

- Stacked notification layout
- Slide and fade entry animations
- Success, error, and info variants
- Hover interactions
- Responsive mobile layout
- Pure CSS implementation

## Usage

Open `demo.html` in a browser to view the animated notification stack demo.

## Files

- `demo.html` → Notification structure
- `style.css` → Notification animations and styles
- `README.md` → Documentation
