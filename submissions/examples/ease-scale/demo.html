Scale Utilities

Atomic scaling utility classes for sizing elements uniformly or stretching them along the X or Y axes. Hover over cards to see scaling transitions.

Uniform Scale (ease-scale-*)

50%
.ease-scale-50

Scale Down (0.5x)

Shrinks the element uniformly to half its native dimensions.

75%
.ease-scale-75

Scale Down (0.75x)

Shrinks the element uniformly to 75% of its native dimensions.

90%
.ease-scale-90

Scale Down (0.9x)

Shrinks the element uniformly to 90% of its native dimensions.

110%
.ease-scale-110

Scale Up (1.1x)

Enlarges the element uniformly to 110% of its native dimensions.

125%
.ease-scale-125

Scale Up (1.25x)

Enlarges the element uniformly to 125% of its native dimensions.

150%
.ease-scale-150

Scale Up (1.5x)

Enlarges the element uniformly to 150% (1.5x) of its native dimensions.

Horizontal Scale (ease-scale-x-*)

X
.ease-scale-x-50

Compress X (0.5x)

Shrinks the width of the element to 50% while height remains unchanged.

X
.ease-scale-x-125

Stretch X (1.25x)

Enlarges the width of the element to 125% while height remains unchanged.

X
.ease-scale-x-150

Stretch X (1.5x)

Enlarges the width of the element to 150% while height remains unchanged.

Vertical Scale (ease-scale-y-*)

Y

Y
.ease-scale-y-125

Stretch Y (1.25x)

Enlarges the height of the element to 125% while width remains unchanged.

Y
.ease-scale-y-150

Stretch Y (1.5x)

Enlarges the height of the element to 150% while width remains unchanged.

Developer Guide

Combine uniform or single-axis scale utilities with animations, hovers, or active clicks to create interactive pop-outs: 

<button 
  class="btn ease-transition ease-duration-fast hover:ease-scale-115 active:ease-scale-95"
>
  Click Me
</button>
