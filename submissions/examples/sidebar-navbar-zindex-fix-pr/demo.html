Framework z-index Scale (from variables.css)

Token Value Used By
--ease-z-dropdown 10 Dropdowns
--ease-z-sticky 50 Sticky headers
--ease-z-overlay 100 Navbar glass sticky
--ease-z-modal 1000 Modals, sidebar (fixed)
--ease-z-toast 9999 Toasts, tooltips
How to test:
1. Open the "Before" sidebar — notice the top of the sidebar is hidden behind the navbar bar.
2. Open the "After" sidebar — the sidebar renders completely above the navbar, with a semi-transparent backdrop.
3. The fix uses --ease-z-modal (1000) instead of the hardcoded 100, which is appropriate since the mobile sidebar functions as a full-screen modal-like overlay.
