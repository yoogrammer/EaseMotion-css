# Paginated Content

## What does this do?
A paginated content viewer with next/previous navigation, page number display, and placeholder content for each page.

## How is it used?
Open demo.html in a browser. Use the Previous and Next buttons to navigate between pages. The current page number is displayed.

## Why is this useful?
Demonstrates a pagination pattern for breaking long content into digestible chunks, commonly used in documentation, tutorials, and article viewers.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
