# Levitating Card Hover Showcase

Issue: [#10383](https://github.com/SAPTARSHI-coder/EaseMotion-css/issues/10383)

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Features

- Pure CSS implementation
- Floating hover animation
- Smooth shadow transitions
- Multiple hover intensity variants
- Responsive card layouts
- Feature card showcase
- No JavaScript required
- Reusable utility class

## Usage

```html
<div class="ease-card-float">
  Card Content
</div>
```

## Main Class

```css
.ease-card-float
```

## Hover Variants

```html
<div class="ease-card-float hover-soft"></div>

<div class="ease-card-float hover-medium"></div>

<div class="ease-card-float hover-strong"></div>
```

## Why It Fits EaseMotion CSS

- Human-readable class names
- Animation-first philosophy
- Lightweight implementation
- Easy integration
- Reusable card interactions
- Practical real-world UI pattern
