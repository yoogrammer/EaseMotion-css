# Kubernetes Cluster Topology Component

## What does this do?

Visualizes Kubernetes infrastructure resources and traffic flow using a topology-style layout.

## How is it used?

```html
<div class="node ingress">Ingress</div>
<div class="node service">Service</div>
<div class="node pod">Pod A</div>
```

## Why is it useful?

Kubernetes environments contain interconnected resources that are easier to understand through visual diagrams than plain documentation. This component provides a reusable topology visualization for DevOps dashboards, architecture diagrams, cloud documentation, and educational content using pure HTML and CSS.
