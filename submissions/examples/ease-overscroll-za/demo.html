ease-overscroll utilities

Control overscroll behavior to prevent scroll chaining.

ease-overscroll-contain-za

Scroll inside this box. When you reach the bottom, the bounce is contained — the page behind does not scroll.

This is useful for modals, sidebars, and any nested scrollable region.

ease-overscroll-none-za

No overscroll effect at all — no rubber-band bounce on any axis.

Useful for precise scroll containers like code editors or maps.

ease-overscroll-auto-za

Default browser behavior — overscroll causes rubber-band bounce and may trigger pull-to-refresh.

Restores default if previously overridden.
