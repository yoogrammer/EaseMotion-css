Usage

<!-- Basic skew -->
<div class="ease-skew-hover">Hover to Skew</div>

<!-- Horizontal skew -->
<div class="ease-skew-x-10">Skew 10° on X-axis</div>

<!-- Vertical skew -->
<div class="ease-skew-y-15">Skew 15° on Y-axis</div>

<!-- Both axes -->
<div class="ease-skew-xy-10">Skew 10° on both axes</div>

<!-- Direction variants -->
<div class="ease-skew-left">Skew left -15°</div>
<div class="ease-skew-right">Skew right 15°</div>

<!-- Speed variants -->
<div class="ease-skew-x-10-fast">Fast skew</div>
<div class="ease-skew-x-10-slow">Slow skew</div>

⚡ Skew on Hover Features

  • 📐 Element skews on hover
  • 🎯 Horizontal, vertical, and both axes options
  • 📊 Multiple angle levels (5°, 10°, 15°, 20°)
  • ➡️ Direction variants (left, right, up, down)
  • ⚡ Speed variations (fast, normal, slow)
  • 💫 Smooth CSS transitions
