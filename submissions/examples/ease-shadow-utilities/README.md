# Shadow Utilities

## What does this do?

Provides reusable utility classes for applying different levels of box-shadow.

## How is it used?

```html
<div class="shadow-sm">Content</div>
<div class="shadow-md">Content</div>
<div class="shadow-lg">Content</div>
<div class="shadow-xl">Content</div>
```

## Why is it useful?

Shadow utilities help create depth and visual hierarchy while keeping styling consistent and easy to reuse.
