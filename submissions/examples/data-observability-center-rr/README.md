# Data Observability Command Center Component

## What does this do?

Provides a centralized monitoring interface for data platform health, dataset freshness, quality checks, incidents, and operational reliability.

## How is it used?

```html
<div class="dataset-card healthy">
    <h3>Customer Analytics</h3>
</div>
```

## Why is it useful?

Modern data teams require visibility into dataset quality, freshness, and operational reliability. This component provides a reusable observability dashboard for monitoring data health and incidents while demonstrating enterprise-grade data platform design patterns.
