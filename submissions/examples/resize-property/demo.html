Resize Property

Making elements user-resizable with pure CSS — no JavaScript required.

resize: both

Drag the handle to resize in both directions

I'm resizable both horizontally and vertically. Grab the bottom-right corner and drag!

The resize: both value allows resizing in all directions.

resize: horizontal

Resize width only

I can only be resized horizontally. Height stays fixed.

Useful for sidebars and panels where you want width control but fixed height.

resize: vertical

Resize height only

I can only be resized vertically. Width stays fixed.

Great for textareas, code editors, and expandable sections.

resize: none

Default — no resize handle

I'm not resizable at all. This is the default behavior for most elements.

No resize handle appears on hover or click.

Textarea (inherently resizable)

Textareas have resize: both by default
