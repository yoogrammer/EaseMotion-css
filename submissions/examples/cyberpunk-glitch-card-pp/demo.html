Technical Specifications

  • Image Glitch Slicing: Created using two duplicate image layers absolute-positioned on top of the main photo. On hover, these layers activate clip-path: inset(...) keyframe cycles, slicing the image into shifting horizontal blocks tinted red/cyan.
  • Chromatic Aberration: Achieved on the name and title texts by animating red and cyan offsets within text-shadow properties at steps keyframes.
  • CRT Sweep Line: Made using a moving horizontal glow panel scrolling via linear-gradients and linear vertical translation keyframes.
  • Accessibility Support: The card supports keyboard navigation via tabindex="0" and focus styles. It fully honors user reduced-motion preferences, disabling all glitching, blinking, and sweeping animations under @media (prefers-reduced-motion: reduce).
