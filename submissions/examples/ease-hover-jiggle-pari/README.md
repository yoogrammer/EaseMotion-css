## ease-hover-jiggle

**What does this do?**
A micro-interaction that makes elements continuously animate on hover —
pulse, jiggle, bounce, or shake — drawing attention without being distracting.

**How is it used?**
<button class="hover-pulse">Subscribe Now</button>
<button class="hover-jiggle">Notification</button>
<button class="hover-bounce">Bounce</button>
<button class="hover-shake">Shake</button>

**Why is it useful?**
Provides continuous looping hover animations using pure CSS keyframes,
perfect for CTAs, notification bells, and important links —
consistent with EaseMotion CSS's animation-first philosophy.
