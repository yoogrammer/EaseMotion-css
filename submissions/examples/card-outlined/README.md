# ease-card-outlined

Outlined card variant with transparent background and 2px border. Perfect for minimal designs, overlays, and varied backgrounds.

## Usage

```html
<div class="ease-card ease-card-outlined">Outlined card</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-card-outlined` | Default (2px neutral border) |
| `ease-card-outlined-primary` | Primary color border |
| `ease-card-outlined-success` | Success green border |
| `ease-card-outlined-danger` | Danger red border |
| `ease-card-outlined-dashed` | Dashed border style |

## Notes

- Contrasts with default card (solid bg + 1px border)
- Great for image galleries, overlay content, and card groups
- Combine with `ease-card-hover`, `ease-card-shadow` for interactivity
