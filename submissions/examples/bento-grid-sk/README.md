# Bento Grid Layout

A modern, highly responsive bento grid layout built for landing pages and feature showcases. This component leverages CSS Grid to create a dynamic and visually engaging layout with variable card sizes and smooth entrance animations.

## Key Features
- **Responsive CSS Grid**: Automatically adapts from a complex multi-column layout on desktop to a simplified stack on mobile.
- **Variable Card Sizes**: Uses `grid-column` and `grid-row` spans to create the signature "bento" look.
- **Entrance Animations**: Cards fade and slide into position when the page loads.
- **Hover Interactions**: Subtle scale and shadow effects that respond to user interaction.
- **Dark Mode Support**: Fully compatible with EaseMotion's dark mode design tokens.

## How to Use
1. Include `easemotion.css` in your project.
2. Structure your HTML using the `.ease-bento-grid` and `.ease-bento-card` classes.
3. Apply the custom styles from `style.css`.

## Directory Structure
- `demo.html`: Interactive demo showcasing the bento grid.
- `style.css`: Styles for the grid layout and animations.
