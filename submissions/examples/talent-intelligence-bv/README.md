# Talent Intelligence Dashboard Example

This example showcases a **Talent Intelligence** SaaS dashboard built with **EaseMotion CSS**. It demonstrates:

- A modern glassmorphic UI with vibrant gradients and smooth micro‑animations.
- Over 15 distinct UI sections and more than 25 reusable components/cards.
- Responsive layouts for desktop, tablet, and mobile devices.
- Accessibility considerations such as sufficient color contrast and ARIA labels.
- Motion effects using the `sweepProgress` keyframe defined in the example's `style.css`.

## Directory Structure

```
/talent-intelligence-bv/
│   demo.html   # Interactive demo page
│   style.css   # Custom CSS variables and animations for this example
│   README.md   # This documentation
```

## How to Run

Open `demo.html` in a modern browser. No external dependencies are required; all styling is handled by the included CSS.

## Key Features

- **Candidate Pipeline** with animated progress chart.
- **ATS Integration** status badge.
- **Talent Analytics** cards (Time‑to‑Hire, Offer Acceptance, Source Quality).
- **Diversity Metrics** with animated charts.
- **Job Ads Dashboard** and **Settings** panel.

Built with **EaseMotion CSS** – a lightweight, utility‑first CSS framework for motion‑rich interfaces.
