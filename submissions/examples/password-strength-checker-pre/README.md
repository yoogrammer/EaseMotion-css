## password-strength-checker

**What does this do?**
A real-time password strength checker with an animated bar
and criteria checklist using EaseMotion CSS utilities.

**How is it used?**
Open demo.html and type a password to see:
- Animated strength bar (red/orange/yellow/green)
- Live criteria checklist with ease-badge icons
- Strength label updates in real-time

**Why is it useful?**
Demonstrates EaseMotion CSS ease-fade-in, ease-badge,
ease-rounded-full, and ease-transition utilities in a
practical form UI — consistent with the animation-first philosophy.
