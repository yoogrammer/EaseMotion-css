Animated Gradient Border

A rotating conic-gradient border. Pure CSS — no JS required. Works on cards, buttons, inputs, and any element.

Cards

Default

Primary purple gradient rotating continuously around the border.

Rainbow

Full spectrum gradient sweeping around the border.

Slow Spin

Same effect at a slower, more subtle rotation speed.

Buttons

Hover-Only Variant

Activates on Hover

The gradient border only spins when you hover. Subtle and performant.

Clean at Rest

No animation when idle — great for content-heavy UIs where constant motion is distracting.

How to Use

html
<!-- Card with animated gradient border -->
<div class="gradient-border">
  <h3>Title</h3>
  <p>Content</p>
</div>

<!-- Hover-only variant -->
<div class="gradient-border-hover">
  <h3>Title</h3>
  <p>Content</p>
</div>

<!-- Button -->
<button class="gradient-border-btn">Click me</button>
css vars
/* Customize the gradient and speed */
.gradient-border {
  --gb-color-1: #6c63ff;   /* start color  */
  --gb-color-2: #a78bfa;   /* mid color    */
  --gb-color-3: #6c63ff;   /* end color    */
  --gb-speed:   3s;         /* rotation speed */
  --gb-width:   2px;        /* border width   */
  --gb-radius:  12px;       /* border radius  */
}
