# RAG Pipeline Visualization Component

## What does this do?

Visualizes how Retrieval-Augmented Generation (RAG) systems process user queries through embeddings, vector databases, retrieval systems, and large language models.

## How is it used?

```html
<div class="stage vector">
    Vector Database
</div>

<div class="stage llm">
    Large Language Model
</div>
```

## Why is it useful?

RAG has become one of the most important AI architecture patterns for building intelligent assistants and knowledge-based systems. This component helps explain how retrieval and generation work together using pure HTML and CSS.
