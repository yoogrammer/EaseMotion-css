## ease-rotate-icon-emphasis

**What does this do?**
Animates icon components with rotate effects during emphasis interactions like hover.

**How is it used?**
<button class="icon-btn rotate-emphasis">⚙️</button>
<button class="icon-btn rotate-wiggle">🔔</button>
<button class="icon-btn rotate-flip">🔄</button>
<button class="icon-btn rotate-bounce">⭐</button>

**Why is it useful?**
Adds expressive micro-interactions to icon buttons using pure CSS keyframes,
making UI feel responsive and alive — consistent with EaseMotion CSS's emphasis animation philosophy.
