CSS image-set() Responsive Images

Serving the right image for every screen density and format

Density tiers

image-set(url(img.jpg) 1x, url(img@2x.jpg) 2x, url(img@3x.jpg) 3x)

1x / 2x / 3x

Format fallback

image-set(url(img.avif) type("image/avif"), url(img.webp) type("image/webp"), url(img.jpg) type("image/jpeg"))

avif > webp > jpeg

Combined

image-set(url(img.avif) type("image/avif") 2x, url(img.webp) type("image/webp") 2x, url(img.jpg) 2x, url(img.jpg) 1x)

format + density
