# Ease Hidden & Visible Display Utilities

1. **What does this do?** Adds display utility classes to show, hide, or make elements invisible while preserving layout space.

2. **How is it used?**
```html
   <!-- Completely hide element -->
   <div class="ease-hidden">Hidden</div>

   <!-- Hide but preserve space -->
   <div class="ease-invisible">Invisible</div>

   <!-- Make visible -->
   <div class="ease-visible">Visible</div>

   <!-- Animated fade in -->
   <div class="ease-visible-fade">Fades in</div>
```

3. **Why is it useful?** Essential display utilities with responsive variants and optional fade animations — pure CSS, zero JavaScript, fits EaseMotion CSS's utility-first philosophy.
