# Interactive Copy-to-Clipboard Code Block

## What does this do?

Adds a modern code block component with a hover-revealed copy button and animated clipboard feedback.

---

## How is it used?

```html
<div class="code-block">
    <button class="copy-btn">📋</button>

    <pre>
        <code>
npm install easemotion-css
        </code>
    </pre>
</div>
```

---

## Why is it useful?

Developers frequently copy code snippets from documentation.

This component:

- Reduces friction when copying code
- Improves documentation usability
- Provides visual feedback through micro-interactions
- Works well with dark-themed documentation
- Aligns with EaseMotion CSS's animation-first philosophy
