# Ease Progress Ring

## What does this do?

The Ease Progress Ring component provides a circular progress indicator built entirely with CSS using conic gradients and smooth animations.

---

## Features

- Circular progress visualization
- Pure CSS implementation
- Responsive design
- Multiple size variants
- Multiple color variants
- Smooth entrance animation
- Dashboard-friendly UI

---

## Usage

```html
<div class="progress-ring progress-75">
  <span>75%</span>
</div>
```

### Available Progress Variants

```html
progress-45
progress-60
progress-75
progress-80
progress-90
progress-100
```

### Size Variants

```html
small
large
```

### Color Variants

```html
success
danger
```

---

## Why is it useful?

Circular progress indicators are commonly used in:

- Analytics dashboards
- Course completion tracking
- Profile completion systems
- Task management tools
- KPI monitoring
- Loading states

This component aligns with EaseMotion CSS's animation-first philosophy by providing a visually engaging and reusable progress indicator without requiring JavaScript.
