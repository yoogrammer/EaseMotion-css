# Animated Search Bar Component

A responsive animated search bar component built using pure HTML and CSS.

## Features

- Expanding search input
- Animated focus effects
- Hover interactions
- Icon transitions
- Responsive layout
- Smooth open and close animations
- Pure CSS implementation

## Usage

Open `demo.html` in a browser to view the animated search bar component.

## Files

- `demo.html` → Search bar structure
- `style.css` → Search bar animations and styles
- `README.md` → Documentation
