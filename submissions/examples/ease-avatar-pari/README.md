## ease-avatar

**What does this do?**
A CSS avatar component with size variants, circle variant, color themes,
and a stacked group with hover expand animation.

**How is it used?**
<!-- Base -->
<div class="ease-avatar">AB</div>

<!-- Sizes -->
<div class="ease-avatar ease-avatar-sm">S</div>
<div class="ease-avatar ease-avatar-lg">L</div>

<!-- Circle -->
<div class="ease-avatar ease-avatar-circle">AB</div>

<!-- Group -->
<div class="ease-avatar-group">
  <div class="ease-avatar ease-avatar-circle">AB</div>
  <div class="ease-avatar ease-avatar-circle green">CD</div>
  <div class="ease-avatar ease-avatar-circle count">+3</div>
</div>

**Why is it useful?**
Provides a complete avatar system with smooth hover animations and stacked
group interactions — consistent with EaseMotion CSS's component philosophy.
