# ease-search-pr

Search input with integrated icon, clear button, and focus glow.

## Features

- Search icon positioned inside input on left
- Clear button appears when text is entered
- Focus glow ring with icon color change
- Expandable width variant
- Compact size for tight spaces
- Navbar style with subtle background
- Dark theme variant

## Files

- `style.css` — the search input styles
- `demo.html` — working demo with 5 variants
- `README.md` — usage documentation

## Usage

```html
&lt;div class="search-pr"&gt;
  &lt;svg class="search-pr-icon"&gt;🔍&lt;/svg&gt;
  &lt;input type="text" class="search-pr-input" placeholder="Search..."&gt;
  &lt;button class="search-pr-clear"&gt;✕&lt;/button&gt;
&lt;/div&gt;
