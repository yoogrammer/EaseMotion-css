# Neon Glowing Radio Buttons

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A CSS-only radio button collection featuring neon glow effects, animated selection states, multiple color variants, and modern glassmorphism styling.

## Features

- CSS-only implementation
- Neon glow effect
- Animated selection indicator
- Cyan variant
- Purple variant
- Pink variant
- Green variant
- Glassmorphism panel
- Hover interactions
- Responsive layout
- No JavaScript required

## Usage

```html
<label class="radio-option cyan">
  <input type="radio" name="group">
  <span class="radio-circle"></span>
  <span>Option</span>
</label>
```

## Main Classes

```css
.radio-option
.radio-circle
.cyan
.purple
.pink
.green
```

## Why It Fits EaseMotion CSS

- Animation-first interaction
- Human-readable class names
- Reusable form component
- Lightweight implementation
- Modern visual effects
