## ease-hero

**What does this do?**
A full-width hero section component with centered content,
overlay support, and built-in entrance animations.

**How is it used?**
<section class="ease-hero purple">
  <div class="ease-hero-content">
    <span class="hero-tag">Tag</span>
    <h1 class="hero-title">Title</h1>
    <p class="hero-subtitle">Subtitle</p>
    <div class="hero-actions">
      <button class="hero-btn">CTA</button>
    </div>
  </div>
</section>

<!-- Image background with overlay -->
<section class="ease-hero image-bg">
  <div class="ease-hero-overlay"></div>
  <div class="ease-hero-content">...</div>
</section>

Variants: purple, dark, light, image-bg

**Why is it useful?**
Provides a complete hero section with staggered entrance animations,
overlay support, and color themes — consistent with EaseMotion CSS's
component and animation philosophy.
