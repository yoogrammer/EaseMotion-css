\# ease-squish-card



Squish animation effect for card emphasis on click.



\## Classes



| Class | Description | Speed |

|-------|-------------|-------|

| `ease-squish` | Normal squish effect | 0.4s |

| `ease-squish-fast` | Fast squish effect | 0.2s |

| `ease-squish-slow` | Slow squish effect | 0.8s |

| `ease-squish-stretch` | Stretch squish effect | 0.4s |

| `ease-squish-bounce` | Squish with bounce | 0.5s |



\## Usage



```html

<div class="ease-squish">

&#x20;   Click to squish

</div>
