# ease-bounce

Infinite vertical bounce animation loop. A classic attention-grabbing effect for icons, badges, and call-to-action elements.

## Usage

```html
<div class="ease-bounce">Bounces</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-bounce` | Default bounce animation (infinite loop) |

## Notes

- Loops continuously by default
- Configurable iterations via `--ease-animation-iterations`
- Respects `prefers-reduced-motion`
