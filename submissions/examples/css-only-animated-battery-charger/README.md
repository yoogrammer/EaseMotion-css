# CSS-Only Animated Battery Charger

A modern, fluid animated battery charger component built entirely with CSS. It simulates a battery filling up with glowing liquid and rising bubbles, using only CSS gradients, variables, and keyframe animations.

## Features
- ✨ **100% CSS-only**: No JavaScript logic required.
- 🔋 **Fluid Charging Animation**: Keyframe animations control the height and color of the battery level, moving from a low state to a fully charged state over an 8-second loop.
- 🫧 **Dynamic Bubbles**: Individual bubble elements rise up through the battery liquid, utilizing CSS custom properties (`--i`) to randomize their speed and animation delay.
- 🎨 **Modern Aesthetics**: Uses backdrop-filter for a frosted glass effect and bright, neon drop-shadows for a cyberpunk feel.

## Files
- `demo.html`: Structure containing the battery shell, liquid level, and bubble elements.
- `style.css`: All the styling rules, complex gradients, and synchronized keyframe animations.

## Usage
Simply copy the HTML structure and link the CSS file to use this component. The `.battery-container` manages the layout and styling.
