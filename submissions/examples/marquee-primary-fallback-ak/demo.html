Direct color comparison

Wrong #6366f1
Correct #6c63ff

The difference is subtle but important — #6c63ff is EaseMotion's exact brand primary.

Why the fallback matters

CSS custom property fallbacks are used when the property is not defined (e.g., missing theme token, scoped override). The wrong fallback #6366f1 makes the marquee appear in a color that doesn't match EaseMotion's brand, which is especially noticeable in the scrolling marquee text.

/* Before */
.ease-marquee-primary {
  color: var(--ease-color-primary, #6366f1);
}

/* After */
.ease-marquee-primary {
  color: var(--ease-color-primary, #6c63ff);
}
