# User Profile Widget

## What does this do?

A responsive user profile widget for displaying profile information, statistics, and action buttons.

## How is it used?

```html
<div class="profile-widget">
  <div class="avatar">P</div>
  <h2>User Name</h2>
  <p class="role">Frontend Developer</p>
</div>
```

## Why is it useful?

User profile widgets are commonly used in social platforms, dashboards, portfolio websites, SaaS products, and community applications. This component provides a lightweight and responsive solution using only HTML and CSS.
