How it works

  • 3D Environment: Set up using perspective: 1400px and transform-style: preserve-3d.
  • Physical Overlap: In the resting state, cards are stacked with offset translateZ and subtle rotations on the Z-axis.
  • 3D Fan Out: Hovering the stack translates and rotates each card outward along a 3D arc, spreading them flat and readable.
  • Inspection Elevate: Hovering/focusing an individual card triggers a reset of Y/Z angles and lifts it forward via translate3d(..., ..., 140px) with a `z-index` override.
  • Accessibility: Full keyboard support is included using tabindex="0" and CSS :focus-visible rules.
