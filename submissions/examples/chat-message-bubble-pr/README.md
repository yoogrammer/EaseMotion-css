# Chat Message Bubble Component

## What does this do?

A responsive chat message bubble component for displaying sent and received messages in messaging interfaces.

## How is it used?

```html
<div class="message sent">
  <p>Hello there!</p>
  <span>10:16 AM</span>
</div>
```

## Why is it useful?

Chat interfaces are commonly used in messaging applications, support systems, collaboration tools, and social platforms. This component provides a clean and responsive solution using only HTML and CSS.
