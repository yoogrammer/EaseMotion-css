# Neon Scan Line Card Effect

## Description

A futuristic card animation featuring a glowing scan line that continuously moves across the card.

## Features

- Cyberpunk-inspired effect
- Pure CSS implementation
- Lightweight and reusable
- Smooth neon glow animation

## Files

- demo.html
- style.css
- script.js

## Usage

Open `demo.html` directly in a browser.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
