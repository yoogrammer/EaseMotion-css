# CSS Design Token Explorer Component

## What does this do?

Provides a visual explorer for common design system tokens including colors, typography, spacing, shadows, and border radius values.

## How is it used?

```html
<div class="color-card primary">
    <span>Primary</span>
    <small>#2563EB</small>
</div>
```

## Why is it useful?

Design tokens are the foundation of modern design systems. This component helps developers and designers visualize token values and understand how design systems are structured using pure HTML and CSS.
