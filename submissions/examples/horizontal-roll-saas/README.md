# EaseMotion: SaaS System Horizontal Roll Modal

A modern design framework engineered for productivity workspaces, setup menus, and cloud parameter adjustments with zero JavaScript pipeline overhead.

## Interaction Blueprint

- **Rotational Surface Roll**: Anchored to a strict geometric left pivot (`--roll-origin`), the component rotates on deployment from an edge-on position (`-95deg`) down to a standard flat viewport aspect.
- **Synchronized Slide Stagger**: Internal control modules and parameters translate smoothly forward across the X-axis (`--stagger-drift-x`), timing their emergence to merge effortlessly with the rolling panel reveal motion.

## Exposed Custom CSS Variables

Customize these root-level variable bounds inside `style.css` to fit your interface needs:

| Variable             | Default Metric | Functional Intent                                                 |
| :------------------- | :------------- | :---------------------------------------------------------------- |
| `--roll-duration`    | `0.75s`        | Complete lifecycle opening transition sequence timing window.     |
| `--roll-angle-start` | `-95deg`       | Geometric boundary vector constraint for hidden staging profiles. |
| `--roll-origin`      | `left center`  | Structural anchor pin trajectory focus point.                     |
| `--stagger-drift-x`  | `-30px`        | Horizontal displacement shift mapping distance for nested items.  |
