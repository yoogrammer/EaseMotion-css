Tabs Focus-Visible Demo

Self-contained demo of the keyboard-accessible tabs pattern from components/tabs.css. Use Tab to navigate between labels and observe the focus outline. Linked to issue #5509.

Overview

Use Tab to move focus to the next label. The framework's :focus-visible rule chain surfaces an outline on the visible label whenever a hidden input receives keyboard focus.

Specs

Three tabs, keyboard-navigable, no JavaScript.
