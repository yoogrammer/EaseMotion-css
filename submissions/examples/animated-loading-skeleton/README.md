# Animated Loading Skeleton

## What does this do?
Displays placeholder skeleton blocks with a shimmer animation to indicate content loading for text, avatar, and card layouts.

## How is it used?
Add the skeleton classes to any HTML element:

    <div class="skeleton-card">
      <div class="skeleton-avatar"></div>
      <div class="skeleton-lines">
        <div class="skeleton-line skeleton-line--long"></div>
      </div>
    </div>

## Why is it useful?
Provides a visual loading state that reduces perceived wait time and improves UX while content is being fetched. Works with both light and dark modes.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
