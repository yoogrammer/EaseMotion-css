Usage

Drop in a class. Done.

Link style.css then add the appropriate class to any element.

Button

/* HTML */ <button class="btn swing-button"> Click me </button>

Card

/* HTML */ <div class="card swing-card"> ... </div>

Badge

/* HTML */ <span class="badge swing-badge"> ● Live </span>

Any element

/* HTML — works on any element */ <a class="swing-hover">Nav link</a> /* Speed modifiers */ <div class="swing-hover swing-quick">...</div> <div class="swing-hover swing-slow">...</div>

Core keyframe logic

@keyframes swing-pendulum { 0% { transform: rotate(0deg); } /* Start at rest */ 10% { transform: rotate(-18deg); } /* Full amplitude swing left */ 25% { transform: rotate(13deg); } /* Return, damped by ~28% */ 40% { transform: rotate(-9deg); } /* Third arc */ 55% { transform: rotate(6deg); } /* Fourth arc */ 68% { transform: rotate(-3.5deg); }/* Nearly settled */ 80% { transform: rotate(1.8deg); } /* Micro-correction */ 100% { transform: rotate(0deg); } /* Natural rest */ } .swing-button { transform-origin: top center; /* Pivot from the TOP, not the middle */ display: inline-block; /* Required for transform */ } .swing-button:hover { animation: swing-pendulum 0.85s cubic-bezier(0.36,0.07,0.19,0.97) both; }
