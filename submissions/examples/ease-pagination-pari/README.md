## ease-pagination

**What does this do?**
A pagination component with hover animations, active and disabled states,
rounded variant, and color themes.

**How is it used?**
<ul class="ease-pagination">
  <li class="ease-page-item ease-page-disabled">
    <a class="ease-page-link">← Prev</a>
  </li>
  <li class="ease-page-item ease-page-active">
    <a class="ease-page-link">1</a>
  </li>
  <li class="ease-page-item">
    <a class="ease-page-link">2</a>
  </li>
</ul>

Variants: rounded, green, orange
States: ease-page-active, ease-page-disabled

**Why is it useful?**
Provides an animated, accessible pagination component using pure CSS —
consistent with EaseMotion CSS's lightweight component philosophy.
