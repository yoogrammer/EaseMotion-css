 
# ease-typewriter

Text animates as if being typed character by character.

## Classes

| Class | Speed | Delay |
|-------|-------|-------|
| `ease-typewriter` | 2 seconds | None |
| `ease-typewriter-fast` | 1 second | None |
| `ease-typewriter-slow` | 4 seconds | None |
| `ease-typewriter-delay-2` | 2 seconds | 2 seconds |

## Usage

```html
<div class="ease-typewriter">Hello, World!</div>
<div class="ease-typewriter-fast">Fast typing</div>
<div class="ease-typewriter-slow">Slow typing</div>
<div class="ease-typewriter-delay-2">Starts after 2s</div>
