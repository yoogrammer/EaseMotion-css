Rotate Animation — Design Token

Submission proposing that easemotion/rotate.css use the --ease-speed-slow design token for the .ease-rotate animation, so the rotation speed is themable from :root. Linked to issue #5574.

A
.ease-rotate

Why this matters

Every other animation in the framework — fade, slide, zoom, bounce, hover — references --ease-speed-* tokens. .ease-rotate is the only animation that hardcodes its duration. Switching to the token brings it in line with the rest of the framework and lets consumers override the rotation speed from a single CSS variable.
