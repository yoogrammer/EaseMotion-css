# ease-list-inside / ease-list-outside Utilities

Controls `list-style-position` on ordered and unordered lists.

## Usage

```html
<ul class="ease-list-inside">
  <li>Bullet wraps with text</li>
</ul>

<ol class="ease-list-outside">
  <li>Bullet stays in margin</li>
</ol>
```

## Classes

| Class               | CSS Property                 |
| ------------------- | ---------------------------- |
| `ease-list-inside`  | list-style-position: inside  |
| `ease-list-outside` | list-style-position: outside |
