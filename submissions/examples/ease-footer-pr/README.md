# Ease Footer Component

A modern, responsive website footer component with multiple columns, link groups, newsletter signup, and social media icons.

## Features

- **4-Column Layout**: Brand column, two link columns, and newsletter signup
- **Social Media Icons**: Row of social media icons with hover effects
- **Copyright Bar**: Bottom bar with copyright text and legal links
- **Fully Responsive**:
  - Desktop: 4-column grid
  - Tablet (≤768px): 2-column grid
  - Mobile (≤480px): Stacked single column
- **Link Hover Animation**: Smooth underline slide-in effect on hover
- **Newsletter Form**: Email input with submit button
