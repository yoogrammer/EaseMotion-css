# Tabs Panel

## What does this do?
A tabbed panel component with tab navigation and content panels that switch on click with an active tab indicator.

## How is it used?
Open demo.html in a browser. Click any tab to view its corresponding content panel. The active tab is highlighted with an accent color underline.

## Why is this useful?
Demonstrates a clean tabbed interface pattern for organizing content into switchable panels, commonly used in documentation, settings, and dashboard UIs.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
