# Liquid Button Effect

A button with a gooey/liquid fill animation using CSS transforms and border-radius morphing. The fill rises like a wave and rotates with organic motion on hover.

## Demo

Open `demo.html` in your browser to see:
- Default liquid fill with wave animation
- Blue, glow, and color variants
- Large, small, outline, and rounded sizes
