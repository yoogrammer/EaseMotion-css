How It Works

The card shrinks from scale(1) to scale(0.7) while sliding down by 20px and fading out to opacity(0).

@keyframes ease-zoom-out-down {
  from {
    transform: scale(1) translateY(0);
    opacity: 1;
  }
  to {
    transform: scale(0.7) translateY(20px);
    opacity: 0;
  }
}

Uses ease-in timing for a natural exit feel.
