# Interactive Motion Playground & Live CSS Generator

### 1. What does this do?
This is a self-contained, responsive design-playground page that lets developers interactively test every EaseMotion CSS utility/entrance/looping/hover animation class with customizable duration, delay, direction, iteration count, and easing functions, displaying a real-time animation preview and generating standard HTML/CSS code snippet styles to copy-paste.

### 2. How is it used?
Open the `demo.html` file directly in any modern web browser to load the playground:
- **Choose target elements:** Switch preview modes between a Card, Button, Avatar, or Text.
- **Select any animation:** Pick any of the 25+ framework animations from the structured class list.
- **Tweak parameters:** Change timing function presets (standard, elastic, bounce, expo), duration/delay times, direction, and repeat counts.
- **Live Curve Easing Visualizer:** Observe the generated cubic-bezier curve plotted live on a dynamic coordinate grid canvas.
- **Grab generated CSS & HTML:** Toggle between HTML and CSS code tabs, click "Copy" to copy optimized class syntax with override styles immediately into your project.

### 3. Why is it useful?
It fits EaseMotion CSS's philosophy by simplifying the developer workflow: rather than guessing durations or manually testing multiple timing functions in your source code, you can customize and visual-test styles before copying the drop-in HTML components, offering a rich developer utility that enhances dogfooding and highlights the entire framework API on one responsive screen.
