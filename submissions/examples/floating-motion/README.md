# Floating Motion Animation

## What does this do?

Adds reusable floating motion animations that create smooth vertical movement for UI elements.

---

## How is it used?

```html
<div class="ease-float">
    Content
</div>

<div class="ease-float-slow">
    Content
</div>

<div class="ease-float-fast">
    Content
</div>
```

---

## Why is it useful?

Floating animations are commonly used for hero illustrations, icons, feature cards, and call-to-action elements.

This utility expands the EaseMotion CSS animation collection with subtle motion effects that help attract attention without becoming distracting.
