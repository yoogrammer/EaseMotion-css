# SOC Monitoring Dashboard Component

## What does this do?

Displays a Security Operations Center (SOC) dashboard with threat metrics, alerts, incidents, and monitoring status.

## How is it used?

```html
<div class="card threats">
    <h2>124</h2>
    <p>Threats Detected</p>
</div>

<div class="panel">
    <h3>Critical Alerts</h3>
</div>
```

## Why is it useful?

Security teams use SOC dashboards to monitor threats, alerts, incidents, and infrastructure health in real time. This component provides a clean cybersecurity dashboard built entirely with HTML and CSS.
