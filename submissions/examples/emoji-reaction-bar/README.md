# Animated Emoji Reaction Bar

A CSS-only emoji reaction bar example for EaseMotion CSS.

This demo shows a set of emoji reaction buttons with smooth hover animations, bounce effects, soft shadows, and tooltip labels.

## Features

- Pure HTML and CSS
- No JavaScript required
- Bounce animation on hover
- Tooltip label on hover/focus
- Responsive layout
- Beginner-friendly example
