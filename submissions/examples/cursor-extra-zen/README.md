## What does this do?

Provides additional cursor utility classes beyond the ones in EaseMotion core (pointer, default, not-allowed).

## How is it used?

```html
<div class="cursor-grab">Drag me</div>
<div class="cursor-wait">Loading...</div>
```

## Why is it useful?

Grab cursors are essential for draggable UI elements and sliders. Text and wait cursors cover common UX states. These complement the existing cursor utilities.
