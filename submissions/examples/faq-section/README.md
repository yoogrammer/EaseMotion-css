# FAQ Section

A clean, CSS-only collapsible FAQ section for the EaseMotion CSS documentation site.

## Preview
Open `demo.html` in a browser to see it in action.

## Features
- Pure HTML `<details>` + `<summary>` — zero JavaScript
- Uses EaseMotion CSS custom properties (`--ease-color-primary`)
- Animated open/close indicator (+ / −)
- Fully composable with existing `ease-*` classes

## Usage
Copy the `.faq-section` HTML structure into your page and link `style.css`.
