Frequently Asked Questions

How does EaseMotion CSS differ from Tailwind?

EaseMotion uses human-readable class names like ease-fade-in and is animation-first. No build step or config needed — just link one file.

Can I use it with React or Vue?

Yes. Install via npm and import the stylesheet in your component. Works with any framework that renders standard class attributes.

Does it work without a build step?

Yes. Link the CDN stylesheet and start using classes immediately. No bundler or compiler required.

How do I submit a new component?

Add your demo inside submissions/examples/your-feature/ with demo.html, style.css, and README.md. The maintainer reviews and integrates it.

Can I override the default colors and tokens?

Yes. Override any --ease-* CSS custom property in your :root to theme the entire framework without touching any source file.
