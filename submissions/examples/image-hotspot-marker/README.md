# Image Hotspot Marker

An image with interactive marker dots positioned at specific locations. Hovering over a dot reveals a tooltip with the label. Dots scale up on hover for visual feedback. The tooltip follows the dot position relative to the image container.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser. Hover over the colored dots on the image to see location labels.

## Accessibility notes

Each dot is a `<button>` with descriptive text. Tooltips are revealed on hover.
