📊 ease-z-index utilities

Z-index utilities for controlling stacking order

Z-Index Stacking Order

The higher the z-index, the closer to the top (front).

z-index: 0
(Back)
z-index: 10
(Middle)
z-index: 50
(Front)

Available Z-Index Values

ease-z-0 → z-index: 0
ease-z-10 → z-index: 10
ease-z-20 → z-index: 20
ease-z-50 → z-index: 50
ease-z-100 → z-index: 100
ease-z-top → z-index: 9999
⚠️ Note: z-index only works on positioned elements (position: relative, absolute, fixed, or sticky). Make sure your element has a position property set for z-index to take effect.

Usage

<!-- Z-index utilities require position property -->
<div style="position: relative;">
    <div class="ease-z-0">Lowest stacking</div>
    <div class="ease-z-10">Higher stacking</div>
    <div class="ease-z-100">Even higher</div>
    <div class="ease-z-top">Highest stacking (modal, tooltip, etc.)</div>
</div>

<!-- Common use cases -->
<div class="ease-z-top">Modal overlay</div>
<div class="ease-z-100">Dropdown menu</div>
<div class="ease-z-50">Sticky header</div>
