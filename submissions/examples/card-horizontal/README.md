# ease-card-horizontal

Horizontal flex-based card layout with wrapping, gap, and padding. Arranges content side-by-side and collapses to vertical on mobile.

## Usage

```html
<div class="ease-card ease-card-horizontal">
  <img src="..." />
  <div class="ease-card-body">Content</div>
</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-card-horizontal` | Horizontal card (stacks on mobile) |

## Notes

- Collapses to vertical at `max-width: 640px`
- Use with `ease-card-body` for text content
- Ideal for profile cards, media cards, and product listings
