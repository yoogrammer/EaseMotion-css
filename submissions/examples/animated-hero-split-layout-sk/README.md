# Animated Hero Split Layout

**What does this do?**  
A two-column hero section with content on the left and abstract gradient decorations on the right. When the user hovers over the hero container, the left column lifts upward and the decorative shapes pulse with a subtle glow animation.

**How is it used?**  
Open `demo.html` in any modern browser. The hero responds to hover across the entire container and collapses to a single centered stack on mobile viewports (≤720px). Colors and sizing can be adjusted via CSS custom properties or direct style edits.

**Why is it useful?**  
Hero sections are the first thing visitors see. Subtle motion cues on hover — content lift and an organic pulse — draw attention without being distracting, guiding the eye naturally across the layout. The pure-CSS approach keeps dependencies at zero and makes the pattern trivial to drop into any project.
