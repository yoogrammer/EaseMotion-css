# CSS-only Notification Bell Component

An elegant, animated notification bell component built entirely with CSS. No JavaScript required.

## Overview
This component features a sleek notification bell with an unread badge. When hovered, the bell smoothly swings to simulate a ringing effect, while the unread badge pulses to catch the user's attention.

## Features
- ✨ 100% CSS-only (no JavaScript)
- 🔔 Smooth ringing/swinging animation on hover
- 🔴 Pulsing unread badge animation
- 🎨 Modern dark mode aesthetic
- 📱 Fully responsive

## Files
- `demo.html`: The HTML structure of the notification bell.
- `style.css`: The CSS styling and keyframe animations.

## Usage
Simply copy the HTML structure and link the CSS file to use this component in your projects. The `.ease-bell-container` acts as the main wrapper, and you can adjust its `border-radius` or padding to fit your needs.

## Customization
You can easily customize the colors by changing the CSS variables or hex values in `style.css`:
- Notification badge color: `#ef4444` (Red)
- Icon color: `#cbd5e1` to `#f8fafc` on hover
- Background colors: `#1e293b` to `#26334a` on hover
