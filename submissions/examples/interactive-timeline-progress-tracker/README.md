# Interactive Timeline Progress Tracker

## Description

A multi-step animated progress timeline with milestone activation effects and dynamic progress line filling.

## Features

- Animated milestone activation
- Dynamic progress line
- Checkmark completion state
- Interactive step progression
- Lightweight implementation

## Files

- demo.html
- style.css
- script.js

## Usage

Open `demo.html` and click the Next Step button to advance through the timeline.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
