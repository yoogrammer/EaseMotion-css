Introduction to Layout

This chapter covers the fundamental principles of CSS layout models — from the classic box model to modern grid systems.

The Box Model

Every element in CSS is a rectangular box composed of content, padding, border, and margin areas.

Content Area

The innermost region where text and images live. Sized by width / height or intrinsic dimensions.

Padding & Border

Padding sits inside the border; both expand the visual footprint without affecting the content size directly (when box-sizing: border-box is used).

Normal Flow

Block elements stack vertically; inline elements flow horizontally. This is the default arrangement before any positioning or flex/grid intervention.

Flexbox Deep Dive

Flexbox solves one-dimensional layout challenges with alignment, distribution, and ordering capabilities.

Main Axis & Cross Axis

The flex-direction property determines the main axis (row or column). The cross axis runs perpendicular.

Alignment Properties

justify-content aligns along the main axis; align-items and align-self handle the cross axis.

CSS Grid

Grid handles two-dimensional layouts with rows and columns simultaneously. It is the most powerful layout tool in CSS.
