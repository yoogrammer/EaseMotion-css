# Gallery Image Lightbox

A lightbox gallery component with next/previous navigation, keyboard support, and thumbnail strip navigation at the bottom.

## Features

- Grid of thumbnails with hover effects
- Lightbox overlay with fade-in and image scale transition
- Previous / Next navigation via buttons and arrow keys
- Keyboard `Escape` to close
- Bottom thumbnail strip for quick navigation
- Image counter display

## Custom Properties

| Property                     | Default            | Description                     |
| ---------------------------- | ------------------ | ------------------------------- |
| `--lightbox-duration`        | `0.35s`            | Transition duration             |
| `--lightbox-overlay-color`   | `rgba(0,0,0,0.88)` | Overlay background color       |
| `--lightbox-bg`              | `#0f0f1a`          | Page background                 |
| `--lightbox-nav-color`       | `#ffffff`          | Navigation button color         |
| `--lightbox-radius`          | `12px`             | Image and item border radius    |
