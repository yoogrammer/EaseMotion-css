# ease-card-neumorphic

Neumorphic (soft UI) card variant with raised/pressed shadow effects.

## Usage

```html
<div class="ease-card ease-card-neumorphic">Soft UI card</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-card-neumorphic` | Default neumorphic (medium shadow) |
| `ease-card-neumorphic-sm` | Subtle shadow |
| `ease-card-neumorphic-lg` | Heavy shadow for more depth |
| `ease-card-neumorphic-inset` | Inset shadow for pressed look |

## Notes

- Best on light gray backgrounds (not pure black/white)
- Hover lift is built into the base `.ease-card-neumorphic` class
- Dark mode automatically adjusts shadows
