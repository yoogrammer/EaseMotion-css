# Rock Paper Scissors

## What does this do?
A classic Rock Paper Scissors game with computer AI, animated result display, running score tracker, and best-of series mode.

## How is it used?
Choose Rock, Paper, or Scissors to play against the computer. The result is shown with icons and a message. Track wins, losses, and draws. Enable best-of mode for a series.

## Why is this useful?
A fun, polished take on a classic game that's great for quick entertainment or settling disputes.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to play.
