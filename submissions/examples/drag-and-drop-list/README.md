# Drag and Drop List

A reorderable task list using the native HTML Drag and Drop API. Items can be dragged to new positions within the list. Visual feedback includes a reduced opacity on the dragged item and a highlight indicator on the drop target.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser. Drag any list item to reorder it.

## Accessibility notes

Uses the native `draggable` attribute and Drag and Drop API. Keyboard-based reordering is not implemented in this example.
