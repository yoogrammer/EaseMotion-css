# Climate Weather Widget Showcase

Issue: [#10488](https://github.com/SAPTARSHI-coder/EaseMotion-css/issues/10488)

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A modern animated weather dashboard card featuring current weather information, detailed metrics, and a five-day forecast.

Inspired by modern weather applications and dashboard interfaces.

## Features

- Pure HTML and CSS
- Current temperature display
- Weather condition icon
- Location information
- Humidity statistics
- Wind speed information
- Pressure information
- 5-day weather forecast
- Animated weather icon
- Floating gradient effects
- Hover interactions
- Responsive design
- Weather-specific gradient themes

## Usage

```html
<div class="weather-card sunny">
  ...
</div>
```

## Weather Themes

```css
.sunny
.cloudy
.rainy
```

## Main Classes

```css
.weather-card
.weather-header
.weather-stats
.forecast-grid
.forecast-day
```

## Why It Fits EaseMotion CSS

- Animation-first design
- Human-readable class names
- Reusable dashboard component
- Practical real-world use case
- Responsive and modern UI
- Lightweight implementation
