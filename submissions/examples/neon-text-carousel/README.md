# Neon Text Carousel

## What it does

Animated rotating word carousel that cycles through different words with fade/slide transitions. Each word has its own neon gradient color.

## How to use it

<div class="ease-neon-carousel">
    <div class="ease-carousel-track">
        <span class="ease-carousel-word">Word 1</span>
        <span class="ease-carousel-word">Word 2</span>
        <span class="ease-carousel-word">Word 3</span>
    </div>
</div>

## Variants

- ease-carousel-fast - 4s cycle
- ease-carousel-slow - 12s cycle
- ease-carousel-pause - Paused state

## Why it fits EaseMotion CSS

- Pure CSS animation
- Gradient text colors
- Smooth transitions
- Speed control
