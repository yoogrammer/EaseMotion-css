# Sandbox Component Showcase: Pure-CSS Recursive Directory Tree-View Explorer

## Overview
A zero-dependency, pure-CSS interactive tree-view file explorer and directory navigation panel isolated within an examples subdirectory. It handles recursive folder disclosure pathways using form input switches, managing multi-tier states cleanly without executing script lines.

## 📁 Sandbox Configuration Files
* `demo.html` — Interactive sidebar tree template hosting deeply layered nested lists to evaluate cascading expansion parameters.
* `style.css` — Scoped modifier layout layer setting high-performance max-height scaling curves linked back to global tokens.

## 🚀 Key Layout Mechanics
1. **Recursive State Inputs:** Avoids initializing global state-mapping arrays or listeners by embedding a hidden `input[type="checkbox"]` switch inside each directory tier node block.
2. **Dynamic Height Disclosure Transitions:** Solves tree truncation limits by animating `max-height` and `opacity` properties over clean cubic-bezier acceleration coordinates (`cubic-bezier(0.4, 0, 0.2, 1)`). This maps expansion limits smoothly on the compositor track.
3. **Sandbox Compliance Safeguard:** Fully satisfies repository security policies by placing every asset line strictly inside `submissions/examples/tree-view-explorer/`.
