# Magnetic Button Hover Effect

This submission resolves Feature Request #11232 by adding a smooth, interactive "magnetic pull" effect for buttons.

## What it does
When the cursor hovers near or over an element with the `.btn-magnetic` class, the element gently translates a few pixels in the direction of the mouse pointer. When the cursor leaves, it gracefully snaps back into place using a bouncy cubic-bezier transition.

## How to use it
In this demo, the effect is applied by adding the `.btn-magnetic` class to standard `.ease-btn` elements. Include `style.css` and `script.js` to enable the effect.

## Why it fits
EaseMotion CSS focuses on bringing premium animations to developers effortlessly. A magnetic hover effect adds an interactive, high-end feel to CTAs perfectly aligning with the framework's philosophy.
