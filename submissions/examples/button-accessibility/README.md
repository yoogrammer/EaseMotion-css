# Button Accessibility Improvement

## Overview
This submission improves button accessibility and usability using reusable utility classes.

## Features
- Accessible button states
- Hover and focus effects
- Disabled state support
- Multiple variants (primary, success, outline)

## Usage
Add classes like:

```html
<button class="ease-btn ease-btn-primary">Click Me</button>
Variants
Primary Button
Success Button
Outline Button
Disabled Button
