# ease-glass-tilt

Premium glassmorphism card with 3D tilt and lift effect on hover.

## Files

| File | Description |
|------|-------------|
| 📄 [demo.html](./demo.html) | Interactive demo |
| 🎨 [style.css](./style.css) | Glass tilt styles |
| 📖 [README.md](./README.md) | Documentation |

## Classes

| Class | Color |
|-------|-------|
| `ease-glass-tilt` | Blue/Purple gradient |
| `ease-glass-tilt-purple` | Purple/Pink gradient |
| `ease-glass-tilt-teal` | Teal/Blue gradient |
| `ease-glass-tilt-amber` | Amber/Red gradient |

## Features

- 🔮 Glassmorphism (backdrop-filter: blur)
- 🎭 3D perspective tilt (rotateX + rotateY)
- ✨ Smooth lift effect (translateY)
- 💎 Glowing gradient border
- 🌟 Deepened shadow on hover

## Usage

```html
<!-- Basic glass tilt card -->
<div class="ease-glass-tilt">
    <h2>Card Title</h2>
    <p>Card content here</p>
    <button>Action</button>
</div>

<!-- With badge and price -->
<div class="ease-glass-tilt">
    <div class="badge">POPULAR</div>
    <h2>Premium Plan</h2>
    <div class="price">$19<span>/month</span></div>
    <button>Get Started</button>
</div>

<!-- Color variants -->
<div class="ease-glass-tilt-purple">Purple variant</div>
<div class="ease-glass-tilt-teal">Teal variant</div>
