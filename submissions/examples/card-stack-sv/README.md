# Stacked Cards with Staggered Hover Lift

## What does this do?

Creates a stack of overlapping cards where hovering a card lifts it while nearby cards subtly shift to create depth.

## How is it used?

```html
<div class="stack-wrapper">
  <div class="card">...</div>
  <div class="card">...</div>
  <div class="card">...</div>
</div>
```

## Why is it useful?

Layered card stacks are commonly used in dashboards, portfolios, previews, and onboarding flows. The staggered hover interaction adds depth and visual interest while remaining lightweight and dependency-free.
