Usage

<!-- Basic neon glow -->
<div class="ease-neon-glow">Neon content</div>

<!-- Color variants -->
<div class="ease-neon-glow-pink">Pink neon</div>
<div class="ease-neon-glow-green">Green neon</div>
<div class="ease-neon-glow-blue">Blue neon</div>

<!-- Different speeds -->
<div class="ease-neon-glow-fast">Fast pulse</div>
<div class="ease-neon-glow-slow">Slow pulse</div>

<!-- Different intensities -->
<div class="ease-neon-glow-subtle">Subtle glow</div>
<div class="ease-neon-glow-strong">Strong glow</div>

<!-- On buttons and inputs -->
<button class="ease-neon-glow">Neon Button</button>
<input class="ease-neon-glow" placeholder="Neon Input">

⚡ Neon Glow Features

  • 💡 Soft pulsing border glow animation
  • 🎨 Multiple color variants (blue, pink, green, red, purple, yellow, cyan, orange, white)
  • ⚡ Speed variations (fast, normal, slow)
  • 📊 Intensity levels (subtle, normal, strong)
  • 🎯 Focus indicator variant
  • 💫 Pure CSS, no JavaScript
