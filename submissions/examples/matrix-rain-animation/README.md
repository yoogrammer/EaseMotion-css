# Matrix Rain Animation

A lightweight Matrix-style rain animation utility built using pure CSS.

## Features

- Cascading green character streams
- Multiple falling columns
- Different animation speeds
- Smooth fade-in and fade-out effect
- Pure CSS implementation

## Usage

```html
<div class="matrix-container">
  <div class="column column1">...</div>
  <div class="column column2">...</div>
</div>
```

## Why It Fits EaseMotion CSS

- Human-readable naming
- Animation-first utility
- Lightweight implementation
- Reusable hacker-style background effect
- Pure CSS ambient animation
