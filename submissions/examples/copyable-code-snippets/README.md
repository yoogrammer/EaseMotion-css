# Copy Code Button

## Description

This example demonstrates a reusable copy button for documentation code examples.

## Features

- Copy code instantly
- Uses Clipboard API
- Visual success feedback
- Reusable component pattern

## How It Works

Clicking the copy button copies the code snippet to the clipboard.

A temporary success message is displayed after copying.

## Files

- demo.html
- style.css
- README.md

## Use Case

Useful for documentation websites where developers frequently copy code examples.
