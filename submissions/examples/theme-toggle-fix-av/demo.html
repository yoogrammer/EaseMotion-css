Theme Toggle Fix

Circular sweep reveal with zero flicker — using View Transitions API + suppressed CSS transitions during the sweep.

🐛 The Bug

Two transition systems were firing at the same time: the View Transitions API circular sweep AND individual CSS transitions on body, header, sidebar, etc. Both competing caused flickering mid-animation.

✅ The Fix

A data-transitioning attribute is added to <html> during the sweep. CSS transitions on all affected elements are suppressed via that attribute, so only the View Transitions API runs. Removed after .finished resolves.

🎯 Result

Smooth circular sweep reveal on every toggle. No flash. No flicker. CSS transitions still work perfectly for hover and focus effects outside of theme switching.

data-theme="dark" | data-theme="light"
