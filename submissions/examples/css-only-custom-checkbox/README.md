# CSS-only Custom Animated Checkbox

### 1. What does this do?
A beautiful, custom-designed CSS checkbox that replaces the boring native browser checkbox. It features a satisfying "pop" scale animation when checked, along with an animated checkmark icon drawn using CSS borders, and includes focus states for accessibility.

### 2. How is it used?
```html
<label class="custom-checkbox">
  <input type="checkbox" />
  <span class="checkmark"></span>
  <span class="label-text">I agree to the terms</span>
</label>
```

### 3. Why is this useful?
Styling default checkboxes consistently across browsers is notoriously difficult and usually looks outdated. Developers often need a reliable, drop-in snippet to create polished, branded forms. This component provides a highly reusable, accessible, and purely CSS-driven solution for modern web apps.
