Component

ease-progress-bar

Composable progress bar component. Drop in a wrapper and a fill, set a CSS variable, toggle a class — done.

Variants

Color Variants

Apply a color variant class to .ease-progress-bar. Default is primary.

// .ease-progress-bar + variant
Primary .ease-progress-primary 72%
Success .ease-progress-success 88%
Warning .ease-progress-warning 55%
Danger .ease-progress-danger 32%
Modifier

Animated Striped

Add .ease-progress-animated to the bar for moving diagonal stripes. Works on any color variant.

// .ease-progress-bar.ease-progress-animated
Primary animated 65%
Success animated 80%
Warning animated 45%
Sizes

Size Variants

Apply size modifiers to the .ease-progress wrapper.

// .ease-progress-sm / (default) / .ease-progress-lg / .ease-progress-xl
.ease-progress-sm — 6px 60%
Default — 10px 60%
.ease-progress-lg — 16px 60%
.ease-progress-xl — 24px 60%
Layout

Labeled & Stacked

Use .ease-progress-labeled to put an inline label beside the bar. Use .ease-progress-stack + .ease-progress-row for named multi-bar layouts.

// .ease-progress-labeled
78%
55%
92%
// .ease-progress-stack (named rows)
CSS 94%
JavaScript 81%
Accessibility 67%
Performance 43%
State

Indeterminate

Use when the total progress value is unknown (e.g. an async operation without a known end). Add .ease-progress-indeterminate to the wrapper track.

// .ease-progress.ease-progress-indeterminate
Primary
Success
Interactive

Dynamic Value

Update --ease-progress-value via JS — the CSS transition handles the animation automatically.

// Live value update
Progress 0%
