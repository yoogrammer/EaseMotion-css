# Animated Skeleton Loader (`ease-skeleton-cv`)

A lightweight animated skeleton loader built with **HTML5** and **Pure CSS**.

## Features

- ✨ CSS shimmer effect
- 🃏 Card loader
- 📋 List loader
- 👤 Profile loader
- 📱 Responsive
- No JavaScript
- No external dependencies

## Folder Structure

```text
ease-skeleton-cv/
├── demo.html
├── style.css
└── README.md
```

## Usage

Open `demo.html` directly in any modern browser.

## Browser Support

Works in all modern browsers.
