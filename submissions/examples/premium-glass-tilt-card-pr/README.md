# Premium Glass Tilt Card (-pr)

A highly aesthetic, premium glassmorphism card featuring 3D tilt hover effects and deep drop shadows.

## What does this do?
Provides a luxurious, translucent card component that reacts to user interaction with a 3D perspective pop.

## How is it used?
```html
<div class="glass-tilt-card">
  <div class="glass-card-content">
    <!-- content here -->
  </div>
</div>
```

## Why is it useful?
It brings modern, cutting-edge iOS/macOS style glassmorphism into the EaseMotion framework, perfect for pricing tiers, featured content, or dashboard widgets.
