# Animated Scan Line Overlay

## What it does

CRT-style animated scan line overlay that moves from top to bottom with a glowing trail.

## How to use it

<div class="ease-scan-overlay"></div>

## Variants (SCSS)

- ease-scan-fast - 1.5s animation
- ease-scan-slow - 6s animation
- ease-scan-paused - Paused state
- ease-scan-glow - Blurred glow effect
- ease-scan-subtle - Reduced opacity

## Features

- Moving bright scan line
- Static CRT scan lines (repeating gradient)
- Speed control
- Pause/Resume toggle
- SCSS variables for customization

## Why it fits EaseMotion CSS

- Pure CSS effect
- Retro CRT aesthetic
- Speed control
- Easy to customize colorss
