Card Z-Index Collision Fix

In the core framework, hovering over a card assigns it z-index: 2. If that card lives inside a complex grid where neighboring columns possess higher z-indices, the card will pop "underneath" its neighbors.

This patch dynamically introduces an elevated Z-Index token to ensure cards always overlap sibling containers when hovered!
