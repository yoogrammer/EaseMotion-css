# File Upload Widget

## What does this do?

A responsive file upload widget that provides a drag-and-drop style interface for selecting files.

## How is it used?

```html
<input type="file" id="file-upload">

<label for="file-upload" class="upload-btn">
  Choose File
</label>
```

## Why is it useful?

File upload interfaces are commonly used in dashboards, forms, SaaS applications, portfolio websites, and document management systems. This component provides a lightweight and responsive solution using only HTML and CSS.
