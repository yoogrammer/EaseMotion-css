How It Works

This uses the CSS @property rule to register a custom property --num with syntax: "<integer>". This lets CSS animate integer values directly in keyframes.

The counter-reset property maps the animated value to a CSS counter, and counter(num) in the ::after pseudo-element displays the current number.

Set the target value with --target as an inline style. The animation goes from 0 to that value.
