# ease-neon-glow

Soft glowing border animation that pulses rhythmically — creates beautiful neon glow effects.

## Files

| File | Description |
|------|-------------|
| 📄 [demo.html](./demo.html) | Interactive demo |
| 🎨 [style.css](./style.css) | Neon glow styles |
| 📖 [README.md](./README.md) | Documentation |

## Classes

| Class | Color | Speed |
|-------|-------|-------|
| `ease-neon-glow` | Blue | 2s |
| `ease-neon-glow-pink` | Pink | 2s |
| `ease-neon-glow-green` | Green | 2s |
| `ease-neon-glow-red` | Red | 2s |
| `ease-neon-glow-purple` | Purple | 2s |
| `ease-neon-glow-yellow` | Yellow | 2s |
| `ease-neon-glow-cyan` | Cyan | 2s |
| `ease-neon-glow-orange` | Orange | 2s |
| `ease-neon-glow-white` | White | 2s |
| `ease-neon-glow-fast` | Blue | 1s |
| `ease-neon-glow-slow` | Blue | 3s |
| `ease-neon-glow-subtle` | Blue (subtle) | 2s |
| `ease-neon-glow-strong` | Blue (strong) | 2s |

## Usage

```html
<!-- Basic neon glow -->
<div class="ease-neon-glow">Neon content</div>

<!-- Color variants -->
<div class="ease-neon-glow-pink">Pink neon</div>
<div class="ease-neon-glow-green">Green neon</div>

<!-- Different speeds -->
<div class="ease-neon-glow-fast">Fast pulse</div>
<div class="ease-neon-glow-slow">Slow pulse</div>

<!-- On buttons -->
<button class="ease-neon-glow">Neon Button</button>
