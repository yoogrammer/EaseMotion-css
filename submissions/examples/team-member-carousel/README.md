# Team Member Card Carousel

A responsive team member showcase component with profile cards, hover animations, and social icon reveal effects.

## Features

- Team member profile cards
- Responsive layout
- Hover lift animation
- Social icon reveal effect
- Featured member highlighting
- Pure HTML and CSS

## Files

- demo.html
- style.css
- README.md

## Usage

Open demo.html directly in any browser.

## Why it fits EaseMotion CSS

This component focuses on smooth interactions, reusable UI patterns, lightweight animations, and responsive design principles that align with EaseMotion CSS.
