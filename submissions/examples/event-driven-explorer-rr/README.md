# Event-Driven Architecture Explorer Component

## What does this do?

Provides a visual interface for monitoring event producers, brokers, topics, consumers, and message throughput in event-driven systems.

## How is it used?

```html
<div class="node broker">
    Event Broker
</div>
```

## Why is it useful?

Modern distributed systems increasingly rely on event-driven architectures and streaming platforms. This component provides a reusable visualization interface for understanding event flow, message processing, and consumer relationships while demonstrating modern asynchronous architecture design patterns.
