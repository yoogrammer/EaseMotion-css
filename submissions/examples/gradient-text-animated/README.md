# ease-gradient-text-animated

Animated gradient text effect. Applies a rotating background-position gradient to create a flowing color animation on text.

## Usage

```html
<h1 class="ease-gradient-text-animated">Animated Gradient</h1>
```

## Variants

| Class | Description |
|---|---|
| `ease-gradient-text-animated` | Animated flowing gradient text |

## Notes

- Uses `background-clip: text` for the gradient text effect
- Animation continuously shifts the gradient position
- Pairs with text-stroke and fill-none for outlined animated text
