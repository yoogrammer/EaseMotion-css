# Animated Kanban Task Board

## What does this do?

Provides a modern Kanban-style workflow board with animated task cards and responsive columns.

## Features

- To Do column
- In Progress column
- Done column
- Animated task cards
- Hover interactions
- Responsive dashboard layout

## How is it used?

Open `demo.html` directly in a browser.

## Why is it useful?

It demonstrates a common project-management interface used in productivity applications while showcasing card animations and dashboard layouts.
