🚀 Getting Started with EaseMotion CSS

A beginner-friendly guide to build animated interfaces

📦 Step 1: Add the CDN Link

Add this single line to your HTML <head> section:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/easemotion-css/easemotion.min.css" />

✅ No build steps. No configuration. Just one line and you're ready!

✨ Step 2: Your First Animation

Add ease-fade-in class to any element for a smooth fade effect:

<h1 class="ease-fade-in">Hello, EaseMotion!</h1>

👆 This text faded in!

🔘 Step 3: Your First Button

Use ease-btn with variants like ease-btn-primary:

<button class="ease-btn ease-btn-primary ease-btn-pill">
    Get Started
</button>

🃏 Step 4: Your First Card

Add ease-card class for beautifully styled cards:

<div class="ease-card ease-card-shadow ease-card-hover">
    <div class="ease-card-header">
        <h3>Card Title</h3>
    </div>
    <div class="ease-card-body">
        <p>Card content goes here</p>
    </div>
    <div class="ease-card-footer">
        <button class="ease-btn ease-btn-primary ease-btn-sm">Action</button>
    </div>
</div>

✨ Feature Card

Beautiful cards with zero configuration!

🎨 Step 5: Combine Classes

EaseMotion classes are composable — stack them freely!

<div class="ease-center ease-fade-in ease-slide-up">
    <h1>Build faster.</h1>
    <button class="ease-btn ease-btn-primary ease-hover-grow">
        Get Started
    </button>
</div>

🚀 Build faster.

📝 Complete Example

Here's a complete HTML file to get you started:

<!DOCTYPE html>
<html>
<head>
    <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/easemotion-css/easemotion.min.css" />
</head>
<body>
    <div class="ease-center ease-fade-in">
        <h1>Welcome to EaseMotion!</h1>
        <p class="ease-slide-up ease-delay-200">Animation-first CSS for humans</p>
        <button class="ease-btn ease-btn-primary ease-hover-grow">
            Get Started →
        </button>
    </div>
</body>
</html>
