# ease-appearance-none Utility

Removes default browser styling from form elements (`<input>`, `<select>`, `<button>`, `<textarea>`, etc.). Useful for creating custom-styled form controls.

## Usage

```html
<select class="ease-appearance-none">
  <option>Custom styled select</option>
</select>
<button class="ease-appearance-none">Custom button</button>
```

## Classes

| Class                  | CSS Property                                                      |
| ---------------------- | ----------------------------------------------------------------- |
| `ease-appearance-none` | appearance: none; -webkit-appearance: none; -moz-appearance: none |
