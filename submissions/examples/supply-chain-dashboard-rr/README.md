# Supply Chain Visibility Dashboard Component

## What does this do?

Displays a supply chain monitoring dashboard with inventory metrics, shipment tracking, warehouse status, and delivery performance indicators.

## How is it used?

```html
<div class="card inventory">
    <h2>24.8K</h2>
    <p>Inventory Units</p>
</div>

<div class="panel">
    <h3>Recent Shipments</h3>
</div>
```

## Why is it useful?

Supply chain teams rely on visibility dashboards to track inventory, logistics operations, warehouse performance, and delivery efficiency. This component provides a clean enterprise dashboard built entirely with HTML and CSS.
