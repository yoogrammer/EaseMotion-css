# Color Palette Generator

## What does this do?
Generates random 5-color palettes with hex and RGB display. Individual colors can be locked to preserve them during regeneration.

## How is it used?
Open demo.html in a browser. Click "Generate" for new colors. Click the lock icon to preserve a color, then generate again.

## Why is this useful?
Demonstrates color generation, lock/unlock state management, and luminance-based text contrast.

## Tech Stack
- HTML
- CSS
- Vanilla JavaScript

## Preview
