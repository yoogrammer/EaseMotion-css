Image-Orientation Utility

Controls whether the browser respects EXIF orientation metadata in images or ignores it.

img-orient-none (ignore EXIF orientation)

<img class="img-orient-none" src="...">
Without class
img-orient-none

img-orient-from-image (respect EXIF)

<img class="img-orient-from-image" src="...">
Without class
img-orient-from-image
