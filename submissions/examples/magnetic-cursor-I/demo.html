Usage

<button class="ease-magnetic-cursor">Hover Me</button>

<!-- Different intensities -->
<div class="ease-magnetic-subtle">Subtle pull</div>
<div class="ease-magnetic-strong">Strong pull</div>

<!-- Different directions -->
<div class="ease-magnetic-up">Pulls up</div>
<div class="ease-magnetic-right">Pulls right</div>

<!-- Different speeds -->
<div class="ease-magnetic-fast">Fast pull</div>
<div class="ease-magnetic-slow">Slow pull</div>

🧲 Magnetic Cursor Features

  • 🧲 Elements move toward cursor on hover
  • 🎯 Multiple intensity levels (subtle, normal, strong)
  • ➡️ Directional variants (up, down, left, right)
  • ⚡ Speed variations (fast, normal, slow)
  • 🎨 Works on buttons, cards, icons, links, images
  • 💫 Pure CSS, no JavaScript required for basic effect
