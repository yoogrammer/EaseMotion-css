## ease-accordion

**What does this do?**
A CSS-only accordion/FAQ component using native HTML details and summary
elements with smooth open/close animations and color theme variants.

**How is it used?**
<div class="ease-accordion">
  <details>
    <summary>Question here?</summary>
    <div class="accordion-body">Answer here.</div>
  </details>
</div>

Color themes: add class `green` or `orange` to `.ease-accordion`

**Why is it useful?**
Provides a fully accessible, zero-JavaScript accordion component
with animated arrows and smooth panel transitions — consistent
with EaseMotion CSS's lightweight, pure CSS philosophy.
