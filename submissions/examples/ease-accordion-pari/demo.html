What is EaseMotion CSS?
EaseMotion CSS is a curated CSS animation framework focused on smooth, lightweight utility classes for modern web components.
How do I use it?
Simply add the utility class to your HTML element. No JavaScript required — pure CSS animations powered by keyframes and transitions.
Is it free to use?
Yes! EaseMotion CSS is open source and free to use in both personal and commercial projects.

Green Theme

What animations are included?
Entrance, exit, emphasis, hover, and utility animations covering bounces, fades, slides, glows, and more.
Can I contribute?
Absolutely! Submit your animation inside submissions/examples/ and open a Pull Request. The maintainer reviews and integrates the best ones.

Orange Theme

Does it work without JavaScript?
Yes! The accordion uses the native HTML details and summary elements — no JS needed at all.
Which browsers are supported?
Chrome, Firefox, Safari, and Edge — all modern browsers support the details/summary elements natively.
