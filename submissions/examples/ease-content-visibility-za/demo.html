ease-content-visibility utilities

Lazy-render off-screen content for better performance.

ease-content-visibility-auto-za

Defers rendering and painting until the element is near the viewport. Use on long content sections, lists, cards, and below-the-fold content to speed up initial page load.

ease-content-visibility-hidden-za

Skips rendering entirely — the element takes no layout or paint cost. Unlike display: none, the element still has layout size for containment purposes.

ease-content-visibility-visible-za

Forces the element to render normally, overriding any inherited content-visibility values. Useful for ensuring critical content is always rendered.
