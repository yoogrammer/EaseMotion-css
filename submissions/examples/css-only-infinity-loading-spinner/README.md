# CSS-Only Infinity Loading Spinner

A sleek, animated infinity loading spinner built completely with CSS. It visually connects two spinning rings to form an endless loop, ideal for modern loading screens.

## Features
- ✨ **100% CSS-only**: Relies purely on CSS keyframe animations.
- ♾️ **Infinity Loop Animation**: Two intersecting circular borders rotate in opposing directions to simulate a continuous infinity symbol.
- 🎨 **Modern Gradients and Glow**: Uses vibrant colors and drop-shadows for a neon aesthetic.
- 📱 **Responsive**: Automatically scales for smaller devices.

## Files
- `demo.html`: The HTML structure of the loader.
- `style.css`: The styling rules, structural borders, and keyframe animations that power the continuous loop.

## Usage
Include the HTML in your project and link the CSS file. The loader will start animating automatically, creating an endless infinity loading effect.
