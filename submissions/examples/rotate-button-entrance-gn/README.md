# Rotate Button Entrance

1. **What does this do?** Animates buttons with a rotate entrance effect on page load using pure CSS keyframes.
2. **How is it used?**
```html
   <button class="rotate-entrance">Click Me</button>
   <button class="rotate-entrance-right">Click Me</button>
   <button class="rotate-entrance-bounce">Click Me</button>
```
3. **Why is it useful?** Adds engaging entrance motion to buttons with zero JavaScript, fitting EaseMotion CSS's animation-first philosophy.
