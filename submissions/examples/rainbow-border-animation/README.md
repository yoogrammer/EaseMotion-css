# Rainbow Border Animation

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A reusable `.rainbow-border` utility class that creates a rotating rainbow gradient border using pure CSS. Ideal for premium buttons, badges, cards, and highlighted UI elements.

## Features

- Reusable `.rainbow-border` utility
- Rotating gradient border
- Infinite rainbow color cycle
- CTA button examples
- Premium badge examples
- Glassmorphism card example
- Hover animations
- Responsive layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<button class="rainbow-border">
  Get Started
</button>
```

## Main Classes

```css
.rainbow-border
.cta-btn
.badge
.card
```
