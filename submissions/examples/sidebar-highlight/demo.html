Getting Started

EaseMotion CSS is an animation-first CSS framework with zero dependencies.

Simply add the CDN link to your HTML file and start using classes like ease-fade-in and ease-slide-up.

No build steps, no configuration, just write HTML and it works.

Installation

Add the CDN link to your HTML file:

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/easemotion-css/easemotion.min.css" />

Or install via npm: npm install easemotion-css

Animations

EaseMotion provides 20+ animation classes including:

  • ease-fade-in - Fade in animation
  • ease-slide-up - Slide up from bottom
  • ease-zoom-in - Zoom in effect
  • ease-bounce - Bouncing animation

Components

Ready-to-use components include:

  • Buttons - 6 variants (primary, success, danger, outline, ghost)
  • Cards - 12 variants with different styles
  • Navbar - Responsive navigation component

Customization

Override CSS variables to theme the entire framework:

:root {
    --ease-color-primary: #f97316;
    --ease-speed-fast: 100ms;
    --ease-radius-md: 1rem;
}

API Reference

All classes follow the ease-* naming convention.

Utilities include: layout, spacing, flexbox, grid, and animation timing classes.

Check the full documentation for complete API details.
