# Animated Step Progress Component

A responsive animated multi-step progress component built using pure HTML and CSS.

## Features

- Active and completed step states
- Animated progress line
- Hover interactions
- Responsive mobile layout
- Smooth transitions and animations
- Pure CSS implementation

## Usage

Open `demo.html` in a browser to view the animated step progress component.

## Files

- `demo.html` → Component structure
- `style.css` → Progress animations and styles
- `README.md` → Documentation
