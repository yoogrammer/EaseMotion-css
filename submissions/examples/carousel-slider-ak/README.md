# Carousel Slider

## What does this do?
An image carousel slider with next/previous navigation buttons and dot indicators. Auto-rotates every 3 seconds with pause on hover.

## How is it used?
Open demo.html in a browser. Use the arrow buttons or click the dots to navigate slides. The carousel auto-advances and pauses on hover.

## Why is this useful?
Demonstrates a common carousel/slider pattern for showcasing multiple pieces of content in a limited space, commonly used in hero sections and galleries.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
