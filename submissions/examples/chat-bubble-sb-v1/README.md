# Chat Bubble Messages

A self-contained sent and received chat bubble component for conversation UIs, support panels, and AI assistant interfaces.

## Features

- `.ease-chat-sent` right-aligns outgoing messages.
- `.ease-chat-recv` left-aligns incoming messages.
- Timestamps are included inside each bubble.
- Bubble widths wrap cleanly on small screens.
- Distinct corner radii make message direction obvious.

## Usage

Open `demo.html` in a browser to view the conversation layout. Copy the `.ease-chat-row`, `.ease-chat-msg`, `.ease-chat-sent`, and `.ease-chat-recv` classes into any chat-style UI.
