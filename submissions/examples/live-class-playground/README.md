# Live Class Playground

An interactive EaseMotion CSS playground for previewing class combinations before
copying markup into a project.

## Files

- `demo.html` - Static playground UI, class controls, live preview, generated snippet, and copy action.
- `style.css` - Demo-specific layout and visual styling.

## Features

- Searchable animation class picker.
- Toggle groups for hover and visual utility classes.
- Duration and delay sliders that update the preview and generated HTML.
- Replay button for one-shot entrance animations.
- Copy button for the generated snippet.

## Example Output

```html
<div class="ease-fade-in ease-hover-glow ease-rounded-lg" style="animation-duration: 600ms; animation-delay: 0ms;">
  Demo Element
</div>
```

Open `demo.html` in a browser to try the playground.
