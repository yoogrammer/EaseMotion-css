# Animation Docs Explorer

### 1. What does this do?
This is a searchable animation reference catalog and side-by-side comparison explorer. It allows developers to browse, filter, inspect transition metadata, compare animations in real-time, and copy HTML/CSS code snippets directly.

### 2. How is it used?
Open `demo.html` directly in any modern web browser to access the dashboard:
- **Instant Search**: Type queries in the sidebar to instantly filter the framework animations (25+ classes).
- **Category Pills**: Filter by Entrance, Looping, Hover / Interaction, and Accessibility Safe classes.
- **Compare View**: Switch from standard view to "Compare View" to select and watch two animations play side-by-side.
- **Metadata Panel**: Inspect class timing-functions, speeds, trigger requirements, and motion safety ratings (A+ Safe to C Risk).
- **Code Block**: Inspect and copy generated HTML and CSS code templates directly into your project.

### 3. Why is it useful?
It fits EaseMotion CSS's philosophy by simplifying class discovery. Instead of digging through CSS files or checking online docs, developers can run this offline utility to immediately find the exact class name they need, visually contrast effects side-by-side, verify accessibility compliance levels, and copy ready-made dogfooded snippets.
