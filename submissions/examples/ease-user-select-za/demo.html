ease-user-select utilities

Control text selection behavior on any element.

ease-user-select-none-za

You cannot select any text in this paragraph. Useful for buttons, icons, and draggable elements.

ease-user-select-all-za

Single click anywhere in this paragraph selects all text. Great for code blocks or copyable content.

ease-user-select-text-za

Only text nodes can be selected. The padding/margin area around inline elements cannot start a selection.

ease-user-select-contain-za

Selection is confined inside this box. Dragging outside the boundary does not extend the selection to outer elements.

ease-user-select-auto-za

Browser default behavior (the baseline). Selection works normally based on the element type.
