# Underline Slide-In Hover Effect

## 1. What does this do?

This effect reveals a smooth bottom underline sliding from left to right when a text link is hovered.

## 2. How is it used?

Apply the class to any link or inline text element:

```html
<a href="#" class="hover-underline-slide">Home</a>
```

## 3. Why is it useful?

It aligns perfectly with EaseMotion CSS's philosophy of providing lightweight, animation-first micro-interactions that elevate standard UI layouts (such as header navbars and link menus) without adding heavy dependencies or JavaScript.
