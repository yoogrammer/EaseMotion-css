 
# Line Height Utilities

Line height utilities: tight, normal, relaxed, loose.

## Files

- demo.html - Interactive demo
- style.css - Line height styles
- README.md - Documentation

## Classes

| Class | Value | Use Case |
|-------|-------|----------|
| `ease-leading-tight` | 1.25 | Headings, compact text |
| `ease-leading-normal` | 1.5 | Body text, default |
| `ease-leading-relaxed` | 1.75 | Readable long content |
| `ease-leading-loose` | 2.0 | Quotes, poetic text |

## Usage

```html
<p class="ease-leading-tight">Compact heading text</p>
<p class="ease-leading-normal">Normal body text</p>
<p class="ease-leading-relaxed">Spacious reading</p>
<p class="ease-leading-loose">Maximum spacing</p>
