# Text Blur Reveal

## 1. What does this do?

This adds a smooth focus-reveal effect to text components. The text starts with a soft blur and lower opacity, and transitions into a sharp, clear, and fully opaque state when hovered.

## 2. How is it used?

Simply apply the `.blur-reveal-text` class to any text element:

```html
<div class="blur-reveal-text">
  🔍 Reveal Hidden Insight
</div>
```

## 3. Why is it useful?

It matches EaseMotion CSS's micro-interaction philosophy by creating an elegant "spotlight" focus on hover, using performant hardware-accelerated filters without JavaScript.
