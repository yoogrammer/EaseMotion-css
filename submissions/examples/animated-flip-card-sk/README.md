# Animated Flip Card

A 3D credit-card-style flip card with a gradient front (chip, number, expiry) and a back (magnetic strip, CVV) that rotates 180 degrees on hover using CSS `perspective` and `rotateY`. Open `demo.html` directly in a browser to see the effect. This is ideal for payment UIs, onboarding demos, or any project needing a polished card-flip interaction.
