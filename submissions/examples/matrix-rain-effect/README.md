# Matrix Rain Effect

## What it does

Classic Matrix-style falling code rain effect with trailing characters and green glow.

## How to use it

Canvas automatically animates with falling characters.

## Features

- Japanese and alphanumeric characters
- Bright white head with fading green trail
- Adjustable rain density (More/Less buttons)
- Responsive to window resize
- 60fps animation

## Controls

- More Rain - Increases number of falling columns
- Less Rain - Decreases column count
- Reset - Restarts all drops

## Why it fits EaseMotion CSS

- Pure Canvas animation
- Classic cyberpunk aesthetic
- Density control
- Smooth performance
