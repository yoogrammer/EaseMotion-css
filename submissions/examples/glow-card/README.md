# Pure CSS Animated Glow Border Card

## What does this do?

A premium, GPU-accelerated neon gradient border card component with an ambient animated back-glow that intensifies dynamically on mouseover interaction.

## How is it used?

1. Link the component stylesheet:
   `<link rel="stylesheet" href="style.css">`
2. Add the markup structure to your page:
   ```html
   <div class="ease-glow-wrapper">
     <div class="ease-glow-card-inner">
       <h3>Your Title</h3>
       <p>Your content here...</p>
     </div>
   </div>
   ```

## Why is it useful?

- **GPU-Accelerated**: Relies on background positioning and opacity shifts for silky-smooth 60FPS animations.
- **Micro-Interactions**: Ambient blur increases on hover, providing rich visual feedback.
- **Zero JavaScript**: Pure CSS implementation that works without scripts or external dependencies.

## Tech Stack

- HTML
- CSS (Pure CSS, no JS)
