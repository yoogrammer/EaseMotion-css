# Ease Sticky Zd

## What does this do?
A pin board sticky note component with drop-in animation, colored variants (yellow, blue, pink, green), pushpin detail, hover lift effect, and staggered entrance — pure HTML and CSS.

## How is it used?
```html
<div class="st-grid">
  <div class="st-note st-yellow">
    <div class="st-pin"></div>
    <div class="st-title">Design Review</div>
    <div class="st-body">Review the new dashboard mockups.</div>
    <span class="st-tag">Design</span>
  </div>
  <div class="st-note st-blue">...</div>
</div>
```

Colors: st-yellow, st-blue, st-pink, st-green

## Why is it useful?
Sticky notes are a familiar visual metaphor for task boards, brainstorming tools, project planning interfaces, and kanban-style layouts. This component provides a polished, ready-to-use sticky note with realistic drop-in animation, pushpin detail, alternating rotation for a natural pin board look, hover lift, and staggered grid — all pure CSS with prefers-reduced-motion support.
