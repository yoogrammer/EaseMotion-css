 
# Avatar Component

Avatar component with size variants, circle variant, and avatar group for stacked avatars.

## Files

- demo.html - Interactive demo
- style.css - Avatar styles
- README.md - Documentation

## Classes

- ease-avatar - Base avatar (48px)
- ease-avatar-xs/sm/lg/xl - Size variants
- ease-avatar-circle - Circle variant
- ease-avatar-group - Stacked avatar group
- ease-avatar-primary/success/danger/warning/info/dark - Color variants
- ease-avatar-hover - Hover scale effect

## Usage

```html
<!-- Basic avatar -->
<div class="ease-avatar">👤</div>

<!-- Size variants -->
<div class="ease-avatar-xs">XS</div>
<div class="ease-avatar-lg">LG</div>

<!-- Circle avatar -->
<div class="ease-avatar-circle">👤</div>

<!-- Avatar group -->
<div class="ease-avatar-group">
    <div class="ease-avatar-circle">👤</div>
    <div class="ease-avatar-circle">👤</div>
</div>
