Usage

<!-- Basic avatar -->
<div class="ease-avatar">👤</div>

<!-- Size variants -->
<div class="ease-avatar-xs">XS</div>
<div class="ease-avatar-sm">SM</div>
<div class="ease-avatar-lg">LG</div>

<!-- Circle variant -->
<div class="ease-avatar-circle">👤</div>

<!-- Avatar group (stacked) -->
<div class="ease-avatar-group">
    <div class="ease-avatar-circle">👤</div>
    <div class="ease-avatar-circle">👤</div>
    <div class="ease-avatar-circle">👤</div>
</div>

<!-- Color variants -->
<div class="ease-avatar-primary">P</div>
<div class="ease-avatar-success">S</div>

👤 Avatar Features

  • 👤 Base avatar component with multiple sizes (XS, SM, MD, LG, XL)
  • 🟢 Circle/rounded variant
  • 📚 Avatar group for stacked avatars
  • 🎨 Color variants (primary, success, danger, warning, info, dark)
  • 📷 Image avatar support
  • 🟢 Status indicators (online, offline, away, busy)
  • ✨ Hover scale effect
  • 📱 Fully responsive
