ease-pointer-events & touch-action utilities

These utilities control whether an element captures pointer input and how touch gestures are handled. The overlay demo below shows how .ease-pointer-events-none lets clicks pass through decorative layers to interactive content underneath.

Pointer-events overlay demo

Click the green button through the purple overlay. When the overlay uses .ease-pointer-events-none, clicks reach the button. Toggle to .ease-pointer-events-auto and the overlay blocks them.
