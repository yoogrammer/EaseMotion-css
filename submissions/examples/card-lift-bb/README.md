# Card Lift

**What does this do?**  
Lifts a card element upward by 8px on hover while simultaneously deepening the drop shadow, creating a convincing 3D depth effect.

**How is it used?**  
```html
<div class="lift-card">
  <h3>Title</h3>
  <p>Content</p>
</div>
```

**Why is it useful?**  
Cards are the most common container in UI. Adding physical depth on hover makes them feel interactive and invites clicks. The lift effect is more intentional than a simple scale — it communicates "this card is clickable" while keeping the element's footprint identical (no reflow). Pairs naturally with shadow cards and hover glow.
