# Skeleton Loader Components

A responsive skeleton loading component built using pure HTML and CSS.

## Features

- Smooth shimmer animation
- Avatar placeholder
- Text placeholders
- Card loading layout
- Responsive structure
- Pure CSS implementation

## Usage

Open `demo.html` in a browser to view the skeleton loader demo.

## Files

- `demo.html` → Demo structure
- `style.css` → Skeleton loader styles
- `README.md` → Documentation
