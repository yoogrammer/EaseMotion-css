# Renewable Energy Grid Dashboard

This example showcases a **Renewable Energy Distribution Monitor** built with **EaseMotion CSS**. It demonstrates:

- A deep-sea dark theme with electric green and blue accents to convey energy stability and flow.
- A central network visualization using CSS animations for energy flow and rotating nodes.
- High-contrast gauges and metrics suited for a control room environment.
- Responsive CSS Grid layouts.

## Directory Structure

```
/energy-grid-bv/
│   demo.html   # Interactive demo page
│   style.css   # Custom CSS variables and animations for this example
│   README.md   # This documentation
```

## How to Run

Open `demo.html` in a modern browser. No external dependencies are required.

## Key Features

- **Energy Flow Animation**: Uses CSS keyframes to simulate energy transfer along connecting lines.
- **Glassmorphic Cards**: `eg-glass` class combining backdrop filters with subtle box-shadows.
- **Progress Meters**: Simple HTML/CSS bars for capacities and reserves.
- **Weather / Anomaly Widgets**: Status-driven coloring.

Built with **EaseMotion CSS**.
