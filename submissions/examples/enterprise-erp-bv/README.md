# Enterprise ERP Digital Operations Platform

## What does this do?

The Enterprise ERP Digital Operations Platform is a large-scale operations dashboard designed for global logistics, supply chain monitoring, resource capacity allocation, and financial ledger audits. It features modules for procurement queues, logistics factory hubs, stock monitoring progress bars, compliance tracking matrices, and executive pdf auditors.

## How is it used?

Open `demo.html` directly in any modern browser:

- **Switch Perspective**: Use the sidebar options to pivot the dashboard view between Logistics Overview, Procurement Queue, and Ledger Audits.
- **Select Logistics Hubs**: Click hub cards (Shanghai, Rotterdam, Chicago) to load specific inventories and audit ledger logs.
- **Auditing Ledger**: Type audit notes in the text input inside the inventory panel and press Enter to instantly append logs.
- **Approve Procurement**: Click "Approve Purchase" on pending list items. The card will lock as approved, updating the total procured COGS immediately.
- **Capacity Forecast**: Click "Refresh Forecast" to sweep the laser line down the resource chart and re-allocate bar heights.
- **Parameter Sliders**: Drag the Budget Cap slider to dynamically adjust the operational targets and remaining fund calculations.

## Why is it useful?

Managing corporate resources, approving raw material supply chains, and tracking compliance across logistics centers requires consolidated dashboard environments to avoid discrepancies.

This portal solves ERP workflows by:

1. Displaying logistics hub specifications and audit history side-by-side using high-performance grid columns.
2. Integrating a responsive budget tracker that updates progress indicator metrics and caps dynamically.
3. Incorporating interactive viewpoint controls to pivot information maps from logistics hubs to billing ledgers.

## Tech Stack

- HTML5
- CSS3 (runway bar height transitions, laser sweeps, pulsing beacons, progress bar tracks, layout grids)
- JavaScript (hub metadata loaders, audit note logs, spent budget calculators, approval status handlers)
