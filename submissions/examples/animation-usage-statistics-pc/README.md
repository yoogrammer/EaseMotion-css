# Animation Usage Statistics Dashboard

## What does this do?

This example demonstrates a visual dashboard that highlights the popularity of different animation styles using animated statistic cards and progress indicators.

## How is it used?

Open `demo.html` in a browser to view the dashboard.

The example showcases animation popularity through percentage-based progress bars and animated cards.

## Why is it useful?

Developers can quickly identify commonly used animation patterns and compare their popularity through a simple visual interface.

This concept can be adapted for analytics dashboards, UI reports, admin panels, and performance summaries.

## Tech Stack

- HTML
- CSS
