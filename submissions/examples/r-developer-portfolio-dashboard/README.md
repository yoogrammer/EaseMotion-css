# Developer Portfolio Analytics Dashboard

A responsive dashboard that helps developers track portfolio growth, project statistics, GitHub activity, coding streaks, technology usage, and learning progress.

## Features

- Portfolio analytics overview
- GitHub contribution tracking
- Coding streak statistics
- Project showcase cards
- Technology stack insights
- Learning progress tracker
- Achievement badges
- Activity timeline
- Responsive dashboard design
