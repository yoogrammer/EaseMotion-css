# Marquee Scrolling Text Banner

A horizontally scrolling text banner that continuously loops. Two variants are shown: one scrolling left-to-right and one in reverse. The text pauses on hover for readability. Uses CSS translateX animation on a duplicated inline content block.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser to see the marquee banners scroll. Hover to pause.

## Accessibility notes

The marquee is purely decorative text. Reduced motion disables the scroll animation.
