# Animated Corner Border Button

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A collection of animated corner border buttons using CSS pseudo-elements and transitions. The borders expand from the corners on hover, creating a modern interactive effect.

## Features

- Animated corner border drawing
- Hover lift effect
- Neon variant
- Gradient variant
- Glassmorphism variant
- Responsive layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<button class="corner-btn">
  Button
</button>
```

## Main Classes

```css
.corner-btn
.neon
.gradient
.glass
```

## Why It Fits EaseMotion CSS

- Animation-first interaction
- Human-readable class names
- Reusable button component
- Lightweight implementation
- Modern hover effects
