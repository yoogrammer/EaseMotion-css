# Animation Showcase

**What does this do?**
Adds an interactive showcase for previewing EaseMotion CSS animation classes in real time.

**How is it used?**

```html
<div class="em-fade-in">Fade In</div>
<div class="em-slide-left">Slide Left</div>
<div class="em-zoom-in">Zoom In</div>
<div class="em-bounce">Bounce</div>
```

Users can select an animation from the showcase panel, adjust the duration, and instantly preview the effect before using it in their projects.

**Why is it useful?**
The Animation Showcase helps developers quickly explore available EaseMotion animations without manually editing code. It improves discoverability, simplifies testing, and aligns with EaseMotion CSS's goal of providing simple, readable, and reusable motion utilities.
