# Ease Opacity Utilities

Opacity utility classes for controlling element transparency in EaseMotion-css.

## Included Utilities

- `ease-opacity-0` — fully transparent
- `ease-opacity-25` — 25% opacity
- `ease-opacity-50` — 50% opacity
- `ease-opacity-75` — 75% opacity
- `ease-opacity-100` — fully opaque

## Usage

```html
<button class="ease-opacity-50" disabled>Disabled button</button>

<div class="overlay ease-opacity-75">Semi-transparent overlay</div>
```

## Why is it useful?

These utilities provide a consistent way to dim content for disabled states, modal overlays, and subtle visual hierarchy without writing custom opacity rules.
