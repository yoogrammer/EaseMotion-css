# Animated Stack of Photos Component

## Description
An interactive UI module presenting multiple elements arranged in an overlapping 3D layer sequence. Upon component target hover, cards layout parameters transform dynamically utilizing spring-loaded cubic-bezier timing values.

## Features
- **3 Variants Presets Available**: Includes multi-axis fanning angles, card separation spacing spreads, and layered card elevators.
- **Elastic Transitions**: Smooth translation effects using custom spring-back parameter maps.
- **No JS Dependencies**: Leverages performance-optimized hardware-accelerated CSS engine parameters natively.
