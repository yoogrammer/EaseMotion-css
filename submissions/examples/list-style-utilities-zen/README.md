# List Style Utilities

CSS utility classes for list marker styles.

## Classes

- `.list-none` — `list-style: none`
- `.list-disc` — `list-style: disc`
- `.list-decimal` — `list-style: decimal`
- `.list-square` — `list-style: square`

## Why

Quickly reset or apply list marker styles without writing custom CSS. Useful for styling navigation menus (list-none), content lists (list-disc), numbered steps (list-decimal), and decorative lists (list-square).
