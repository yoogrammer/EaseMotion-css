# Floating Orbit Modal for Modern SaaS Layouts

A modern **Floating Orbit Modal** built entirely with **HTML & CSS** for SaaS dashboards, landing pages, and web applications.

## Features

- Pure HTML & CSS
- Floating Orbit animation
- Modern SaaS UI
- Fully Responsive
- Keyboard Accessible
- CSS Custom Properties
- No JavaScript animations
- Lightweight & Reusable

## Files

```
demo.html
style.css
README.md
```

## CSS Variables

Customize the component by editing the variables inside `:root`.

```css
:root{

--primary:#4f46e5;
--duration:.5s;
--radius:20px;
--orbit-size:16px;

}
```
