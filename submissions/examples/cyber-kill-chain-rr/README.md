# Cyber Kill Chain Visualization Component

## What does this do?

Visualizes the Cyber Kill Chain framework, showing the sequential stages attackers follow during a cyber attack.

## How is it used?

```html
<div class="stage exploit">
    Exploitation
</div>

<div class="stage control">
    Command & Control
</div>
```

## Why is it useful?

The Cyber Kill Chain is widely used in cybersecurity training, threat analysis, and incident response. This component provides a clear visual representation of attack progression using pure HTML and CSS.
