All Property Utility

Reset every CSS property on an element with a single class. These correspond to the CSS-wide keyword values for the `all` shorthand.

Default (no reset)

This card has a link and styled text. All properties are inherited normally.

all-unset

<div class="card all-unset">...</div>

This card has a link. Everything resets to inherited/natural values — borders, padding, colors are gone.

all-initial

<div class="card all-initial">...</div>

This card has a link. All properties revert to their CSS initial values.

all-revert

<div class="card all-revert">...</div>

This card has a link. Properties revert to the browser/user-agent stylesheet.
