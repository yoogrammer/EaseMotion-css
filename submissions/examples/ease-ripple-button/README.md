# Ease Ripple Button

A reusable ripple-effect button component for EaseMotion CSS.

## Features

- Animated ripple click effect
- Smooth hover elevation
- Active press feedback
- Keyboard focus support
- Reduced motion accessibility
- Mobile responsive

## Variants

- Basic Button
- Outline Button
- Filled Button
- Icon Button

## Files

- demo.html
- style.css
- README.md

## Usage

Include `style.css` and use the provided button classes.

```html
<button class="ease-ripple-button ease-ripple-basic">
  Click Me
</button>
