 
# Background Gradient Utilities

Background gradient utilities: primary gradients, direction gradients, and color combinations.

## Files

- demo.html - Interactive demo
- style.css - Gradient utilities styles
- README.md - Documentation

## Classes

### Basic Gradients
| Class | Description |
|-------|-------------|
| `ease-bg-gradient-primary` | Primary to secondary |
| `ease-bg-gradient-secondary` | Secondary to accent |
| `ease-bg-gradient-accent` | Accent gradient |

### Direction Gradients
| Class | Direction |
|-------|-----------|
| `ease-bg-gradient-to-r` | Left to right |
| `ease-bg-gradient-to-l` | Right to left |
| `ease-bg-gradient-to-b` | Top to bottom |
| `ease-bg-gradient-to-t` | Bottom to top |
| `ease-bg-gradient-to-tr` | Top right |
| `ease-bg-gradient-to-br` | Bottom right |

### Color Combinations
| Class | Colors |
|-------|--------|
| `ease-bg-gradient-sunset` | Sunset colors |
| `ease-bg-gradient-ocean` | Ocean colors |
| `ease-bg-gradient-forest` | Forest colors |
| `ease-bg-gradient-aurora` | Aurora colors |
| `ease-bg-gradient-candy` | Candy colors |
| `ease-bg-gradient-fire` | Fire colors |

## Usage

```html
<div class="ease-bg-gradient-primary">Primary gradient</div>
<div class="ease-bg-gradient-to-r">Left to right gradient</div>
<div class="ease-bg-gradient-sunset">Sunset gradient</div>
