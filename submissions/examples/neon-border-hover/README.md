# Luminous Neon Border Showcase

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Features

- Pure CSS implementation
- Animated neon border effect
- Works on buttons
- Works on cards
- Multiple neon color themes
- Glow intensity variants
- Responsive layout
- Keyboard accessible

## Usage

```html
<button class="ease-neon-border">
  Click Me
</button>
```

## Color Variants

```html
<div class="ease-neon-border neon-cyan"></div>
<div class="ease-neon-border neon-purple"></div>
<div class="ease-neon-border neon-green"></div>
<div class="ease-neon-border neon-pink"></div>
```

## Main Class

```css
.ease-neon-border
```

## Why It Fits EaseMotion CSS

- Human-readable class names
- Animation-first philosophy
- Lightweight implementation
- Easy customization
- Modern visual effects
- Reusable across multiple UI components
