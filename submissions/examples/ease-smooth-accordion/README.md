# EaseAccordion Component (`ease-smooth-accordion`)

A zero-dependency, pure CSS layout module for displaying responsive, smoothly animating interactive accordion panels.

## Usage

```html
<div class="ease-accordion-item">
  <input type="checkbox" id="unique-id" class="ease-accordion-toggle">
  <label for="unique-id" class="ease-accordion-header">
    Header Title Text
    <span class="ease-accordion-icon"></span>
  </label>
  <div class="ease-accordion-content">
    <p>Hidden body text goes here.</p>
  </div>
</div>
