About This Fix

In core/animations.css, the class .ease-zoom-out references keyframe ease-kf-zoom-out (line 679), but the keyframe was defined as ease-zoom-out (line 650) — missing the -kf- infix. This mismatch causes the animation to silently fail.

This demo uses the corrected @keyframes kf-zoom-out-demo name so the animation class can resolve it properly. The same pattern should be applied to core/animations.css: rename @keyframes ease-zoom-out → @keyframes ease-kf-zoom-out.
