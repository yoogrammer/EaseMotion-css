# Interactive Animation Playground

An interactive playground for experimenting with CSS animations in real time.

## Features

- Live animation preview
- Animation selector
- Duration control
- Delay control
- Iteration count control
- Easing selector
- Copy generated CSS
- Responsive design

## Controls

- Animation Type
- Duration
- Delay
- Iterations
- Easing Function

## Usage

Open `demo.html` in a browser and customize animation settings to preview changes instantly.
