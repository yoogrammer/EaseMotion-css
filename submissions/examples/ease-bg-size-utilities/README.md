# Ease Background Size Utilities

A collection of lightweight background-size utility classes for controlling image scaling behavior.

## Features

- Pure CSS utilities
- Lightweight and reusable
- Cross-browser compatible
- Beginner-friendly implementation

## Included Utilities

- `ease-bg-cover`
- `ease-bg-contain`
- `ease-bg-auto`

## Usage

```html
<div class="ease-bg-cover"></div>

<div class="ease-bg-contain"></div>

<div class="ease-bg-auto"></div>
```

## Utility Details

- `ease-bg-cover` fills the container while preserving aspect ratio
- `ease-bg-contain` keeps the full image visible
- `ease-bg-auto` uses the default background-size behavior
