CSS `flex-wrap` utility classes for the EaseMotion CSS framework.

```html

```

## Utility Classes

| Class | Declaration |
|-------|-------------|
| `.flex-wrap` | `flex-wrap: wrap;` |
| `.flex-nowrap` | `flex-wrap: nowrap;` |
| `.flex-wrap-reverse` | `flex-wrap: wrap-reverse;` |
| `.flex-wrap-initial` | `flex-wrap: initial;` |
| `.flex-wrap-inherit` | `flex-wrap: inherit;` |
| `.flex-wrap-unset` | `flex-wrap: unset;` |
| `.flex-wrap-revert` | `flex-wrap: revert;` |
| `.flex-wrap-balance` | `flex-wrap: balance;` |
| `.flex-wrap-ghost-left` | `flex-wrap: <flex-wrap>;` |

## Responsive Variants

Prefix with `sm-`, `md-`, `lg-` for responsive behavior:

```html
<div class="util-class sm-util-class md-util-class lg-util-class">
  Responsive Flex Wrap
</div>
```

## Dark Mode

Use the `-dark` variant:

```html
<div class="util-class-dark">
  Dark mode Flex Wrap
</div>
```

## Reduced Motion

Use the `-nomotion` variant:

```html
<div class="util-class-nomotion">
  No motion Flex Wrap
</div>
```

## Framework Integration

All utilities use EasMotion design tokens from `core/variables.css`: `--ease-primary`, `--ease-secondary`, `--ease-accent`, `--ease-success`, `--ease-danger`, `--ease-warning`, `--ease-info`.
