# Layered Image Stack Hover

A CSS-only image stack component where layered cards spread apart on hover.

## Features

- Pure CSS
- Transform-based hover animation
- Responsive layout
- No JavaScript required

## Usage

```html
<div class="image-stack">
  <div class="stack-card">Image 1</div>
  <div class="stack-card">Image 2</div>
  <div class="stack-card">Image 3</div>
</div>
```

Hover over the stack to see the cards spread apart into a layered gallery effect.
