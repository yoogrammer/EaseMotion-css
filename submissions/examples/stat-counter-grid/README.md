# Stat Counter Grid

Stats and counters displayed in responsive grid layouts with large numbers, labels, icons, and animated appearance.

## Features

- 2×2, 3×2, and 4×1 responsive grid variants
- Large formatted numbers with tabular-nums
- Colored SVG icons with decorative background circles
- Trend indicators (up/down) with pill badges
- Card hover elevation with translateY effect
- Icon scale animation on hover
- Staggered count-up fade-in animation
- Fully responsive with breakpoints at 700px and 450px
- `prefers-reduced-motion` support

## Files

- `demo.html` — Three grid variants with sample stats
- `style.css` — Self-contained CSS (176 lines)
