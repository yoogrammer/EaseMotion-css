# CSS 3D Interactive Flip Card

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A CSS-only 3D interactive flip card showcase demonstrating perspective transforms, front/back card faces, hover-triggered animations, and reusable card layouts.

## Features

- CSS-only 3D flip animation
- Perspective-based layout
- Front and back card faces
- Hover interaction
- Profile card example
- Product card example
- Glassmorphism variant
- Neon variant
- Responsive layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="flip-card">
  <div class="flip-card-inner">
    ...
  </div>
</div>
```

## Main Classes

```css
.flip-card
.flip-card-inner
.flip-card-front
.flip-card-back
.neon
.glass
```

## Why It Fits EaseMotion CSS

- Animation-first design
- Human-readable class names
- Reusable card component
- Demonstrates CSS 3D transforms
- Lightweight implementation
