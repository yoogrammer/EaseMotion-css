# Heartbeat Animation

## What does this do?

Creates a realistic heartbeat-style animation using a double-pulse rhythm followed by a brief resting period.

## How is it used?

```html
<div class="heartbeat-element">
  Content
</div>
```

## Why is it useful?

The heartbeat animation draws attention to important interactive elements such as like buttons, health indicators, notification badges, and status icons. The double-beat motion feels more natural than a standard pulse animation and fits EaseMotion CSS's animation-first philosophy.
