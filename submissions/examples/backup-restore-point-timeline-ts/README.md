# Backup Restore Point Timeline

## What does this do?

Shows automatic, manual, failed, and verified backup restore points in a timeline.

## How is it used?

```html
<article class="point verified">
  <span>09:00</span><h2>Verified backup</h2>
</article>
```

## Why is it useful?

It gives admin panels a clear way to review backup history and restore readiness.
