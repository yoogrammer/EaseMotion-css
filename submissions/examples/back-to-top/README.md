# Back to Top Button

## What does this do?
Adds a fixed-position button that appears when the user scrolls down and smoothly scrolls back to the top of the page on click.

## How is it used?
```html
<button class="back-to-top" aria-label="Back to top">↑</button>
```
Toggle the `visible` class with a scroll listener to show/hide it.

## Why is it useful?
Long documentation pages benefit from quick navigation shortcuts. This fits EaseMotion CSS's philosophy of composable, human-readable classes — `back-to-top` describes exactly what the component does.
