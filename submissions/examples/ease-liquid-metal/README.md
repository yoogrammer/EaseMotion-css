# ease-liquid-metal

Mercury-like flowing surface effect using animated multi-stop metallic gradients.

## Usage

```html
<div class="ease-liquid-metal">Silver</div>
<div class="ease-liquid-metal ease-liquid-metal--gold">Gold</div>
<div class="ease-liquid-metal ease-liquid-metal--copper">Copper</div>
```

## Why it fits EaseMotion CSS

Pure CSS `background-position` animation on a multi-stop gradient. No JS, no SVG.
