 
# Glassmorphism Hover Card

Glassmorphism hover card with blur, transparency, and subtle elevation.

## Files

- demo.html - Interactive demo
- style.css - Glass card styles
- README.md - Documentation

## Classes

- ease-glass-card - Base glass card
- ease-glass-card-blur-sm/lg - Blur intensities
- ease-glass-card-lift-sm/lg - Lift amounts
- ease-glass-card-fast/slow - Speed variants
- glass-card-primary/success/danger/warning/dark - Color variants

## Usage

```html
<div class="ease-glass-card">
    <h3>Card Title</h3>
    <p>Card content here</p>
</div>

<!-- Color variants -->
<div class="glass-card-primary ease-glass-card">Primary</div>
