# ParkEasy – Smart Parking Meter App Landing Page

A complete, ready‑to‑copy landing page for a smart parking meter mobile app built entirely with **EaseMotion CSS** utility classes – no custom keyframes.

## Sections
- **Header** – clean navigation
- **Hero** – digital parking meter mockup with download CTA
- **Features** – extend time remotely, smart alerts, payment history
- **City Partnerships Logos** – five partner cities
- **Coverage Areas** – eight major Indian cities
- **Pricing Model** – pay as you go and monthly pass
- **User Reviews** – app testimonials
- **App Store CTAs** – blue call‑to‑action section with iOS and Android buttons
- **Footer**

## EaseMotion Classes Showcased
- **Layout:** `ease-container`, `ease-flex`, `ease-grid`, `ease-gap-*`, `ease-justify-between`, `ease-items-center`, `ease-max-w-*`, `ease-mx-auto`
- **Typography:** `ease-text-*`, `ease-font-bold`, `ease-font-semibold`, `ease-text-gray-*`, `ease-text-blue-*`, `ease-text-white`
- **Spacing:** `ease-py-*`, `ease-px-*`, `ease-p-*`, `ease-mb-*`, `ease-mt-*`
- **Components:** `ease-btn`, `ease-btn-primary`, `ease-btn-lg`, `ease-card`
- **Hover Effects:** `ease-hover-scale-105`, `ease-hover-shadow-xl`, `ease-hover-text-blue-700`, `ease-hover-opacity-100`
- **Animations:** `ease-fade-in`, `ease-slide-up`, `ease-delay-*`, `ease-transition`
- **Utilities:** `ease-rounded-xl`, `ease-shadow-2xl`, `ease-shadow-sm`, `ease-bg-*`, `ease-w-full`, `ease-opacity-80`

## How to Use
1. Copy `demo.html` into your project.
2. Make sure the path to `easemotion.css` is correct.
3. Open in any modern browser – no build step required.
