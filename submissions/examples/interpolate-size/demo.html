CSS interpolate-size

Finally animate between height: auto and fixed values — no max-height hacks.

Accordion (height: auto)

interpolate-size: allow-keywords is a CSS property that lets you transition or animate between a <length> value and the auto keyword — something that was previously impossible without JavaScript or max-height workarounds.

The browser needs a numeric start and end value to interpolate. Since auto is a keyword (not a number), CSS couldn't animate it. Developers had to use max-height hacks with arbitrary values — which caused timing mismatches or cut-off content.

Chrome 132+, Edge 132+. Firefox and Safari support are in development. In unsupported browsers, the accordion still works — it just opens instantly without animation.
