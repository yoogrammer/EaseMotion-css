# ease-decoration-style Utilities

Controls the `text-decoration-style` property — the shape of the decoration line.

## Usage

```html
<span class="ease-decoration-underline ease-decoration-wavy"
  >Wavy underline</span
>
<span class="ease-decoration-line-through ease-decoration-double"
  >Double strikethrough</span
>
```

## Classes

| Class                    | CSS Property                  |
| ------------------------ | ----------------------------- |
| `ease-decoration-solid`  | text-decoration-style: solid  |
| `ease-decoration-double` | text-decoration-style: double |
| `ease-decoration-dotted` | text-decoration-style: dotted |
| `ease-decoration-dashed` | text-decoration-style: dashed |
| `ease-decoration-wavy`   | text-decoration-style: wavy   |
