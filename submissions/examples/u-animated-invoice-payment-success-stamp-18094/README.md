# Animated Invoice Payment Success Stamp

## Overview

Animated Invoice Payment Success Stamp is a lightweight CSS animation that simulates a successful invoice payment using a bouncing PAID stamp effect.

## Features

- CSS-only animation
- Payment confirmation effect
- Invoice success visualization
- Responsive design
- Easy customization
- Accessibility support

## Usage

```html
<div class="invoice-paid">
  <span class="invoice">📄</span>
  <span class="paid-stamp">PAID</span>
</div>
```

## Accessibility

Supports:

```css
@media (prefers-reduced-motion: reduce)
```

to disable animations for users who prefer reduced motion.

## Browser Support

- Chrome
- Firefox
- Edge
- Safari
