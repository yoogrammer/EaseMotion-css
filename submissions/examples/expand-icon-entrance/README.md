# Expand Icon Entrance

A smooth expand-style entrance animation for icon components.

## Features

- Lightweight pure CSS animation
- Smooth expanding entrance effect
- Cross-browser compatible
- Reusable utility class

## Usage

```html
<div class="ease-expand-icon-entrance">
  ⭐
</div>
```

## Animation Details

- Icon smoothly expands during entrance
- Uses CSS keyframes with easing
- Lightweight and modern utility animation
