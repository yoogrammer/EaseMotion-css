# Hover Animation Showcase

A collection of simple hover-triggered animations for common UI elements.

## Included Examples

### Button Hover

Scales the button when the user hovers over it.

### Card Hover

Lifts the card slightly upward on hover.

### Image Hover

Zooms the image smoothly when hovered.

## Usage

Open `demo.html` in a browser and hover over each element to see the animations.

## Features

* Pure CSS
* Beginner-friendly
* No JavaScript required
* Easy to customize
