# Animated Alerts & Banners Pack

A collection of lightweight, pure-CSS alert and banner components featuring fluid entrance animations.

## Features
- Base `.em-alert` styling with 4 core variants:
  - `.em-alert-success` (Green)
  - `.em-alert-danger` (Red)
  - `.em-alert-warning` (Amber)
  - `.em-alert-info` (Blue)
- Pure-CSS entrance animation helpers:
  - `.em-alert-slide-in` (Smooth slide-down + fade)
  - `.em-alert-fade-in` (Clean opacity transition)
- Responsive flex layout with optional dismissible action trigger support.

## Usage
Add classes directly to your alert element container:

```html
<div class="em-alert em-alert-success em-alert-slide-in">
  <span><strong>Success:</strong> Action completed.</span>
</div>
```
