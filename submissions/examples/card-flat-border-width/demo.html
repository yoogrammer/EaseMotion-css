Fix #3904

ease-card-flat Border-Width Fix

The framework .ease-card-flat sets border-color: transparent but the 1px border width from .ease-card still occupies space, causing a 1px layout shift when toggling the class. This example demonstrates the recommended fix:

/* Recommended fix in components/cards.css */
.ease-card-flat {
  border-color: transparent;
  border-width: 0;  /* remove 1px border to prevent layout shift */
}

Before (border: 1px solid transparent)

1px transparent border still occupies space.

width includes 1px border

After (border-width: 0)

No 1px border — no layout shift.

width excludes border

✅ Fix Demonstrated

See style.css for the override. The permanent fix in components/cards.css should add border-width: 0 to .ease-card-flat.
