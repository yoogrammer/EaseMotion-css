Zoom-Out Animation — Design Token

Submission proposing that easemotion/zoom.css use the --ease-speed-medium and --ease-ease-out design tokens for the .ease-zoom-out animation, replacing the hardcoded 0.6s ease-out. Linked to issue #5575.

B
.ease-zoom-out

Why this matters

.ease-zoom-in and .ease-contract-image-entrance in the same file already use the framework's design tokens. .ease-zoom-out was the only holdout, with a hardcoded duration and easing. Switching to the tokens brings it in line with the rest of the file and lets consumers override the timing from a single source.
