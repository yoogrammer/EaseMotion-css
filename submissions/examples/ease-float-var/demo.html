Fix #3896

ease-float CSS Variable Override

The framework .ease-float hardcodes 3s ease-in-out. This example shows the correct override:

:root { --ease-speed-medium: 500ms; }
.ease-float {
  animation: ease-kf-float var(--ease-speed-medium) var(--ease-ease) infinite;
}
