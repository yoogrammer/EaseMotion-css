Responsive Spacing Utilities

Padding & margin variants that activate at specific breakpoints. Resize the browser window to see each class take effect.

— px

Active breakpoint

xs < 640px sm ≥ 640px md ≥ 768px lg ≥ 1024px xl ≥ 1280px

Padding — all axes p-*

Base: ease-p-2 (8 px) → sm: ease-sm-p-4 (16 px) → md: ease-md-p-6 (24 px) → lg: ease-lg-p-8 (32 px) → xl: ease-xl-p-10 (40 px)
