🖱️ Pointer/Cursor Utilities

  • 🖱️ ease-pointer - Shows hand cursor (cursor: pointer)
  • 🚫 ease-no-pointer - Disables pointer events (pass-through)
  • 🖱️ ease-cursor-default - Explicitly sets default cursor
