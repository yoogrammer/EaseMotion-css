The Pure CSS :has() Sibling Formula

Historically, CSS could only target elements downstream (following siblings). Selecting preceding siblings required JavaScript. With the modern CSS :has() selector, we target the element immediately preceding the hovered card. 

/* 1. Target the hovered card (tilts inward) */
.ease-kinetic-item:hover {
  transform: translateZ(50px) rotateX(10deg) rotateY(-10deg);
}

/* 2. Target the succeeding sibling (pushed right) */
.ease-kinetic-item:hover + .ease-kinetic-item {
  transform: translateZ(20px) rotateY(15deg) translateX(12px);
}

/* 3. Target the preceding sibling (pushed left using :has) */
.ease-kinetic-item:has(+ .ease-kinetic-item:hover) {
  transform: translateZ(20px) rotateY(-15deg) translateX(-12px);
}
