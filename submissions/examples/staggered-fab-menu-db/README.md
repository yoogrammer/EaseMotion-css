# Staggered FAB Menu Component

1. What does this do? Implements a Floating Action Button (FAB) that reveals sub-options sequentially with a staggered transition.
2. How is it used? Nest `.fab-menu` items alongside the checkbox toggler `.fab-checkbox` and `.fab-main` in the layout.
3. Why is it useful? It showcases how to write staggered delays (`transition-delay`) to coordinate animations between sibling elements.
