Keyboard Shortcut Cheat Sheet

Reference panel for common developer and productivity shortcuts — pure HTML & CSS, accessible and responsive.

Navigation

  • Open Command Palette Ctrl + K
  • Quick Open Ctrl + P
  • Go to File Ctrl + T

Editing

  • Format Document Ctrl + Shift + I
  • Toggle Comment Ctrl + /
  • Multi-cursor add Alt + Click

Search

  • Find in Files Ctrl + Shift + F
  • Find Ctrl + F
  • Replace Ctrl + H

Actions

  • Show Suggestions / Quick Fix Alt + Enter
  • Run Task / Build Ctrl + Shift + B
  • Toggle Terminal Ctrl + `

Tips & Shortcuts

Use this panel as a quick reference or embed it into docs and dashboards. The layout collapses to a single column on narrow screens and retains keyboard-orderable markup for accessibility.

Legend
Ctrl — Control key
Shift — Shift key
Alt — Alt / Option key
