# Glassmorphic Credit Card Component

1. What does this do? Creates an interactive, 3D-hoverable credit card that flips front-to-back using a checkbox-hack toggler.
2. How is it used? Place the `.card-container` containing the flip checkbox and the labels in your HTML.
3. Why is it useful? It showcases advanced CSS 3D transforms (`backface-visibility`, `preserve-3d`) and glassmorphism styling patterns for cards.
