Design Token (CSS Variable) Tree Shaking

This demo proves that the core framework components function perfectly fine with a drastically reduced set of CSS variables.

By actively pruning hundreds of lines of completely unused alpha-colors and massive, unused box-shadows from core/variables.css, we can massively reduce the framework's baseline payload size, safely saving downstream end-users from downloading thousands of unnecessary "dead bytes."
