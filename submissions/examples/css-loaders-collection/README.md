# CSS Loaders Collection

## What does this do?

A collection of 8 lightweight, pure CSS loading animations including Spinner, Pulse, Bouncing Dots, Ripple, Wave, Dual Ring, Orbit, and Skeleton loaders. Each loader is built using only HTML and CSS with no JavaScript dependencies.

## How is it used?

Add any loader markup to your page:

```html
<div class="em-loader-spinner"></div>
```

Or use other variants:

```html
<div class="em-loader-pulse"></div>

<div class="em-loader-dots">
  <div></div>
  <div></div>
  <div></div>
</div>
```

All loaders can be customized through CSS variables such as size, speed, and color.

## Why is it useful?

Provides modern loading indicators for applications, dashboards, forms, and content-loading states without requiring JavaScript. The collection offers multiple visual styles so developers can choose the loader that best matches their design system while keeping performance high and implementation simple.

## Tech Stack

- HTML
- CSS Animations
- CSS Variables
- No JavaScript
- No Frameworks
