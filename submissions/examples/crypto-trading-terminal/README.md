# Crypto Trading Terminal

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A professional crypto trading dashboard showcasing market data, portfolio analytics, watchlists, activity feeds, animated charts, and premium trading terminal UI patterns using pure HTML and CSS.

## Features

- Market overview cards
- Crypto price widgets
- Portfolio analytics
- Trading watchlist
- Activity feed
- Animated performance chart
- Profit/Loss indicators
- Glassmorphism panels
- Responsive layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="coin-card">
  <h3>BTC</h3>
</div>
```

## Main Classes

```css
.coin-card
.panel
.chart
.metric
.watchlist
.feed-item
```

## Why It Fits EaseMotion CSS

- Animation-first design

- Reusable dashboard components

- Lightweight implementation
