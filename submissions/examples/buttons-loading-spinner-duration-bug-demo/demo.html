Bug: hardcoded 0.7s spin duration

The loading spinner uses 0.7s directly instead of a CSS variable.

Source

.ease-btn-loading::after {
  animation: ease-kf-btn-spin 0.7s linear infinite;  /* hardcoded */
}

Fix

.ease-btn-loading::after {
  animation: ease-kf-btn-spin var(--ease-btn-spin-speed, 0.7s) linear infinite;
}
