# Badge Selectors Showcase

Issue: [#8371](https://github.com/SAPTARSHI-coder/EaseMotion-css/issues/8371)

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A collection of reusable badge patterns demonstrating status indicators, notification badges, role badges, product labels, skill tags, and animated badges using pure HTML and CSS.

## Features

- Status badges
- Notification badges
- Product badges
- Role badges
- Animated pulse badges
- Skill tags
- Responsive layout
- Hover interactions
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<span class="badge success">
  Success
</span>
```

## Main Classes

```css
.badge
.success
.warning
.danger
.info
.pulse
.tag
.count-badge
```

## Why It Fits EaseMotion CSS

- Human-readable class names
- Reusable UI patterns
- Lightweight implementation
- Animation-first interactions
- Useful for dashboards and applications
