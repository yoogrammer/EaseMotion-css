# Dashboard Statistic Widgets

Responsive animated dashboard statistic widgets built using pure HTML and CSS.

## Features

- Animated metric cards
- Hover glow effects
- Trend indicators
- Responsive dashboard layout
- Smooth transitions
- Pure CSS implementation

## Usage

Open `demo.html` in a browser to view the dashboard statistic widgets.

## Files

- `demo.html` → Dashboard widget structure
- `style.css` → Widget styles and animations
- `README.md` → Documentation
