# Popover API Demo

## What does this do?
Demonstrates the native HTML Popover API with animated open/close effects using CSS.

## How is it used?
Add `popover` attribute and `popovertarget` on trigger buttons:

    <button popovertarget="my-popover">Open</button>
    <div id="my-popover" popover>Content</div>

## Why is it useful?
Native popovers with no JavaScript required, built-in light dismiss and accessibility.

## Tech Stack
- HTML
- CSS (Popover API)

## Preview
Open demo.html directly in your browser to see the effect.
