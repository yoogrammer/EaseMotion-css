# ease-border-beam

Animated rotating border beam using `conic-gradient` and `@property`. Zero JavaScript required.

## Usage

```html
<div class="ease-border-beam">
  Card with animated border
</div>
```

## Color Variants

| Class | Description |
|---|---|
| `ease-border-beam` | Default purple → pink beam |
| `ease-border-beam-blue` | Cyan → blue beam |
| `ease-border-beam-green` | Green → teal beam |
| `ease-border-beam-gold` | Amber → yellow beam |
| `ease-border-beam-pink` | Rose → pink beam |

## Other Variants

| Class | Description |
|---|---|
| `ease-border-beam-fast` | 1.5s rotation speed |
| `ease-border-beam-slow` | 6s rotation speed |
| `ease-border-beam-thick` | 3px border width |
| `ease-border-beam-hover` | Beam activates on hover only |

## Browser Support

Requires `@property` support (Chrome 85+, Edge 85+). Falls back to a static gradient border in unsupported browsers.
