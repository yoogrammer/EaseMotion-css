# Dashboard Analytics Neon Glow Modal

A modern **Pure CSS Neon Glow Modal** designed for **Dashboard Analytics** interfaces. The component features a smooth neon glow animation, responsive layout, keyboard accessibility, and customizable CSS custom properties without requiring JavaScript.

---

## ✨ Features

- Pure HTML & CSS (No JavaScript)
- Neon Glow animation
- Responsive layout
- Keyboard accessible
- CSS custom properties
- Dashboard Analytics inspired design
- Easy to customize

---

## 📂 Files

- `demo.html`
- `style.css`

---

## ⚙️ CSS Custom Properties

```css
:root{
    --duration:1s;
    --easing:ease-in-out;

    --primary:#00e5ff;
    --secondary:#00bcd4;
    --bg:#08111f;
}
```

You can customize:

- Animation duration
- Animation easing
- Glow intensity
- Theme colors

---

## 🚀 Usage

Open `demo.html` directly in your browser.

No JavaScript or build tools are required.
