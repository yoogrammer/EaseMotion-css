Button Spinner Duration Token Fix

Spinner duration is now customisable via --ease-btn-spinner-duration. Default is 0.7s.

Default (0.7s)

Fast (0.3s via inline token)

Slow (1.5s via inline token)

Global override via :root

Set --ease-btn-spinner-duration on :root to change all spinners globally.

:root {
  --ease-btn-spinner-duration: 1s;
}
