# Edge Computing Architecture Visualization Component

## What does this do?

Visualizes how data moves from edge devices through local and regional edge infrastructure before reaching cloud platforms and analytics services.

## How is it used?

```html
<div class="node edge">
    Local Edge Node
</div>

<div class="node cloud">
    Cloud Platform
</div>
```

## Why is it useful?

Edge computing reduces latency by processing data closer to devices instead of sending everything directly to the cloud. This component helps explain edge infrastructure architectures for IoT systems, cloud platforms, technical documentation, and educational content using pure HTML and CSS.
