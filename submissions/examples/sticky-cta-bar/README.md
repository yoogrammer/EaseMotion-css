# Sticky CTA Bar

A fixed bottom CTA bar that slides in once the user scrolls past 400px. Uses a smooth `transform: translateY` transition.

## Usage

```html
<div class="cta-bar" id="ctaBar">
  <span class="cta-text">Your offer text</span>
  <button class="cta-btn">Action</button>
</div>
```

Toggle `.visible` to show/hide. JavaScript controls the scroll threshold.

## Why it fits EaseMotion CSS

Demonstrates fixed positioning, slide-in transitions, and dark-theme surface tokens — common patterns in conversion-focused landing pages built with EaseMotion CSS.
