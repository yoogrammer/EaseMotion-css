# Text Align Utilities

## What does this do?

Provides reusable utility classes for text alignment including left, center, right, and justified text.

## How is it used?

```html
<p class="text-left">Left aligned text</p>
<p class="text-center">Center aligned text</p>
<p class="text-right">Right aligned text</p>
<p class="text-justify">Justified text</p>
```

## Why is it useful?

These utilities make text alignment simple and readable while following EaseMotion CSS's philosophy of human-friendly class names and reusable styling patterns.
