# Soft Focus Ring

What does this do? It presents a lightweight card pattern with a subtle glow on hover and focus to make interactive content feel more polished.

How is it used? Add the `.soft-card`, `.soft-pill`, and `.soft-action` classes to your HTML structure to create a simple card with a highlighted button state.

Why is it useful? It fits EaseMotion CSS by showing how small, readable motion and focus states can improve interface clarity without adding complexity.
