# Radial Option Selector Dial Component

1. What does this do? Creates a circular option dial that rotates to reflect user selection alongside displaying an interactive content panel.
2. How is it used? Place the `.dial-wrapper` containing radio inputs, a selector ring, and label positions in your mockup.
3. Why is it useful? It demonstrates how to combine CSS transformations (`rotate`) and counter-transformations on child nodes so that text labels remain upright while their parent container rotates.
