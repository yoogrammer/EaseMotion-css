# Glassmorphism Shimmer

## What does this do?
Frosted glass cards with a shimmering light overlay that glides across the surface on hover.

## How is it used?
Add the class to any HTML element:

    <div class="glass-card">
      <div class="glass-card-content">
        <h3>Card Title</h3>
        <p>Card content here.</p>
      </div>
    </div>

## Why is it useful?
Combines the popular glassmorphism aesthetic with an interactive shimmer highlight, making cards feel alive and premium. Great for portfolios, dashboards, and feature grids.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
