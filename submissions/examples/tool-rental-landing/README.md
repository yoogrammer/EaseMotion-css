# Rent-a-Tool

A responsive landing page for Rent-a-Tool built with EaseMotion CSS framework. This example showcases how to build a complete, production-ready landing page using only EaseMotion utility classes and minimal custom CSS.

## Live Demo
Open \demo.html\ in any modern web browser to view the landing page. No build tools or server required.

## Features

- **Pure EaseMotion CSS** - Uses the EaseMotion CDN for all animations and utility classes
- **Responsive Layout** - Adapts seamlessly to mobile, tablet, and desktop viewports
- **Dark Theme Design** - Modern dark color scheme with custom CSS variables
- **Smooth Animations** - Entrance animations using \ase-fade-in\, \ase-slide-up\, and delay utilities
- **Interactive Elements** - Hover effects, button animations, and navigation links
- **Zero JavaScript** - No JavaScript required for functionality or animations
- **Multiple Sections** - Hero, services/products, features, about, and footer sections
- **Accessible Markup** - Semantic HTML with proper heading hierarchy

## EaseMotion Classes Used

This page demonstrates the following EaseMotion CSS framework classes:

- \ase-fade-in\ - Fade in entrance animation
- \ase-slide-up\ - Slide up entrance animation
- \ase-delay-{100,200,300}\ - Staggered animation delays
- \ase-btn\, \ase-btn-primary\, \ase-btn-lg\, \ase-btn-sm\ - Button components
- \ase-btn-hover\, \ase-hover-lift\ - Button hover effects
- \ase-text-hover-glow\ - Text glow on hover
- \ase-container\ - Responsive container
- \ase-flex\, \ase-items-center\, \ase-justify-between\ - Flexbox utilities
- \ase-gap-{3,6}\ - Gap spacing
- \ase-text-{center,lg,xl,2xl,3xl,4xl,5xl}\ - Typography scale
- \ase-font-{bold,semibold}\ - Font weight
- \ase-py-16\, \ase-px-8\, \ase-px-4\, \ase-py-4\ - Padding utilities
- \ase-mb-{2,4,6,8,12}\, \ase-mt-2\ - Margin utilities
- \ase-opacity-{70,80,90}\ - Opacity utilities
- \ase-text-muted\ - Muted text color

## Customization

To customize this landing page for your own brand:

1. Edit the CSS variables in \style.css\ to match your brand colors
2. Replace the placeholder content in \demo.html\ with your own text
3. Adjust the hero section headline and description
4. Modify the product or service cards to showcase your offerings
5. Update contact information and hours of operation
