# Timeline Component (timeline-ru)

## What does this do?

Provides responsive timeline components for milestones, roadmaps, portfolios, and project history.

## How is it used?

Include the timeline HTML structure and style.css.

Example:

```html
<div class="timeline">
    <div class="timeline-item">
        <div class="marker"></div>
        <div class="content">
            Content
        </div>
    </div>
</div>
