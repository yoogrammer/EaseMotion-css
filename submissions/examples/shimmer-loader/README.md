# Aurora Shimmer Loader Showcase

Issue: #10385

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Features

- Pure CSS implementation
- Animated shimmer sweep
- Responsive layout
- Loading card examples
- Profile skeleton example
- No JavaScript required
- Lightweight and reusable
- Modern skeleton loading pattern

## Usage

```html
<div class="ease-loader-shimmer"></div>
```

## Example

```html
<div class="skeleton-line ease-loader-shimmer"></div>

<div class="skeleton-image ease-loader-shimmer"></div>
```

## Main Class

```css
.ease-loader-shimmer
```

## Why It Fits EaseMotion CSS

- Human-readable class names
- Animation-first philosophy
- Lightweight implementation
- Easy integration
- Reusable loading states
- Practical real-world UI pattern
