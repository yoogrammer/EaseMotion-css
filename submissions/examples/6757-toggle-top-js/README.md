\# Toggle to Top Button (#6757)



\## What does this do?

Provides an explicit, floating navigation layout utility that returns the viewport layout cleanly to the designated coordinate destination via scroll animations.



\## How is it used?

Bind the target identifier class configuration safely on your structural root node:



```html

<html class="ease-scroll-container-js" id="top-js">

&#x20; <a href="#top-js" class="toggle-to-top-js">

&#x20;   </a>

</html>
