# Two-Factor Authentication (2FA) Verification UI Showcase

## What does this do?

Provides a modern authentication interface for OTP-based two-factor verification.

## Features

- 6-digit OTP input boxes
- Verification button
- Success feedback state
- Countdown timer
- Device verification information
- Recovery code notice
- Responsive layout
- Authentication-focused animations

## How is it used?

Open `demo.html` directly in a browser.

## Why is it useful?

It demonstrates a common security-focused UI pattern used in authentication systems while showcasing subtle form, validation, and feedback animations.
