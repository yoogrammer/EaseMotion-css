## ease-repo-hygiene

**What does this do?**
A visual guide demonstrating repo hygiene best practices and
the contributor flow for EaseMotion CSS.

**How is it used?**
Open demo.html in a browser to see the contributor checklist,
step-by-step flow, and quick git command reference.

**Why is it useful?**
Helps new contributors understand the correct workflow,
reducing vague PRs, naming conflicts, and contribution errors —
improving overall repo health and maintainer experience.
