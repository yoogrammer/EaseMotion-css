Customizable Tokens

This effect is highly customizable using native CSS custom properties at the element layer.

--shimmer-color-1 The primary starting gradient tone (default: #3b82f6 blue).
--shimmer-color-2 The secondary gradient tone (default: #ec4899 pink).
--shimmer-color-highlight The sweeping light sweep highlight (default: #ffffff).
--shimmer-duration Sweep animation cycle length (default: 4 seconds).
