# CSS Quotes Smart Typography Example (#9161)

## Description
This example demonstrates how to use the `quotes` property to automatically apply language-specific quotation marks.

## Usage
Add the `.quotes-en` or `.quotes-fr` class to a parent container and use `::before`/`::after` with `open-quote`/`close-quote` keywords.

## Why EaseMotion CSS?
Proper typography is essential for motion-first design interfaces, ensuring that internationalized content remains visually polished.
