# Heartbeat Animation

This example demonstrates the usage of the `ease-heartbeat` animation utility.

## Description

The `ease-heartbeat` utility class applies a double-pulse scaling animation to an element, mimicking a human heartbeat. 

This animation is perfect for "Like" or "Favorite" actions, health-related metrics, or drawing urgent, organic attention to a specific UI element without relying on linear flashing.

## Usage

Include the `style.css` containing the animation, then apply the class to your HTML element.

```html
<div class="ease-heartbeat" style="color: red;">
  ❤️
</div>
```

## Preview

Open `demo.html` in your browser to test out the heartbeat animation on a sample icon component.
