# Shadow-Lift Hover Card

## 1. What does this do?

This adds a smooth elevation effect (`translateY` offset) combined with a soft, expanded drop shadow when a card component is hovered.

## 2. How is it used?

Apply the class directly to any card container element:

```html
<div class="shadow-lift-card">
  <h3>Card Title</h3>
  <p>Card content goes here.</p>
</div>
```

## 3. Why is it useful?

It fits EaseMotion CSS's philosophy by using clean, declarative transition classes to create realistic elevation micro-interactions on surfaces, prioritizing performance and user experience using raw CSS properties.
