CSS light-dark() Color Function

Define both light and dark colors in one declaration. Toggle below to switch.

Feature

Dynamic Theming

The light-dark() function accepts two color values — the first is used in light mode, the second in dark mode. No duplicate declarations needed.

Usage

Single Declaration

color: light-dark(#1a1a2e, #e2e8f0); — that is all you need for text that adapts to both themes automatically.

Controls

Form Controls


Integration

Works with Variables

light-dark() can be used inside custom properties, making it easy to build a complete theme system with minimal code.
