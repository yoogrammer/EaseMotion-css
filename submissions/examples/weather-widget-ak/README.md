# Weather Widget

## What does this do?
A weather widget displaying mock current weather with city name, temperature, condition icon, humidity, and wind speed data with a refresh button that randomizes values.

## How is it used?
Open demo.html in a browser. Click the Refresh button to randomize the weather values for demonstration purposes.

## Why is this useful?
Demonstrates a compact data-display card layout with dynamic content updates, suitable for dashboard or weather app interfaces.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
