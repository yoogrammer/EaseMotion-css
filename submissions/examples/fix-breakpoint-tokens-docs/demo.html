Breakpoint Token Limitation

⚠️ Known CSS Limitation
var(--ease-bp-md) does not work inside @media queries in any browser. CSS custom properties cannot be used as media query values.
✅ Correct Usage
Copy the raw pixel value directly into your @media rule:

@media (min-width: 768px) { ... }
