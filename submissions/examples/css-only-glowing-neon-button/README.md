# CSS-only Glowing Neon Button

### 1. What does this do?
A cyberpunk-inspired glowing neon button component. It features animated borders and intense multi-layered box-shadows on hover to simulate a realistic neon tube glow.

### 2. How is it used?
```html
<a href="#" class="neon-button">
  <span></span>
  <span></span>
  <span></span>
  <span></span>
  ENTER THE MATRIX
</a>
```

### 3. Why is this useful?
Neon aesthetics are highly requested for dark-mode web apps, portfolios, and gaming interfaces. This pure CSS implementation offers high performance and visual impact without relying on images or JavaScript.
