# AeroTherm Pro — Advanced HVAC Business Landing

A premium, interactive, full-page home services (HVAC) landing page designed using **EaseMotion CSS** framework classes.

## Showcase Features

*   **Responsive Glassmorphism Navigation**: Implements `ease-navbar-glass` along with `ease-navbar-glass-sticky` for a modern, responsive sticky navbar.
*   **Emergency Service Call-To-Action**: Features an alert banner with a pulsing indicator (`ease-ping`) and an attention-grabbing shake button (`ease-shake-subtle`).
*   **Comprehensive Services Grid**: Grid card system featuring detailed information on heating, cooling, and air quality using `.ease-card-hover` and `.ease-card-glow` interactions.
*   **Interactive Estimator Card**: Fully working JavaScript estimator card using `.ease-field`, `.ease-input`, and `.ease-select` controls.
*   **Interactive Booking/Scheduling Form**: Secure contact service scheduler using standard framework form components (`.ease-field`, `.ease-input`, `.ease-select`, `.ease-textarea`, `.ease-btn-block`).
*   **Entrance and Scroll-Triggered Reveal Animations**: Polished user experience built using composable scroll-reveal animations (`ease-reveal`, `ease-reveal-up`, `ease-reveal-left`, `ease-reveal-right`) and entrance delay modifiers (`ease-delay-200`).
*   **Premium Composed Footer**: Structured newsletter sign-up and social media navigation built utilizing the `.ease-footer` suite.

## EaseMotion CSS Classes Showcased

### Layout & Spacing
*   `.ease-container` — Constrains layout width to the design grid.
*   `.ease-center` — Centers contents using standard flex variables.
*   `.ease-flex` — Applies standard display flex.
*   `.ease-flex-col` — Vertical flexbox container direction.
*   `.ease-flex-wrap` — Wraps overflow elements safely.
*   `.ease-justify-center` / `.ease-justify-between` — Horizontally aligns items.
*   `.ease-items-center` / `.ease-items-start` — Vertically aligns items.
*   `.ease-gap-2` / `.ease-gap-4` / `.ease-gap-8` / `.ease-gap-10` — Flexbox/Grid row and column spacing.
*   `.ease-grid` — Enables standard CSS Grid layout.
*   `.ease-grid-cols-2` / `.ease-grid-cols-12` — Set specific column layouts.
*   `.ease-col-span-5` / `.ease-col-span-7` — Responsive column span configurations.
*   `.ease-stack` / `.ease-stack-sm` — Flexbox stack helpers.
