# Rotate Button Entrance

## What does this do?

Adds a button entrance animation where the button rotates and scales into view.

## How is it used?

```html
<button class="rotate-button-entrance">
  Get Started
</button>
```

## Why is it useful?

Rotate button entrance animations help draw attention to important call-to-action buttons, onboarding actions, and interactive controls while keeping the motion lightweight and reusable.
