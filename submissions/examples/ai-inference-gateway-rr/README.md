# AI Inference Gateway Dashboard Component

## What does this do?

Provides a monitoring interface for AI inference gateways, displaying request routing, endpoint health, latency metrics, and traffic statistics.

## How is it used?

```html
<div class="endpoint-card">
    <h3>GPT-4 Endpoint</h3>
</div>
```

## Why is it useful?

Modern AI platforms rely on inference gateways to manage model routing, traffic distribution, caching, and endpoint health. This component provides a reusable infrastructure monitoring interface while demonstrating enterprise AI operations dashboard patterns.
