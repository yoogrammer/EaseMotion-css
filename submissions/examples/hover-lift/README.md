# ease-hover-lift

Lift effect on hover with translateY(-4px) and shadow. A simpler variant of ease-hover-lift-shadow.

## Usage

```html
<div class="ease-card ease-hover-lift">Lifts on hover</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-hover-lift` | Default lift with shadow |
| `ease-hover-lift-sm` | Subtle lift |
| `ease-hover-lift-lg` | Strong lift |

## Notes

- Applies `translateY(-4px)` and `box-shadow: var(--ease-shadow-xl)`
- Simpler alternative to `ease-hover-lift-shadow`
- Great for cards, buttons, and panels
- Respects `prefers-reduced-motion`
