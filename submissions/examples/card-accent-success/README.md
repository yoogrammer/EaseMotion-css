# ease-card-accent-success

Card variant with a 4px green left border accent for success/confirmation states. Distinct from the primary-colored `ease-card-accent`.

## Usage

```html
<div class="ease-card ease-card-accent-success">Success card</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-card-accent-success` | Green left accent border |
| `ease-card-accent-danger` | Red left accent border |
| `ease-card-accent-warning` | Yellow left accent border |

## Notes

- 4px left border accent using semantic color tokens
- Pairs with card backgrounds for status messages
- Great for form feedback, notifications, and status panels
