opacity:0 + 16ms delay prevents FOUC

.ease-animate-ready pre-sets opacity: 0 and adds a 16ms animation-delay so the browser paints the hidden state before the animation starts. Works with .ease-fade-in, .ease-slide-up, and .ease-zoom-in.
