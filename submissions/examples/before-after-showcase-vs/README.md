# Before & After Showcase Example

## What does this do?
This example provides a modern before-and-after comparison layout that highlights visual improvements between an outdated interface and a refined modern UI. It uses CSS-only mockup cards, glassmorphism-inspired panels, and responsive split-screen behavior.

## How is it used?
You can customize labels, headings, descriptions, and mockup card content directly in `demo.html`. Colors, spacing, panel effects, and animations are controlled in `style.css`, making it easy to adapt the showcase for different design transformations or portfolio case studies.

## Why is it useful?
Before-and-after comparisons are useful for product redesign presentations, UX case studies, design system documentation, and client deliverables. This component communicates impact quickly by showing clear visual contrast between legacy and improved interfaces.
