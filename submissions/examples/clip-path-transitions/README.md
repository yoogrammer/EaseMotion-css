# Clip Path Transitions

## What does this do?
Elements transition between clip-path shapes on hover, revealing their full content.

## How is it used?
Add a clip-path class to a card:

    <div class="clip-card clip-circle"><span>Content</span></div>

## Why is it useful?
Creative hover reveals for galleries, portfolio items, and interactive cards.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
