What is EaseMotion CSS?
EaseMotion is a pure CSS animation utility library. It provides ready-to-use classes for common animations with zero JavaScript required.
How do I install it?
Simply link the easemotion.css file in your HTML and add the utility classes to your elements. No build step or npm install needed.
Does it support dark mode?
Yes! EaseMotion uses CSS custom properties for theming, making it easy to adapt to dark mode using standard CSS media queries or class-based approaches.

Flush variant

Is it accessible?
Absolutely. The accordion uses native <details> and <summary> HTML elements, which are fully accessible by default — keyboard navigable and screen-reader friendly.
Can I use it with React or Vue?
Yes! Since it's pure CSS, it works with any framework. Just apply the classes to your elements as you would any utility CSS.
Is it free to use?
EaseMotion CSS is fully open source and free to use in personal and commercial projects.
