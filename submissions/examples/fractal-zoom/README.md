# Fractal Zoom Animation

## What does this do?

Adds reusable fractal zoom animations that create the illusion of continuously zooming into an element.

---

## How is it used?

```html
<div class="ease-fractal-zoom-mt">
    Content
</div>

<div class="ease-fractal-zoom-soft-mt">
    Content
</div>

<div class="ease-fractal-zoom-fast-mt">
    Content
</div>
```

---

## Why is it useful?

Fractal zoom effects create a sense of depth and recursive motion that can be used in hero sections, creative landing pages, loading screens, and showcase components.

The animation expands the EaseMotion CSS animation collection with a unique motion style while remaining simple and reusable.
