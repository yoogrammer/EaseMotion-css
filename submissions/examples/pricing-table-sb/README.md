# CSS-only Pricing Table Component

A fully responsive, CSS-only pricing table component for showcasing multiple pricing plans in a clean and modern layout.

This feature includes:

- Plan cards with titles, pricing, and feature lists.
- Call-to-action buttons for each plan.
- A "featured" or "recommended" plan style with highlighting and scale effects.
- Smooth hover elevation animations for cards.
- A responsive layout that wraps cards on smaller screens.
- Dark mode compatibility.

## Why is this useful for EaseMotion CSS?

Pricing sections are essential for SaaS products, portfolios, business websites, and e-commerce platforms. A CSS-only pricing table offers a lightweight, dependency-free solution that is easily customizable. It seamlessly integrates with EaseMotion CSS's goal of providing developer-friendly, human-readable, and animation-focused UI utilities.
