# Shockwave Propagation

## Feature Description

Shockwave Propagation creates a cascading wave animation across sibling elements.

When one item is hovered, nearby items react sequentially with delayed animations, creating the illusion of a travelling shockwave.

## Usage

```html
<div class="ease-shockwave-group">
  <div class="ease-shockwave-item">1</div>
  <div class="ease-shockwave-item">2</div>
  <div class="ease-shockwave-item">3</div>
</div>
