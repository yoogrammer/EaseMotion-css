# Social Links

**What does this do?**
Creates a horizontal row of social media icon links with platform-specific hover animations. Each icon changes color and glow based on its respective platform, providing immediate visual recognition and interactive feedback.

**How is it used?**

```html
<div class="social-links">

  <a href="#" class="social-link github">
    <i class="fab fa-github"></i>
  </a>

  <a href="#" class="social-link linkedin">
    <i class="fab fa-linkedin-in"></i>
  </a>

  <a href="#" class="social-link twitter">
    <i class="fab fa-x-twitter"></i>
  </a>

</div>
```

**Why is it useful?**
Social media links appear in portfolios, landing pages, documentation sites, blogs, and personal websites. Developers often need a clean and reusable way to display these links while preserving each platform's visual identity.

The hover animations provide immediate feedback, improve discoverability, and make social sections feel more engaging without requiring JavaScript. The component is lightweight, customizable, and suitable for a wide variety of modern interfaces.
