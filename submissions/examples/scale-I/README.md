 
# Scale Transform Utilities (ease-scale)

Scale transform utilities for consistent sizing — 75%, 90%, 100%, 110%, and 125%.

## Files

| File | Description |
|------|-------------|
| 📄 [demo.html](./demo.html) | Interactive demo |
| 🎨 [style.css](./style.css) | Scale styles |
| 📖 [README.md](./README.md) | Documentation |

## Classes

### Static Scale
| Class | Scale |
|-------|-------|
| `ease-scale-75` | 75% |
| `ease-scale-90` | 90% |
| `ease-scale-100` | 100% |
| `ease-scale-110` | 110% |
| `ease-scale-125` | 125% |

### Hover Scale
| Class | Hover Scale |
|-------|-------------|
| `ease-scale-hover-110` | 110% on hover |
| `ease-scale-hover-125` | 125% on hover |

## Usage

```html
<!-- Scale down -->
<div class="ease-scale-75">75% of original size</div>

<!-- Scale up -->
<div class="ease-scale-125">125% of original size</div>

<!-- Hover scale -->
<div class="ease-scale-hover-110">Scales to 110% on hover</div>
