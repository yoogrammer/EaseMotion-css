# Staggered Dropdown Menu

1. What does this do? Creates a dropdown menu where items reveal sequentially with a staggered animation effect when the menu is opened.
2. How is it used? Apply `.dropdown` to the container and `.stagger-1`, `.stagger-2`, etc., to each menu item.
3. Why is it useful? It provides a more polished and modern feel to navigation menus compared to static reveals, guiding the user's eye through the options.
