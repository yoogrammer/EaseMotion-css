## What does this do?
The shake effect shakes a component horizontally or vertically, continually

## How is it used?
The class can be directly applied to any html element:

```html
<div class="horizontal-shake">
  Box
</div>
```
```html
<div class="vertical-shake">
  Box
</div>
```

## Why is it useful?
The Shake Effect provides a simple and recognizable way to draw user attention to an element. It can be used to indicate invalid form inputs, failed actions, warnings, notifications, or to highlight elements that require immediate attention.

This animation aligns with EaseMotion CSS's animation-first philosophy by providing a reusable, composable utility that developers can apply with a single class instead of creating custom keyframes for every project.
