Ease Copy Button AK

What does this do?

Adds a reusable SVG-based copy-to-clipboard button for code blocks with visual feedback and hover interactions.

---

How is it used?

<div class="code-container">
  <pre><code>npm install easemotion-css</code></pre>

  <button class="copy-btn" aria-label="Copy code">
    <!-- SVG Copy Icon -->
  </button>
</div>

The button copies the code snippet to the clipboard when clicked and temporarily displays a copied-state indicator for user feedback.

---

Why is it useful?

This component improves developer experience by allowing users to quickly copy code snippets without manually selecting text. The SVG-based icon keeps the UI modern and lightweight while maintaining accessibility and responsive interaction behavior.

Features

- SVG-based copy icon
- Clipboard API support
- Hover and focus interaction states
- Temporary copied-state feedback
- Responsive and lightweight UI
- Dark-theme-friendly styling
- Easy integration into documentation/code block layouts
