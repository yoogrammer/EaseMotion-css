# Expanding Search Input

A CSS-only search field that smoothly expands when focused.

## Features

- Pure CSS
- Focus-based expansion animation
- Accessible markup
- Responsive behavior
- No JavaScript required

## Usage

```html
<input
  type="search"
  class="search-input"
  placeholder="Search..."
  aria-label="Search"
>
```

The input expands smoothly on focus, providing additional space for user input while maintaining a compact default appearance.
