# Animated Toggle Switch

## What it does

CSS-only 3D toggle switch with smooth sliding animation and multiple themes.

## How to use it

<label class="ease-toggle">
    <input type="checkbox">
    <span class="ease-toggle-slider"></span>
</label>

## Variants

- ease-toggle-large - Larger size (80x44px)
- ease-toggle-small - Smaller size (44x24px)
- ease-toggle-pink - Pink/purple gradient
- ease-toggle-green - Green gradient
- ease-toggle-glow - Neon glow effect
- ease-toggle-pulse - Pulse animation on toggle
- ease-toggle-icon - Icon-based (moon/sun)

## Why it fits EaseMotion CSS

- Pure CSS, zero JavaScript
- 3D sliding animation
- Multiple color themes
- Size variants
- Accessible (uses checkbox)
