Bounce Animation Utilities

Attention-grabbing bounce animations with configurable intensity, speed, and hover interactions.

Continuous Bounce

.animate-bounce makes the element bounce up and down continuously.

🚙
.animate-bounce Default bounce (20px, 0.6s)
⚡
.animate-bounce .bounce-fast Fast bounce (0.3s)
🔴
.animate-bounce .bounce-slow Slow bounce (1s)
📢
style="--bounce-height: 40px" High bounce (40px)

Single Bounce

.animate-bounce-once bounces exactly once on page load. Refresh to replay.

👋
.animate-bounce-once Entrance bounce
✔
.animate-bounce-once .bounce-fast Fast entrance

Hover Bounce

.hover-bounce triggers a bounce when you hover over the card.

🔔
Hover to bounce
.hover-bounce
✨
Fast hover bounce
.hover-bounce .bounce-fast

CSS Custom Properties

Customize bounce behavior by overriding these variables.

--bounce-height

Default: 20px

Controls the bounce distance. Higher values = more energetic bounce.

--bounce-speed

Default: 0.6s

Sets the animation duration. Use 0.3s for fast, 1s for slow.

❤️

Accessibility

The component respects prefers-reduced-motion — all bounce animations are disabled for users who prefer reduced motion. On small screens, the bounce height is halved to reduce intensity.
