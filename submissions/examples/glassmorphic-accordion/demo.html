What is EaseMotion CSS?

EaseMotion CSS is an animation-first utility library focused on readable, composable motion primitives for the web.

Does it require JavaScript?

Most animations are pure CSS. This accordion uses native <details> — zero JS required.

Is reduced motion supported?

Yes — all transitions respect prefers-reduced-motion and fall back to instant open/close.
