# Testimonial Carousel

A simple testimonial carousel with fade transitions between slides. Shows navigation dots (clickable) and prev/next arrow buttons. The current slide fades in while others remain hidden.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser. Use the arrows or click the dots to navigate testimonials.

## Accessibility notes

Arrow buttons and dot indicators are `<button>` elements. Carousel navigation is keyboard accessible if the buttons are focused.
