Usage

<!-- Auto-stagger parent (up to 6 children) -->
<div class="ease-stagger-auto">
    <div>Item 1</div>
    <div>Item 2</div>
    <div>Item 3</div>
</div>

<!-- Different effects -->
<div class="ease-stagger-fade-up">...</div>
<div class="ease-stagger-fade-left">...</div>
<div class="ease-stagger-zoom">...</div>

<!-- Different speeds -->
<div class="ease-stagger-fast">...</div>
<div class="ease-stagger-slow">...</div>

<!-- Custom stagger using CSS variables -->
<div style="--stagger-delay: 100ms; --base-delay: 50ms;">
    <div style="--i: 1;">Item 1</div>
    <div style="--i: 2;">Item 2</div>
</div>

⚡ Staggered System Features

  • 🎯 Auto-stagger for up to 6 children using nth-child
  • ⚡ Multiple stagger effects (fade-up, fade-left, zoom)
  • 🎮 Speed variations (fast: 40ms, normal: 80ms, slow: 150ms)
  • 💪 Custom CSS variables support (--stagger-delay, --base-delay, --i)
  • ✨ Smooth cubic-bezier easing
  • 🚀 No JavaScript required
