# Shake Image Utility Animation

## What does this do?
Applies a horizontal shake effect with slight rotation to image components,
triggered either on load/click or on hover.

## How is it used?

**On click / programmatic trigger:**
```html
<img class="shake-image" src="photo.jpg" alt="..." />
```

**On hover:**
```html
<img class="shake-image-hover" src="photo.jpg" alt="..." />
```

## Why is it useful?
Shake animations draw attention to images during error states, alerts, or
interactive feedback. Pure CSS, no JS, with full prefers-reduced-motion support.
