Documentation Navbar Redesign

This proposal adds a dedicated, high-contrast action button for the Animation Builder to the website's top navigation bar. It is styled to stand out as a primary action, ensuring that new and returning developers can quickly build custom motion profiles.

EaseMotion CSS
  • Home
  • Animation Preview
  • Animation Builder
✨ Design Highlights:
  • High-contrast primary theme color background (#6c63ff in dark mode, #4f46e5 in light mode).
  • Elastic scale shrink (transform: scale(0.95)) and darken state on click/active.
  • Centralized variable compatibility: integrates seamlessly with custom CSS custom properties.
  • Optimized padding & line-height adjustments for consistent cross-browser baseline alignment.
