# Animated Hamburger Menu Icon

A three-line hamburger icon that animates into a close (X) icon when clicked, and back to a hamburger when clicked again. The top and bottom lines rotate to form the X, while the middle line fades out. Includes a state indicator below.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser. Click the hamburger icon to see it animate to an X and back.

## Accessibility notes

The button has an `aria-label="Toggle menu"`. Reduced motion disables the line transitions.
