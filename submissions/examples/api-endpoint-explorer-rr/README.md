# API Endpoint Explorer Component

## What does this do?

Provides a clean interface for displaying API endpoints, HTTP methods, authentication requirements, and response previews.

## How is it used?

```html
<span class="method get">GET</span>

<h3>/api/users</h3>
```

## Why is it useful?

API documentation is a critical part of modern software development. This component offers a structured and visually appealing way to showcase endpoints, making developer documentation easier to read and navigate using pure HTML and CSS.
