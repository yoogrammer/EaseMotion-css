Usage

<!-- Add dir="rtl" to html element for RTL languages -->
<html lang="ar" dir="rtl">

<!-- Use logical properties in CSS -->
.element {
    margin-inline-start: 1rem;     /* instead of margin-left */
    margin-inline-end: 1rem;       /* instead of margin-right */
    padding-inline-start: 1rem;    /* instead of padding-left */
    padding-inline-end: 1rem;      /* instead of padding-right */
    inset-inline-start: 0;         /* instead of left */
    inset-inline-end: 0;           /* instead of right */
}

🔄 RTL Support Features

  • 🌍 Supports Arabic, Hebrew, Urdu, and other RTL languages
  • 📐 Uses CSS logical properties instead of physical properties
  • 🔄 Automatic layout mirroring with dir="rtl"
  • 📱 Fully responsive across all devices
  • 🎯 No JavaScript required for RTL layout
  • 💫 Seamless integration with existing components
