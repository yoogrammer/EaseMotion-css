# ease-color-cycle

Animated color cycling on text, background or border using CSS keyframes. Zero JavaScript required.

## Usage

```html
<!-- Text color cycling -->
<p class="ease-color-cycle">Color Cycle</p>

<!-- Background color cycling -->
<div class="ease-color-cycle-bg">Background</div>

<!-- Border color cycling -->
<div class="ease-color-cycle-border">Border</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-color-cycle` | Cycles text color |
| `ease-color-cycle-bg` | Cycles background color |
| `ease-color-cycle-border` | Cycles border color |
| `ease-color-cycle-fast` | 2s cycle speed |
| `ease-color-cycle-slow` | 8s cycle speed |
| `ease-color-cycle-neon` | Neon color theme |
| `ease-color-cycle-pastel` | Pastel color theme |
| `ease-color-cycle-earth` | Earth tone theme |
| `ease-color-cycle-pause` | Pauses on hover |

## Custom Properties

| Property | Default | Description |
|---|---|---|
| `--ease-cycle-1` | `#6366f1` | First color |
| `--ease-cycle-2` | `#ec4899` | Second color |
| `--ease-cycle-3` | `#f97316` | Third color |
| `--ease-cycle-4` | `#22c55e` | Fourth color |
| `--ease-cycle-speed` | `4s` | Animation duration |
