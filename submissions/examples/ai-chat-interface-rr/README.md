# Modern AI Chat Interface Component

## What does this do?

Provides a responsive AI chat interface with user messages, assistant messages, and an animated typing indicator.

## How is it used?

```html
<div class="message ai">
    AI response
</div>

<div class="message user">
    User message
</div>
```

## Why is it useful?

AI-powered applications are becoming increasingly common. This component provides a reusable chat interface pattern suitable for AI assistants, chatbots, customer support systems, and conversational web applications while showcasing EaseMotion CSS animation capabilities.
