# Animated Timeline Component

A modern vertical timeline component that showcases milestones with smooth animations and clean styling.

## Features

- Vertical timeline layout
- Animated milestone cards
- Responsive design
- Hover interactions
- Easy customization
- Demonstrates EaseMotion CSS animation usage
