# Scale Shadow Hover

A lightweight hover utility that combines subtle scaling with enhanced shadow depth for a smooth interactive effect.

## Features

* Pure CSS hover animation
* Smooth scale interaction
* Dynamic shadow enhancement
* Lightweight and reusable
* No JavaScript required

## Files

* `demo.html`
* `style.css`

## Usage

```html
<div class="ease-scale-shadow-hover">
  Hover Me
</div>
```

## Concepts Used

* CSS transitions
* transform: scale()
* box-shadow animation
* easing functions
