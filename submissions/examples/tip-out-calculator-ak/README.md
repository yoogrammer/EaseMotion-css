# Tip-Out Calculator

## What does this do?
A restaurant tip-out calculator with configurable staff roles and tip-out percentages, per-person payout calculation, and percentage total validation.

## How is it used?
Enter total tips for the shift. Add or remove staff roles (bartender, server, busser, runner, host) with their tip-out percentages. The calculator shows per-person payout. A warning appears if percentages don't add up to 100%.

## Why is this useful?
This helps restaurant managers and staff fairly distribute tip pools based on predetermined percentages for each role.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to use the calculator.
