# Secure Password Strength Meter Component

## What does this do?

Provides an animated password strength meter with visual feedback, strength indicators, and password requirement validation.

## Features

- Strength progress bar
- Animated fill effect
- Password requirement checklist
- Strong password indicator
- Responsive layout

## How is it used?

Open `demo.html` directly in a browser.

## Why is it useful?

It demonstrates a common authentication and security-related UI pattern while showcasing smooth animations and validation feedback.
