# Component Showcase Gallery

## Overview

A visual gallery that allows developers to browse, preview, search, and filter EaseMotion CSS components from a single interface.

---

## Features

- Live component previews
- Search functionality
- Category filtering
- Copy HTML button
- Copy CSS class button
- Responsive grid layout
- Interactive hover effects
- Component descriptions
- Mobile-friendly design

---

## Folder Structure

```text
component-showcase-gallery/
├── demo.html
├── style.css
├── script.js
└── README.md
```

---

## Usage

1. Open `demo.html` in a browser.
2. Browse available components.
3. Search by component name.
4. Filter by category.
5. Copy HTML snippets.
6. Copy CSS class names.

---

## Categories Included

- Buttons
- Cards
- Alerts

Additional categories can be added easily by extending the component data array inside `script.js`.

---

## Why It Fits EaseMotion CSS

This gallery provides a centralized visual reference for components, making it easier for developers to:

- Discover available UI elements
- Learn through live examples
- Experiment quickly
- Reduce documentation navigation
- Improve onboarding experience

The design aligns with EaseMotion CSS's goal of creating a simple and visual-first developer experience.
