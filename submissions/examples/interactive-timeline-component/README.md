# Interactive Timeline Component

A responsive animated timeline component built using pure HTML and CSS.

## Features

- Vertical responsive timeline
- Smooth entry animations
- Hover interactions
- Timeline milestone indicators
- Responsive mobile layout
- Pure CSS implementation

## Usage

Open `demo.html` in a browser to view the interactive timeline demo.

## Files

- `demo.html` → Timeline structure
- `style.css` → Timeline animations and styles
- `README.md` → Documentation
