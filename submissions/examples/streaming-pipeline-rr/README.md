# Real-Time Data Streaming Pipeline Component

## What does this do?

Visualizes how real-time events flow from producers through streaming infrastructure into analytics and storage systems.

## How is it used?

```html
<div class="node broker">
    Kafka Event Broker
</div>

<div class="node processor">
    Stream Processing Engine
</div>
```

## Why is it useful?

Modern applications generate continuous streams of data from websites, mobile apps, IoT devices, and services. This component provides a clear visualization of how those events move through streaming platforms, processing engines, storage systems, and analytics tools using pure HTML and CSS.
