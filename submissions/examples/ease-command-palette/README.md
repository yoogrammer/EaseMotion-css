\# ease-command-palette



CMD+K style command palette overlay with searchable commands and keyboard navigation.



\## Features



\- ⌨️ Keyboard accessible (arrow keys, Enter, ESC)

\- 🔍 Search/filter commands

\- 🎨 Backdrop blur effect

\- 📱 Responsive design

\- ⚡ Animated slide-down entrance



\## Usage



```html

<!-- Trigger button -->

<button onclick="openPalette()">Open Palette</button>



<!-- Keyboard shortcut -->

Press Cmd+K (Mac) or Ctrl+K (Windows/Linux)
