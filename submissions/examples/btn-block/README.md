# ease-btn-block

Full-width block button (`width: 100%`). Pairs with any button variant `.ease-btn-primary`, `.ease-btn-outline`, etc.

## Usage

```html
<button class="ease-btn ease-btn-primary ease-btn-block">Full width</button>
```

## Variants

| Class | Description |
|---|---|
| `ease-btn-block` | Full-width block button |

## Notes

- Stretches to fill parent container width
- Essential for forms, CTAs, and mobile layouts
- Composable with any button style variant
