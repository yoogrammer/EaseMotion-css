Text Scramble Effect

Characters scramble with random symbols before decoding to reveal the real text. Triggers on page load and on hover.

📄 How to Use

Add ease-scramble and a data-text attribute to any element:

<h1 class="ease-scramble" data-text="Hello World">
  Hello World
</h1>

<p class="ease-scramble ease-scramble--emerald" data-text="EaseMotion CSS">
  EaseMotion CSS
</p>
