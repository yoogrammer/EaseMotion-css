# ease-hover-morph-card

Card morphing hover effect — transitions from square/rectangle to circle on hover with a scale-up.

## Usage

```html
<div class="ease-card ease-hover-morph-card">Hover to morph</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-hover-morph-card` | Default morph (0.3s, scale 1.08) |
| `ease-morph-slow` | Slow transition (0.8s) |
| `ease-morph-fast` | Fast transition (0.15s) |
| `ease-morph-scale-lg` | Extra scale (1.15x) |

## Notes

- Pairs well with `ease-card`, avatar images, or team member cards
- Use `ease-card-shadow` for depth on hover
- Respects `prefers-reduced-motion`
