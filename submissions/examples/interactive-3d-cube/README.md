# Interactive 3D Cube

## What does this do?
A CSS 3D cube that interactively rotates based on mouse position.

## How is it used?
Structure a cube with six faces in a preserving-3d container:

    <div class="cube">
      <div class="cube-face front">Front</div>
      ...
    </div>

## Why is it useful?
Engaging 3D interaction for hero sections, product showcases, and creative portfolios.

## Tech Stack
- HTML
- CSS (3D transforms)
- Vanilla JS (mouse tracking)

## Preview
Open demo.html directly in your browser to see the effect.
