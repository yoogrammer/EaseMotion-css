# Holographic Hover Card Effect

A lightweight CSS hover effect that adds a holographic shine animation and glowing highlight to cards. On hover, the card scales slightly while a cyan shimmer sweeps across its surface, creating a modern and interactive visual experience.

## Features

* CSS-only implementation
* Smooth holographic shine animation
* Subtle glow and scale effect on hover
* Easy to customize and integrate
* Perfect for cards, buttons, and UI components
