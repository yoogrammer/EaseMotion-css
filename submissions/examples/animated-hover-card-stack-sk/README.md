# Animated Hover Card Stack

**What does this do?**  
Creates a stacked card layout where cards fan out and scale up on hover using smooth CSS transitions with a spring-like cubic-bezier curve.

**How is it used?**  
Wrap `.stack-card` elements inside a `.card-stack` container. Each card uses `--index` and `--bg` custom properties. On hover, cards spread apart with staggered rotation and translate transforms.

**Why is it useful?**  
Provides an engaging, interactive card reveal pattern for feature showcases, portfolio items, or onboarding steps — using pure CSS with no JavaScript required.
