# Notification Dropdown Panel

A bell icon with a red badge count that opens a dropdown panel showing notifications. Each notification has an indigo dot, text, and relative timestamp. Includes a Mark All Read button that hides the dots. Clicking outside closes the dropdown.

## EaseMotion CSS classes used

- `ease-flex` — page-level centering
- `ease-center` — vertical and horizontal centering

## How to run

Open `demo.html` in a browser. Click the bell to open the notification panel. Click outside to close.

## Accessibility notes

The bell is a `<button>` element. The badge count is visible text. The Mark All Read button is a `<button>`.
