# Activity Feed Component

## What does this do?

A responsive activity feed component for displaying recent user actions, updates, notifications, or events.

## How is it used?

```html
<div class="activity-item">
  <div class="avatar">A</div>

  <div class="content">
    <h3>Alice Johnson</h3>
    <p>Created a new project.</p>
    <span>2 minutes ago</span>
  </div>
</div>
```

## Why is it useful?

Activity feeds are widely used in dashboards, collaboration tools, project management applications, admin panels, and social platforms. This component provides a lightweight solution using only HTML and CSS.
