# Responsive Animated Masonry Grid

A responsive gallery layout using CSS columns and animated details overlays.

---

## What is Included

1. **`demo.html`**: A clean masonry gallery layout containing varying card sizes, image backgrounds, and sliding detail templates.
2. **`style.css`**: Column count allocations, break avoidance properties, scale transforms, and overlay opacity transitions.

---

## Verification Steps

1. Open `demo.html` in a web browser.
2. Verify cards are arranged cleanly in columns.
3. Resize the window to verify the column counts change to 2 on tablets and 1 on mobile screens.
4. Hover over a card — confirm that the background scales slightly and the details overlay slides up from the bottom.
