# ease-fill-current / ease-stroke-current

SVG icon color utilities using `currentColor`.

## Usage

```html
<svg class="ease-fill-current ease-text-primary" viewBox="0 0 24 24">
  <path d="..." />
</svg>
<svg
  class="ease-stroke-current ease-text-success"
  viewBox="0 0 24 24"
  fill="none"
>
  <path stroke-width="2" d="..." />
</svg>
```

## Classes

| Class                 | CSS Property         |
| --------------------- | -------------------- |
| `ease-fill-current`   | fill: currentColor   |
| `ease-stroke-current` | stroke: currentColor |
