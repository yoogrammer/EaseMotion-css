# Commission Calculator

## What does this do?
A tiered commission calculator that computes earnings based on sale amount and commission tier structure, with optional base salary and visual sales target progress bars.

## How is it used?
Enter a sale amount, select a commission tier, and optionally toggle base salary. The calculator shows commission earned, base salary, and total earnings with target progress bars.

## Why is this useful?
This helps sales professionals and managers quickly estimate commission-based earnings across different tier structures and track progress toward sales targets.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to use the calculator.
