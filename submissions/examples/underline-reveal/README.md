# Hover Underline Reveal Effect

A clean, modern navigation link hover effect where an accent underline scales smoothly outwards from the center.

## Features
- **Centering Animation:** Line grows outward symmetrically from the middle.
- **Performance Optimized:** Uses CSS transforms (`scaleX`) instead of changing width property to prevent browser layout lag.
- **Cubic Bezier Easing:** Custom cubic curve creates a crisp, premium movement response.

## How to Use

Include the utility class `.ease-link-underline-reveal` on your anchor tags:

```html
<a href="#" class="ease-link-underline-reveal">Link Text</a>
