# Jello Hover Animation

## 1. What does this do?
Adds an interactive, physics-based "jello" wobble effect to components when hovered, morphing geometry via shrinking/stretching skew transformations.

## 2. How is it used?
Attach the interactive utility handler class onto any clickable or highlighted element layer:
```html
<button class="jello-hover-trigger">Interact</button>
