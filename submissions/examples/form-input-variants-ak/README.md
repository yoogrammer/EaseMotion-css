Form Input Variants AK

What does this do?

Demonstrates various form input style variants including outlined, filled, underlined, and with icons, using CSS custom properties.

---

How is it used?

<input class="input input--outlined" placeholder="Enter text" />
<input class="input input--filled" placeholder="Enter text" />
<input class="input input--underlined" placeholder="Enter text" />

Apply the variant modifier class to the input element.

---

Why is it useful?

Provides a reusable form input system with consistent styling, focus states, and theming across different visual variants without duplicating base styles.

Features

- Three visual variants (outlined, filled, underlined)
- Focus ring with custom property color
- Disabled state styling
- Placeholder styling
- Consistent padding and border-radius
- Custom property theming
- Accessible focus indicators
