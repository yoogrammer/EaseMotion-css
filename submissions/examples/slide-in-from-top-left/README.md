# ease-slide-in-from-top-left

Diagonal corner entrance animation: element slides in from the top-left corner to its natural position with fade-in.

## Usage

```html
<div class="ease-slide-in-from-top-left">Enters from top-left corner</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-slide-in-from-top-left` | Diagonal from top-left corner |
| `ease-slide-in-from-top-right` | Diagonal from top-right corner |
| `ease-slide-in-from-bottom-left` | Diagonal from bottom-left corner |
| `ease-slide-in-from-bottom-right` | Diagonal from bottom-right corner |

## Notes

- Slides in diagonally from corner
- All corner directions available
- Great for quadrant reveals, modal entrances, and staggered corner content
- Respects `prefers-reduced-motion`
