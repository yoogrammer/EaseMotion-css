# CSS Scroll-Driven Staggered Reveal Example

Scroll-triggered staggered card reveal using CSS scroll-driven animations without JavaScript.

## Features

- Cards reveal with staggered delays using scroll-driven animation timelines
- No JavaScript — pure CSS `animation-timeline: scroll()`
- Opacity + translateY entrance effect
- Dark theme with `prefers-reduced-motion` support

## Files

- `demo.html` — Scrollable card list
- `style.css` — Self-contained CSS (100 lines)

## Preview

A vertical list of cards that fade and slide up one by one as the page scrolls.
