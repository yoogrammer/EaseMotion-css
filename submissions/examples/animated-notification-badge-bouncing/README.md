# Animated Notification Badge

## What it does

Animated badge for notifications, messages, or alerts with bounce, pulse, shake, glow, and pop effects.

## How to use it

<div class="badge-container">
    <div class="icon">🔔</div>
    <div class="ease-notification-badge">3</div>
</div>

## Variants

- ease-badge-bounce - Bounce animation on update
- ease-badge-pulse - Continuous pulsing
- ease-badge-shake - Shaking effect
- ease-badge-glow - Neon glow pulse
- ease-badge-pop - Pop-in animation
- ease-badge-spin - Spin on update

## Why it fits EaseMotion CSS

- Pure CSS animations
- Multiple attention-grabbing effects
- Perfect for notification counts
- Easy to integrate
