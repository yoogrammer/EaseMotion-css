# Decision Randomizer

## What does this do?
A fun decision-making tool with two modes: a spinning wheel that lands on a random choice, and a card flip reveal animation.

## How is it used?
Enter your choices (one per line), select a mode (Spin Wheel or Card Flip), and click Randomize! The result is shown prominently with a history log of past decisions.

## Why is this useful?
Perfect for indecisive moments, group activities, or adding an element of chance to decision-making in a visually engaging way.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to randomize decisions.
