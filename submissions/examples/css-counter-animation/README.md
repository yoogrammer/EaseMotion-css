# CSS Counter Animation

## What does this do?
Uses CSS `counter()` / `counter-increment` to auto-number elements with hover animations.

## How is it used?
Add class `counter-step` inside a parent with `counter-reset`:

    <div class="counter-list">
      <div class="counter-step">Item</div>
    </div>

## Why is it useful?
Clean auto-numbered lists for steps, timelines, and ordered content without hardcoded numbers.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
