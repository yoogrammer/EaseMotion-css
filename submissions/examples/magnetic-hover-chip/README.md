# Magnetic Hover Chip

Issue: #ISSUE_NUMBER

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A reusable chip/tag component featuring magnetic hover movement, glow effects, glassmorphism styling, and status variants built entirely with HTML and CSS.

## Features

- Magnetic hover interaction
- Smooth lift animation
- Glow effects
- Glassmorphism styling
- Success variant
- Warning variant
- Danger variant
- Info variant
- Responsive layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<span class="magnetic-chip">
  Launch Ready
</span>
```

## Main Classes

```css
.magnetic-chip
.success
.warning
.danger
.info
```

## Why It Fits EaseMotion CSS

- Animation-first design
- Human-readable class names
- Reusable utility component
- Modern interaction pattern
- Lightweight implementation
