# Pulse Glow Animation

This example demonstrates the usage of the `ease-pulse-glow` animation utility.

## Description

The `ease-pulse-glow` utility applies a continuous, soft glowing pulse effect using CSS `box-shadow`. 

This micro-interaction is excellent for drawing a user's attention to primary call-to-action buttons, notification badges, or live recording indicators without being jarring.

## Usage

Include the `style.css` containing the animation, then apply the class to your HTML element.

```html
<!-- Primary Button -->
<button class="ease-pulse-glow">
  Subscribe Now
</button>

<!-- Notification Indicator -->
<div class="ease-pulse-glow" style="width: 12px; height: 12px; border-radius: 50%; background: blue;"></div>
```

## Preview

Open `demo.html` in your browser to see the effect applied to a button and a notification badge.
