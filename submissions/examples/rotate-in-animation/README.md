# rotate-in animation

## What does this do?
This adds a rotate-in animation that spins an element from 0° to 360° while fading and scaling it into view.

## How is it used?
Apply the class directly to any HTML element:

```html
<div class="rotate-in">
  Icon
</div>
```
```html
<div class="rotate-in-loader">
  Loading
</div>
```

## Why is it useful?
It makes UI elements look smooth and animated. It is useful for icons, loaders and decorative elements to improve visual appearance without using JavaScript.
