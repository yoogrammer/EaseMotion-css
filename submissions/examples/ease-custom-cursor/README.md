# Custom Cursor

## What does this do?
This feature allows you to visually replace the default browser cursor with a modern styled cursor effect like a glowing dot, ring, or hover-expanding indicator using pure CSS utilities.

## How is it used?

Apply the class directly to any HTML element:

```html
<body class="ease-cursor-none">

  <div class="ease-cursor ease-cursor-glow"></div>

  <section class="ease-center">
    Content Here
  </section>

</body>
```

## Why is it useful?
This utility improves UI experience by: Creating a modern, premium feel UI and Adding interactive hover feedback
