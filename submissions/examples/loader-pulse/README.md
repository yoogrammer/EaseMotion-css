# ease-loader-pulse

Pulsing opacity loader — a solid colored circle that fades in/out. A simpler alternative to spinning loaders.

## Usage

```html
<div class="ease-loader-pulse"></div>
```

## Variants

| Class | Description |
|---|---|
| `ease-loader-pulse` | Default pulsing loader |
| `ease-loader-pulse-sm` | Small variant |
| `ease-loader-pulse-lg` | Large variant |

## Notes

- Add inline size for custom dimensions
- Uses `--ease-color-primary` for fill color
- Respects `prefers-reduced-motion`
