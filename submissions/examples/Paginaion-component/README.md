# CSS-only Pagination Component

A fully CSS-only, lightweight, and highly polished pagination component for navigating between pages.

This feature includes:-

- Previous and next buttons
- Numbered page links
- Visually distinct active/current page state
- Disabled state for unavailable previous/next actions
- Smooth hover micro-interactions and keyboard `focus-visible` styles
- An ellipsis item for long page ranges
- A compact, mobile-friendly flexbox layout
- CSS Variables for effortless theming and dark mode compatibility

## Why is this useful for EaseMotion CSS?

Pagination is a core navigation pattern for any content-heavy interface, including blogs, data tables, documentation sites, product listings, and dynamic dashboards.

Adding this component ensures EaseMotion CSS can handle complex navigation structures right out of the box. Keeping with its philosophy of zero-dependency, animation-first, and highly readable CSS implementations, this component empowers developers to drop in beautiful, accessible pagination with zero Javascript overhead.
