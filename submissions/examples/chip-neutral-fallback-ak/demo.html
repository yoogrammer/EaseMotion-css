Chip Neutral Fallback Mismatch

Neutral color fallback values in the chip component mismatch actual framework design tokens.

Issue #5753

The Problem

When a chip uses var(--ease-chip-neutral-bg, #f3f4f6), the hardcoded fallback #f3f4f6 may not match the framework's actual --ease-color-neutral-100 value. If the variable is undefined, the chip shows an incorrect color.
