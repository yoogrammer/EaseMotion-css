# CSS-Only Glowing Holographic Card

A stunning, interactive holographic card component built completely with CSS. It features a sweeping multi-color glow effect, 3D transform on hover, and text styling that simulates holographic foil.

## Features
- ✨ **100% CSS-only**: No JavaScript required for the interactivity or the holographic glow.
- 🌈 **Holographic Glow Layer**: Uses animated complex `linear-gradient` with `mix-blend-mode: color-dodge` and `filter: hue-rotate` to create a realistic, dynamic foil effect.
- 📐 **3D Perspective**: Built with CSS 3D transforms (`rotateX`, `rotateY`, `translateZ`) to provide depth when hovering.
- 📱 **Responsive**: Easily scales and adapts for any layout.

## Files
- `demo.html`: The HTML structure of the card.
- `style.css`: The styling rules, gradients, and keyframe animations that power the holographic effect.

## Usage
Include the HTML in your project and link the CSS file. Hovering over the card will trigger the 3D rotation, the sweeping holographic gradient, and the animated gradient text effect.
