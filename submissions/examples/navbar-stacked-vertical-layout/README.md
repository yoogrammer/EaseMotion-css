# Navbar Stacked Vertical Layout

Issue: [#11468](https://github.com/SAPTARSHI-coder/EaseMotion-css/issues/11468)

Built for EaseMotion CSS — a lightweight CSS framework focused on reusable animations, interactive effects, and human-readable class naming.

## Overview

A reusable vertical stacked navigation layout featuring sidebar navigation, active states, hover interactions, dashboard content sections, and responsive behavior.

## Features

- Vertical sidebar navigation
- Stacked menu items
- Active navigation state
- Hover slide animation
- Icon and label support
- Dashboard layout
- Glassmorphism styling
- Responsive design
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<nav class="nav-menu">
  <a class="nav-item active">
    Dashboard
  </a>
</nav>
```

## Main Classes

```css
.sidebar
.nav-menu
.nav-item
.active
.content
.hero-card
```

## Why It Fits EaseMotion CSS

- Reusable layout pattern
- Human-readable class names
- Animation-first interactions
- Modern dashboard styling
- Lightweight implementation
