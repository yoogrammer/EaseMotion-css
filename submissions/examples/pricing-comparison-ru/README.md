# Pricing Comparison Component

## What does this do?

Provides a responsive pricing comparison section with multiple plans, feature comparisons, hover animations, and a highlighted recommended plan.

## How is it used?

```html
<div class="pricing-card">
    <h2>Starter</h2>
    <div class="price">$9/month</div>
</div>
```

## Why is it useful?

Pricing comparison sections are one of the most common UI patterns in SaaS products, startups, developer tools, and subscription platforms. This component provides a reusable, responsive, and visually engaging solution for displaying plans and features.
