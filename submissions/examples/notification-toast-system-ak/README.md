# Toast Notification System

## What does this do?
A toast notification system with success, error, info, and warning variants. Notifications auto-dismiss after 3 seconds with a progress bar.

## How is it used?
Open demo.html in a browser. Click any button to trigger a toast notification of that type. Each toast slides in and auto-dismisses.

## Why is this useful?
Demonstrates a non-intrusive notification pattern for providing user feedback, with timed auto-dismiss and visual progress indication.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser.
