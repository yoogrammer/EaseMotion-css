# Animated Weather Card

## What it does

Weather-themed cards with unique animations for each condition (sunny, rainy, cloudy, storm, snow, night).

## How to use it

<div class="ease-weather-card ease-weather-sunny">
    <div class="ease-weather-icon">☀️</div>
    <div class="ease-weather-temp">72°F</div>
    <div class="ease-weather-condition">Sunny</div>
</div>

## Weather Types

- ease-weather-sunny - Pulsing sun with glow
- ease-weather-rainy - Falling raindrops
- ease-weather-cloudy - Drifting cloud
- ease-weather-storm - Flashing lightning effect
- ease-weather-snow - Gentle falling snow
- ease-weather-night - Pulsing moon glow

## Why it fits EaseMotion CSS

- Pure CSS animations
- Unique animations per weather type
- Hover lift effect
- Beautiful gradient backgrounds
