# Ease Hover Sweep

## What does this do?
Full background color sweeps in from one side on hover.

## How is it used?

Add one of the sweep classes to any HTML element:

```html
<button class="sweep-btn sweep-left">Sweep Left</button>
<button class="sweep-btn sweep-right">Sweep Right</button>
<button class="sweep-btn sweep-top">Sweep Top</button>
```

## Why is it useful?
It provides a clean hover feedback pattern for buttons and CTA elements using only CSS, and fits EaseMotion CSS's animation-first, no-JavaScript approach.
