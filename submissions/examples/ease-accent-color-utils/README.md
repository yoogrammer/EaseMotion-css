# ease-accent-\* Color Utilities

Sets the `accent-color` on form elements (checkboxes, radios, switches, etc.) to a semantic palette color.

## Usage

```html
<input type="checkbox" class="ease-accent-primary" checked />
<input type="radio" class="ease-accent-success" name="opt" checked />
<input type="range" class="ease-accent-warning" />
```

## Classes

| Class                 | Accent Color |
| --------------------- | ------------ |
| `ease-accent-primary` | #6366f1      |
| `ease-accent-success` | #22c55e      |
| `ease-accent-danger`  | #ef4444      |
| `ease-accent-warning` | #f59e0b      |
| `ease-accent-info`    | #3b82f6      |
