# Student Productivity Workspace Dashboard

A responsive student productivity dashboard designed to help learners manage study progress, tasks, habits, deadlines, and achievements.

## Features

- Productivity overview
- Study progress tracking
- Habit tracker
- Weekly goals
- Task management
- Achievement badges
- Upcoming deadlines
- Responsive design
- EaseMotion CSS animation showcase
