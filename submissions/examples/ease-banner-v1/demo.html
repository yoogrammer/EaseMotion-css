🎯 ease-banner

Top-of-page banner/announcement component with slide-down animation

Banner Variants

  • Default - Purple gradient banner (first one)
  • Info - Blue banner (dismissible)
  • Success - Green banner
  • Warning - Yellow banner
  • Error - Red banner (dismissible)

Create Your Own Banner

Usage

<!-- Basic banner -->
<div class="ease-banner">
    <div class="ease-banner-content">
        <span>🎉</span>
        <p>Your message here</p>
    </div>
</div>

<!-- Dismissible banner -->
<div class="ease-banner ease-banner-info">
    <div class="ease-banner-content">
        <span>ℹ️</span>
        <p>Info message</p>
    </div>
    <button class="ease-banner-close" onclick="this.parentElement.remove()">✕</button>
</div>

<!-- Color variants -->
<div class="ease-banner ease-banner-success">...</div>
<div class="ease-banner ease-banner-warning">...</div>
<div class="ease-banner ease-banner-error">...</div>
