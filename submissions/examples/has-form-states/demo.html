:has() Parent-Aware Form Validation

Groups use :has(:user-invalid) on the fieldset to show group-level error states. Animated validation feedback appears as you type.

How it works: Type invalid data (e.g. single-letter name, non-numeric ZIP) and see the group highlight using :has(:user-invalid). The :user-invalid pseudo-class fires after the user has interacted with the field.
