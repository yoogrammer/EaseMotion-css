# Email Subscribe Card

A subscription card with animated input focus and a submit button featuring loading/success states.

## Features

- Card entrance animation on page load
- Input border expands and glows on focus
- Button transitions through loading (spinner) and success (checkmark) states

## CSS Custom Properties

| Property | Default | Description |
|---|---|---|
| `--sub-duration` | `0.35s` | Transition duration for input/border animations |
| `--sub-card-bg` | `#ffffff` | Card background color |
| `--sub-input-border` | `#d1d5db` | Input default border color |
| `--sub-focus-color` | `#6366f1` | Input focus border / glow color |
| `--sub-btn-bg` | `#6366f1` | Submit button background |
| `--sub-btn-color` | `#ffffff` | Submit button text color |
| `--sub-radius` | `12px` | Border radius for card, input, and button |
