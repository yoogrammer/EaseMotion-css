Sidebar Mobile Animation Fix

Issue #8478 — display:none kills the transform transition on mobile

How to test: Click Open in each demo below. The broken sidebar snaps in instantly. The fixed sidebar slides in with the spring animation. You can also resize the browser to a narrow width or use Chrome DevTools → Device Toolbar to simulate mobile.
Broken Current behaviour — display:none → display:block (no animation)
EaseMotion
Dashboard Components Animations Settings

The sidebar uses display:none for the hidden state. When opened, the element is inserted into the render tree in one frame — straight to its final position. The transition: transform 0.6s defined in sidebar.css never fires because there is no "before" state to animate from. The sidebar just snaps in.

Fixed Proposed fix — visibility:hidden → visibility:visible (spring animation plays)
EaseMotion
Dashboard Components Animations Settings

The sidebar uses visibility:hidden for the hidden state. The element stays in the render tree, so the browser can compute its starting transform: translateX(-100%). When opened, the transition fires and the sidebar slides in with the cubic-bezier(0.34, 1.56, 0.64, 1) spring easing. pointer-events:none prevents the hidden sidebar from intercepting taps while off-screen.

The fix — mobile breakpoint in sidebar.css
/* mobile breakpoint */
@media (max-width: 768px) {
  .ease-sidebar {
    position: fixed; top: 0; left: 0;
    height: 100vh; z-index: 100;
    width: var(--ease-sidebar-width, 260px);

-   display: none;            /* removes element from render tree */
+   visibility: hidden;        /* keeps element in render tree     */
+   pointer-events: none;      /* prevents blocking touch events    */

    transform: translateX(-100%);
    transition:
      transform 0.6s cubic-bezier(0.34, 1.56, 0.64, 1),
+     visibility 0.6s,           /* instant-on, delayed-off           */
      box-shadow 0.3s ease;
  }

  .ease-sidebar-layout.ease-sidebar-open .ease-sidebar {
-   display: block;
+   visibility: visible;
+   pointer-events: auto;
    transform: translateX(0);
    box-shadow: 12px 0 36px rgba(0, 0, 0, 0.15);
  }
}
