# ease-zoom-out

Zoom-out entrance animation from scale(1.5) to scale(1). Creates a dramatic reveal effect for content.

## Usage

```html
<div class="ease-zoom-out">Zooms into view</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-zoom-out` | Default zoom-out entrance |
| `ease-zoom-out-sm` | Subtle zoom (1.2x) |
| `ease-zoom-out-lg` | Strong zoom (2x) |

## Notes

- Starts zoomed in and settles to normal size
- Great for hero text, modal content, and reveal animations
- Respects `prefers-reduced-motion`
