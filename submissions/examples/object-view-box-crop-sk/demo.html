CSS Image Cropping

Demonstrating the power of the object-view-box CSS property. This allows you to crop, zoom, and pan an image natively in CSS without modifying the source file or using complex wrapper hacks.

Original Image

The full, unmodified source image using standard object-fit: cover.

Zoom In

inset(10%) crops 10% from all sides, effectively zooming in without changing the container size.

Top Half Crop

inset(0% 0% 50% 0%) crops the bottom 50%, focusing entirely on the top half of the image.

Interactive Crop

Hover to animate! object-view-box is animatable, allowing for smooth pan and zoom effects on interaction.

Browser Support Note: object-view-box is a modern CSS feature (supported in Chrome 104+, Edge 104+). In unsupported browsers, the images will gracefully fall back to displaying the full original image via object-fit: cover.
