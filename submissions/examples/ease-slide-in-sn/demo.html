CSS Only - No JavaScript - Smooth Transforms

Slide-In Animation Utilities

Slide-in-left and slide-in-right animations for notification panels, sidebars, and content reveals.

Slide-In Left

.animate-slide-in-left slides element from left to its natural position. Refresh to replay.

←
.animate-slide-in-left Default (100px, 0.5s)
←
.animate-slide-in-left.slide-fast Fast (0.2s)
←
.animate-slide-in-left.slide-slow Slow (1s)
←
.animate-slide-in-left.slide-far Far (200px)

Slide-In Right

.animate-slide-in-right slides element from right to its natural position.

→
.animate-slide-in-right Default (100px, 0.5s)
→
.animate-slide-in-right.slide-fast Fast (0.2s)
→
.animate-slide-in-right.slide-slow Slow (1s)
→
.animate-slide-in-right.slide-far Far (200px)

CSS Custom Properties

--slide-duration

Default: 0.5s

Controls the slide-in animation speed.

--slide-distance

Default: 100px

Controls how far the element slides from its starting position.

Use Cases

Notification Panels

Slide notifications in from the right side of the screen for a modern alert experience.

Sidebar Navigation

Slide sidebars in from the left for mobile navigation drawers and off-canvas menus.

Content Reveals

Stagger slide-in animations for content sections as they enter the viewport.

Hero Sections

Slide hero text and CTAs from the left or right for engaging landing page entrances.

Accessibility

All animations respect prefers-reduced-motion. Slide distance is reduced on mobile for better UX.
