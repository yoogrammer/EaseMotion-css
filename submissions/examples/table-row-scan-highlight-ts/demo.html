Usage

Add class="scan-row" and tabindex="0" to any <tr>. Wrap the table in class="scan-table-wrapper" for responsive overflow. Use scan-row-success, scan-row-warning, scan-row-danger, or scan-row-neutral to change the highlight color. Speed can be adjusted with scan-row-fast or scan-row-slow.
