EaseMotion Framework Installation Guide

Installation and usage examples for popular frameworks.

React

Install

npm install easemotion-css

Usage

import "easemotion-css";

export default function App() {
  return (
    <div className="ease-fade-in">
      Hello React
    </div>
  );
}
