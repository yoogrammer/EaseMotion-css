# Scale Card Entrance Animation

## What does this do?
Animates a card component into view with a smooth scale-up entrance effect with a subtle spring bounce.

## How is it used?
Add the class to any card element:

```html
<div class="scale-card-entrance">
  Your card content here
</div>
```

## Why is it useful?
Card entrance animations improve perceived responsiveness and visual polish.
This animation fits EaseMotion CSS's philosophy of lightweight, reusable utility 
classes — pure CSS, no JS required, with full `prefers-reduced-motion` support.
