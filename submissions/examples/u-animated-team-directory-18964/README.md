# Animated Team Directory

A responsive team member directory component with animated profile cards, status indicators, and role badges.

## Features

- Animated card reveal effects
- Online, away, and offline status indicators
- Hover lift interactions
- Responsive grid layout
- Role badges
- Modern dashboard-style UI

## Files

- demo.html
- style.css

## Usage

Open `demo.html` in a browser.

## Customization

You can customize:

- Member information
- Role colors
- Status indicators
- Animation speed
- Card styling

## Use Cases

- Team pages
- Contributor showcases
- Company directories
- Community member listings
- Dashboard user management
