## What does this do?

Provides `ease-object-fit` and `ease-object-position` utility classes for controlling how replaced elements (images, videos) fit within their container.

## How is it used?

```html
<img class="ease-object-cover-za" src="..." />
<img class="ease-object-contain-za" src="..." />
<img class="ease-object-top-za" src="..." />
```

## Why is it useful?

EaseMotion core has no object-fit or object-position utilities. These are essential for responsive images, video backgrounds, avatars, and any scenario where replaced elements need consistent sizing without distorting aspect ratio.
