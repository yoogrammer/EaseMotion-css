# Interactive Pricing Card Toggle

A responsive pricing card component with animated monthly/yearly billing toggle, featured plan highlighting, and hover effects.

## Features

- Monthly and yearly pricing switch
- Three pricing tiers
- Featured plan badge
- Smooth hover animations
- Responsive layout
- Pure HTML, CSS and minimal JavaScript

## Files

- demo.html
- style.css
- README.md

## Usage

Open `demo.html` in any browser.
