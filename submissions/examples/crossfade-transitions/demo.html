CSS crossfade() Transitions

Smooth image crossfading gallery with no JavaScript

How it works

background-image: crossfade(url(a), url(b), percentage) blends two images. The percentage controls which image is dominant. CSS animation cycles through the sequence by changing the percentage and swapping image URLs via custom properties.

0% (first) 30% 60% 100% (second)
