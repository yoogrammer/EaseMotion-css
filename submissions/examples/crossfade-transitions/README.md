# crossfade-transitions

Showcases CSS `crossfade()` for smooth image-to-image transitions in a gallery layout without JavaScript.

## How It Works

Four images crossfade cyclically using `crossfade()` as `background-image`. CSS animations control the fade timing and sequence. Each image fades in, holds, then fades out. The `crossfade()` function blends two images at configurable percentages.

## Usage

Open `demo.html` in a supporting browser (Chrome/Edge). Placeholder gradient swatches represent images — swap in real URLs via CSS custom properties. Reduce-motion query freezes the current frame.
