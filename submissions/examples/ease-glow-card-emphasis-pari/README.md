## ease-glow-card-emphasis

**What does this do?**
Animates card components with a glowing shadow effect during hover/emphasis interactions.

**How is it used?**
<div class="glow-card glow-purple">Content</div>
<div class="glow-card glow-green">Content</div>
<div class="glow-card glow-orange">Content</div>
<div class="glow-card glow-pink">Content</div>

**Why is it useful?**
Adds vibrant, color-coded glow emphasis to cards using pure CSS keyframes and transitions,
making UI components feel interactive and alive — consistent with EaseMotion CSS's emphasis animation philosophy.
