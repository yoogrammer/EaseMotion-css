## What does this do?

This component provides a file upload area with drag-and-drop support, click-to-browse fallback, and image preview showing the filename and file size after selection.

## How is it used?

Include the `style.css` and the HTML structure. Users can drag an image onto the drop zone or click to open a file picker. Once selected, the image is previewed with metadata, and a Remove button clears the selection.

## Why is this useful?

Drag-and-drop file upload with instant preview gives users immediate visual feedback, improving the upload experience in forms, profile editors, and media galleries.

## Tech Stack

- HTML
- CSS
- JavaScript (vanilla)
