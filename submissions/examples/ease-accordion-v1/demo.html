EaseMotion CSS is a utility-first CSS framework focused on animations and motion design. It provides pre-built classes for common UI patterns, letting you "write UI like you describe it in English."
You can either download the source, use the CDN link, or install via npm: npm install easemotion-css. All styles are pure CSS with no JavaScript dependencies.
