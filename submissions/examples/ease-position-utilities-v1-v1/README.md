# Position Utilities

## What does this do?

Provides reusable utility classes for common CSS position values.

## How is it used?

```html
<div class="position-static">Static element</div>
<div class="position-relative">Relative element</div>
<div class="position-absolute">Absolute element</div>
<div class="position-fixed">Fixed element</div>
<div class="position-sticky">Sticky element</div>
```

## Why is it useful?

Position utilities help developers quickly control element positioning using clear and readable class names. They are useful for layouts, overlays, sticky headers, fixed badges, and absolutely positioned UI elements.
