# Animated Border Pulse

## What it does

Creates a pulsing border animation for loading states, notifications, or attention-grabbing elements.

## How to use it

<div class="ease-border-pulse">Content</div>

## Variants

- ease-border-fast - 0.8s cycle
- ease-border-slow - 3s cycle
- ease-border-blue - Blue color theme
- ease-border-green - Green success theme
- ease-border-rainbow - Cycling rainbow colors
- ease-border-thick - 3px border width

## Why it fits EaseMotion CSS

- Pure CSS animation
- Multiple color variants
- Speed control options
- Perfect for loading states
