ease-will-change utilities

Hint the browser about upcoming animations for smoother rendering.

ease-will-change-transform-za

Use on elements that will be transformed (translate, scale, rotate, etc.). Promotes to a compositor layer for GPU-accelerated movement.

ease-will-change-opacity-za

Use on elements that will fade in/out. Opacity changes are GPU-friendly with will-change.

ease-will-change-scroll-position-za

Use on scroll containers where scroll position will change, like sticky headers or scroll-driven animations.

ease-will-change-contents-za

Use on containers whose content will change (e.g., dynamic lists, updating text). Tells the browser to offload rendering.

ease-will-change-auto-za

Resets will-change to default — the browser manages layer promotion on its own.
