# Animated Flip Card

## What it does

3D flip card that rotates on hover to reveal back content. Perfect for product cards, team profiles, or interactive elements.

## How to use it

<div class="ease-flip-card">
    <div class="ease-flip-card-inner">
        <div class="ease-flip-front">Front</div>
        <div class="ease-flip-back">Back</div>
    </div>
</div>

## Variants

- ease-flip-fast - 0.4s flip speed
- ease-flip-slow - 0.8s flip speed
- ease-flip-vertical - Flips vertically (X-axis)
- ease-flip-glow - Neon glow border
- ease-flip-click - Click to flip (not hover)

## Why it fits EaseMotion CSS

- Pure CSS 3D transforms
- Multiple flip directions
- Speed control
- Glass morphism styling
