## Overview

A retro vaporwave-inspired background featuring a glowing striped sunset, neon perspective grid, animated stars, synthwave typography, and futuristic gradients using pure HTML and CSS.

## Features

- Vaporwave sunset
- Striped sun effect
- Neon horizon grid
- Animated glow effects
- Floating stars
- Synthwave typography
- Responsive layout
- Pure HTML and CSS
- No JavaScript required

## Usage

```html
<div class="sun">
  <div class="sun-stripes"></div>
</div>
```
