# Animation: Expandable Feature Cards

A card expansion interaction that reveals additional content when hovered. The card smoothly grows in height while hidden content fades and slides into view.

## Features

- Pure CSS interaction
- Smooth card expansion
- Slide-up content reveal
- Modern SaaS-style design
- Responsive layout
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="feature-card">
    <h3>Performance</h3>

    <p class="hidden-content">
        Optimized animations and lightweight styles.
    </p>
</div>
```

Hovering the card expands its height and reveals additional information.

## Why is it useful?

Expandable cards allow interfaces to present more information without overwhelming the user. They work particularly well for feature showcases, pricing pages, dashboards, and product landing pages while maintaining a clean visual hierarchy.
