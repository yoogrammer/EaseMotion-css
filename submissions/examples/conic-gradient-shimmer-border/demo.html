How it works under the hood

Traditional CSS gradients are static because the browser does not know how to interpolate color angles. By using CSS Houdini @property, we register --shimmer-angle so browser layout engines can interpolate the rotation value on the GPU compositor thread, bypassing layout repaints entirely. 

/* 1. Register the custom angle property */
@property --shimmer-angle {
  syntax: '<angle>';
  inherits: false;
  initial-value: 0deg;
}

/* 2. Apply dynamic conic-gradient background */
.ease-shimmer-card {
  background: conic-gradient(from var(--shimmer-angle), transparent 50%, var(--primary) 90%, var(--accent) 100%);
  animation: ease-shimmer-rotate 3s linear infinite;
}

/* 3. Rotate via keyframes */
@keyframes ease-shimmer-rotate {
  0%   { --shimmer-angle: 0deg; }
  100% { --shimmer-angle: 360deg; }
}
