# Animated 3D Perspective Card

**What does this do?**  
Creates a 3D tilt card that rotates in response to hover position using CSS perspective transforms, with a shine sweep effect.

**How is it used?**  
Wrap any element in `.perspective-wrap` with `perspective: 1000px`. Add `.perspective-card` for the 3D transform target and `.card-shine` for the light sweep overlay.

**Why is it useful?**  
Adds an immersive, interactive 3D card effect ideal for product showcases, feature highlights, or pricing cards without JavaScript.
