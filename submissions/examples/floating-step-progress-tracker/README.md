# Animation: Floating Step Progress Tracker

A multi-step progress tracker that uses floating milestone animations and content reveals to visually communicate workflow progress.

## Features

- Pure CSS interaction
- Floating step animation
- Hover-based information reveal
- Gradient progress connector
- Responsive layout
- Reduced motion support
- No JavaScript required

## Usage

```html
<div class="step">

    <div class="step-circle">1</div>

    <h3>Planning</h3>

    <div class="step-info">
        Define goals and requirements.
    </div>

</div>
