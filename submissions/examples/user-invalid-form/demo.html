CSS :user-invalid / :user-valid

Form validation that appears only after user interaction

Please enter at least 2 characters.
Please enter a valid email address.
Message must be at least 10 characters.

Note: :user-invalid and :user-valid are supported in Chromium 125+ and Firefox 88+. Unlike :invalid/:valid, these pseudo-classes only apply after the user has interacted with the field — preventing the "red flash" on page load for required fields.
