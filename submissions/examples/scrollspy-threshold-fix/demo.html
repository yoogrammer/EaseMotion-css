Utilities (tall section)

This section is intentionally very tall — just like the real Utilities section in docs/index.html.

With threshold: 0.4 (buggy), 40% of this section's height must be visible before the sidebar link activates. On a normal screen, that 40% threshold is never reached, so the active state never fires.

Switch to AFTER (fixed) above and scroll — the sidebar link activates immediately when the section enters the top of the viewport.

Animations (tall section)

Another tall section. Same problem occurs here in buggy mode.

The fix uses threshold: 0.1 + rootMargin: '-10% 0px -80% 0px' so the link activates as soon as the top of the section enters the upper portion of the viewport.
