# Voting Poll Creator

## What does this do?
An interactive poll creation tool that lets users create custom polls with multiple options, cast votes, and see live results with animated percentage bars.

## How is it used?
Enter a poll question, add/remove options, then start the poll. Click any option to vote. Results update in real-time with a highlighted leader.

## Why is this useful?
Perfect for quick team decisions, classroom activities, or social gatherings where you need instant feedback with visual results.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to start creating polls.
