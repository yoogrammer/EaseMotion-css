# Kanban Board

## What
A kanban task board with three columns (To Do, In Progress, Done), task creation, and column-to-column movement via buttons.

## How
Uses a JavaScript object to store tasks per column. Buttons trigger `moveTask` to transfer tasks between columns, and `render` re-renders the DOM from the data model.

## Why
Showcases a practical project management interface with clear column separation and intuitive task movement.
