# Distributed System Communication Map Component

## What does this do?

Visualizes communication between frontend applications, gateways, services, infrastructure components, and databases within a distributed system.

## How is it used?

```html
<div class="node gateway">
    API Gateway
</div>

<div class="node service">
    Auth Service
</div>
```

## Why is it useful?

Distributed systems often involve many interconnected services and infrastructure components. This component provides a clean architecture-style visualization for documentation, engineering blogs, technical portfolios, system design presentations, and educational content using pure HTML and CSS.
