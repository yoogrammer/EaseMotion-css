# ease-content-\* Align-Content Utilities

Controls the `align-content` property on multi-line flex containers and grid layouts.

## Usage

```html
<div class="ease-flex ease-flex-wrap ease-content-center" style="height: 300px">
  <div>Item 1</div>
  <div>Item 2</div>
  <div>Item 3</div>
</div>
```

## Classes

| Class                  | CSS Property                 |
| ---------------------- | ---------------------------- |
| `ease-content-start`   | align-content: flex-start    |
| `ease-content-center`  | align-content: center        |
| `ease-content-end`     | align-content: flex-end      |
| `ease-content-between` | align-content: space-between |
| `ease-content-around`  | align-content: space-around  |
| `ease-content-evenly`  | align-content: space-evenly  |
