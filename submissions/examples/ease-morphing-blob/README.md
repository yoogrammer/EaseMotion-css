# ⚡ Ease Morphing Blob Animation Component

A 100% pure CSS, endlessly mutating organic shape component for EaseMotion CSS designed to provide abstract, living backdrops or creative layout focal points.

## ✨ What it does
Utilizes complex eight-value fractional `border-radius` parsing configurations combined with an infinite keyframe loop to interpolate smooth, flowing geometric transitions completely natively—with zero JavaScript overhead or main-thread rendering tax.

## 🚀 How to Use
Drop any combination of sizing and color gradient utility modifiers onto your markup target node elements:

```html
<div class="ease-blob ease-blob--indigo ease-blob--lg"></div>

<div class="ease-blob ease-blob--rose ease-blob--md"></div>

<div class="ease-blob ease-blob--emerald ease-blob--sm"></div>
