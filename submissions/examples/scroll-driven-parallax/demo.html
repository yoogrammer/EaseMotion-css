Behind the Scenes

Each layer uses a scroll() animation timeline tied to the nearest scroll container. The sky layer scrolls slowly, mountains at a medium pace, and the foreground at full speed, creating a depth illusion.

This uses the CSS Scroll-Driven Animations specification. The animation-range property defines which scroll range the animation covers, and each layer's animation-duration (in terms of scroll distance) is set to auto via the timeline.

.layer-sky { animation-timeline: scroll(); animation-range: 0 100vh; }
.layer-mountains { animation-timeline: scroll(); animation-range: 0 100vh; }
.layer-foreground { animation-timeline: scroll(); animation-range: 0 100vh; }

When prefers-reduced-motion: reduce is active, all parallax animations are disabled and the hero content stays fixed.
