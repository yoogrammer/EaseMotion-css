Translate Utilities

Atomic translation utility classes for shifting elements along the horizontal (X), vertical (Y), or both axes simultaneously. Hover over cards to see movements.

Horizontal Translate (ease-translate-x-*)

X
.ease-translate-x-4

Translate X (1rem)

Translates the element 1rem (16px) to the right on the horizontal axis.

X
.ease-translate-x-12

Translate X (3rem)

Translates the element 3rem (48px) to the right on the horizontal axis.

X
.ease-translate-x-full

Translate X Full (100%)

Translates the element by 100% of its own width to the right.

X
.ease-translate-x-neg-4

Translate X Negative (-1rem)

Translates the element 1rem (16px) to the left on the horizontal axis.

X
.ease-translate-x-neg-full

Translate X Negative Full (-100%)

Translates the element by 100% of its own width to the left.

Vertical Translate (ease-translate-y-*)

Y
.ease-translate-y-4

Translate Y (1rem)

Translates the element 1rem (16px) downwards on the vertical axis.

Y
.ease-translate-y-12

Translate Y (3rem)

Translates the element 3rem (48px) downwards on the vertical axis.

Y
.ease-translate-y-full

Translate Y Full (100%)

Translates the element by 100% of its own height downwards.

Y
.ease-translate-y-neg-4

Translate Y Negative (-1rem)

Translates the element 1rem (16px) upwards on the vertical axis.

Y
.ease-translate-y-neg-full

Translate Y Negative Full (-100%)

Translates the element by 100% of its own height upwards.

Diagonal Translate Shorthands (ease-translate-*)

XY
.ease-translate-half

Translate Diagonal (50%)

Translates 50% along both X and Y axes, moving the element diagonally right and down.

XY
.ease-translate-full

Translate Diagonal Full (100%)

Translates 100% along both X and Y axes, moving the element diagonally right and down.

XY
.ease-translate-neg-half

Translate Diagonal Neg (-50%)

Translates -50% along both X and Y axes, moving the element diagonally left and up.

XY
.ease-translate-neg-full

Translate Diagonal Neg Full (-100%)

Translates -100% along both X and Y axes, moving the element diagonally left and up.

Developer Guide

Apply the translate classes together with transitions or hover states to easily move your components around: 

<div class="ease-relative ease-overflow-hidden">
  <div 
    class="box ease-translate-x-full ease-translate-y-neg-4" 
  >
    Centered Element
  </div>
</div>
