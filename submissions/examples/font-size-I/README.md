 
# Font Size Scale Utilities

Responsive font-size utilities using CSS clamp() — xs, sm, base, lg, xl, 2xl, 4xl.

## Files

- demo.html - Interactive demo
- style.css - Font size styles
- README.md - Documentation

## Classes

| Class | Default Size | Range |
|-------|--------------|-------|
| `ease-text-xs` | 0.75rem | 0.7rem - 0.875rem |
| `ease-text-sm` | 0.875rem | 0.8rem - 1rem |
| `ease-text-base` | 1rem | 0.9rem - 1.125rem |
| `ease-text-lg` | 1.125rem | 1rem - 1.25rem |
| `ease-text-xl` | 1.25rem | 1.125rem - 1.5rem |
| `ease-text-2xl` | 1.5rem | 1.25rem - 2rem |
| `ease-text-4xl` | 2rem | 1.75rem - 3rem |

## Usage

```html
<p class="ease-text-xs">Extra small text</p>
<p class="ease-text-sm">Small text</p>
<p class="ease-text-base">Base text</p>
<p class="ease-text-lg">Large text</p>
<p class="ease-text-xl">Extra large text</p>
