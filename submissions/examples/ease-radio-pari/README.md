## ease-radio

**What does this do?**
Custom styled radio buttons with animated fill on selection and a card variant.

**How is it used?**
<div class="ease-radio-group">
  <label class="ease-radio">
    <input type="radio" name="group" />
    <span class="radio-circle"></span>
    Option
  </label>
</div>

Color variants: add class `green`, `orange`, or `pink` to `.ease-radio`
Card variant: add class `card-radio` to `.ease-radio`
Horizontal layout: add class `horizontal` to `.ease-radio-group`

**Why is it useful?**
Provides fully accessible, animated custom radio buttons using pure CSS —
consistent with EaseMotion CSS's lightweight component philosophy.
