# Filter Blur Transition

## What does this do?
Images transition between blurred and clear states on hover using CSS filter: blur().

## How is it used?
Add class `filter-image` to any element:

    <div class="filter-image"></div>

## Why is it useful?
Creative image reveals, loading states, and privacy blur effects.

## Tech Stack
- HTML
- CSS (no frameworks, no JavaScript)

## Preview
Open demo.html directly in your browser to see the effect.
