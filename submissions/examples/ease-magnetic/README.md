# Ease Magnetic

A magnetic hover interaction where the button subtly pulls toward the cursor using CSS variables and minimal JavaScript.

## Features

- Cursor-follow magnetic effect
- Smooth snap-back animation
- CSS-driven transforms
- Minimal JS (only updates CSS variables)

## Files

- `demo.html`
- `style.css`

## Concepts Used

- CSS custom properties
- `transform: translate()`
- Mouse position tracking
- Smooth transitions
