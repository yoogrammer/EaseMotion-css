# Flip Image Emphasis

A lightweight hover animation that applies a subtle 3D flip and scale effect to images.

## Usage

```html
<img
  src="image.jpg"
  class="flip-image-emphasis"
  alt="Demo"
/>
```

## Integration Note

The class name `.flip-image-emphasis` does not follow the `ease-*` naming convention required for framework integration. This class must be renamed to `.ease-flip-image-emphasis` (or a shorter approved name) in both `style.css` and `demo.html` before this submission can be promoted to core.
