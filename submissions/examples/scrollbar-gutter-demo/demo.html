CSS scrollbar-gutter: stable

Preventing layout shift when scrollbars appear

scrollbar-gutter: stable

This panel uses scrollbar-gutter: stable. The gutter space is always reserved, so content never shifts when a scrollbar appears.

default (auto)

This panel uses the default scrollbar-gutter: auto. Watch the content shift left/right when the scrollbar appears or disappears.

Try it: Click "Add line" to add content until a scrollbar appears. Watch the default panel shift its content inward when the scrollbar appears — the stable panel stays put.
