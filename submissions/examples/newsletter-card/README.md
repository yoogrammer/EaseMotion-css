# Newsletter/Email Digest Card Component

A self-contained user experience module engineered for managing content presentation layouts, inline preference controls, and collapsible vertical article slots.

## Modular States
- **Collapsible Context Drawers:** Transitions content using combined max-height boundaries safely.
- **Scale Action Unsubscribe Filters:** Micro-interaction state modifiers providing immediate verification hints on execution.
- **Flexible Structural Metadata Labels:** Absolute tracking labels staying out of the way of underlying flex grids.

## Usage Layout Structure
```html
<div class="ease-newsletter-card">
  <div class="ease-digest-header"> ... </div>
  <div class="ease-digest-full-content"> ... </div>
</div>
```
