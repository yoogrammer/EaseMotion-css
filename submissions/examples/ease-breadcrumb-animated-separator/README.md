# ease-breadcrumb Animated Separator

Adds animated separator icons to breadcrumb navigation.

## Features

- Animated chevron separator
- Smooth scale effect
- Rotation transition
- Pure CSS
- Lightweight and reusable

## Usage

```html
<nav class="ease-breadcrumb">
    <a href="#">Home</a>
    <span class="separator">›</span>
    <a href="#">Products</a>
</nav>
```

## Why it fits EaseMotion CSS

This enhancement introduces subtle motion to an otherwise static UI pattern. The separator animation provides visual feedback while maintaining readability and simplicity.
