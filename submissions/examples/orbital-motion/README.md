# Orbital Motion Animation Utility

## Overview

Orbital Motion Animation Utility introduces orbit-style motion effects that simulate an element rotating around a central point.

The utility includes three animation variants:

- ease-orbit-spin
- ease-orbit-reverse
- ease-orbit-fast

---

## Features

- Pure CSS implementation
- Lightweight and reusable
- Modern motion effect
- Works with cards, icons, loaders, and UI elements
- Interactive preview and copy functionality

---

## Usage

### Orbit Spin

```html
<div class="ease-orbit-spin"></div>
```

### Orbit Reverse

```html
<div class="ease-orbit-reverse"></div>
```

### Orbit Fast

```html
<div class="ease-orbit-fast"></div>
```

---

## Included Variants

| Class | Description |
|---------|------------|
| ease-orbit-spin | Standard orbital rotation |
| ease-orbit-reverse | Reverse orbital rotation |
| ease-orbit-fast | Fast orbital rotation |
