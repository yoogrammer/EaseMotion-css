# ease-glow-shadow-exit

Glow exit animation where the box-shadow glow dissipates as the element fades out. A unique glowing dismiss effect.

## Usage

```html
<div class="ease-glow-shadow-exit">Glow fades away</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-glow-shadow-exit` | Default glow shadow exit |

## Notes

- Box-shadow glow dissipates from 12px to 0px
- Element fades out simultaneously
- Great for glowing buttons, cards, and emphasis elements
- Respects `prefers-reduced-motion`
