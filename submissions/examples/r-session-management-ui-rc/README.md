# Session Management UI Showcase

## What does this do?

Provides a modern interface for managing active sessions, reviewing login history, and controlling device access.

## Features

- Active session cards
- Current device indicator
- Revoke session actions
- Login activity timeline
- Responsive design
- Security-focused UI patterns

## How is it used?

Open `demo.html` directly in a browser.

## Why is it useful?

It demonstrates a common account-security interface used in modern web applications while showcasing smooth card interactions and timeline animations.
