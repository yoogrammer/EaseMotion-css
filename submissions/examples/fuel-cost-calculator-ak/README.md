# Fuel Cost Calculator

## What does this do?
A fuel cost calculator that computes total trip cost, fuel needed, and cost per km/mile with dual-unit efficiency support (MPG or L/100km) and optional two-vehicle comparison mode.

## How is it used?
Enter distance, fuel efficiency, and fuel price. Toggle between MPG and L/100km. Enable two-vehicle comparison to compare costs side by side.

## Why is this useful?
This helps drivers plan trips and compare vehicle fuel economy to make cost-effective travel decisions.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to use the calculator.
