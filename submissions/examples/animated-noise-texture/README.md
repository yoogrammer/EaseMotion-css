# Animated Noise Texture

## What it does

Real-time film grain / static noise overlay effect using Canvas API. Creates authentic analog texture.

## How to use it

<canvas id="noiseCanvas" class="ease-noise-canvas"></canvas>

## Features

- Real-time noise generation
- Adjustable intensity (Low/Medium/High)
- Toggle on/off
- Mix-blend-mode overlay
- 60fps performance

## Variants

- ease-noise-high - 0.6 opacity
- ease-noise-low - 0.15 opacity

## Why it fits EaseMotion CSS

- Canvas-based real-time effect
- Adjustable intensity
- Retro aesthetic
- Performance optimized
