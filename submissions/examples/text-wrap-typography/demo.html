text-wrap: pretty — Paragraph Optimization

Default

CSS text wrapping controls how inline content breaks across lines. The default algorithm wraps text greedily, breaking at the earliest opportunity. This can leave a single word dangling on the last line — an orphan. In long paragraphs, a few extra words on the last line can significantly improve readability and visual balance.

text-wrap: stable — Content Stability

With text-wrap: stable, the browser avoids re-wrapping text when content changes elsewhere on the page. This prevents layout shifts when toggling visibility of elements. Try resizing the viewport — the text maintains stable line breaks.
