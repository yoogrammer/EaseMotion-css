# Z-Index Utilities

## What does this do?

Provides reusable utility classes for controlling stacking order using z-index values.

## How is it used?

```html
<div class="z-10">Content</div>
<div class="z-30">Content</div>
<div class="z-50">Content</div>
```

## Why is it useful?

Z-index utilities make it easier to manage overlapping elements with clear and readable class names while maintaining reusable styling patterns.
