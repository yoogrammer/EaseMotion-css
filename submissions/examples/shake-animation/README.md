# Shake Animation Demo

## What does this do?

Provides reusable horizontal and vertical shake animations that can be used to draw user attention to important UI elements.

---

## How is it used?

Horizontal shake:

```html
<div class="shake-horizontal">
    Content
</div>
```

Vertical shake:

```html
<div class="shake-vertical">
    Content
</div>
```

---

## Why is it useful?

Shake animations are commonly used to:

- Indicate validation errors
- Highlight warnings
- Draw attention to notifications
- Emphasize failed actions
- Improve visual feedback

---

## Features

- Horizontal shake animation
- Vertical shake animation
- Form validation example
- Notification example

- Accessibility support
