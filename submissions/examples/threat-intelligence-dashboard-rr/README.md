# Cybersecurity Threat Intelligence Dashboard Component

## What does this do?

Provides a cybersecurity-focused dashboard for monitoring risk scores, threat feeds, indicators of compromise (IOCs), threat actors, and attack source activity.

## How is it used?

```html
<div class="card risk">
    <h2>82</h2>
    <span>Risk Score</span>
</div>
```

## Why is it useful?

Threat intelligence platforms help security teams proactively identify cyber threats and monitor threat actor activity. This component provides a modern cybersecurity dashboard built entirely with HTML and CSS.
