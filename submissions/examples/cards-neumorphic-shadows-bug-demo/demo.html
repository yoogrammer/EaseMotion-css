Bug: hardcoded rgba shadow values

Neumorphic card uses hardcoded rgba(15, 23, 42, 0.12) / rgba(255, 255, 255, 0.78) instead of CSS variables.

Neumorphic Card

Shadows are hardcoded.

Source

.ease-card-neumorphic {
  box-shadow:
    8px 8px 20px rgba(15, 23, 42, 0.12),
    -8px -8px 20px rgba(255, 255, 255, 0.78);
}

Fix

.ease-card-neumorphic {
  box-shadow:
    var(--ease-shadow-neumorphic-dark, 8px 8px 20px rgba(15, 23, 42, 0.12)),
    var(--ease-shadow-neumorphic-light, -8px -8px 20px rgba(255, 255, 255, 0.78));
}
