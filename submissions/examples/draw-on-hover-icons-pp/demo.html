Technical Specifications

  • pathLength="100" Standardization — Overrides custom vector geometries to uniform 100 stroke-units, making dashoffset mappings completely size-independent.
  • Sequenced Stroke Draw — Orchestrates path animations (icon draws instantly; card outline follows with a transition-delay) to build a multi-layered effect.
  • Interactive drop-shadow() Filters — Shifts layout styles from dim outlines to glowing vector projections using GPU compositor-backed drop-shadow filters.
  • Reduced Motion Support — Integrates prefers-reduced-motion overrides that skip the vector tracing, presenting static glowing colors immediately on hover.
