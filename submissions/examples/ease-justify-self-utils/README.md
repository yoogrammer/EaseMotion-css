# ease-justify-self utilities

Controls `justify-self` for individual grid/flexbox item alignment along the inline axis.

## Usage

```html
<div class="ease-grid">
  <div class="ease-justify-self-center">Centered</div>
</div>
```

## Classes

| Class                       | CSS Property            |
| --------------------------- | ----------------------- |
| `ease-justify-self-auto`    | `justify-self: auto`    |
| `ease-justify-self-start`   | `justify-self: start`   |
| `ease-justify-self-center`  | `justify-self: center`  |
| `ease-justify-self-end`     | `justify-self: end`     |
| `ease-justify-self-stretch` | `justify-self: stretch` |
