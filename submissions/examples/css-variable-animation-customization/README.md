# CSS Variable-Based Animation Customization

A customizable animation system using CSS custom properties for durations, easing, scaling, and motion behavior.

## Features

- CSS variable-based animation control
- Global duration customization
- Configurable easing and scaling
- Theme-friendly motion system
- Pure HTML and CSS implementation
- Lightweight and reusable structure

## Usage

Developers can customize animation behavior globally using CSS variables:

```css
:root {
  --ease-duration: 1s;
  --ease-easing: ease-in-out;
  --ease-scale: 1.08;
}
