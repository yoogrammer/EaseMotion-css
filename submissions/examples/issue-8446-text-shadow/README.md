# Issue 8446: Document text-shadow utilities

This submission documents the newly added `.text-shadow-*` classes that were missing from the main documentation.

**What does this do?**
It proposes how `.text-shadow-sm`, `.text-shadow-md`, and `.text-shadow-lg` can be structured and demonstrated.

**How is it used?**
Apply the utility classes directly to text elements (e.g. `<h1 class="text-shadow-lg">`).

**Why is it useful?**
Ensures users are aware of the existing utility classes available in the framework for adding depth to typography.
