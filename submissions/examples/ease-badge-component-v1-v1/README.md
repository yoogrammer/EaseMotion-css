# Badge Component

## What does this do?

Provides reusable badge styles for showing labels, statuses, categories, and small UI indicators.

## How is it used?

```html
<span class="badge badge-primary">Primary</span>
<span class="badge badge-success">Success</span>
<span class="badge badge-warning">Warning</span>
<span class="badge badge-danger">Danger</span>
<span class="badge badge-neutral">Neutral</span>
```

## Why is it useful?

Badges are commonly used in dashboards, documentation, cards, tables, and navigation interfaces to communicate short status or category information. This component keeps those labels consistent, readable, and easy to reuse.
