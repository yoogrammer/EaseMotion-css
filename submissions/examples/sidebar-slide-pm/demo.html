Sidebar Navigation Component

About

A vertical sidebar navigation component with slide-in animation, collapsible mode, active states, and mobile off-canvas overlay. Built with CSS transitions and custom properties.

Features

Fixed Collapsible Active State Mobile Slide-in Overlay Reduced Motion

Usage

Apply class .sidebar to the sidebar container, .sidebar-item to nav links, and .sidebar-collapsed for icon-only mode. On mobile, add .open to slide in.
