# ease-bounce-button-exit

Button exit animation that squish-bounces through multiple scale values with overshoot before shrinking to zero scale and fading out.

## Usage

```html
<button class="ease-btn ease-bounce-button-exit">Bounces out</button>
```

## Variants

| Class | Description |
|---|---|
| `ease-bounce-button-exit` | Default bounce exit animation |

## Notes

- Multiple bounce overshoots before disappearing
- Great for dismiss animations on buttons and badges
- Respects `prefers-reduced-motion`
