Fix #2775: build:scss compiles a Sass partial

This is a build-tooling fix (no visual demo). The build:scss script in package.json compiles scss/_index.scss — a file prefixed with _, which Sass treats as a "partial" not meant for direct compilation.

Proposed change

1. Rename: scss/_index.scss → scss/index.scss

2. Update package.json:
"build:scss": "sass scss/_index.scss dist/easemotion.scss.css"
"build:scss": "sass scss/index.scss dist/easemotion.scss.css"

See README.md for full details and rationale.
