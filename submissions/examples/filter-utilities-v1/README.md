# Filter Utilities

CSS utility classes for applying common filter effects.

## Utilities

- `ease-blur-sm`
- `ease-blur`
- `ease-blur-lg`
- `ease-brightness-50`
- `ease-brightness-150`
- `ease-grayscale`
- `ease-sepia`

## Usage

```html
<img src="image.jpg" class="ease-blur">
<img src="image.jpg" class="ease-sepia">
```
