Callout & Alert

Contextual messages for info, success, warning, and error states — with border-left accent, icon slot, and dismiss animation.

Color Variants (dismissible)

ℹ
Information
Your account has been updated successfully. New features are available in your dashboard.
✓
Success
All changes have been saved. Your deployment is now live at the production URL.
⚠
Warning
Your API key will expire in 7 days. Generate a new key to avoid service interruptions.
✕
Error
Failed to connect to the database. Please check your connection string and try again.

Non-Dismissible (simple message)

ℹ
This is a simple info callout without a title or dismiss button.
✓
Operation completed successfully.
⚠
Your session will expire in 5 minutes.
✕
An unexpected error occurred. Please refresh the page.

Usage

<div class="ease-callout ease-callout-info"> <span class="ease-callout-icon">ℹ</span> <div class="ease-callout-body"> <div class="ease-callout-title">Info</div> Message text here. </div> <button class="ease-callout-dismiss">&times;</button> </div> <script> document.querySelectorAll('.ease-callout-dismiss') .forEach(btn => { btn.addEventListener('click', (e) => { const callout = e.currentTarget.closest('.ease-callout'); callout.classList.add('hiding'); setTimeout(() => callout.remove(), 350); }); }); </script>
