# ease-card-accent

Accent border card variants using colored left borders for semantic status indication.

## Usage

```html
<div class="ease-card ease-card-accent">Primary accent</div>
<div class="ease-card ease-card-accent-success">Success</div>
```

## Variants

| Class | Description |
|---|---|
| `ease-card-accent` | Primary color left border |
| `ease-card-accent-success` | Green success border |
| `ease-card-accent-danger` | Red danger border |
| `ease-card-accent-warning` | Amber warning border |
| `ease-card-accent-info` | Light primary info border |
| `ease-card-accent-neutral` | Gray neutral border |

## Notes

- Composable with `ease-card-hover`, `ease-card-shadow`, etc.
- Dark mode automatically adjusts accent colors
- Useful for dashboards, notification lists, and status feeds
