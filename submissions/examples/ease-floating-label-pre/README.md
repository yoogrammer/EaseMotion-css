## ease-floating-label

**What does this do?**
A modern input field where the placeholder transforms into a
floating label when the user focuses or enters text.

**How is it used?**
<div class="floating-input">
  <input type="text" required />
  <label>Name</label>
</div>

Color variants: green, orange
States: error, success

**Why is it useful?**
Improves form usability with a smooth label animation inspired
by Material Design — consistent with EaseMotion CSS's
animation-first philosophy.
