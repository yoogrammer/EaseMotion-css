# Animated FAQ Accordion

A responsive FAQ accordion component with smooth interactions and EaseMotion CSS animation examples.

## Features

- Expand/collapse FAQ items
- Responsive layout
- Clean card design
- EaseMotion animation classes
- Easy customization

- SaaS landing pages
- Product FAQs
- Portfolio websites
