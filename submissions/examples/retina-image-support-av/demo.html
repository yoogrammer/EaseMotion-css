srcset Retina Image Refactor

By strictly enforcing the native HTML srcset attribute across all framework documentation and demo files, we mathematically guarantee that visual assets remain perfectly crisp and professional on high-density Retina and 4K displays.

❌ Legacy `src` Attribute

Uses a single, low-resolution 1x image file. When viewed on a modern MacBook Retina display or 4K monitor, the browser violently stretches the missing pixels, resulting in an incredibly blurry, cheap-looking image.

✅ Modern `srcset` Attribute

Uses the native HTML srcset. The browser intelligently detects if the user has a high-density (2x or 3x) display and natively downloads the much sharper, high-resolution version of the exact same image.
