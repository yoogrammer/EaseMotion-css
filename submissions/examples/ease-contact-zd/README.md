# Ease Contact Zd

## What does this do?
An animated contact form component with input fields, textarea, submit button, focus animations, and success state — pure HTML and CSS.

## How is it used?
```html
<form class="cf-form">
  <div class="cf-group">
    <label class="cf-label" for="input-id">Label</label>
    <input class="cf-input" id="input-id" type="text" placeholder="Placeholder">
  </div>
  <div class="cf-group">
    <label class="cf-label" for="msg-id">Message</label>
    <textarea class="cf-textarea" id="msg-id" placeholder="Your message..."></textarea>
  </div>
  <button class="cf-btn" type="submit">Send Message</button>
</form>
```

## Why is it useful?
Contact forms are a standard element on every website. This component provides a polished, animated form with focus effects, entrance animations, and a success state — ready to use with your preferred form backend.
