# typewriter-text

1. What does this do?
   Animates text to type itself out character by character with a 
   blinking cursor — like a classic typewriter effect. Includes a 
   one-time and looping variant, plus slow and fast speed modifiers.

2. How is it used?
   Apply `.typewriter-text` for a one-time typing effect:
```html
   <h1 class="typewriter-text">Hello, I am a Developer.</h1>
```
   Or use `.typewriter-loop` for continuous back-and-forth typing:
```html
   <h2 class="typewriter-loop">Building beautiful UIs.</h2>
```
   Add `.typewriter-fast` or `.typewriter-slow` to adjust speed.

3. Why is it useful?
   Typewriter effects are one of the most requested text animations 
   for hero sections, portfolios, and landing pages. This pure CSS 
   implementation needs zero JavaScript and fits perfectly into 
   EaseMotion CSS's goal of expressive, utility-first animations.
