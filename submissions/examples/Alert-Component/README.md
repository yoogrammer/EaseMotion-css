# Animated Alert Component

**What does this do?**  
Creates reusable alert boxes for displaying status messages, notifications, warnings, errors, and informational content. Each alert includes subtle entrance animations and semantic color styling for improved user feedback.

**How is it used?**

```html
<div class="alert alert-success">
  Profile updated successfully.
</div>

<div class="alert alert-danger">
  Something went wrong.
</div>

<div class="alert alert-warning">
  Storage space is running low.
</div>

<div class="alert alert-info">
  New update available.
</div>
```

**Why is it useful?**  
Alerts are one of the most common UI patterns used across dashboards, forms, admin panels, landing pages, and web applications. They provide immediate visual feedback to users and help communicate important information clearly.

This component offers:

- Semantic status variants
- Smooth entrance animations
- Clean and modern styling
- Responsive behavior
- Pure CSS implementation

The feature aligns with EaseMotion CSS because it is:

- Human-readable
- Animation-first
- Composable
- Reusable
- Lightweight

Potential standardized framework classes:

- ease-alert
- ease-alert-primary
- ease-alert-secondary
- ease-alert-success
- ease-alert-danger
- ease-alert-warning
- ease-alert-info
