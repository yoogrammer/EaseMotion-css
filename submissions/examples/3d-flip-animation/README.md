# 3D Flip Entrance Animation

## What does this do?

Adds reusable 3D flip entrance animations for modern UI components.

---

## How is it used?

```html
<div class="ease-flip-x">
    Content
</div>

<div class="ease-flip-y">
    Content
</div>
```

---

## Why is it useful?

3D flip effects are commonly used in dashboards, feature cards, onboarding sections, and landing pages.

This animation expands EaseMotion CSS beyond fade and slide effects while remaining simple, readable, and reusable.
