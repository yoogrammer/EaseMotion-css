# Gradient Spinner Loader

A collection of CSS-only rotating gradient loaders built using conic gradients and smooth infinite animations.

## Features

- Pure CSS
- No JavaScript required
- Smooth infinite rotation
- Lightweight implementation
- Responsive examples
- Multiple color variants
- Size modifiers
- Easy customization

## Basic Usage

```html
<div class="ease-gradient-loader"></div>
```

## Available Variants

### Default

```html
<div class="ease-gradient-loader"></div>
```

### Purple Variant

```html
<div class="ease-gradient-loader loader-purple"></div>
```

### Success Variant

```html
<div class="ease-gradient-loader loader-success"></div>
```

### Large Spinner

```html
<div class="ease-gradient-loader loader-large"></div>
```

### Small Spinner

```html
<div class="ease-gradient-loader loader-small"></div>
```

## Main Class

```css
.ease-gradient-loader
```

## Customization

You can customize the loader through CSS variables:

```css
:root {
  --loader-size: 64px;
  --loader-thickness: 6px;
  --loader-speed: 1s;
}
```

## Why It Fits EaseMotion CSS

- Human-readable class naming
- Animation-first design
- Lightweight implementation
- Modern UI pattern
- Easy integration into existing projects
- Flexible and customizable
