# Ease Margin Utilities

## What does this do?

Provides utility classes for applying common margin patterns, including automatic centering and directional spacing.

## How is it used?

```html
<div class="ease-m-auto"></div>

<div class="ease-mx-auto"></div>

<div class="ease-mt-4"></div>

<div class="ease-mb-4"></div>

<div class="ease-ml-4"></div>

<div class="ease-mr-4"></div>
```

## Why is it useful?

Margin utility classes help developers quickly apply consistent spacing without writing custom CSS for every element. They improve productivity and maintain a uniform design system.
