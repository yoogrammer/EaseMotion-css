# E-Sports Arena

A responsive landing page for E-Sports Arena built with EaseMotion CSS framework.

## Live Demo
Open \demo.html\ in any modern web browser to view the landing page.

## Features

- **Pure EaseMotion CSS** - Uses the EaseMotion CDN
- **Responsive Layout** - Adapts to all viewports
- **Dark Theme Design** - Modern dark color scheme
- **Smooth Animations** - \ase-fade-in\, \ase-slide-up\, delays
- **Zero JavaScript** - No JS required
- **Multiple Sections** - Hero, services, features, about, footer

## EaseMotion Classes Used

- \ase-fade-in\ - Fade in animation
- \ase-slide-up\ - Slide up animation
- \ase-delay-{100,200}\ - Animation delays
- \ase-btn\, \ase-btn-primary\, \ase-btn-lg\ - Buttons
- \ase-btn-hover\, \ase-hover-lift\ - Hover effects
- \ase-container\ - Container
- \ase-flex\, \ase-items-center\, \ase-justify-between\ - Flex
- \ase-grid\, \ase-grid-cols-{2,3}\ - Grid
- \ase-text-{center,lg,xl,2xl,3xl,5xl}\ - Typography
- \ase-font-{bold,semibold}\ - Font weight
- \ase-py-16\, \ase-px-8\, \ase-px-4\, \ase-py-4\ - Padding
- \ase-mb-{2,4,6,8,12}\, \ase-mt-2\ - Margin
- \ase-opacity-{70,80}\ - Opacity
- \ase-bg-surface\ - Background section
- \ase-text-hover-glow\ - Hover glow
