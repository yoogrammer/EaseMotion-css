ease-inset utilities

Shorthand utilities for top, right, bottom, left positioning.

inset: 0 (all sides)

Stretched to all edges

inset-x: 0 (left + right)

Stretched horizontally only

inset-y: 0 (top + bottom)

Stretched vertically only

Inset utilities work great with position: absolute or position: fixed to stretch elements across their positioned container. Used extensively for overlays, pseudo-element covers, and full-bleed backgrounds.
