# Audio Waves Loader

## What does this do?

This demo creates an animated audio equalizer loader using pure CSS. Multiple vertical bars continuously grow and shrink at different intervals, simulating the appearance of sound waves or music frequencies.

The effect is achieved using staggered animation delays and varying bar heights to produce a dynamic, rhythmic loading animation.

## How is it used?

```html id="x8k2vn"
<div class="audio-loader-container">
  <div class="equalizer-bar bar-1"></div>
  <div class="equalizer-bar bar-2"></div>
  <div class="equalizer-bar bar-3"></div>
  <div class="equalizer-bar bar-4"></div>
  <div class="equalizer-bar bar-5"></div>
  <div class="equalizer-bar bar-6"></div>
  <div class="equalizer-bar bar-7"></div>
</div>
```

Each bar shares the same animation while using different animation delays to create a natural wave-like motion.

The animation runs automatically and requires no JavaScript.

## Why is it useful?

Audio-wave and equalizer loaders are a visually engaging alternative to traditional spinners. They are commonly used for:

- Music and audio applications
- Voice assistants
- Podcast platforms
- Streaming interfaces
- Loading indicators
- Modern dashboard experiences

Because the effect is built entirely with CSS, it is lightweight, easy to customize, and fits EaseMotion CSS's animation-first philosophy by providing expressive motion through simple, reusable classes.
