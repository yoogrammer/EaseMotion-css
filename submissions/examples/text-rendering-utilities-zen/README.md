# Text Rendering Utilities

CSS utility classes for controlling text rendering hints.

## Classes

- `.text-render-auto` — `text-rendering: auto`
- `.text-render-speed` — `text-rendering: optimizeSpeed`
- `.text-render-legibility` — `text-rendering: optimizeLegibility`
- `.text-render-geometric-precision` — `text-rendering: geometricPrecision`

## Why

Fine-tune how the browser renders text — prioritize speed for UI labels or legibility for body text with kerning and ligatures.
