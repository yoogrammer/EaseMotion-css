# CSS-only Avatar and Avatar Group Component

A versatile, CSS-only avatar and avatar group component for displaying user profiles.

This feature includes:

- Image avatars with fallback initials styling.
- Three sizes: small, medium, and large.
- Shape variants: default circular and rounded-square.
- Online/offline status indicator dots.
- Avatar groups with overlapping stacked avatars.
- Overflow count item (e.g., `+3`) for long lists of users.
- Smooth hover and focus interactions for group items.
- Dark mode compatibility.

## Why is this useful for EaseMotion CSS?

Avatars are foundational building blocks for modern interfaces such as dashboards, team rosters, profile pages, comment threads, and chat applications.

By adding a dedicated Avatar component with grouping and status features, developers can easily plug this into their EaseMotion CSS projects without needing custom CSS overrides or complex JavaScript rendering logic. It strictly follows the library's design philosophy of being zero-dependency, animation-friendly, and highly readable.
