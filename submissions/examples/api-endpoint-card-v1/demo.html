API Endpoint Card

Developer-friendly API documentation component — Issue #1178

GET /api/v1/users

Returns a paginated list of all registered users.

Auth Required 200 OK 
{
  "success": true,
  "users": [...]
}
POST /api/v1/users

Create a new user account.

Auth Required 201 Created 
{
  "success": true,
  "id": "usr_abc123"
}
PUT /api/v1/users/:id

Update an existing user's profile information.

Auth Required 200 OK 
{
  "success": true,
  "updated": "profile"
}
DELETE /api/v1/users/:id

Permanently delete a user account.

Auth Required 401 Unauthorized 
{
  "success": false,
  "error": "Invalid token"
}
