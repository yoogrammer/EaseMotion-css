# Ease Testimonial Zd

## What does this do?
An animated testimonial/quote card component with star ratings, author avatars, quote decoration, and entrance animations — pure HTML and CSS.

## How is it used?
```html
<div class="tm-grid">
  <div class="tm-card">
    <div class="tm-quote-icon" aria-hidden="true">"</div>
    <div class="tm-text">Testimonial text here...</div>
    <div class="tm-stars">
      <span class="tm-star" aria-label="Star">★</span>
      <span class="tm-star" aria-label="Star">★</span>
      <span class="tm-star" aria-label="Star">★</span>
      <span class="tm-star" aria-label="Star">★</span>
      <span class="tm-star" aria-label="Star">★</span>
    </div>
    <div class="tm-author">
      <img class="tm-avatar" src="avatar.jpg" alt="Name" width="40" height="40">
      <div class="tm-info">
        <div class="tm-name">Author Name</div>
        <div class="tm-role">Role, Company</div>
      </div>
    </div>
  </div>
</div>
```

## Why is it useful?
Testimonial sections are a standard part of marketing pages, landing sites, and product showcases. This component provides a polished, cohesive set of testimonial cards with subtle animations that enhance the browsing experience without overwhelming the content.
