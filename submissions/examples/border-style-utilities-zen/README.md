## What does this do?

Provides utility classes for common `border-style` values: solid, dashed, dotted, double, and none.

## How is it used?

```html
<div class="border-solid">Solid border</div>
<div class="border-dashed">Dashed border</div>
```

## Why is it useful?

EaseMotion includes border-width and border-color utilities but lacks border-style classes. These are needed for composing complete border styles without inline CSS.
