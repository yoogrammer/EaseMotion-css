# CSS 3D Perspective Tilt Accordion for Marketing Landing Page Layouts

## Overview

This project demonstrates a modern marketing landing page accordion featuring a smooth **3D Perspective Tilt** opening animation built entirely with HTML and CSS. It provides an engaging FAQ section with subtle depth effects, responsive layouts, and accessible interactions without requiring JavaScript.

---

## Features

- Pure HTML5 and CSS3 implementation
- CSS-only 3D Perspective Tilt animation
- Responsive FAQ accordion layout
- Marketing landing page inspired design
- Smooth open and close transitions
- Hover elevation effects
- Keyboard-accessible using native `<details>` and `<summary>`
- CSS custom properties for easy customization
- No JavaScript or external libraries required

### Animation

- Accordion panels open with a smooth 3D perspective tilt.
- Subtle rotation on both the X and Y axes creates a realistic depth effect.
- Panels slightly scale and lift while opening.
- Hover interactions add gentle elevation.
- The expand icon transitions from a plus to a minus for better visual feedback.

## Customization

The component uses CSS custom properties for quick styling.

```css
:root{
    --primary:#6366f1;
    --primary-light:#818cf8;
    --surface:#ffffff;
    --text:#1f2937;
    --muted:#64748b;
    --radius:20px;
}
```

You can customize:

- Primary colors
- Background
- Typography
- Border radius
- Shadows
- Accordion spacing
- Animation duration
- Perspective depth
- Hover effects
