accent-color Themed Form Controls

Use a single CSS property — accent-color — to theme native form controls. Pick a color below to see it applied instantly.

Checkboxes

Radio Buttons

65

Progress & Meter

Code

/* Single line themes all native form controls */
.theme-8681 {
  accent-color: #6c63ff;
}

<!-- Apply to any ancestor -->
<body class="theme-8681">
  <input type="checkbox" checked>
  <input type="radio" checked>
  <input type="range">
  <progress value="70" max="100"></progress>
  <meter value="0.6"></meter>
</body>
