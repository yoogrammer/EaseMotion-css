 
# Submission Walkthrough Demo

This folder contains a demo showing how the submission walkthrough will look in CONTRIBUTING.md.

## Files

| File | Description |
|------|-------------|
| 📄 [demo.html](./demo.html) | Preview of walkthrough in CONTRIBUTING.md |
| 🎨 [style.css](./style.css) | Styling for the preview |
| 📖 [README.md](./README.md) | Documentation |

## Walkthrough Steps Covered

1. Fork the repository
2. Clone your fork
3. Create a new branch
4. Create submission folder with 3 files
5. Add, commit, and push
6. Submit Pull Request

## How to Create the Actual GIF/Video

### Recommended Tools:
- **ScreenToGif** (Windows) - Free, easy to use
- **OBS Studio** (Windows/Mac/Linux) - Professional screen recorder
- **Loom** (Browser extension) - Quick recording
- **Kap** (Mac) - Open source

### Steps to Create:
1. Record your screen following the walkthrough script
2. Export as GIF (for GitHub) or MP4
3. Upload to GitHub or cloud storage
4. Embed in CONTRIBUTING.md using `![Walkthrough](link.gif)`
