📹 Submission Walkthrough

How to contribute to EaseMotion CSS - Step by Step Guide

1

Fork the Repository

Click the "Fork" button on GitHub to create your own copy of EaseMotion CSS.

📋 Click "Fork" → Your GitHub account
2

Clone Your Fork

git clone https://github.com/YOUR-USERNAME/EaseMotion-css.git cd EaseMotion-css
3

Create a New Branch

git checkout -b feature/your-feature-name
4

Create Your Submission Folder

mkdir submissions/examples/your-feature-name cd submissions/examples/your-feature-name 
submissions/examples/your-feature-name/
├── demo.html     ← Interactive demo
├── style.css     ← CSS styles
└── README.md     ← Documentation
5

Add Your Files

git add submissions/examples/your-feature-name/ git commit -m "feat: add your feature description" git push origin feature/your-feature-name
6

Submit a Pull Request

Go to GitHub → Click "Compare & pull request" → Add title and description → Create PR

✅ Title: feat: add your feature (Fixes #issue-number) ✅ Wait for maintainer review
