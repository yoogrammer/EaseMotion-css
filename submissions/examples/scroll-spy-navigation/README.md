# Scroll Spy Sidebar Navigation

A documentation-style sidebar navigation system that automatically tracks the user's current reading position and highlights the active section while scrolling.

## Features

- Scroll Spy navigation
- Automatic active section detection
- Sidebar auto-scroll synchronization
- Active navigation highlighting
- Smooth scrolling behavior
- Responsive layout
- Uses Intersection Observer API
- No external dependencies

## Usage

Open `demo.html` in a browser.

### Test the functionality

1. Scroll through the content sections.
2. Observe the active sidebar item updating automatically.
3. Continue scrolling to lower sections such as:
   - Pricing
   - FAQ
   - Contributing
4. Notice that the sidebar automatically scrolls to keep the active navigation item visible.

## Files

- `demo.html` – Demo implementation
- `style.css` – Styling and navigation behavior

## Inspiration

Inspired by modern documentation websites such as:

- React Documentation
- Next.js Documentation
- Tailwind CSS Documentation
- MDN Web Docs

These websites automatically synchronize navigation state with the currently visible content section.

## Technical Details

The implementation uses:

- Intersection Observer API
- Active navigation state management
- Programmatic sidebar scrolling
- Responsive documentation layout
