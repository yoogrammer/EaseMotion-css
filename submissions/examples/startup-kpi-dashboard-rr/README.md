# Startup KPI Analytics Dashboard Component

## What does this do?

Displays key startup metrics including Monthly Recurring Revenue (MRR), Annual Recurring Revenue (ARR), growth rate, churn rate, and customer performance indicators.

## How is it used?

```html
<div class="card mrr">
    <h2>$48.2K</h2>
    <span>Monthly Recurring Revenue</span>
</div>
```

## Why is it useful?

Startups and SaaS businesses rely on KPI dashboards to track revenue, customer acquisition, retention, and growth. This component provides a modern analytics dashboard built entirely with HTML and CSS.
