# CSS-only Liquid Fill Button

### 1. What does this do?
An interactive CSS-only button that features a fluid, liquid-filling wave animation upon hover, creating a highly dynamic micro-interaction.

### 2. How is it used?
```html
<a href="#" class="liquid-button">
  <span>Hover Me</span>
  <div class="liquid"></div>
</a>
```

### 3. Why is it useful?
It adds an organic, fluid animation style to the framework's collection of buttons. Since it requires no JavaScript or SVG filters to achieve the wave effect, it strictly adheres to the project's CSS-only and animation-first philosophy.
