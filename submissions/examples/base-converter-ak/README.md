# Base Converter

## What does this do?
A number base converter supporting binary, octal, decimal, and hexadecimal with real-time bidirectional conversion and step-by-step conversion explanation.

## How is it used?
Enter a number, select source base and target base. The result updates in real time. Use quick preset buttons for common conversions (Bin→Dec, Dec→Hex, etc.). View detailed conversion steps.

## Why is this useful?
This helps students, programmers, and engineers quickly convert between number bases and understand the conversion process through step-by-step explanation.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to use the converter.
