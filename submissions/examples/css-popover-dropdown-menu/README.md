# CSS Popover Dropdown Menu Example

A dropdown menu using the CSS `popover` API — no JavaScript required for open/close behavior.

## Features

- Uses the HTML `popover` attribute for declarative show/hide
- Smooth entrance animation with `@keyframes`
- Click-outside-to-close behavior (native popover API)
- Custom styling with arrow indicator
- Dark theme with `prefers-reduced-motion` support

## Files

- `demo.html` — Popover dropdown layout
- `style.css` — Self-contained CSS (85 lines)

## Preview

A CSS popover-based dropdown menu that opens on button click and closes on outside click — no JS needed.
