# ease-btn-outline

Outline button variant with a bordered background and hover fill effect. A standard secondary button pattern.

## Usage

```html
<button class="ease-btn ease-btn-outline">Outline button</button>
```

## Variants

| Class | Description |
|---|---|
| `ease-btn-outline` | Outline button with hover fill |

## Notes

- Border with transparent background in resting state
- Hover fills with primary color
- Standard secondary button pattern in every design system
