# CSS-Only Animated Camera Shutter

Pure CSS animated camera shutter effect — a circular iris that opens and closes like a DSLR camera lens.

## Features

- Concentric blade segments that rotate outward to open
- Smooth CSS `clip-path` animation for the aperture
- Toggle button to open/close the shutter
- Dark theme with `prefers-reduced-motion` support

## Files

- `demo.html` — Interactive shutter with toggle button
- `style.css` — Self-contained CSS (100 lines)
