# Ease Wavy Zd

## What does this do?
An animated wavy section divider component with SVG waves, layered effects, flipped orientation, and gradient backgrounds — pure HTML and CSS.

## How is it used?
```html
<!-- Single wave divider -->
<div class="wv-divider wv-divider-wave">
  <svg viewBox="0 0 960 120" preserveAspectRatio="none">
    <path d="M0,80 C160,160 320,0 480,80 C640,160 800,0 960,80 L960,120 L0,120 Z"></path>
  </svg>
</div>

<!-- Flipped wave -->
<div class="wv-divider wv-divider-wave wv-flip">
  <svg viewBox="0 0 960 120" preserveAspectRatio="none">
    <path d="M0,80 C160,160 320,0 480,80 C640,160 800,0 960,80 L960,120 L0,120 Z"></path>
  </svg>
</div>
```

## Why is it useful?
Section dividers are a standard design element on landing pages, portfolios, and marketing sites. This component provides reusable SVG wave dividers with layered and gradient variants.
