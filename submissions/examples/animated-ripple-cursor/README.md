# Animated Ripple Cursor

## What it does

Cursor-following ripple effect with decaying waves. Moving mouse creates small ripples; clicking creates larger bursts.

## How to use it

Canvas automatically tracks cursor and creates ripples.

## Features

- Real-time cursor tracking
- Movement creates small ripples
- Click creates larger ripple bursts
- 3 color modes (Purple/Blue/Rainbow)
- Trailing dot effect
- Clear button to reset

## Color Modes

- Purple - Default neon purple
- Blue - Cyan/teal theme
- Rainbow - Cycling HSL colors

## Why it fits EaseMotion CSS

- Canvas-based real-time animation
- Interactive cursor tracking
- Multiple color options
- Smooth 60fps performance
