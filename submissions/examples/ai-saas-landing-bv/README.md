# AI SaaS Landing Platform

## What does this do?

The AI SaaS Landing Platform is a modern marketing startup landing page showcasing product capabilities, pricing tiers, customer success stories, and conversion-focused Call-to-Actions (CTAs) utilizing advanced CSS transitions.

## How is it used?

Open `demo.html` directly in any modern browser:

- **Interactive Telemetry Dashboard**: Check out the live-refresh stats box on the hero section simulating active LLM workers and token consumption.
- **Bento Grid Features**: Observe the bento cards grid displaying animated workflows (connecting node pipelines) and projected latency meters.
- **Pricing Selector**: Toggle the monthly vs yearly billing switch to dynamically update pricing cards with discounts.
- **Testimonial Slider**: Click "Next Testimonial" to trigger quote updates.
- **FAQ Accordion**: Click on headers to expand FAQ descriptions.

## Why is it useful?

Creating premium SaaS marketing platforms requires high visual fidelity, structured grids, and smooth interactivity to capture customer conversions.

This showcase solves this by:

1. Demonstrating how to build responsive 3D-feeling bento grid layouts without heavy JavaScript libraries.
2. Presenting accessible billing switches and collapsible FAQs that respect user motion controls.
3. Leveraging keyframe floaters and gradient transitions for an interactive, modern user experience.

## Tech Stack

- HTML5
- CSS3 (bento layouts, pricing toggles, accordion transitions, floaters)
- JavaScript (simulated telemetry loops, node flowchart lighting, accordion expansions)
