 
# Pulse Ripple Loading Button

Pulse ripple loading animation — continuous expanding ring effect for submit/loading states.

## Files

- demo.html - Interactive demo
- style.css - Pulse button styles
- README.md - Documentation

## Classes

- ease-pulse-btn - Base pulse button
- ease-pulse-fast/slow - Speed variants
- ease-pulse-ripple-subtle/strong - Ripple intensity
- ease-pulse-double - Double ripple effect

## Color Variants

- pulse-btn-primary/success/danger/warning/dark/gradient

## Size Variants

- pulse-btn-sm - Small
- pulse-btn - Normal
- pulse-btn-lg - Large

## Usage

```html
<button class="ease-pulse-btn">Submit</button>
<button class="pulse-btn-primary ease-pulse-btn">Primary</button>
<button class="ease-pulse-fast">Fast Pulse</button>
