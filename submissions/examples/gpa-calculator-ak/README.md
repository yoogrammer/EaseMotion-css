# GPA Calculator

## What does this do?
A college GPA calculator that tracks courses per semester, converts letter grades to GPA points, and displays semester and cumulative GPA with trend visualization.

## How is it used?
Add semesters and courses with credits and letter grades. The calculator shows current semester GPA, cumulative GPA, and a visual GPA trend across semesters.

## Why is this useful?
Students can track their academic performance across semesters, experiment with grade changes, and visualize their GPA progression over time.

## Tech Stack
- HTML
- CSS (no frameworks)
- Vanilla JavaScript

## Preview
Open demo.html directly in your browser to use the calculator.
