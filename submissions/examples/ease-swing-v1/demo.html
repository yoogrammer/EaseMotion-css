Usage

<!-- Basic swing (continuous) -->
<div class="ease-swing">Swinging content</div>

<!-- Different angles -->
<div class="ease-swing-small">5° swing</div>
<div class="ease-swing-large">25° swing</div>

<!-- Different speeds -->
<div class="ease-swing-fast">Fast swing</div>
<div class="ease-swing-slow">Slow swing</div>

<!-- One-shot on hover -->
<div class="ease-swing-hover">Swing on hover</div>

<!-- Custom angle using CSS variable -->
<div class="ease-swing-custom" style="--ease-swing-angle: 20deg;">20° swing</div>

⚡ Pendulum Swing Features

  • 🔄 Anchor point at transform-origin: top center
  • 📐 Configurable swing angle via --ease-swing-angle
  • ⚡ Speed variations (fast, normal, slow)
  • 🎯 One-shot mode (on hover) and continuous mode
  • 💫 Smooth easing in/out motion
