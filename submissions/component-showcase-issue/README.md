# Component Showcase

## What does this do?
A live showcase page displaying all EaseMotion CSS components 
(buttons and cards) with copyable HTML snippets.

## How is it used?
Open demo.html in your browser to see all components live.
Copy any HTML snippet directly into your project!

## Why is it useful?
Developers can quickly preview all available components in one 
place without reading through the entire documentation.

## Tech Stack
- HTML
- CSS (EaseMotion CSS framework)

## Preview
Open demo.html directly in your browser to see all components.
