What is EaseMotion CSS?

EaseMotion CSS is a human-readable, animation-first CSS framework. Zero dependencies. No build step required.

How do I install it?

Install via npm: npm install ease-motion-css, or download the CSS file directly from the GitHub releases.

Is JavaScript required?

No! EaseMotion CSS is pure CSS. Some components use the <details> / <summary> pattern or hidden checkbox/radio inputs for interactivity.
