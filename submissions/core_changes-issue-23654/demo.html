ease-scroll-snap

Zero-JS carousels and sliders using native CSS scroll-snap. Horizontal, vertical, snap-start, snap-center — all with a single utility class.

.ease-scroll-snap-x + .ease-scroll-snap-item — swipe or scroll to snap between slides.
