Toast / Notification Component

Positioned notification messages — pure CSS.

Type Variants

ℹ
Info

This is an informational message.

✓
Success

Your changes have been saved successfully.

⚠
Warning

Your session will expire in 5 minutes.

✕
Error

Something went wrong. Please try again.

Simpler (no title / no icon)

Simple message without a title.

Operation completed.

Positioned Toasts

Add toast-top-right, toast-top-left, toast-bottom-right, or toast-bottom-left to position the toast container.

<div class="toast-container toast-top-right">
  <div class="toast toast-info">...</div>
</div>
