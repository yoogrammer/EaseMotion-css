Dead Keyframe #4571

The orphaned @keyframes easeZoomIn { scale(0) → scale(1) } in submissions/examples/ease-zoom-in/style.css exists as a community example but is never referenced by any framework class. The actual framework uses @keyframes ease-kf-zoom-in { scale(0.85) → scale(1) } shown below.

✦

Resolution

The dead easeZoomIn keyframe should be removed from the example, or the example should be updated to reference the canonical ease-kf-zoom-in keyframe name. The official easemotion/zoom.css already uses the correct keyframe.
