# Range Slider

1. **What does this do?** Provides a styled range slider using native `<input type="range">` with custom track fill, thumb, 4 color variants (primary, success, warning, danger), 3 sizes (sm, base, lg), and disabled state — pure CSS.

2. **How is it used?**
   ```html
   <input type="range" class="range" min="0" max="100" value="50">
   <input type="range" class="range range-success range-lg" min="0" max="100" value="75">
   ```

3. **Why is it useful?** Range sliders let users select numeric values through a visual draggable control. Pure CSS styling of native `input[type=range]` ensures accessibility while providing a polished appearance with dark mode support.
