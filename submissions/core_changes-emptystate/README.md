# Empty State Component

1. **What does this do?** Provides an empty state placeholder with icon, title, description text, optional action button, and 3 size variants (sm, base, lg) — perfect for empty inboxes, search results, and dashboards.

2. **How is it used?**
   ```html
   <div class="empty-state">
     <div class="empty-icon">📭</div>
     <h3 class="empty-title">No messages</h3>
     <p class="empty-text">Messages will appear here.</p>
   </div>
   ```

3. **Why is it useful?** Empty states improve UX by explaining what goes in an empty area and guiding users toward the next action. Pure CSS, dark mode, dashed border styling, size variants.
