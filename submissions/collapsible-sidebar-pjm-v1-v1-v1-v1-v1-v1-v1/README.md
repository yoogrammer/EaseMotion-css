# Collapsible Sidebar PJM

## What does this do?

Provides a collapsible navigation sidebar with a built-in toggle button. Users can expand or collapse the sidebar to control how much screen space is dedicated to navigation versus content.

## How is it used?

1. Open the page.
2. Click the ☰ (menu) button in the sidebar header.
3. The sidebar will collapse or expand with a smooth transition.
4. When collapsed, the full logo text **"EaseMotion CSS"** is shortened to **"EM"** to save space.
5. Click the button again to restore the sidebar and full logo text.

## Features

* Collapsible sidebar navigation
* Toggle button for easy interaction
* Dynamic logo text switching (`EaseMotion CSS` ↔ `EM`)
* Clean layout suitable for documentation websites
* Smooth user experience for both large and small screens

## Why is it useful?

This component helps maximize content visibility while keeping navigation easily accessible. It is especially useful for documentation portals, dashboards, admin panels, and developer tools where screen space is valuable. The collapsible behavior improves usability and creates a cleaner, more responsive interface consistent with EaseMotion CSS design principles.
