# CSS Gradient Text Mask Reveal Demo

An animation workspace showcasing smooth typography revealing masks engineered via linear tracking gradients and hardware-accelerated positioning states.

## Features
- **Linear Gradient Masking:** Utilizes `-webkit-mask-image` combined with shifting position matrices to reveal characters smoothly.
- **GPU Accelerated Keyframes:** Smooth execution through hardware-friendly transforms and interpolation layers.
- **Interactive Control Mechanics:** Built-in speed adjustments and manual loop replays to inspect motion details.
